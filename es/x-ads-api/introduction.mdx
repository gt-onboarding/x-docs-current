<div id="connect-to-the-x-ads-platform">
  ## Conéctate con la plataforma publicitaria de X
</div>

La API de X Ads conecta a los desarrolladores con la plataforma publicitaria de X para crear soluciones que satisfagan las necesidades de los anunciantes de X en todo el mundo.
