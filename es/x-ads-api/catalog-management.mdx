<div id="overview">
  ## Descripción general
</div>

La Catalog API es una solución de comercio que ofrece a los anunciantes la capacidad de configurar feeds de productos, agrupar productos en conjuntos y gestionar de forma integral los productos del catálogo. La Catalog API será una herramienta central que permite la gestión programática de catálogos y otorga a los anunciantes un mayor control sobre cómo se importan y actualizan sus catálogos.

Un catálogo contiene todo lo relativo al producto y al conjunto de productos del usuario y está asociado al handle del usuario. Actualmente, **un usuario solo puede crear un catálogo**. 

La Catalog API admite 2 formas de incorporar productos:

1.  Scheduled Feed: el usuario puede añadir una URL de feed (CSV, TSV y XML) que se recuperará periódicamente (X actualmente admite un archivo de hasta 8 GB)

2.  Batch Products API: el usuario puede ver, crear, actualizar y eliminar atributos de productos con solicitudes JSON por lotes

Los comercios de X pueden crear conjuntos de productos basados en reglas de filtro. Al hacerlo, podrán adjuntar metadatos adicionales a sus productos y crear conjuntos de productos especiales para mostrar estos productos a través de funciones de comercio orgánicas o anuncios dinámicos de productos. La Catalog API y X Shopping Manager admiten dos tipos de conjuntos de productos. 

1.  Manual: el usuario puede seleccionar hasta 50 productos y establecer un nombre y una descripción

2.  Filter: el usuario puede añadir hasta 30 filtros para generar automáticamente conjuntos con productos que cumplan estos filtros. Los tipos de filtros incluyen precio, categoría de producto de Google, tipo de producto, inventario, precio de oferta y campos personalizados.

Catalog API incluye 4 endpoints para gestionar el catálogo:

* [Product Catalogs](https://developer.x.com/en/docs/x-ads-api/catalog-management/api-reference/product-catalogs) \- Crear un contenedor para la información de productos
* [Products](https://developer.x.com/en/docs/x-ads-api/catalog-management/api-reference/products) -  Añadir, actualizar, ver y eliminar productos de los catálogos de forma sincrónica
* [Product Sets](https://developer.x.com/en/docs/x-ads-api/catalog-management/api-reference/product-sets) -  Clasificar productos en grupos basados en la información de los productos
* [Schedule Feeds](https://developer.x.com/en/docs/twitter-ads-api/catalog-management/api-reference/scheduled-feeds) \- Configurar la ingestión asíncrona de datos de productos basados en archivos en los catálogos

<div id="prerequisites-for-the-catalog-api">
  ### Requisitos previos para la API de Catálogo
</div>

Los endpoints de la API de Catálogo están disponibles actualmente solo a través de un programa de acceso anticipado. Para solicitar acceso, comunícate con tu representante de X. 

Debes aceptar los términos de servicio a través de [X Shopping Manager](https://ads.x.com/shopping_manager).

Para obtener más información sobre X Shopping Manager y las especificaciones de productos, consulta la [guía de especificaciones de productos](https://business.x.com/en/help/shopping-specs.html).
