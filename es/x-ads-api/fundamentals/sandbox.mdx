El sandbox es un entorno para que pruebes y desarrolles tu implementación de Ads API.

Las campañas creadas a través del sandbox no se entregarán a los usuarios y se pueden crear dinámicamente registros de prueba para objetos como Funding Instruments.

Usar el sandbox es tan sencillo como utilizar el nombre de host correspondiente de ads-api-sandbox.x.com, pero también incluye algunas funcionalidades únicas que solo se encuentran en el sandbox.

Realiza una solicitud inicial a [POST accounts](/es/x-ads-api/campaign-management#post-accounts) para crear una nueva cuenta de anunciante. La respuesta de esta llamada contendrá un account&#95;id que se puede usar para solicitudes posteriores. Estos endpoints permiten a los usuarios crear y probar con varias cuentas de anunciante.

<div id="sandbox-only-endpoint-list">
  ### Lista de endpoints disponibles solo en el sandbox:
</div>

* [POST accounts](/es/x-ads-api/campaign-management#post-accounts) Crear una cuenta en el sandbox
* [POST accounts/:account&#95;id/features](/es/x-ads-api/campaign-management#post-accounts-account-id-features) Añadir una característica de cuenta a una cuenta en el sandbox
* [POST accounts/:account&#95;id/funding&#95;instruments](/es/x-ads-api/campaign-management#post-accounts-account-id-funding-instruments) Añadir un instrumento de financiación a una cuenta en el sandbox
* [DELETE accounts/:account&#95;id](/es/x-ads-api/campaign-management#delete-accounts-account-id) Eliminar una cuenta en el sandbox
* [DELETE accounts/:account&#95;id/features](/es/x-ads-api/campaign-management#delete-accounts-account-id-features) Eliminar una característica de cuenta de una cuenta en el sandbox
* [DELETE accounts/:account&#95;id/funding&#95;instruments/:funding&#95;instrument&#95;id](/es/x-ads-api/campaign-management#delete-accounts-account-id-funding-instruments-funding-instrument-id) Eliminar un instrumento de financiación de una cuenta en el sandbox
