El sandbox es un entorno donde puedes probar y desarrollar tu implementación de Ads API.

Las campañas creadas en el sandbox no se servirán a los usuarios y se pueden crear dinámicamente registros de prueba para objetos como Funding Instruments.

Usar el sandbox es tan sencillo como llamar al nombre de host correspondiente, ads-api-sandbox.x.com, pero también incluye algunas funcionalidades exclusivas que solo están disponibles en el sandbox.

Realiza una solicitud inicial a [POST accounts](/es/x-ads-api/campaign-management#post-accounts) para crear una nueva cuenta de anunciante. La respuesta de esta llamada contendrá un account&#95;id que podrá utilizarse en solicitudes posteriores. Estos endpoints permiten a los clientes crear y realizar pruebas con múltiples cuentas de anunciante.

<div id="sandbox-only-endpoint-list">
  ### Lista de endpoints exclusivos del entorno sandbox:
</div>

* [POST accounts](/es/x-ads-api/campaign-management#post-accounts) Crear una cuenta en el entorno sandbox
* [POST accounts/:account&#95;id/features](/es/x-ads-api/campaign-management#post-accounts-account-id-features) Agregar una función de cuenta a una cuenta en el entorno sandbox
* [POST accounts/:account&#95;id/funding&#95;instruments](/es/x-ads-api/campaign-management#post-accounts-account-id-funding-instruments) Agregar un instrumento de financiación a una cuenta en el entorno sandbox
* [DELETE accounts/:account&#95;id](/es/x-ads-api/campaign-management#delete-accounts-account-id) Eliminar una cuenta en el entorno sandbox
* [DELETE accounts/:account&#95;id/features](/es/x-ads-api/campaign-management#delete-accounts-account-id-features) Eliminar una función de cuenta de una cuenta en el entorno sandbox
* [DELETE accounts/:account&#95;id/funding&#95;instruments/:funding&#95;instrument&#95;id](/es/x-ads-api/campaign-management#delete-accounts-account-id-funding-instruments-funding-instrument-id) Eliminar un instrumento de financiación de una cuenta en el entorno sandbox
