El tipo de moneda se identifica mediante [ISO-4217](http://en.wikipedia.org/wiki/ISO_4217). Es una cadena de tres letras como `USD` o `EUR`. El valor de una moneda se representa en micros. Para USD, 5,50 $ se codifica como 5,50 * 1e6, es decir, 5.500.000. Para representar un &quot;valor entero&quot;, debes multiplicar el valor local en micros por 1e6 (1&#95;000&#95;000) para todas las monedas.

La Ads API reconoce las monedas oficiales de todos los países donde el producto X SMB Ads está disponible. Para obtener una lista completa de estos países, haz clic [aquí](https://business.x.com/en/help/overview/about-eligibility-for-twitter-ads.html).

La mayoría de las monedas tienen una &quot;precisión&quot; de dos decimales (como USD: 1,23 $).

* `AED`, `VEF`, `PKR`, `IDR`, `HKD`, `QAR`, `PHP`, `NZD`, `INR`, `EUR`, `ARS`, `THB`, `MXN`, `EGP`, `DZD`, `SAR`, `LBP`, `COP`, `TRY`, `SGD`, `MAD`, `AUD`, `USD`, `MYR`, `CAD`, `BRL`, `GBP`.

Las siguientes monedas no usan decimales:

* `VND`, `KRW`, `CLP`, `JPY`

Y estas usan tres decimales:

* `KWD`, `IQD`, `BHD`, `TND`

La utilización de las monedas se realiza mediante [instrumentos de financiación](/es/x-ads-api/introduction).
