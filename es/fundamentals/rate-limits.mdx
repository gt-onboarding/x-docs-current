Cada día, miles de desarrolladores realizan solicitudes a la plataforma para desarrolladores de X. Para gestionar el enorme volumen de estas solicitudes, se establecen límites en la cantidad de solicitudes que se pueden realizar. Estos límites nos ayudan a ofrecer la API confiable y escalable en la que nuestra comunidad de desarrolladores confía.

Cada una de nuestras API utiliza los límites de frecuencia de diferentes maneras. Para obtener más información sobre estas diferencias entre plataformas, consulta las páginas específicas de límites de frecuencia dentro de nuestras secciones específicas de API:

* [X API v2](/es/x-api/fundamentals/rate-limits)
* [X API: Enterprise](/es/x-api/enterprise-gnip-2.0/fundamentals/rate-limits)
* [X Ads API](/es/x-ads-api/fundamentals/rate-limiting)
