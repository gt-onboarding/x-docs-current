<div id="api-key-and-secret">
  ### API Key and Secret
</div>

La API Key y el Secret (también conocidos como Consumer Key y Consumer Secret) son las credenciales más fundamentales necesarias para acceder a la X API. Estas credenciales actúan como el nombre de usuario y la contraseña de tu Aplicación de X y son utilizadas por la X API para entender de qué Aplicación provienen las solicitudes. 

Estas credenciales pueden ser utilizadas por los [endpoints de autenticación](/es/resources/fundamentals/authentication/api-reference) para generar credenciales adicionales, como [Tokens de acceso y Secrets de usuario](/es/resources/fundamentals/authentication/oauth-1-0a/api-key-and-secret) y [Bearer Tokens](/es/resources/fundamentals/authentication/oauth-2-0/bearer-tokens). También necesitas usar estas credenciales junto con Tokens de acceso y otros parámetros de autorización para [autorizar solicitudes](/es/resources/fundamentals/authentication/oauth-1-0a/authorizing-a-request) que requieren autenticación con Contexto de usuario de OAuth 1.0a. 

<div id="how-to-acquire-an-api-key-and-secret">
  #### Cómo obtener una API Key y Secret
</div>

Para obtener una API Key y Secret de X, sigue estos pasos:

1. [Regístrate para obtener una cuenta de desarrollador de X](https://developer.x.com/en/apply-for-access)
2. Crea una [X App](/es/resources/fundamentals/developer-apps) dentro del [portal para desarrolladores](/es/resources/fundamentals/developer-portal). Ten en cuenta que, si deseas usar [X API v2](/es/x-api/introduction), debes añadir tu X App a un [Proyecto](/es/resources/fundamentals/projects).
    

Cuando crees tu X App, verás tu API Key y Secret, junto con un Bearer Token. Ten en cuenta que solo mostramos estas credenciales una vez, así que asegúrate de guardarlas en tu gestor de contraseñas o en algún lugar seguro.

Tenemos más recomendaciones sobre cómo manejar tus claves y tokens en nuestra página de [mejores prácticas de autenticación](/es/resources/fundamentals/authentication/guides/authentication-best-practices), incluidos detalles sobre lo que debes hacer si tus credenciales se ven comprometidas.
 

<div id="how-to-find-and-regenerate-your-api-key-and-secret-after-app-creation">
  #### Cómo encontrar y regenerar tu API Key y Secret después de crear tu Aplicación
</div>

Si ya creaste una Aplicación y necesitas encontrar o regenerar tu API Key y Secret, sigue estos pasos:

1. Accede al portal para desarrolladores
2. Despliega el menú &#39;Projects and Apps&#39; en la barra lateral de navegación
3. Abre la Aplicación que está asociada con la API Key y Secret que deseas encontrar o regenerar
4. Ve a la pestaña &#39;Keys and tokens&#39;

Ahí encontrarás todas las credenciales asociadas con tu Aplicación. 
 

<div id="how-to-use-your-api-key-and-secret">
  #### Cómo usar tu API Key y Secret
</div>

Si solo estás explorando la X Developer Platform, te recomendamos utilizar una [herramienta o biblioteca](/es/resources/tools-and-libraries) para ver qué está disponible en la plataforma. Estas herramientas gestionan la autenticación de forma eficaz y pueden ahorrarte mucho tiempo y frustración. En particular, recomendamos [comenzar con Postman](/es/tutorials/postman-getting-started) o [Insomnia](https://insomnia.rest/) para desarrolladores principiantes. 

Si te interesa crear una solicitud desde cero, lee nuestra guía sobre [cómo autorizar una solicitud OAuth 1.0a](/es/resources/fundamentals/authentication/oauth-1-0a/authorizing-a-request).
