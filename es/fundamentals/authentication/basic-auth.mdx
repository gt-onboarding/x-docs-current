<div id="basic-authentication">
  ## Autenticación básica
</div>

Muchas de las API Enterprise de X requieren el uso de autenticación básica HTTP (HTTP Basic Authentication). Para realizar correctamente una solicitud a una API que requiere autenticación básica, debes enviar una combinación válida de dirección de correo electrónico y contraseña como encabezado de autorización en cada solicitud. La combinación de correo electrónico y contraseña es la misma que usarás para acceder a la [Enterprise API console](https://console.gnip.com/) y se puede modificar desde esta consola. 

Al crear una solicitud utilizando autenticación básica, asegúrate de agregar el encabezado HTTP Authentication: Basic con las credenciales codificadas sobre HTTPS.

En el siguiente ejemplo de solicitud cURL, deberás reemplazar `<email_address>` y `<password>` con tus credenciales antes de enviar la solicitud:

```bash
curl -v --compressed -u<email_address>:<password>
    "https://gnip-api.x.com/search/30day/accounts/<account-name>/prod/counts.json?query=from%3Axdevelopers"
```
