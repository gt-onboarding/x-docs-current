Empieza a usar el SDK de TypeScript para X API en tu proyecto.

<div id="install">
  ## Instalación
</div>

<CodeGroup>
  ```bash npm theme={null}
  npm install @xdevplatform/xdk
  ```

  ```bash yarn theme={null}
  yarn add @xdevplatform/xdk
  ```

  ```bash pnpm theme={null}
  pnpm add @xdevplatform/xdk
  ```
</CodeGroup>

<div id="typescript-support">
  ## Compatibilidad con TypeScript
</div>

El SDK está escrito en TypeScript e incluye definiciones de tipos completas. No se requieren paquetes de tipos adicionales.

<div id="requirements">
  ## Requisitos
</div>

* Node.js 16+
* TypeScript 4.5+ (si utilizas TypeScript)
