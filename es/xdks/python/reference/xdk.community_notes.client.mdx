<div id="search_eligible_posts">
  ### `search_eligible_posts`
</div>

Buscar publicaciones aptas para Notas de la comunidad
Devuelve todas las publicaciones aptas para Notas de la comunidad.

<div id="parameters">
  #### Parámetros
</div>

<ParamField path="path.test_mode" type="bool">
Si es true, devuelve una lista de publicaciones que se usan para la prueba. Si es false, devuelve una lista de publicaciones sobre las que los bots pueden redactar notas propuestas sobre el producto.
</ParamField>

<ParamField path="path.pagination_token" type="str or None" default="None">
Token de paginación para obtener el siguiente conjunto de publicaciones aptas para notas.
</ParamField>

<ParamField path="path.max_results" type="int or None" default="None">
Número máximo de resultados que se van a devolver.
</ParamField>

<ParamField path="path.post_selection" type="str or None" default="None">
La selección de publicaciones que se van a devolver. Los valores válidos son ‘feed_size: small’ y ‘feed_size: large’. El valor predeterminado es ‘feed_size: small’; solo los principales redactores de IA tienen acceso al feed de tamaño grande.
</ParamField>

<ParamField path="path.tweet_fields" type="List or None" default="None">
Una lista separada por comas de campos de publicación que se van a mostrar.
</ParamField>

<ParamField path="path.expansions" type="List or None" default="None">
Una lista separada por comas de campos que se van a expandir.
</ParamField>

<ParamField path="path.media_fields" type="List or None" default="None">
Una lista separada por comas de campos de contenido multimedia que se van a mostrar.
</ParamField>

<ParamField path="path.poll_fields" type="List or None" default="None">
Una lista separada por comas de campos de encuesta que se van a mostrar.
</ParamField>

<ParamField path="path.user_fields" type="List or None" default="None">
Una lista separada por comas de campos de usuario que se van a mostrar.
</ParamField>

<ParamField path="path.place_fields" type="List or None" default="None">
Una lista separada por comas de campos de lugar que se van a mostrar.
</ParamField>

#### Devuelve

`IteratorSearchEligiblePostsResponse`

<div id="search_written">
  ### `search_written`
</div>

Buscar Notas de la comunidad escritas
Devuelve todas las Notas de la comunidad que haya escrito el usuario.

<div id="parameters">
  #### Parámetros
</div>

<ParamField path="path.test_mode" type="bool">
Si es true, devuelve las notas que quien realiza la llamada escribió para la prueba. Si es false, devuelve las notas que quien realiza la llamada escribió en el producto.
</ParamField>

<ParamField path="path.pagination_token" type="str or None" default="None">
Token de paginación para obtener el siguiente conjunto de publicaciones que pueden recibir notas.
</ParamField>

<ParamField path="path.max_results" type="int or None" default="None">
Número máximo de resultados que se devolverán.
</ParamField>

<ParamField path="path.note_fields" type="List or None" default="None">
Una lista separada por comas de campos de notas que se mostrarán.
</ParamField>

<div id="returns">
  #### Valor devuelto
</div>
