<div id="search">
  ### `search`
</div>

Buscar comunidades
Obtiene una lista de comunidades que coinciden con la consulta de búsqueda especificada.

<div id="parameters">
  #### Parámetros
</div>

<ParamField path="path.query" type="str">
Cadena de consulta para buscar comunidades.
</ParamField>

<ParamField path="path.max_results" type="int or None" default="None">
El número máximo de resultados de búsqueda que se devolverán en una solicitud.
</ParamField>

<ParamField path="path.next_token" type="Any or None" default="None">
Este parámetro se usa para obtener la siguiente «página» de resultados. El valor que se pasa en el parámetro se toma directamente de la respuesta proporcionada por la API y no debe modificarse.
</ParamField>

<ParamField path="path.pagination_token" type="Any or None" default="None">
Este parámetro se usa para obtener la siguiente «página» de resultados. El valor que se pasa en el parámetro se toma directamente de la respuesta proporcionada por la API y no debe modificarse.
</ParamField>

<ParamField path="path.community_fields" type="List or None" default="None">
Una lista separada por comas de campos de Community que se van a mostrar.
</ParamField>
