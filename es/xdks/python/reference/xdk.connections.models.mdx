Este módulo proporciona modelos de Pydantic para las estructuras de datos de las solicitudes y respuestas
de los endpoints de conexiones de la X API. Todos los modelos se generan
a partir de la especificación de OpenAPI y proporcionan seguridad de tipos y validación.
