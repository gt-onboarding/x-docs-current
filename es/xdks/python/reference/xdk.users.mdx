<div id="block_dms">
  ### `block_dms`
</div>

Bloquear MD
Bloquea, para el usuario autenticado, los Mensajes Directos hacia y desde un usuario específico por su id.

<div id="parameters">
  #### Parámetros
</div>

<ParamField path="path.id" type="Any">
  El id del Usuario de destino para el que el usuario autenticado solicita bloquear los mensajes directos (MD).
</ParamField>

#### Devuelve

`BlockDmsResponse` - Datos de la respuesta

<div id="create_bookmark">
  ### `create_bookmark`
</div>

Crear marcador
Agrega una publicación a los marcadores del usuario autenticado.

<div id="parameters">
  #### Parámetros
</div>

<ParamField path="path.id" type="Any">
  El id del usuario de origen autenticado para el que se agregarán marcadores.
</ParamField>

<ParamField path="body.body" type="CreateBookmarkRequest">
  Cuerpo de la solicitud
</ParamField>

<div id="delete_bookmark">
  ### `delete_bookmark`
</div>

Eliminar marcador

  El id del usuario autenticado de origen cuyo marcador se va a eliminar.
</ParamField>

<ParamField path="path.tweet_id" type="Any">
  El id de la Publicación que el usuario de origen está eliminando de sus marcadores.
</ParamField>

`DeleteBookmarkResponse` - datos de la respuesta
