Utilidades de streaming sólidas para el SDK de la X API.

Este módulo proporciona manejo de conexiones de streaming con reconexión automática,
backoff exponencial y gestión de errores completa. Los clientes pueden consumir
endpoints de streaming sin preocuparse por la gestión de las conexiones: el SDK
gestiona toda la recuperación automáticamente.

<div id="xdkstreamingstream_with_retry">
  ### `xdk.streaming.stream_with_retry`
</div>

Transmite datos desde un endpoint con reconexión automática y backoff exponencial.
Esta función gestiona toda la conexión, incluyendo:

- Reconexión automática ante desconexiones
- Backoff exponencial con jitter para los retrasos de reintento
- Clasificación de errores como reintentables vs. fatales
- Callbacks de ciclo de vida para supervisar el estado de la conexión
