<div id="get_analytics">
  ### `get_analytics`
</div>

Obtener estadísticas de contenido multimedia
Recupera datos analíticos del contenido multimedia.

#### Parameters

<ParamField path="path.media_keys" type="List">
Una lista de Media Keys separada por comas. Se permiten hasta 100 elementos en una sola solicitud.
</ParamField>

<ParamField path="path.end_time" type="str">
YYYY-MM-DDTHH:mm:ssZ. La marca de tiempo en UTC que representa el final del intervalo de tiempo.
</ParamField>

<ParamField path="path.start_time" type="str">
YYYY-MM-DDTHH:mm:ssZ. La marca de tiempo en UTC que representa el inicio del intervalo de tiempo.
</ParamField>

<ParamField path="path.granularity" type="str">
La granularidad de los resultados de conteo de búsqueda.
</ParamField>

<ParamField path="path.media_analytics_fields" type="List or None" default="None">
Una lista de campos de MediaAnalytics, separados por comas, para mostrar.
</ParamField>

<div id="returns">
  #### Devuelve
</div>

`GetAnalyticsResponse` - Datos de respuesta
