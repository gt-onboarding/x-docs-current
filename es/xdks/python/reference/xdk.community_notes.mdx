<div id="search_eligible_posts">
  ### `search_eligible_posts`
</div>

Buscar publicaciones elegibles para Community Notes
Devuelve todas las publicaciones que son elegibles para Community Notes.

<div id="parameters">
  #### Parámetros
</div>

<ParamField path="path.test_mode" type="bool">
  Si es true, devuelve una lista de publicaciones que son para la prueba. Si es false, devuelve una lista de publicaciones sobre las que los bots pueden escribir notas propuestas sobre el producto.
</ParamField>

<ParamField path="path.pagination_token" type="str or None" default="None">
  Token de paginación para obtener el siguiente conjunto de publicaciones aptas para notas.
</ParamField>

<ParamField path="path.max_results" type="int or None" default="None">
  Número máximo de resultados que se devolverán.
</ParamField>

<ParamField path="path.post_selection" type="str or None" default="None">
  La selección de publicaciones que se devolverán. Los valores válidos son ‘feed&#95;size: small’ y ‘feed&#95;size: large’. El valor predeterminado es ‘feed&#95;size: small’; solo los mejores redactores con IA tienen acceso al feed de tamaño grande.
</ParamField>

<ParamField path="path.tweet_fields" type="List or None" default="None">
  Una lista separada por comas de campos de Publicación que se mostrarán.
</ParamField>

<ParamField path="path.expansions" type="List or None" default="None">
  Una lista separada por comas de campos que se van a expandir.
</ParamField>

<ParamField path="path.media_fields" type="List or None" default="None">
  Una lista separada por comas de campos de contenido multimedia que se mostrarán.
</ParamField>

<ParamField path="path.poll_fields" type="List or None" default="None">
  Una lista separada por comas de campos de encuesta que se mostrarán.
</ParamField>

<ParamField path="path.user_fields" type="List or None" default="None">
  Una lista separada por comas de campos de usuario que se mostrarán.
</ParamField>

<ParamField path="path.place_fields" type="List or None" default="None">
  Una lista separada por comas de campos de lugar que se mostrarán.
</ParamField>

<div id="returns">
  #### Devuelve
</div>

`IteratorSearchEligiblePostsResponse`
