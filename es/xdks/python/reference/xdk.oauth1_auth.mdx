Este módulo proporciona funcionalidad de autenticación con OAuth 1.0a para flujos de autorización seguros. Incluye generación de tokens de solicitud, generación de URL de autorización, intercambio de tokens de acceso y generación de firmas OAuth 1.0a.

  ### `class xdk.oauth1_auth.OAuth1`
</div>
