Este módulo proporciona funcionalidad de autenticación OAuth 1.0a para flujos
de autorización seguros. Incluye la generación de tokens de solicitud, la
generación de URL de autorización, el intercambio de Token de acceso y la
generación de firmas OAuth 1.
