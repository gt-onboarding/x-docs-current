Este módulo proporciona funcionalidad de autenticación mediante OAuth2 PKCE (Proof Key for Code Exchange)
para flujos de autorización seguros. Incluye generación del `code verifier`,
gestión de tokens y renovación automática de tokens.

<div id="class-xdkoauth2_authoauth2pkceauth">
  ### `class xdk.oauth2_auth.OAuth2PKCEAuth`
</div>

Autenticación OAuth 2 PKCE para la X API.

<div id="parameters">
  #### Parámetros
</div>

<ParamField path="path.base_url" type="str" default="'https://api.x.com'">
La URL base para el endpoint de tokens de la X API (por defecto [https://api.x.com](https://api.x.com)).
</ParamField>

La URL base para la autorización OAuth2 (por defecto [https://x.com/i](https://x.com/i)).
</ParamField>

<ParamField path="path.client_id" type="str or None" default="None">
El ID de cliente para la X API.
</ParamField>

<ParamField path="path.client_secret" type="str or None" default="None">
El secreto de cliente para la X API.
</ParamField>

<ParamField path="path.redirect_uri" type="str or None" default="None">
El URI de redirección para la autorización OAuth2.
</ParamField>

<ParamField path="path.token" type="Dict[str, Any] or None" default="None">
Un diccionario de tokens OAuth2 existente (si está disponible).
</ParamField>

<ParamField path="path.scope" type="str or List[str] or None" default="None">
Cadena de texto con valores separados por espacios o lista de cadenas para los ámbitos (scopes) de autorización OAuth2.
</ParamField>
