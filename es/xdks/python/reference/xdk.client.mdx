Este módulo proporciona la clase principal Client para interactuar con la X API.
Coordina todos los subclientes y se encarga de la autenticación, la gestión de sesiones
y los flujos de OAuth2 PKCE. Toda la funcionalidad se genera a partir de la especificación de OpenAPI.

<div id="parameters">
  #### Parámetros
</div>

<ParamField path="path.base_url" type="str" default="'https://api.x.com'">
La URL base de la X API (de manera predeterminada, [https://api.x.com](https://api.x.com)).
</ParamField>

<ParamField path="path.bearer_token" type="str or None" default="None">
El Bearer Token para la X API (autenticación de solo aplicación).
</ParamField>

<ParamField path="path.access_token" type="str or None" default="None">
El Token de acceso OAuth2 para el contexto de usuario (se puede usar directamente como Bearer Token).
</ParamField>

<ParamField path="path.client_id" type="str or None" default="None">
El ID de cliente para la X API (requerido para el flujo OAuth2 PKCE).
</ParamField>

<ParamField path="path.client_secret" type="str or None" default="None">
El secreto de cliente para la X API.
</ParamField>

<ParamField path="path.redirect_uri" type="str or None" default="None">
La URI de redirección para la autorización OAuth2.
</ParamField>

<ParamField path="path.token" type="Dict[str, Any] or None" default="None">
Un diccionario de un token OAuth2 existente (si está disponible). Si se proporciona, se extraerá el access_token.
</ParamField>

<ParamField path="path.scope" type="str or List[str] or None" default="None">
Cadena de texto separada por espacios o lista de cadenas para los ámbitos de autorización OAuth2.
</ParamField>

<ParamField path="path.authorization_base_url" type="str" default="'https://x.com/i'">
La URL base para la autorización OAuth2 (de manera predeterminada, [https://x.com/i](https://x.com/i)).
</ParamField>

<ParamField path="path.auth" type="OAuth1">
Instancia de OAuth1 para la autenticación OAuth1.0a.
</ParamField>
