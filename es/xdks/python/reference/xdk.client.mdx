Este módulo proporciona la clase principal Client para interactuar con la X API.
Coordina todos los subclientes y gestiona la autenticación, la administración de sesiones,
así como los flujos OAuth2 PKCE. Toda la funcionalidad se genera a partir de la especificación OpenAPI.

<div id="parameters">
  #### Parámetros
</div>

<ParamField path="path.base_url" type="str" default="'https://api.x.com'">
La URL base de la X API (de forma predeterminada [https://api.x.com](https://api.x.com)).
</ParamField>

<ParamField path="path.bearer_token" type="str or None" default="None">
El Bearer Token para la X API (autenticación solo de aplicación).
</ParamField>

<ParamField path="path.access_token" type="str or None" default="None">
El token de acceso OAuth2 para contexto de usuario (se puede usar directamente como Bearer Token).
</ParamField>

<ParamField path="path.client_id" type="str or None" default="None">
El ID de cliente para la X API (requerido para el flujo OAuth2 PKCE).
</ParamField>

<ParamField path="path.client_secret" type="str or None" default="None">
El secreto de cliente para la X API.
</ParamField>

<ParamField path="path.redirect_uri" type="str or None" default="None">
La URI de redirección para la autorización OAuth2.
</ParamField>

<ParamField path="path.token" type="Dict[str, Any] or None" default="None">
Un diccionario de token OAuth2 existente (si está disponible). Si se proporciona, se extraerá el access_token.
</ParamField>

<ParamField path="path.scope" type="str or List[str] or None" default="None">
Cadena separada por espacios o lista de cadenas para los scopes de autorización OAuth2.
</ParamField>

<ParamField path="path.authorization_base_url" type="str" default="'https://x.com/i'">
La URL base para la autorización OAuth2 (de forma predeterminada [https://x.com/i](https://x.com/i)).
</ParamField>

<ParamField path="path.auth" type="OAuth1">
Instancia OAuth1 para la autenticación OAuth1.0a.
</ParamField>
