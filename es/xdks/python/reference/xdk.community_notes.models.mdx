Este módulo proporciona modelos Pydantic para las estructuras de datos de las solicitudes y respuestas
de los endpoints de notas de la comunidad de la X API. Todos los modelos se generan
a partir de la especificación OpenAPI y proporcionan seguridad de tipos y validación.
