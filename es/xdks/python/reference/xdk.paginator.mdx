Utilidades de paginación basada en cursores para el SDK de X API.

Este módulo proporciona una clase Cursor para ofrecer una paginación elegante en
todos los clientes de la API. La clase Cursor permite iterar fácilmente sobre resultados paginados
usando tanto los métodos .pages() como .items(), manteniendo la seguridad de tipos adecuada.

<div id="xdkpaginatorcursor">
  ### `xdk.paginator.cursor`
</div>

Crea un cursor con la inferencia y la validación de tipos adecuadas.
Esta función de fábrica ayuda con la inferencia de tipos para que
obtengas sugerencias de tipo correctas para el tipo de respuesta y
valida que el método sea compatible con la paginación tanto en el
análisis estático como en tiempo de ejecución.

</ParamField>
