<div id="labels_compliance">
  ### `labels_compliance`
</div>

Etiquetas de Publicaciones en streaming (Streaming)
Transmite todos los eventos de etiquetado aplicados a Publicaciones.
Este es un endpoint de streaming que entrega datos en tiempo real a medida que se vuelven disponibles.
Cada elemento entregado representa un único punto de datos del flujo.
La conexión se gestiona automáticamente con una lógica de reintentos con backoff exponencial.
Si el flujo se desconecta, el SDK se volverá a conectar automáticamente sin intervención por parte del cliente.

<div id="parameters">
  #### Parámetros
</div>

<ParamField path="path.backfill_minutes" type="int or None" default="None">
  La cantidad de minutos de backfill solicitada.
</ParamField>

<ParamField path="path.start_time" type="str or None" default="None">
  YYYY-MM-DDTHH:mm:ssZ. La marca de tiempo UTC más temprana a partir de la cual se proporcionarán las etiquetas de la Publicación.
</ParamField>

<ParamField path="path.end_time" type="str or None" default="None">
  YYYY-MM-DDTHH:mm:ssZ. La marca de tiempo UTC más reciente a partir de la cual se proporcionarán las etiquetas de la Publicación.
</ParamField>

<ParamField path="path.stream_config" type="StreamConfig">
  StreamConfig opcional para personalizar el comportamiento de reintento, los tiempos de espera y las devoluciones de llamada (callbacks).
</ParamField>
