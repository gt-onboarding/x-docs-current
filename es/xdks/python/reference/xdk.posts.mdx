  ## Contenido del módulo
</div>

Este módulo proporciona acceso a los endpoints de publicaciones de la X API
y sirve como el punto de entrada principal para toda la funcionalidad relacionada con las publicaciones.

<div id="get_counts_all">
  ### `get_counts_all`
</div>

Obtener la cantidad total de todas las Publicaciones
Recupera la cantidad de Publicaciones que coinciden con una consulta de búsqueda desde el archivo completo.

<div id="parameters">
  #### Parámetros
</div>

<ParamField path="path.query" type="str">
  Una consulta/regla/filtro para encontrar Publicaciones coincidentes. Consulta [https://t.co/rulelength](https://t.co/rulelength) para identificar la longitud máxima permitida de la consulta.
</ParamField>

<ParamField path="path.start_time" type="str or None" default="None">
  YYYY-MM-DDTHH:mm:ssZ. La marca de tiempo UTC más antigua (de los últimos 7 días) a partir de la cual se proporcionarán las Publicaciones. La marca de tiempo tiene granularidad de segundos y es inclusiva (es decir, 12:00:01 incluye el primer segundo del minuto).
</ParamField>

<ParamField path="path.end_time" type="str or None" default="None">
  YYYY-MM-DDTHH:mm:ssZ. La marca de tiempo UTC más reciente hasta la cual se proporcionarán las Publicaciones. La marca de tiempo tiene granularidad de segundos y es exclusiva (es decir, 12:00:01 excluye el primer segundo del minuto).
</ParamField>

<ParamField path="path.since_id" type="Any or None" default="None">
  Devuelve resultados con un ID de Publicación mayor (es decir, más reciente) que el ID especificado.
</ParamField>

<ParamField path="path.until_id" type="Any or None" default="None">
  Devuelve resultados con un ID de Publicación menor (es decir, más antiguo) que el ID especificado.
</ParamField>

<ParamField path="path.next_token" type="Any or None" default="None">
  Este parámetro se usa para obtener la siguiente “página” de resultados. El valor utilizado con el parámetro se toma directamente de la respuesta devuelta por la API y no debe modificarse.
</ParamField>

<ParamField path="path.pagination_token" type="Any or None" default="None">
  Este parámetro se usa para obtener la siguiente “página” de resultados. El valor utilizado con el parámetro se toma directamente de la respuesta devuelta por la API y no debe modificarse.
</ParamField>

<ParamField path="path.granularity" type="str or None" default="None">
  La granularidad de los resultados de conteo de búsqueda.
</ParamField>

<ParamField path="path.search_count_fields" type="List or None" default="None">
