<div id="get_counts_all">
  ### `get_counts_all`
</div>

Obtiene la cantidad de todas las Publicaciones
Obtiene la cantidad de Publicaciones que coinciden con una consulta de búsqueda en todo el archivo.

<div id="parameters">
  #### Parámetros
</div>

<ParamField path="path.query" type="str">
  Una consulta/regla/filtro para buscar Publicaciones que coincidan. Consulta [https://t.co/rulelength](https://t.co/rulelength) para identificar la longitud máxima permitida de la consulta.
</ParamField>

<ParamField path="path.start_time" type="str or None" default="None">
  AAAA-MM-DDTHH:mm:ssZ. La marca de tiempo UTC más antigua (de los últimos 7 días) a partir de la cual se proporcionarán las Publicaciones. La marca de tiempo tiene granularidad de segundos y es inclusiva (es decir, 12:00:01 incluye el primer segundo del minuto).
</ParamField>

<ParamField path="path.end_time" type="str or None" default="None">
  AAAA-MM-DDTHH:mm:ssZ. La marca de tiempo UTC más reciente hasta la cual se proporcionarán las Publicaciones. La marca de tiempo tiene granularidad de segundos y es exclusiva (es decir, 12:00:01 excluye el primer segundo del minuto).
</ParamField>

<ParamField path="path.since_id" type="Any or None" default="None">
  Devuelve resultados con un id de la Publicación mayor que (es decir, más reciente que) el id especificado.
</ParamField>

<ParamField path="path.until_id" type="Any or None" default="None">
  Devuelve resultados con un id de la Publicación menor que (es decir, más antigua que) el id especificado.
</ParamField>

<ParamField path="path.next_token" type="Any or None" default="None">
  Este parámetro se utiliza para obtener la siguiente “página” de resultados. El valor usado con el parámetro se extrae directamente de la respuesta proporcionada por la API y no debe modificarse.
</ParamField>

<ParamField path="path.pagination_token" type="Any or None" default="None">
  Este parámetro se utiliza para obtener la siguiente “página” de resultados. El valor usado con el parámetro se extrae directamente de la respuesta proporcionada por la API y no debe modificarse.
</ParamField>

<ParamField path="path.granularity" type="str or None" default="None">
  La granularidad de los resultados de recuento de búsqueda.
</ParamField>

<ParamField path="path.search_count_fields" type="List or None" default="None">
