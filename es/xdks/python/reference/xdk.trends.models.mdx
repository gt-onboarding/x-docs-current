Este módulo proporciona modelos de Pydantic para las estructuras de datos de las solicitudes y respuestas
para los endpoints de tendencias de la X API. Todos los modelos se generan
a partir de la especificación OpenAPI y proporcionan seguridad de tipos y validación.
