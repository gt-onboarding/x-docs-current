Este módulo proporciona un cliente para interactuar con los endpoints de actividad de la X API.
Las operaciones de streaming en tiempo real devuelven generadores que entregan datos a medida que van llegando.
Las conexiones de streaming se gestionan automáticamente con lógica de reintentos con backoff exponencial para un manejo sólido y confiable.

<div id="stream">
  ### `stream`
</div>

Flujo de actividad (streaming)
Flujo de actividades de X
Este es un endpoint de streaming que proporciona datos en tiempo real a medida que están disponibles.
Cada elemento emitido representa un único dato del flujo.
La conexión se gestiona automáticamente mediante una lógica de reintentos con backoff exponencial.
Si el flujo se desconecta, el SDK se reconectará automáticamente sin intervención del cliente.
