<div id="search">
  ### `search`
</div>

Buscar Communities
Devuelve una lista de Communities que coinciden con la consulta de búsqueda especificada.

<div id="parameters">
  #### Parámetros
</div>

<ParamField path="path.query" type="str">
  Consulta para buscar comunidades.
</ParamField>

<ParamField path="path.max_results" type="int or None" default="None">
  El número máximo de resultados de búsqueda que se devolverán en una solicitud.
</ParamField>

<ParamField path="path.next_token" type="Any or None" default="None">
  Este parámetro se utiliza para obtener la siguiente página de resultados. El valor utilizado con el parámetro se extrae directamente de la respuesta proporcionada por la API y no debe modificarse.
</ParamField>

<ParamField path="path.pagination_token" type="Any or None" default="None">
  Este parámetro se utiliza para obtener la siguiente página de resultados. El valor utilizado con el parámetro se extrae directamente de la respuesta proporcionada por la API y no debe modificarse.
</ParamField>

<ParamField path="path.community_fields" type="List or None" default="None">
  Una lista separada por comas de campos de Community que se mostrarán.
</ParamField>
