  ## Contenido del módulo

Este módulo proporciona acceso a los endpoints de comunidades de la X API
y actúa como el punto de entrada principal para toda la funcionalidad relacionada con las comunidades.

<div id="search">
  ### `search`
</div>

Buscar comunidades
Obtiene una lista de comunidades que coinciden con la consulta de búsqueda indicada.

<div id="parameters">
  #### Parámetros
</div>

<ParamField path="path.query" type="str">
  Consulta para buscar comunidades.
</ParamField>

<ParamField path="path.max_results" type="int or None" default="None">
  El número máximo de resultados de búsqueda que se devolverán en una petición.
</ParamField>

<ParamField path="path.next_token" type="Any or None" default="None">
  Este parámetro se utiliza para obtener la siguiente «página» de resultados. El valor utilizado con el parámetro se toma directamente de la respuesta proporcionada por la API y no se debe modificar.
</ParamField>

<ParamField path="path.pagination_token" type="Any or None" default="None">
  Este parámetro se utiliza para obtener la siguiente «página» de resultados. El valor utilizado con el parámetro se toma directamente de la respuesta proporcionada por la API y no se debe modificar.
</ParamField>

<ParamField path="path.community_fields" type="List or None" default="None">
  Una lista separada por comas de campos de comunidad que se mostrarán.
</ParamField>
