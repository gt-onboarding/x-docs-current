---
title: "StreamClient"
sidebarTitle: "StreamClient"
---

Este módulo proporciona un cliente para interactuar con los endpoints de streaming de la X API.
Las operaciones de streaming en tiempo real devuelven generadores que van entregando datos a medida que llegan.
Las conexiones de streaming se gestionan automáticamente con una lógica de reintentos con backoff exponencial para una gestión robusta.
