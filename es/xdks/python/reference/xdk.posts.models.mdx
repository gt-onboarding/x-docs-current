Este módulo proporciona modelos de Pydantic para las estructuras de datos de las solicitudes y respuestas de los endpoints de Publicaciones de la X API. Todos los modelos se generan a partir de la especificación OpenAPI y ofrecen seguridad de tipos y validación.
