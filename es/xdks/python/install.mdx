El SDK de Python de XDK está disponible directamente desde el repositorio de GitHub y puede instalarse con `pip`.

<div id="prerequisites">
  ## Requisitos previos
</div>

- Python 3.8 o superior.
- `pip` y `venv` para entornos virtuales (recomendado).

<div id="quick-install">
  ## Instalación rápida
</div>

Instala el XDK desde el subdirectorio en GitHub:

```bash
pip install xdk
```

Esto obtiene la versión generada más reciente de la rama `main`.


<div id="development-install">
  ## Instalación para desarrollo
</div>

Para desarrollo o para colaborar:

1. Clona el repositorio:
   ```bash
   git clone https://github.com/xdevplatform/xdk.git
   cd xdk/python
   ```
2. Instala las dependencias en modo editable:
   ```bash
   pip install -e .
   ```
   Esto instala el SDK y sus dependencias en tiempo de ejecución.
3. (Opcional) Instala las dependencias de desarrollo para pruebas y linting:
   ```bash
   pip install -e .[dev]
   ```

<div id="verification">
  ## Verificación
</div>

Comprueba la instalación:

```python
import xdk
print(xdk.__version__)  # Debe imprimir la versión del XDK
```

**Nota:** Dado que el XDK se genera utilizando la especificación OpenAPI, revisa siempre el [registro de cambios de X API](https://docs.x.com/changelog) y las notas de la versión del XDK en el repositorio para conocer cualquier cambio.
Para obtener ejemplos de código detallados que utilizan el XDK de Python, consulta nuestro [repositorio de ejemplos de código en GitHub](https://github.com/xdevplatform/samples/tree/main/python).
