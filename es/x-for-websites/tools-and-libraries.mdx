<div id="official-resources">
  ## Recursos oficiales
</div>

Las siguientes herramientas pueden ayudarte a comenzar con X para sitios web.

<CardGroup cols={3}>
<Card
  title="publish.x.com"
  icon="x-twitter"
  href="https://publish.x.com/"
>
Crea tu propia Publicación, cronología o botón incrustado.
</Card>
<Card
  title="Twitter for WordPress"
  icon="x-twitter"
  href="https://github.com/twitter/wordpress"
>
Un complemento oficial para optimizar tu sitio de WordPress para X, que incluye Publicaciones incrustadas y X Cards.
</Card>
<Card
  title="twemoji"
  icon="x-twitter"
  href="https://twemoji.x.com/"
>
Juego de caracteres emoji de X, gratuito y de código abierto, que incluye una biblioteca de JavaScript para compatibilidad multiplataforma.
</Card>
</CardGroup>
