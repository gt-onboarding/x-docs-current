Puedes usar la API oEmbed para obtener contenido incrustado de forma programática, como [Publicaciones](https://developer.x.com/en/docs/twitter-for-websites/embedded-tweets/overview) y [timelines](https://developer.x.com/en/docs/twitter-for-websites/timelines/overview). 

La respuesta de la API oEmbed proporcionará un fragmento de HTML que se reconocerá automáticamente cuando se [incluya el JavaScript de widgets de X en la página](https://developer.x.com/web/javascript/loading).

Ten en cuenta que se recomienda usar la API para realizar tareas a gran escala y que aconsejamos utilizar nuestra potente herramienta [publish.x.com](https://publish.x.com/#) para incrustar contenido.

<Tabs>
  <Tab title="Cronologías incrustadas">
    El fragmento de HTML devuelto se reconocerá automáticamente como una [cronología incrustada](https://developer.x.com/en/docs/twitter-for-websites/timelines/overview) cuando se incluya en la página el [JavaScript de widgets de X](https://developer.x.com/web/javascript/loading).

    El endpoint oEmbed permite personalizar la apariencia final de una cronología incrustada configurando las propiedades correspondientes en el marcado HTML, que serán interpretadas por el JavaScript de X incluido de forma predeterminada en la respuesta HTML. El formato del marcado devuelto puede cambiar con el tiempo a medida que X agrega nuevas funciones o ajusta la representación de la cronología.

    Para una cronología de X especificada por la URL de la cronología, en un formato JSON compatible con [oEmbed](https://oembed.com/). Se admiten cronologías de usuarios y de Listas. El marcado de la cronología está pensado para almacenarse en caché en tus servidores hasta el tiempo de vida de caché sugerido por la propiedad `cache_age`.

    ## URL del recurso

    *[https://publish.x.com/oembed](https://publish.x.com/oembed)*

    ## Ejemplos de solicitudes

    ```bash
    curl --request GET --url 'https://publish.x.com/oembed?url=https%3A%2F%2Ftwitter.com%2FInterior%2Fstatus%2F507185938620219395'
    twurl -H publish.x.com "/oembed?url=https://x.com/Interior/status/463440424141459456"
    ```

    ```json
    }
    ```

  <Tab title="Publicaciones insertadas">
    El fragmento HTML devuelto se reconocerá automáticamente como una [Publicación incrustada](https://developer.x.com/web/embedded-tweets) cuando [se incluya el JavaScript del widget de X en la página](https://developer.x.com/web/javascript/loading).

    El endpoint oEmbed permite personalizar la apariencia final de una Publicación incrustada configurando las propiedades correspondientes en el marcado HTML que será interpretado por el JavaScript de X incluido con la respuesta HTML de forma predeterminada. El formato del marcado devuelto puede cambiar con el tiempo a medida que X añade nuevas funcionalidades o ajusta su representación de Publicaciones.

    El marcado de respaldo de la Publicación debe almacenarse en caché en sus servidores durante el tiempo de vida de caché sugerido especificado por la propiedad `cache_age`.

    ## URL del recurso

    | `maxwidth`  <br />Int `[220..550]`                                                                                                   | `325`                | El ancho máximo de una Publicación renderizada en píxeles enteros. Un valor proporcionado que quede por debajo o por encima del rango permitido se devolverá, respectivamente, como el ancho mínimo o máximo admitido; el valor de ancho ajustado se reflejará en la propiedad `width` devuelta. Ten en cuenta que X no es compatible con el parámetro oEmbed `maxheight`. Las Publicaciones son fundamentalmente texto y, por lo tanto, tienen una altura impredecible que no se puede escalar como una imagen o un video. Asimismo, la respuesta de oEmbed no proporcionará un valor para `height`. Las implementaciones que requieran alturas uniformes para las Publicaciones deben consultar los parámetros `hide_thread` y `hide_media` indicados a continuación. |
    | `hide_media`  <br />Boolean, String o Int                                                                                            | `false`              | Cuando se establece en `true`, `"t"` o `1`, los enlaces en una Publicación no se expanden a vistas previas de fotos, videos ni enlaces.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
    | `hide_thread`  <br />Booleano, String o Int                                                                                          | `false`              | Cuando se establece en `true`, `"t"` o `1`, no se mostrará una versión contraída de la Publicación anterior en un hilo de conversación cuando la Publicación solicitada sea una respuesta a otra Publicación.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |

    | `align`  <br />Enum `{left,right,center,none}`                                                                                       | `none`               | Especifica si la Publicación incrustada debe quedar flotada a la izquierda, a la derecha o en el centro de la página con respecto al elemento padre.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
    | `lang`  <br />Enum([Idioma](https://developer.x.com/en/docs/twitter-for-websites/twitter-for-websites-supported-languages/overview)) | `en`                 | La solicitud devuelve HTML y una Publicación renderizada en el [idioma de X compatible con las Publicaciones incrustadas](https://developer.x.com/web/overview/languages) especificado.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
    | `theme`  <br />Enum `{light, dark}`                                                                                                  | `light`              | Cuando se configura en `dark`, la Publicación se muestra con texto claro sobre un fondo oscuro.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
