El endpoint de búsqueda de tendencias permite a los desarrolladores obtener las tendencias de una ubicación, especificada mediante el identificador where-on-earth (WOEID).

<Note>
  **Nota**: WOEID es un identificador heredado creado por Yahoo que ha quedado obsoleto. X API utiliza el valor numérico para identificar ubicaciones de tendencias de ciudades y países. Consulta nuestra [entrada de blog antigua](https://blog.x.com/engineering/en_us/a/2010/woeids-in-twitters-trends.html) o los [datos archivados](https://archive.org/details/geoplanet_data_7.10.0.zip0.)
</Note>

El `tweet_count` de las últimas 24 horas también se devuelve para muchas tendencias, si está disponible.

Este endpoint admite autenticación mediante app-auth y tiene un límite de tasa de 75 solicitudes por ventana de 15 minutos.

<div id="getting-started">
  ### Introducción
</div>

Para usar este endpoint, necesitas un [bearer token]([https://developer.x.com(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token)](https://developer.x.com\(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token\)) del [portal para desarrolladores](https://developer.x.com/en/portal/dashboard). Una vez que tengas el bearer token, podrás llamar a la API de uso como se muestra a continuación:

```
      curl 'https://api.x.com/2/trends/by/woeid/26062' --header 'Authorization: Bearer XXXXX'
```

Si la solicitud se completa correctamente, verás la respuesta JSON que se muestra a continuación:

**Configuración de la cuenta**

Para acceder a estos endpoints, necesitarás:

* Una [cuenta de desarrollador](https://developer.x.com/en/portal/petition/essential/basic-info) aprobada.
* Autenticarte con las claves y tokens de una [developer App](/es/resources/fundamentals/developer-apps) que se encuentra dentro de un [Proyecto](/es/resources/fundamentals/projects). 

Obtén más información sobre cómo acceder a los endpoints de X API v2 en nuestra [guía de introducción](/es/x-api/getting-started/getting-access).
