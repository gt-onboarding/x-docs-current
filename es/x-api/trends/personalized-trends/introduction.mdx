El endpoint Personalized Trends te permite obtener tendencias personalizadas para una cuenta de usuario autenticada.

<Note>
  **Nota:** Las tendencias personalizadas solo las devolverá el endpoint para cuentas de usuario suscritas a X Premium.
</Note>

<Note>
  **Configuración de la cuenta**

  Para acceder a este endpoint, necesitarás:

  * Una [cuenta de desarrollador](https://developer.x.com/en/portal/petition/essential/basic-info) aprobada.
  * Autenticarte usando las claves y tokens de una [developer App](/es/resources/fundamentals/developer-apps) que esté dentro de un [Proyecto](/es/resources/fundamentals/projects).

  Obtén más información sobre cómo acceder a los endpoints de la X API v2 en nuestra [guía de introducción](/es/x-api/getting-started/getting-access).
</Note>

<div id="getting-started">
  ### Primeros pasos
</div>

<div id="authentication">
  #### Autenticación
</div>

Puedes autenticar este endpoint ya sea con [Contexto de usuario de OAuth 1.0a](/es/resources/fundamentals/authentication) o con [OAuth 2.0 Authorization Code with PKCE](resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2).
 

<div id="making-a-request">
  #### Realizar una solicitud
</div>

Puedes llamar al endpoint Personalized Trends tal como se muestra a continuación:

`curl 'https://api.x.com/2/users/personalized_trends' --header 'Authorization: ••••••'`

Si la solicitud se completa correctamente, deberías ver la respuesta JSON como se muestra a continuación:

```
{
    "data": [
        {
            "category": "Bitcoin",
            "post_count": "42K posts",
            "trend_name": "MicroStrategy's $42 Billion Bitcoin Investment",
            "trending_since": "5 hours ago"
        },
        {
            "category": "Music",
            "post_count": "2K posts",
            "trend_name": "Drake's 'No Face' Music Video Release",
            "trending_since": "Trending now"
        },
        {
            "category": "Weather",
            "post_count": "1.2K posts",
            "trend_name": "Super Typhoon Leon Impacts Taiwan and Batanes",
            "trending_since": "Trending now"
        }
    ]
}
```

**Nota:** Si el usuario autenticado que realiza la solicitud no está suscrito a X Premium, recibirá la siguiente respuesta:

```
{
    "data": [
        {
            "category": "Unknown",
            "post_count": "Unknown",
            "trending_since": ""
        }
    ]
}
```
