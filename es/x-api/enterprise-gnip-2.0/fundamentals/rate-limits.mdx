<div id="overview">
  ## Descripción general
</div>

Cada día, muchos miles de desarrolladores realizan solicitudes a la X API. Para ayudar a gestionar el elevado volumen de estas solicitudes, se imponen límites a la cantidad de solicitudes que se pueden realizar. Estos límites nos ayudan a ofrecer la API confiable y escalable en la que nuestra comunidad de desarrolladores confía. 

El número máximo de solicitudes permitidas se basa en un intervalo de tiempo, es decir, un período o ventana de tiempo específico. Si un endpoint tiene un límite de tasa de 900 solicitudes/15 minutos, entonces se permiten hasta 900 solicitudes dentro de cualquier intervalo de 15 minutos.
