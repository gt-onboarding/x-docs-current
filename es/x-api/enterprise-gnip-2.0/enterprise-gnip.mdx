Nuestras APIs de Enterprise ofrecen el nivel más alto de acceso y fiabilidad para quienes dependen de los datos de X. Son ideales cuando superas el nivel premium y necesitas un acceso aún más fiable, paquetes personalizados o contratos a largo plazo. El acceso a la Enterprise API incluye administradores de cuentas dedicados y soporte técnico personalizado.

Si has identificado que las soluciones Basic y Pro no son suficientes para cubrir tus necesidades, puedes solicitar acceso a los productos Enterprise en el siguiente enlace:

<Button href="https://developer.x.com/en/products/twitter-api/enterprise/enterprise-api-interest-form">
  Solicitar acceso
</Button>

<div id="available-endpoints">
  ## Endpoints disponibles
</div>

| En tiempo real | [PowerTrack API](/es/x-api/enterprise-gnip-2.0/powertrack-api) | Proporciona una cobertura completa de las Publicaciones públicas en tiempo real, que se pueden filtrar mediante una amplia variedad de operadores y reglas. |
| :--- | :--- | :--- |
| [Decahose API](/es/x-api/enterprise-gnip-2.0/fundamentals/decahose-api) | Transmite una muestra aleatoria del 10% de todas las Publicaciones públicas y sus Me gusta. |
| :--- | :--- | :--- |
| [Account Activity API](/es/x-api/enterprise-gnip-2.0/fundamentals/account-activity) | Permite suscribirse a usuarios y recibir una variedad de actividades diferentes, incluidas Publicaciones, Mensajes Directos, Me gusta, seguimientos (follows) y más. |
| :--- | :--- | :--- |
| Histórico | [Search API](/es/x-api/enterprise-gnip-2.0/fundamentals/search-api) | Busca en los últimos 30 días o en el archivo completo de Publicaciones públicas utilizando herramientas de filtrado avanzadas. |
| [Historical PowerTrack API](/es/x-api/enterprise-gnip-2.0/powertrack-api) | Filtra el archivo completo de Publicaciones públicas mediante un proceso económico de acceso por lotes. |
| Insights | [Engagement API](/es/x-api/enterprise-gnip-2.0/fundamentals/engagement-api) | Mide el volumen de impresiones e interacciones de las Publicaciones públicas. |
| Cumplimiento | [Compliance Firehose API](/es/x-api/enterprise-gnip-2.0/fundamentals/firehouse) | Transmite todos los eventos de cumplimiento. |
| Uso | [Usage API](/es/x-api/enterprise-gnip-2.0/fundamentals/usage) | Proporciona acceso programático a los datos de consumo de actividad en todos los productos Enterprise. |

Obtén más información sobre las URL expandidas, los datos de encuestas y la geolocalización del perfil visitando nuestra [sección de enriquecimientos](/es/x-api/enterprise-gnip-2.0/fundamentals/data-enrichments).
 

<div id="advanced-filtering-tools">
  ### Herramientas avanzadas de filtrado
</div>

La Premium Search API pone a disposición un conjunto avanzado de operadores que no están disponibles con el endpoint de búsqueda estándar. Muchos de estos operadores están disponibles para los usuarios del sandbox gratuito, pero se obtiene un conjunto más sólido de operadores si se utiliza un plan de pago. Para obtener más información, visita nuestra [sección de reglas y filtros](/es/x-api/enterprise-gnip-2.0/fundamentals/rules-filtering#getting-started-with-enterprise-rules-and-queries).
