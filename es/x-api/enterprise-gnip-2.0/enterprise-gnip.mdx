Nuestras APIs Enterprise ofrecen el nivel más alto de acceso y fiabilidad para quienes dependen de los datos de X. Son perfectas a medida que escalas más allá del nivel premium y necesitas un acceso más fiable, paquetes personalizados o contratos a más largo plazo. El acceso a las APIs Enterprise incluye gestores de cuentas dedicados y soporte técnico personalizado.

Si has comprobado que las soluciones Basic y Pro no son suficientes para satisfacer tus necesidades, puedes solicitar acceso a los productos Enterprise en el siguiente enlace:

  Solicitar acceso
</Button>

<div id="available-endpoints">
  ## Endpoints disponibles
</div>

| En tiempo real | [PowerTrack API](/es/x-api/enterprise-gnip-2.0/powertrack-api) | Proporciona una cobertura completa de las Publicaciones públicas en el momento en que se producen, que se pueden filtrar mediante una amplia gama de operadores y reglas. |
| :--- | :--- | :--- |
| [Decahose API](/es/x-api/enterprise-gnip-2.0/fundamentals/decahose-api) | Transmite una muestra aleatoria del 10% de todas las Publicaciones públicas y sus Me gusta. |
| :--- | :--- | :--- |
| [Account Activity API](/es/x-api/enterprise-gnip-2.0/fundamentals/account-activity) | Suscribirse a usuarios y recibir una variedad de actividades diferentes, incluidas Publicaciones, Mensajes Directos, Me gusta, seguimientos y más. |
| :--- | :--- | :--- |
| Histórico | [Search API](/es/x-api/enterprise-gnip-2.0/fundamentals/search-api) | Buscar en los últimos 30 días o en todo el archivo de Publicaciones públicas utilizando herramientas avanzadas de filtrado. |
| [Historical PowerTrack API](/es/x-api/enterprise-gnip-2.0/powertrack-api) | Filtrar el archivo completo de Publicaciones públicas mediante un proceso económico de acceso por lotes. |
| Insights | [Engagement API](/es/x-api/enterprise-gnip-2.0/fundamentals/engagement-api) | Medir el volumen de impresiones e interacciones de las Publicaciones públicas. |
| Cumplimiento | [Compliance Firehose API](/es/x-api/enterprise-gnip-2.0/fundamentals/firehouse) | Transmitir todos los eventos de cumplimiento. |
| Uso | [Usage API](/es/x-api/enterprise-gnip-2.0/fundamentals/usage) | Proporciona acceso programático al consumo de actividad en los productos Enterprise. |

Obtén más información sobre URLs expandidas, datos de encuestas y geolocalización de perfil visitando nuestra [sección de enriquecimientos](/es/x-api/enterprise-gnip-2.0/fundamentals/data-enrichments).
 

<div id="advanced-filtering-tools">
  ### Herramientas avanzadas de filtrado
</div>

La Premium Search API ofrece un conjunto avanzado de operadores que no están disponibles en el endpoint de búsqueda estándar. Muchos de estos operadores están disponibles para los usuarios del sandbox gratuito, pero obtendrás un conjunto más completo de operadores si utilizas un plan de pago. Obtén más información visitando nuestra [sección de reglas y filtros](/es/x-api/enterprise-gnip-2.0/fundamentals/rules-filtering#getting-started-with-enterprise-rules-and-queries).
