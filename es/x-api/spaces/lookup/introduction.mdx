Los endpoints de búsqueda por ID devuelven datos que puedes usar para crear experiencias que muestren información de Spaces a tus usuarios o para obtener detalles sobre uno o más Spaces. Puedes buscar uno o varios Spaces en la misma solicitud.

También puedes buscar Spaces por el ID de su creador. Este endpoint es útil para buscar Spaces de usuarios en una Lista o para determinar si varios hosts recurrentes están en vivo ahora o tienen un Space próximo programado. Se pueden buscar hasta 100 usuarios en una sola solicitud.

**Configuración de la cuenta**

Para acceder a estos endpoints, necesitas:

* Una [cuenta de desarrollador](https://developer.x.com/en/portal/petition/essential/basic-info) aprobada.
* Autenticarte usando las claves y tokens de una [aplicación de desarrollador](/es/resources/fundamentals/developer-apps) que se encuentre dentro de un [Proyecto](/es/resources/fundamentals/projects). 

Obtén más información sobre cómo obtener acceso a los endpoints de X API v2 en nuestra [guía de primeros pasos](/es/x-api/getting-started/getting-access).
