La X API se puede utilizar para recuperar y analizar datos de X de forma programática, así como para crear soluciones para la conversación en X.

A lo largo de los años, la X API ha crecido al incorporar niveles adicionales de acceso para que los desarrolladores puedan ampliar su acceso y así mejorar e investigar la conversación pública.

Recientemente, lanzamos la X API v2. La X API v2 incluye una base moderna, funciones nuevas y avanzadas, y un proceso de incorporación rápido al [acceso Basic](https://developer.x.com/en/portal/products/basic).

Las tres pestañas siguientes explican las diferentes versiones y niveles de acceso de la X API, qué hay de nuevo en la v2 y qué recursos de X puedes recuperar, crear, eliminar y ajustar usando la API.

  <Tab title="Novedades en la v2">
    ## Novedades de la v2

    La X API v2 representa la mayor actualización de la X API desde 2012. Con ella llega un conjunto de funciones nuevas y avanzadas, así como acceso rápido y gratuito a la API.

    Algunas de las funciones disponibles con v2 incluyen lo siguiente:

    ### Nuevos endpoints

    Hemos lanzado un conjunto de endpoints completamente nuevos para X API v2. Puedes ver una lista completa de los endpoints de v2, incluidos los que son nuevos, en nuestra guía del mapa de endpoints de X API.

    [Visita el mapa de endpoints de X API &gt;](/es/x-api/migrate/x-api-endpoint-map)

    ### Nuevos objetos de datos más detallados

    Hemos modernizado nuestros objetos de datos con diversas mejoras nuevas que te permitirán navegar y analizar datos con mayor facilidad.

    [Visita el diccionario de datos &gt;](/es/x-api/fundamentals/data-dictionary)

    ### Nuevos parámetros para ayudarte a recuperar solo los objetos y campos que quieres

    Hemos añadido los parámetros fields y expansions a nuestros endpoints de datos, que te permiten solicitar objetos y campos relacionados más allá de los campos que se devuelven de forma predeterminada.

    [Aprende a usar fields y expansions &gt;](/es/x-api/fundamentals/data-dictionary#how-to-use-fields-and-expansions)

    ### Métricas avanzadas

    Comprende más fácilmente el rendimiento de Publicaciones, usuarios, contenido multimedia y encuestas directamente en tu carga útil solicitando tanto métricas públicas como privadas, incluidas impresiones, reproducciones de vídeo, perfil de usuario y clics en URL, algunas de las cuales se dividen en contexto orgánico y promocionado.

    [Más información sobre métricas &gt;](/es/x-api/fundamentals/metrics)

    ### Filtra e identifica qué Publicaciones contienen distintos temas

    Cuando uses búsqueda de Publicaciones o flujo filtrado, ahora puedes filtrar por tema usando nuestros operadores de entidad y contexto. También hemos incluido estos temas en la carga útil de la Publicación para facilitar el análisis.

    [Más información sobre las anotaciones de Publicaciones &gt;](/es/x-api/fundamentals/post-annotations)

    ### Filtra e identifica qué Publicaciones pertenecen a un hilo de respuesta

    Haz que sea más fácil identificar una Publicación como parte de un hilo de conversación cuando uses búsqueda de Publicaciones, flujo filtrado y consulta de Publicaciones. También hemos añadido la capacidad de determinar si se han configurado las opciones de respuesta de la conversación para una Publicación con el campo de Publicación `reply_settings`.

    [Más información sobre el seguimiento de conversaciones &gt;](/es/x-api/fundamentals/conversation-id)

    ### Y mucho más...

    * Filtrado de spam de alta fiabilidad
    * Las URL acortadas se expanden por completo para facilitar el análisis de URL
    * Objetos de respuesta JSON simplificados al eliminar campos obsoletos y modernizar etiquetas
    * Funcionalidad de recuperación y redundancia para nuestros endpoints de streaming
    * Devolución del 100 % de las Publicaciones públicas y disponibles que coinciden en las consultas de búsqueda
    * Reglas de streaming para que puedas hacer cambios sin interrumpir conexiones
    * Lenguaje de consulta más expresivo para flujo filtrado y búsqueda
    * Especificación OpenAPI para crear nuevas bibliotecas y hacer un seguimiento de cambios de forma más transparente
    * Compatibilidad de la API con nuevas funciones y endpoints con mayor rapidez a medida que nuestra plataforma evoluciona para satisfacer las necesidades de desarrolladores, investigadores, empresas y personas que usan X

  <Tab title="Recursos de la X API">
    ## Recursos de la plataforma X API

    En el ámbito del diseño de API, un recurso es una entidad con datos asociados, relaciones con otros recursos y un conjunto de métodos que operan sobre ella. Por ejemplo, una Publicación es un recurso que puedes crear, eliminar o recuperar usando distintas herramientas, como realizar búsquedas históricas o recuperarlas en tiempo real.

    La X API permite crear, eliminar, obtener o ajustar una variedad de recursos en la plataforma, incluidos los siguientes:

    | Recurso             | Descripción                                                                                                                                                                                            |
    | :------------------ | :----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
    | **Posts**           | Accede a millones de Publicaciones para entender la conversación pública, o crea tus propias Publicaciones para participar en ella.  Disponibilidad actual:  - X API v2  - Enterprise  - Standard v1.1 |
    | **Users**           | Administra o consulta usuarios de X para analizar redes, entender a tu audiencia o fomentar relaciones positivas en línea.  Disponibilidad actual:  - X API v2  - Enterprise  - Standard v1.1          |
    | **Spaces**          | Busca y consulta X Spaces y sus asistentes para ayudar a las personas a encontrar conversaciones de audio interesantes y relevantes.  Disponibilidad actual:  - X API v2                               |
    | **Direct Messages** | Envía y recibe Mensajes Directos para gestionar incidencias de clientes, enviar mensajes de bienvenida o crear interacciones humanas positivas.  Disponibilidad actual:  - Standard v1.1               |
    | **Lists**           | Crea y administra Listas de cuentas para mantenerte al tanto de expertos del sector, voces influyentes u organizar a las cuentas que sigues.  Disponibilidad actual:  - X API v2  - Standard v1.1      |
    | **Trends**          | Identifica primero las tendencias geográficas para detectar movimientos del sector, descubrir temas de actualidad o adelantarte a la última moda.  Disponibilidad actual:  - Standard v1.1             |
    | **Media**           | Sube objetos de contenido multimedia para compartir tu energía creativa, crear experiencias interactivas o desarrollar herramientas de accesibilidad.  Disponibilidad actual:  - Standard v1.1         |
    | **Places**          | Busca lugares para entender qué está pasando en tu vecindario y en todo el mundo.  Disponibilidad actual:  - Standard v1.1                                                                             |
  </Tab>
