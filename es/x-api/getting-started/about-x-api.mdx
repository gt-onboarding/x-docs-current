La X API se puede usar para recuperar y analizar datos de X de forma programática, así como para crear soluciones relacionadas con la conversación en X.

A lo largo de los años, la X API ha crecido incorporando niveles adicionales de acceso para que los desarrolladores puedan escalar su acceso y así mejorar e investigar la conversación pública.

Recientemente, lanzamos la X API v2. La X API v2 incluye una base moderna, funciones nuevas y avanzadas, y un proceso de incorporación rápido a [Basic access](https://developer.x.com/en/portal/products/basic).

Las siguientes tres pestañas explican las diferentes versiones y niveles de acceso de la X API, qué hay de nuevo en v2 y qué recursos de X puedes recuperar, crear, eliminar y modificar mediante la API.

  <Tab title="Novedades en v2">
    ## Novedades de la v2

    X API v2 representa la mayor actualización de X API desde 2012. Incluye una serie de funcionalidades nuevas y avanzadas, además de acceso rápido y gratuito a la API.

    Algunas de las funcionalidades disponibles con la v2 incluyen lo siguiente:

    ### Nuevos endpoints

    Hemos lanzado un conjunto de endpoints completamente nuevos para X API v2. Puedes ver una lista completa de los endpoints de la v2, incluidos los que son nuevos, en nuestra guía del mapa de endpoints de X API.

    [Visita el mapa de endpoints de X API &gt;](/es/x-api/migrate/x-api-endpoint-map)

    ### Nuevos objetos de datos, más detallados

    Hemos modernizado nuestros objetos de datos con varias mejoras nuevas que te permitirán navegar y analizar datos con mayor facilidad.

    [Visita el diccionario de datos &gt;](/es/x-api/fundamentals/data-dictionary)

    ### Nuevos parámetros para ayudarte a recuperar solo los objetos y campos que quieres

    Hemos añadido los parámetros `fields` y `expansions` a nuestros endpoints de datos, que te permiten solicitar objetos relacionados y campos más allá de aquellos que se devuelven de forma predeterminada.

    [Aprende a usar fields y expansions &gt;](/es/x-api/fundamentals/data-dictionary#how-to-use-fields-and-expansions)

    ### Métricas avanzadas

    Comprende más fácilmente el rendimiento de Publicaciones, usuarios, contenido multimedia y encuestas directamente en tu payload solicitando tanto métricas públicas como privadas, incluidas impresiones, reproducciones de video, visitas al perfil de usuario y clics en URL, algunas de las cuales distinguen entre contexto orgánico y promocionado.

    [Más información sobre métricas &gt;](/es/x-api/fundamentals/metrics)

    ### Filtra e identifica qué Publicaciones contienen distintos temas

    Al usar búsqueda de Publicaciones o flujo filtrado, ahora puedes filtrar por tema utilizando nuestros operadores de entidad y contexto. También hemos incluido estos temas en el payload de la Publicación para facilitar el análisis.

    [Más información sobre anotaciones de Publicaciones &gt;](/es/x-api/fundamentals/post-annotations)

    ### Filtra e identifica qué Publicaciones pertenecen a un hilo de respuesta

    Facilita la identificación de una Publicación como parte de un hilo de conversación cuando uses búsqueda de Publicaciones, flujo filtrado y búsqueda de Publicaciones específicas. También hemos añadido la capacidad de determinar si se ha configurado la opción de respuestas de conversación para una Publicación con el campo de la Publicación `reply_settings`.

    [Más información sobre el seguimiento de conversaciones &gt;](/es/x-api/fundamentals/conversation-id)

    ### Y mucho más...

    * Filtrado de spam de alta confianza
    * Las URL acortadas se expanden completamente para facilitar el análisis de URL
    * Objetos de respuesta JSON simplificados mediante la eliminación de campos obsoletos y la modernización de etiquetas
    * Funcionalidad de recuperación y redundancia para nuestros endpoints de streaming
    * Devolución del 100 % de las Publicaciones públicas y disponibles que coinciden con las consultas de búsqueda
    * &quot;Reglas&quot; de streaming para que puedas hacer cambios sin interrumpir las conexiones
    * Lenguaje de consulta más expresivo para el flujo filtrado y la búsqueda
    * Especificación OpenAPI para crear nuevas bibliotecas y hacer un seguimiento de los cambios con mayor transparencia
    * Compatibilidad de la API con nuevas funcionalidades y endpoints de forma más rápida a medida que nuestra plataforma evoluciona para satisfacer las necesidades de desarrolladores, investigadores, empresas y personas que usan X
  </Tab>

  <Tab title="Recursos de la X API">
    ## Recursos de la plataforma X API

    En el ámbito del diseño de API, un recurso es una entidad con datos asociados, relaciones con otros recursos y un conjunto de métodos que operan sobre ella. Por ejemplo, una Publicación es un recurso que puedes crear, eliminar o recuperar usando diversas herramientas, como buscarlas en el historial o recuperarlas en tiempo real.

    La X API permite crear, eliminar, obtener o ajustar una variedad de recursos diferentes en la plataforma, incluidos los siguientes:

    | Resource            | Description                                                                                                                                                                                    |
    | :------------------ | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
    | **Posts**           | Aprovecha millones de Publicaciones para entender la conversación pública o crea las tuyas para participar en ella.  Disponibilidad actual:  - X API v2  - Enterprise  - Standard v1.1         |
    | **Users**           | Administra o consulta usuarios de X para analizar redes, entender a tu audiencia o fomentar relaciones positivas en línea.  Disponibilidad actual:  - X API v2  - Enterprise  - Standard v1.1  |
    | **Spaces**          | Busca y consulta X Spaces y sus participantes para ayudar a las personas a encontrar conversaciones de audio interesantes y relevantes.  Disponibilidad actual:  - X API v2                    |
    | **Direct Messages** | Envía y recibe mensajes directos para gestionar problemas de clientes, enviar mensajes de bienvenida o crear interacciones humanas positivas.  Disponibilidad actual:  - Standard v1.1         |
    | **Lists**           | Crea y administra listas de cuentas para mantener el pulso del sector, seguir voces influyentes u organizar a quién sigues.  Disponibilidad actual:  - X API v2  - Standard v1.1               |
    | **Trends**          | Identifica primero las tendencias geográficas para precisar los movimientos del sector, descubrir temas de actualidad o adelantarte a la última moda.  Disponibilidad actual:  - Standard v1.1 |
    | **Media**           | Sube objetos de contenido multimedia para compartir tu energía creativa, crear experiencias interactivas o desarrollar herramientas de accesibilidad.  Disponibilidad actual:  - Standard v1.1 |
    | **Places**          | Busca lugares para entender qué está pasando en tu vecindario y en todo el mundo.  Disponibilidad actual:  - Standard v1.1                                                                     |
