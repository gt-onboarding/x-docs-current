<div id="apps-using-the-hide-replies-endpoint">
  ## Apps que usan el endpoint hide replies
</div>

Aquí tienes una colección de aplicaciones destacadas creadas por desarrolladores que integraron el endpoint hide replies. Al combinar su tecnología con la X API, crearon experiencias útiles para ayudar a que todos los usuarios mejoren la conversación pública.

![Clarabridge](https://cdn.cms-twdigitalassets.com/content/dam/developer-twitter/hide-replies/clarabridge.jpg.twimg.1920.jpg)

**Clarabridge**

Clarabridge integra hide replies para que las empresas puedan mantener su feed de X limpio e interesante de leer. Las respuestas se pueden ocultar manualmente y los agentes también pueden usar el análisis de texto de Clarabridge para detectar automáticamente lenguaje soez y otro tipo de lenguaje que no se ajusta a la política de la marca.

[Ir al sitio web ▸](https://www.clarabridge.com/ "Ir al sitio web ▸")

[Clarabridge en X](https://x.com/Clarabridge "Clarabridge en X")

![Aplicación de plantilla de Perspective API](https://cdn.cms-twdigitalassets.com/content/dam/developer-twitter/hide-replies/perspective-logo.png.twimg.1920.png)

**Aplicación de plantilla de Perspective API**

Perspective es una inteligencia artificial entrenada por Jigsaw (una unidad dentro de Alphabet) para detectar comentarios tóxicos. Esta aplicación detecta las respuestas que recibe una persona y oculta automáticamente una respuesta en función de la puntuación (“score”) que indica con qué nivel de confianza Perspective considera que un comentario es similar a comentarios tóxicos que ha visto en el pasado. Esta es una aplicación de plantilla de código abierto desarrollada por X. Te animamos a personalizarla para crear herramientas para tus usuarios.

[Usar aplicación ▸](https://quintessential-yoke.glitch.me "Usar aplicación ▸")

[Jigsaw en X](https://x.com/jigsaw "Jigsaw en X")

![Reshuffle](data:image/svg+xml,%3Csvg xmlns=%22http://www.w3.org/2000/svg%22 viewBox=%220 0 150 150%22%3E%3C/svg%3E)

**Reshuffle**

Reshuffle es una plataforma para conectar aplicaciones empresariales con scripts flexibles. Crearon un script de integración para detectar y ocultar respuestas que incluyan palabras clave que definas. Para probarlo, asegúrate de tener una aplicación válida habilitada para hide replies y luego sigue las instrucciones en el repositorio.

[Ir a la aplicación ▸](https://github.com/reshufflehq/reshuffle "Ir a la aplicación ▸")

[Reshuffle en X](https://x.com/reshufflehq "Reshuffle en X")

![Ocultar respuestas no deseadas](data:image/svg+xml,%3Csvg xmlns=%22http://www.w3.org/2000/svg%22 viewBox=%220 0 396 397%22%3E%3C/svg%3E)

**Ocultar respuestas no deseadas**

Dara Oladosu, el desarrollador detrás de la popular aplicación QuotedReplies, creó una aplicación que oculta automáticamente las respuestas que cumplen algunos de los criterios que él ha determinado que tienen más probabilidades de mostrar comportamiento abusivo, incluidas las respuestas que contienen ciertas palabras clave que ha silenciado en el pasado.

[Ir a la aplicación ▸](https://hideunwantedreplies.com/ "Ir a la aplicación ▸")

[Dara Oladosu en X](https://x.com/dara_tobi "Dara Oladosu en X")

![Pandaflow](data:image/svg+xml,%3Csvg xmlns=%22http://www.w3.org/2000/svg%22 viewBox=%220 0 94 94%22%3E%3C/svg%3E)

**Pandaflow**

Pandaflow es una plataforma sin código para automatizar flujos de trabajo. Integra el endpoint para que los usuarios puedan ocultar respuestas de forma programática según un flujo personalizado que definan.

[Ir a la aplicación ▸](https://pandaflow.io/ "Ir a la aplicación ▸")

[Pandaflow en X](https://x.com/pandaflowio "Pandaflow en X")
