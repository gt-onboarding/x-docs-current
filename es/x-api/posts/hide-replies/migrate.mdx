  ## Comparación de los endpoints de ocultar respuestas de X API
</div>

El endpoint de ocultar respuestas de v2 está sustituyendo al endpoint de ocultar respuestas de Labs. Si tienes código, aplicaciones o herramientas que usan la versión de Labs de este endpoint y estás considerando migrar al nuevo endpoint de X API v2, esta guía es para ti.

Para usar la nueva X API v2 (incluido el endpoint de ocultar respuestas), deberás [inscribirte en el nuevo portal para desarrolladores](https://developer.x.com/en/portal/opt-in), crear un [Proyecto](/es/resources/fundamentals/projects) y añadir una Aplicación a ese Proyecto. Luego podrás usar las credenciales asociadas a esa Aplicación para hacer solicitudes al endpoint de ocultar respuestas. Añadir la misma Aplicación que está inscrita para el endpoint de ocultar respuestas de Labs v2 mantendrá a tus usuarios autenticados.

La siguiente tabla compara las diferencias entre Labs y el nuevo endpoint de X API v2:
 

| **Descripción** | **Labs v2** | **X API v2** |
| :--- | :--- | :--- |
| Dominio del host | https://api.x.com | https://api.x.com |
| Ruta del endpoint | /labs/2/tweets/:id/hidden | /2/tweets/:id/hidden |
| Autenticación | Contexto de usuario de OAuth 1.0a | Contexto de usuario de OAuth 1.0a |
| Métodos HTTP admitidos | PUT | PUT |
| Límites de frecuencia de solicitudes predeterminados | 10 solicitudes por 15 minutos (compartidas entre todos los usuarios autenticados) | 50 solicitudes por 15 minutos (para cada usuario autenticado) |
| Puede ocultar respuestas | ✔︎  | ✔︎  |
| Puede volver a mostrar una respuesta previamente oculta | ✔︎  | ✔︎  |
| Puede ocultar o volver a mostrar respuestas varias veces | ✔︎  | ✔︎  |
| Requiere el uso de credenciales de una [aplicación de desarrollador](/es/resources/fundamentals/developer-apps) asociada con un [Proyecto](/es/resources/fundamentals/projects) |     | ✔   |
