  ## Comparación de los endpoints de ocultar respuestas de X API
</div>

El endpoint de ocultar respuestas de v2 está reemplazando el endpoint de ocultar respuestas de Labs. Si tienes código, aplicaciones o herramientas que usan la versión de Labs de este endpoint y estás considerando migrar al endpoint más reciente de X API v2, esta guía es para ti.

Para usar la nueva X API v2 (incluido el endpoint de ocultar respuestas), tendrás que [inscribirte en el nuevo portal para desarrolladores](https://developer.x.com/en/portal/opt-in), crear un [Proyecto](/es/resources/fundamentals/projects) y agregar una aplicación a ese Proyecto. Luego puedes usar las credenciales asociadas con esa aplicación para hacer solicitudes al endpoint de ocultar respuestas. Agregar la misma aplicación que ya está inscrita en el endpoint de Labs v2 de ocultar respuestas mantendrá autenticados a tus usuarios.

La siguiente tabla muestra las diferencias entre Labs y el endpoint más reciente de X API v2:
 

| **Descripción** | **Labs v2** | **X API v2** |
| :--- | :--- | :--- |
| Dominio de host | https://api.x.com | https://api.x.com |
| Ruta del endpoint | /labs/2/tweets/:id/hidden | /2/tweets/:id/hidden |
| Autenticación | Contexto de usuario de OAuth 1.0a | Contexto de usuario de OAuth 1.0a |
| Métodos HTTP compatibles | PUT | PUT |
| Límites de frecuencia de solicitudes predeterminados | 10 solicitudes por 15 minutos (compartidos entre todos los usuarios autenticados) | 50 solicitudes por 15 minutos (para cada usuario autenticado) |
| Puede ocultar respuestas | ✔︎  | ✔︎  |
| Puede mostrar una respuesta que se había ocultado anteriormente | ✔︎  | ✔︎  |
| Puede ocultar o mostrar respuestas varias veces | ✔︎  | ✔︎  |
| Requiere el uso de credenciales de una [developer App](/es/resources/fundamentals/developer-apps) asociada con un [Project](/es/resources/fundamentals/projects) |     | ✔   |
