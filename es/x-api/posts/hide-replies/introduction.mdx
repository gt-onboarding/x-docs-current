Este endpoint te permite ocultar o volver a mostrar respuestas de forma programática usando los criterios que definas. Al igual que en la experiencia principal de X, las respuestas se ocultarán de la conversación principal, pero seguirán siendo visibles en una página independiente. Puedes usar el endpoint para crear aplicaciones que ayuden a las personas a ocultar las respuestas que no consideran valiosas.

El endpoint para ocultar respuestas utiliza autenticación con [Contexto de usuario de OAuth 1.0a](/es/resources/fundamentals/authentication#oauth-1-0a-2) o [OAuth 2.0 Authorization Code with PKCE](resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2). Si se ejecuta correctamente, el endpoint oculta una única respuesta que se publicó en una conversación de una Publicación iniciada por un usuario autenticado. Cada conversación admite ocultar hasta 725 Publicaciones.

<Note>
  **Configuración de la cuenta**

  Para acceder a estos endpoints, necesitarás:

  * Una [cuenta de desarrollador](https://developer.x.com/en/portal/petition/essential/basic-info) aprobada.
  * Autenticarte usando las claves y tokens de una [aplicación de desarrollador](/es/resources/fundamentals/developer-apps) que se encuentre dentro de un [Proyecto](/es/resources/fundamentals/projects).

  Obtén más información sobre cómo acceder a los endpoints de X API v2 en nuestra [guía de primeros pasos](/es/x-api/getting-started/getting-access).
</Note>
