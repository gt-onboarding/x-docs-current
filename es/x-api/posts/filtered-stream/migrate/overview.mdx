<div id="comparing-x-apis-filtered-stream-endpoints">
  ## Comparación de los endpoints de flujo filtrado de la X API
</div>

El grupo de endpoints de flujo filtrado de la X API v2 está reemplazando al endpoint estándar v1.1 [statuses/filter](https://developer.x.com/en/docs/x-api/v1/tweets/filter-realtime/api-reference/post-statuses-filter) y a la [PowerTrack API](/es/x-api/enterprise-gnip-2.0/powertrack-api). Si cuentas con código, aplicaciones o herramientas que usan una versión anterior del endpoint de flujo filtrado y estás considerando migrar al endpoint más reciente de X API v2, esta comparación puede ayudarte a empezar.

Consulta nuestras guías de migración más detalladas para:

[Migrar de la Standard v1.1 a X API v2](x-api/posts/filtered-stream#standard-v1-1-compared-to-x-api-v2)

[Migrar de PowerTrack API a X API v2](/es/x-api/enterprise-gnip-2.0/powertrack-api#powertrack-operators)
