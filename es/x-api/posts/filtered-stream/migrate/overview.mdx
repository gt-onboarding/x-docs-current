<div id="comparing-x-apis-filtered-stream-endpoints">
  ## Comparación de los endpoints de filtered stream de X API
</div>

El grupo de endpoints de filtered stream de v2 está reemplazando [standard v1.1 statuses/filter](https://developer.x.com/en/docs/x-api/v1/tweets/filter-realtime/api-reference/post-statuses-filter) y [PowerTrack API](/es/x-api/enterprise-gnip-2.0/powertrack-api). Si tienes código, aplicaciones o herramientas que usan una versión anterior del endpoint de filtered stream y estás considerando migrar al endpoint más reciente de X API v2, esta comparación puede ayudarte a comenzar.

Consulta nuestras guías de migración más detalladas para:

[Migración de Standard v1.1 a X API v2](x-api/posts/filtered-stream#standard-v1-1-compared-to-x-api-v2)

[Migración de PowerTrack API a X API v2](/es/x-api/enterprise-gnip-2.0/powertrack-api#powertrack-operators)
