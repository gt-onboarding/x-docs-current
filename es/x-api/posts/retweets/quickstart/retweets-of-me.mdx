La consulta de Reposts de mí te permite ver cuáles de tus propias Publicaciones han sido compartidas por otras personas.

Esta funcionalidad proporciona a los usuarios información sobre qué Publicaciones se están compartiendo y generando más interacción con su audiencia, lo que puede ayudar a facilitar interacciones más específicas y a ajustar la estrategia de contenido.

<div id="authentication">
  #### Autenticación
</div>

Como estás realizando solicitudes en nombre de un usuario, debes autenticar estos endpoints con [Contexto de usuario de OAuth 1.0a](/es/resources/fundamentals/authentication) o [OAuth 2.0 Authorization Code with PKCE](/es/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2), y utilizar los tokens de acceso de usuario asociados al usuario en cuyo nombre realizas la solicitud. Puedes generar este token de acceso de usuario mediante el [flujo OAuth de tres patas](https://developer.x.com/resources/fundamentals/authentication/obtaining-user-access-tokens) (OAuth 1.0a) o mediante el [flujo de concesión Authorization Code with PKCE](/es/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-3) (OAuth 2.0).

<div id="making-a-request">
  #### Realizar una solicitud
</div>

Una vez que un usuario se haya autenticado con tu aplicación, puedes llamar a la API Retweets of Me en nombre de ese usuario, como se muestra a continuación:

```bash
      curl 'https://api.x.com/2/users/reposts_of_me' --header 'Authorization: ••••••'
```
