<div id="comparing-x-apis-likes-endpoints">
  ## Comparación de los endpoints de Likes de X API
</div>

Estas guías se centrarán en los siguientes aspectos:

* **Parámetros de solicitud de la API** - El endpoint de X API v2 introduce un nuevo conjunto de parámetros de solicitud. Aunque algunos parámetros resultarán familiares, especialmente para quienes ya trabajan con Labs, hay muchas diferencias importantes, como la introducción de los parámetros [fields](/es/x-api/fundamentals/fields) y [expansions](/es/x-api/fundamentals/expansions).

* **Requisitos de la aplicación y el Proyecto** - Para acceder a X API v2, deberás utilizar credenciales de una [developer App](/es/resources/fundamentals/developer-apps) que esté asociada a un [Proyecto](/es/resources/fundamentals/projects).

<div id="likes-lookup">
  #### Consulta de Me gusta
</div>

**Usuarios a los que les ha gustado una Publicación**

El endpoint de usuarios a los que les ha gustado una Publicación es una funcionalidad nueva en X API v2, que te permite obtener información sobre los usuarios a los que les gusta una Publicación.

| Descripción | X API v2 |
| :--- | :--- |
| Métodos HTTP compatibles | GET |
| Dominio de host | https://api.x.com |
| Ruta del endpoint | /2/tweets/:id/liking&#95;users |
| [Autenticación](/es/resources/fundamentals/authentication) | OAuth 2.0 Bearer Token<br /><br />OAuth 1.0a User Context |
| [Límites de frecuencia](/es/resources/fundamentals/rate-limits) predeterminados para solicitudes | 75 solicitudes por 15 min (por Aplicación)<br /><br />75 solicitudes por 15 min (por usuario) |

**Publicaciones que le han gustado a un usuario**

La siguiente tabla compara el endpoint estándar v1.1 [GET favorites/list](https://developer.x.com/en/docs/twitter-api/v1/tweets/post-and-engage/api-reference/get-favorites-list) y los endpoints de Publicaciones marcadas con Me gusta de X API v2:

| Descripción | Standard v1.1 | X API v2 |
| :--- | :--- | :--- |
| Métodos HTTP compatibles | GET | GET |
| Dominio de host | https://api.x.com | https://api.x.com |
| Ruta del endpoint | /1.1/favorites/list.json | /2/users/:id/liked&#95;tweets |
| [Autenticación](/es/resources/fundamentals/authentication) | OAuth 1.0a User Context | OAuth 2.0 Bearer Token<br /><br />OAuth 1.0a User Context |
| [Límites de frecuencia](/es/resources/fundamentals/rate-limits) predeterminados para solicitudes | 75 solicitudes por 15 min | 75 solicitudes por 15 min (por Aplicación)<br /><br />75 solicitudes por 15 min (por usuario) |
| Formatos de datos | Formato Standard v1.1 | [Formato X API v2](/es/x-api/fundamentals/data-dictionary) (determinado por los parámetros de solicitud fields y expansions; no es compatible con versiones anteriores de los formatos v1.1)<br /><br />Para obtener más información sobre cómo migrar del formato Standard v1.1 al formato X API v2, visita nuestra [guía de migración de formatos de datos](/es/x-api/migrate/data-format-migration). |

<div id="manage-likes">
  #### Administrar Me gusta
</div>

Los endpoints de la X API v2 para administrar Me gusta reemplazan a los endpoints de v1.1 [POST favorites/create](https://developer.x.com/en/docs/twitter-api/v1/tweets/post-and-engage/api-reference/post-favorites-create) y [POST favorites/destroy](https://developer.x.com/en/docs/twitter-api/v1/tweets/post-and-engage/api-reference/post-favorites-destroy).

Las siguientes tablas comparan los endpoints estándar de v1.1 y de la X API v2 para administrar Me gusta:

<div id="like-a-post">
  #### Indicar que te gusta una Publicación
</div>

| Descripción | Standard v1.1 | X API v2 |
| :--- | :--- | :--- |
| Métodos HTTP admitidos | POST | POST |
| Dominio de host | https://api.x.com | https://api.x.com |
| Ruta del endpoint | /1.1/favorites/create.json | /2/users/:id/likes |
| [Autenticación](/es/resources/fundamentals/authentication) | OAuth 1.0a User Context | OAuth 1.0a User Context |
| [Límites de frecuencia predeterminados de solicitudes](/es/resources/fundamentals/rate-limits) | 1000 solicitudes cada 24 horas (por usuario)<br /><br />1000 solicitudes cada 24 horas (por aplicación) | 50 solicitudes cada 15 minutos (por usuario)<br /><br />1000 solicitudes cada 24 horas (por usuario, compartido con DELETE) |

<div id="unlike-a-post">
  #### Quitar el Me gusta de una Publicación
</div>

| Descripción | Standard v1.1 | X API v2 |
| :--- | :--- | :--- |
| Métodos HTTP compatibles | POST | DELETE |
| Dominio del host | https://api.x.com | https://api.x.com |
| Ruta del endpoint | /1.1/favorites/destroy.json | /2/users/:id/likes/:tweet&#95;id |

| [Límites de frecuencia](/es/resources/fundamentals/rate-limits) de solicitud predeterminados | 1000 solicitudes cada 24 horas (por usuario)<br /><br />1000 solicitudes cada 24 horas (por aplicación) | 50 solicitudes cada 15 minutos (por usuario)<br /><br />1000 solicitudes cada 24 horas (por usuario, compartidas con POST) |
