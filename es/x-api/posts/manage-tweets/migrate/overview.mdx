<div id="comparing-x-apis-manage-posts-endpoints">
  ## Comparación de los endpoints de gestión de Publicaciones de la X API
</div>

Los endpoints de gestión de Publicaciones de la versión 2 reemplazarán a los endpoints estándar de la versión 1.1 [POST statuses/update](https://developer.x.com/en/docs/twitter-api/v1/tweets/post-and-engage/api-reference/post-statuses-update) y [POST statuses/destroy/:id](https://developer.x.com/en/docs/twitter-api/v1/tweets/post-and-engage/api-reference/post-statuses-destroy-id). Si tienes código, aplicaciones o herramientas que usan la versión 1.1 de los endpoints de gestión de Publicaciones y estás considerando migrar al endpoint más reciente de la X API v2, entonces este conjunto de guías es para ti.

Las siguientes tablas comparan los endpoints estándar de gestión de Publicaciones de la versión 1.1 y de la X API v2:

<div id="create-a-post">
  ### Crear una Publicación
</div>

| Descripción | v1.1 estándar | X API v2 |
| :--- | :--- | :--- |
| Métodos HTTP admitidos | POST | POST |
| Dominio del host | https://api.x.com | https://api.x.com |
| Ruta del endpoint | /1.1/statuses/update.json | /2/tweets |
| [Autenticación](/es/resources/fundamentals/authentication) | Contexto de usuario de OAuth 1.0a | Contexto de usuario de OAuth 1.0a<br /><br />OAuth 2.0 Authorization Code with PKCE |
| [Límites de frecuencia](/es/x-api/fundamentals/rate-limits) predeterminados de las solicitudes | Ninguno<br /><br />300 solicitudes por ventana de 3 horas por usuario y por aplicación. Se comparte con el endpoint v1.1 POST Retweets. | 200 solicitudes por 15 minutos por usuario<br /><br />300 solicitudes por ventana de 3 horas por usuario y por aplicación. Se comparte con el endpoint v2 POST Retweets. |
| Requiere el uso de credenciales de una [aplicación de desarrollador](/es/resources/fundamentals/developer-apps) asociada a un [Proyecto](/es/resources/fundamentals/projects) |     | ✔   |

<div id="delete-a-post">
  ### Eliminar una Publicación
</div>

| Descripción | Standard v1.1 | X API v2 |
| :--- | :--- | :--- |
| Métodos HTTP compatibles | POST | DELETE |
| Dominio de host | https://api.x.com | https://api.x.com |
| Ruta del endpoint | /1.1/statuses/destroy/:id.json | /2/tweets/:id |
| [Autenticación](/es/resources/fundamentals/authentication) | OAuth 1.0a User Context | OAuth 1.0a User Context<br /><br />OAuth 2.0 Authorization Code with PKCE |
| [Límites de frecuencia](/es/x-api/fundamentals/rate-limits) predeterminados para solicitudes | Ninguno | 50 solicitudes por 15 min por usuario |
| Requiere el uso de credenciales de una [aplicación de desarrollador](/es/resources/fundamentals/developer-apps) asociada a un [Proyecto](/es/resources/fundamentals/projects) |     | ✔   |
