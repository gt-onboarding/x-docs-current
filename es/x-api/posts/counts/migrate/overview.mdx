<div id="comparing-x-apis-post-counts-endpoints">
  ## Comparación de los endpoints de recuento de Publicaciones de X API
</div>

El endpoint de recuento de Publicaciones de X API v2 acabará reemplazando al [endpoint de recuento de la Enterprise Search API](/es/x-api/enterprise-gnip-2.0/fundamentals/search-api#counts-requests-post-count). Si tienes código, aplicaciones o herramientas que usan una versión anterior de un endpoint de recuento de Publicaciones y estás considerando migrar a los endpoints más recientes de X API v2, entonces esta guía es para ti.

Esta página contiene dos tablas comparativas:

* [Recuentos de Publicaciones recientes](#recent)
* [Recuentos de Publicaciones de archivo completo](#full-archive)
* [Comparación de operadores de filtrado](#operator)

<div id="recent-post-counts-comparison">
  ### Comparación de recuentos de Publicaciones recientes
</div>

La versión Enterprise de los endpoints de recuento de Publicaciones te permite obtener recuentos de hasta 30 días o desde el archivo completo. Por lo tanto, el endpoint de recuento de Publicaciones recientes de v2, que analiza un período de 7 días, no es un reemplazo directo de ninguno de los endpoints mencionados.

Sin embargo, para facilitar las comparaciones, analizaremos cómo se compara el endpoint de recuento de Publicaciones recientes de v2 con el endpoint Enterprise de 30 días.

La siguiente tabla compara los distintos tipos de endpoints de recuento de Publicaciones recientes:

| **Descripción** | **Enterprise** | **X API v2** |
| :--- | :--- | :--- |
| Dominio del host | https://gnip-api.x.com | https://api.x.com |
| Ruta del endpoint | /search/30day/accounts/:account&#95;name/:label/counts.json | /2/tweets/counts/recent |
| [Authentication](/es/resources/fundamentals/authentication) | Autenticación básica | OAuth 2.0 Bearer Token |
| Formato de marca de tiempo | YYYYMMDDhhmm | YYYY-MM-DDTHH:mm:ssZ  <br />[ISO 8601 / RFC 3339](https://tools.ietf.org/html/rfc3339#section-5.6) |
| Devuelve recuentos de Publicaciones con una antigüedad máxima de | 31 días | 7 días |
| Métodos HTTP admitidos | GET | GET |
| Límites de frecuencia de solicitudes predeterminados | 20 solicitudes por 1 s, agregadas entre solicitudes de datos de búsqueda y de conteo  <br />El límite de frecuencia por minuto variará según el socio, tal como se especifica en tu contrato. | 180 solicitudes por 15 min por usuario  <br />450 solicitudes por 15 min por Aplicación |
| Permite filtrar mediante [annotations](/es/x-api/fundamentals/post-annotations) |     | ✔   |
| Permite filtrar mediante [conversation&#95;id](/es/x-api/fundamentals/conversation-id) |     | ✔   |
| Nombre de la clave JSON para la matriz de datos de Publicaciones | results | data |
| Granularidad temporal | Día, hora o minuto | Día, hora o minuto |
| Zona horaria | UTC | UTC |
| Parámetros de solicitud para seleccionar el período de tiempo | fromDate  <br />toDate | start&#95;time  <br />end&#95;time |
| Parámetros de solicitud para navegar por id de Publicación |     | since&#95;id  <br />until&#95;id |
| Requiere el uso de credenciales de una [developer App](/es/resources/fundamentals/developer-apps) asociada con un [project](/es/resources/fundamentals/projects) |     | ✔   |

<div id="filtering-operator-comparison">
  ### Comparación de operadores de filtrado
</div>

Las dos versiones diferentes (Enterprise y v2) de los conteos de Publicaciones difieren en qué operadores están disponibles y también presentan distintos niveles de disponibilidad de operadores dentro de cada versión, lo cual se explica a continuación.

Enterprise

* No hay subniveles de operadores de Enterprise. Todos los operadores de Enterprise están disponibles para todos los usuarios de Enterprise.

X API v2

* **Básico:** Estos operadores están disponibles para cualquier usuario de v2.
* **Avanzado:** Estos operadores solo están disponibles para usuarios que han sido aprobados para el acceso Academic Research.

Puedes obtener más información sobre cada uno de estos conjuntos de operadores en sus respectivas guías:

* [Operadores de Enterprise](/es/x-api/enterprise-gnip-2.0/fundamentals/search-api#building-search-queries)
* [Operadores de X API v2](/es/x-api/posts/search/integrate/build-a-query)
