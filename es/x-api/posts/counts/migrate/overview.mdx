<div id="comparing-x-apis-post-counts-endpoints">
  ## Comparación de los endpoints de conteo de Publicaciones de la X API
</div>

El endpoint de conteo de Publicaciones de v2 acabará reemplazando el [endpoint de conteo de la Enterprise Search API](/es/x-api/enterprise-gnip-2.0/fundamentals/search-api#counts-requests-post-count). Si tienes código, aplicaciones o herramientas que usan una versión anterior de un endpoint de conteo de Publicaciones y estás considerando migrar a los endpoints más recientes de X API v2, esta guía es para ti.

Esta página contiene dos tablas comparativas:

* [Conteo de Publicaciones recientes](#recent)
* [Conteo de Publicaciones de archivo completo](#full-archive)
* [Comparación de operadores de filtrado](#operator)

<div id="recent-post-counts-comparison">
  ### Comparación de conteos de Publicaciones recientes
</div>

La versión Enterprise de los endpoints de conteo de Publicaciones te permite obtener conteos para 30 días o desde el archivo completo. Por lo tanto, el endpoint de conteo de Publicaciones recientes de X API v2, que analiza un período de 7 días, no es un reemplazo directo de ninguno de los endpoints mencionados.

Sin embargo, para ayudar con las comparaciones, analizaremos cómo se compara el endpoint de conteo de Publicaciones recientes de X API v2 con el endpoint Enterprise de 30 días.

La siguiente tabla compara los distintos tipos de endpoints de conteo de Publicaciones recientes:

| :--- | :--- | :--- |

| Endpoint path | /search/30day/accounts/:account&#95;name/:label/counts.json | /2/tweets/counts/recent |
| [Authentication](/es/resources/fundamentals/authentication) | Autenticación básica | OAuth 2.0 Bearer Token |
| Timestamp format | YYYYMMDDhhmm | YYYY-MM-DDTHH:mm:ssZ  <br />[ISO 8601 / RFC 3339](https://tools.ietf.org/html/rfc3339#section-5.6) |

<div id="filtering-operator-comparison">
  ### Comparación de operadores de filtrado
</div>

Las dos versiones diferentes (Enterprise y v2) de recuentos de Publicaciones difieren en qué operadores están disponibles y también tienen distintos niveles de disponibilidad de operadores dentro de cada versión, lo cual se explica a continuación.

Enterprise

* No hay subniveles de operadores Enterprise. Todos los operadores Enterprise están disponibles para todos los usuarios de Enterprise.

X API v2

* **Core:** Estos operadores están disponibles para cualquier usuario de v2.
* **Advanced:** Estos operadores solo están disponibles para usuarios que hayan sido aprobados para el acceso Academic Research.

Puedes obtener más información sobre cada uno de estos conjuntos de operadores en sus respectivas guías:

* [Enterprise operators](/es/x-api/enterprise-gnip-2.0/fundamentals/search-api#building-search-queries)
* [X API v2 operators](/es/x-api/posts/search/integrate/build-a-query)
