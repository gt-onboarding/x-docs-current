La Usage API de X API v2 permite a los desarrolladores obtener de forma programática el uso de su proyecto. Con este endpoint, los desarrolladores pueden realizar un seguimiento y supervisar el número de Publicaciones que han recuperado para un ciclo de facturación determinado.

Los desarrolladores pueden usar el endpoint GET para obtener el uso diario del proyecto correspondiente a los últimos 90 días (como máximo). El uso también se puede agregar por aplicación cliente conectada a tu proyecto.

Existe un límite de frecuencia por aplicación de 50 solicitudes cada 15 minutos para este endpoint GET.

<div id="getting-started">
  ### Primeros pasos
</div>

Para utilizar este endpoint, necesitas un [bearer token](https://developer.x.com/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token) del [portal para desarrolladores](https://developer.x.com/en/portal/dashboard). Una vez que tengas el bearer token, podrás llamar a la API de uso como se muestra a continuación:

```
      curl 'https://api.x.com/2/usage/tweets' --header 'Authorization: Bearer XXXXX'
```

Si la solicitud se realiza con éxito, deberías ver la respuesta JSON que se muestra a continuación:

```
{
    "data": {
        "cap_reset_day": 10,
        "project_cap": "5000000000",
        "project_id": "1369785403853424",
        "project_usage": "43435"
    }
}
```

<Note>
  **Configuración de la cuenta**

  Para acceder a estos endpoints, necesitas:

  * Una [cuenta de desarrollador](https://developer.x.com/en/portal/petition/essential/basic-info) aprobada.
  * Autenticarte usando las claves y tokens de una [aplicación de desarrollador](/es/resources/fundamentals/developer-apps) que se encuentre dentro de un [Proyecto](/es/resources/fundamentals/projects). 

  Obtén más información sobre cómo acceder a los endpoints de X API v2 en nuestra [guía de introducción](/es/x-api/getting-started/getting-access).
</Note>
