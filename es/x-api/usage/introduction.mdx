La API de uso en la X API v2 permite a los desarrolladores obtener de forma programática el uso de su Proyecto. Con este endpoint, los desarrolladores pueden hacer seguimiento del número de Publicaciones que han obtenido en un ciclo de facturación determinado.

Los desarrolladores pueden usar el endpoint GET para obtener el uso diario del Proyecto hasta los últimos 90 días. El uso también se puede agregar por aplicación cliente conectada a tu Proyecto.

Hay un límite de frecuencia por aplicación de 50 solicitudes cada 15 minutos para este endpoint GET.

<div id="getting-started">
  ### Primeros pasos
</div>

Para utilizar este endpoint, necesitas un [Bearer Token](https://developer.x.com/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token) del [portal para desarrolladores](https://developer.x.com/en/portal/dashboard). Una vez que tengas el Bearer Token, podrás llamar a la API de uso como se muestra a continuación:

```
      curl 'https://api.x.com/2/usage/tweets' --header 'Authorization: Bearer XXXXX'
```

Si la solicitud se realiza correctamente, verás la respuesta JSON que se muestra a continuación:

```
{
    "data": {
        "cap_reset_day": 10,
        "project_cap": "5000000000",
        "project_id": "1369785403853424",
        "project_usage": "43435"
    }
}
```

<Note>
  **Configuración de la cuenta**

  Para acceder a estos endpoints, necesitas:

  * Una [cuenta de desarrollador](https://developer.x.com/en/portal/petition/essential/basic-info) aprobada.
  * Autenticarte con las claves y tokens de una [aplicación de desarrollador](/es/resources/fundamentals/developer-apps) que se encuentra dentro de un [Proyecto](/es/resources/fundamentals/projects).

  Obtén más información sobre cómo obtener acceso a los endpoints de X API v2 en nuestra [guía de introducción](/es/x-api/getting-started/getting-access).
</Note>
