Verás que todavía tenemos varios elementos marcados como **[Próximamente]**. Si ves algo en esta tabla que esté marcado como **[Reemplazo en consideración]** o **[Sin reemplazo previsto]**, y te gustaría que publiquemos una versión v2 de ese endpoint, háznoslo saber a través de nuestro [foro de la comunidad](https://devcommunity.x.com/) o de tu gestor de cuenta de Enterprise.
