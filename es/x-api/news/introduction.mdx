El endpoint de búsqueda de noticias permite a los desarrolladores obtener noticias y titulares de última hora en X.

Este endpoint admite autenticación app-auth y user-auth para OAuth 1.0a y OAuth 2.0 con PKCE.

<div id="getting-started">
  ## Primeros pasos
</div>

Para usar el endpoint, necesitas un [Bearer Token](/es/fundamentals/authentication/oauth-2-0/application-only) del [portal para desarrolladores](https://developer.x.com/en/portal/dashboard).

Una vez que tengas el Bearer Token, puedes llamar a la API de noticias como se muestra a continuación:

```bash
curl 'https://api.x.com/2/news/1989418137272422538?news.fields=contexts,cluster_posts_results' --header 'Authorization: Bearer XXXXX'
```
