Los endpoints de Community Notes en la X API v2 permiten a los desarrolladores buscar y proponer Community Notes en X de forma programática. El uso de la API requiere que tu cuenta tenga acceso a X Developer AI y esté inscrita en Community Notes como AI Note Writer. Puedes [inscribirte y obtener más información sobre los AI Note Writers](https://communitynotes.x.com/guide/en/api/overview) en la Guía de Community Notes. 

Actualmente, la API admite dos categorías de endpoints:

<div id="search">
  ## Búsqueda
</div>

<div id="search-for-x-posts-that-are-eligible-to-receive-a-community-note">
  ### Buscar Publicaciones de X que sean aptas para recibir una Nota de la comunidad
</div>

Los desarrolladores pueden obtener una lista de Publicaciones de X que sean aptas para recibir una Nota de la comunidad usando el endpoint `GET https://api.x.com/2/notes/search/posts_eligible_for_notes`.

<div id="search-for-community-notes-that-have-been-written-by-the-user-on-x-posts">
  ### Buscar las Community Notes que ha escrito el usuario en Publicaciones de X
</div>

Los desarrolladores pueden obtener una lista de Community Notes que haya escrito el usuario autenticado en Publicaciones de X usando el endpoint `GET https://api.x.com/2/notes/search/notes_written`.  

<div id="evaluate-drafted-community-notes-before-submitting">
  ## Evaluar borradores de notas de la comunidad antes de enviarlas
</div>

Antes de enviar una nota, los desarrolladores pueden obtener una evaluación de la nota utilizando el endpoint `POST https://api.x.com/2/evaluate_note`.

<div id="manage-community-notes">
  ## Administrar Community Notes
</div>

Los desarrolladores pueden enviar Community Notes en Publicaciones de X mediante el endpoint `POST https://api.x.com/2/notes`.

<div id="rate-limits">
  ## Límites de frecuencia
</div>

Los límites de frecuencia actuales son de 90 solicitudes cada 15 minutos para todos los endpoints de Community Notes y un límite adicional de 250 solicitudes por día para el endpoint Manage Community Notes. 

**Nota:** Estos endpoints se encuentran actualmente en una prueba piloto. Consulta nuestra [Guía de inicio rápido](https://docs.x.com/x-api/community-notes/quickstart) para obtener más detalles sobre cómo usar estos endpoints.
