<div id="community-tools-and-libraries-for-v2">
  ## Herramientas y bibliotecas de la comunidad para v2
</div>

Las bibliotecas que se indican aquí han sido creadas por miembros de la comunidad de desarrolladores. Ten en cuenta que pueden encontrarse en distintos grados de cobertura de la API.

Si has creado tu propia biblioteca de X API o una herramienta útil, [háznoslo saber](https://devcommunity.x.com/c/libraries-and-sdks/63) y la añadiremos a esta lista para ayudar a que otros la encuentren. También tenemos algunas [version badges](https://twbadges.glitch.me/) (insignias de versión) que puedes tomar prestadas para usarlas en tus propios archivos README.

¿Buscas inspiración? Puedes explorar y buscar en los temas [X](https://github.com/topics/twitter?o=desc&s=updated) y [X-api-v2](https://github.com/topics/twitter-api-v2?o=desc&s=updated) en GitHub para encontrar ejemplos de código útiles de otros desarrolladores.

**Ir a:** [C# / .NET](#csharp), [Dart / Flutter](#dart), [Go](#go), [Java](#java), [JavaScript (Node.JS) / TypeScript](#nodejs), [Kotlin](#kotlin), [PHP](#php), [PowerShell](#powershell), [Python](#python), [R](#r), [Ruby](#ruby), [Rust](#rust), [Swift](#swift)

<div id="community-tools-and-libraries">
  ## Herramientas y bibliotecas de la comunidad
</div>

Estas son algunas de las muchas bibliotecas mantenidas por la comunidad que dan soporte a X API en varios lenguajes de programación y plataformas. Ten en cuenta que es posible que estos recursos no hayan sido probados por el equipo de Twitter.

Las bibliotecas que se enumeran aquí deberían implementar la mayoría de las funcionalidades de la API estándar **v1.1**, a menos que se indique lo contrario; consulta con sus autores para obtener más detalles y soporte adicional.\
Si has creado una biblioteca que sea compatible con X API **v2**, háznoslo saber a través de nuestros [foros de la comunidad](https://devcommunity.x.com/c/libraries-and-sdks/63) para su posible inclusión en esta página. También puedes usar los foros para informar de cualquier cambio en estas listas.

Si echas en falta una biblioteca o herramienta para tu lenguaje de programación favorito, háznoslo saber a través de la [plataforma de comentarios](https://twitterdevfeedback.uservoice.com/), donde también puedes votar ideas o inspirarte para crear y enviar algo nuevo.
