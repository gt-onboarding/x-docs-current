<div id="comparing-x-apis-list-lookup-endpoints">
  ## Comparación de los endpoints de búsqueda de listas de X API
</div>

El grupo de endpoints de búsqueda de listas de v2 reemplazará los endpoints estándar de v1.1 [GET lists/show](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/get-lists-show) y [GET lists/ownership](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/get-lists-ownerships). Si tienes código, aplicaciones o herramientas que usan alguno de estos endpoints de búsqueda de listas y estás considerando migrar al endpoint más reciente de X API v2, entonces este conjunto de guías es para ti.

Las siguientes tablas comparan los endpoints estándar de v1.1 y los endpoints de listas de X API v2:

**Búsqueda de lista por ID**

|     |     |     |
| :--- | :--- | :--- |
| Descripción | v1.1 estándar | X API v2 |
| Métodos HTTP admitidos | `GET` | `GET` |
| Dominio del host | `https://api.x.com` | `https://api.x.com` |
| Ruta del endpoint | `/1.1/lists/show.json` | `/2/lists/:id` |
| [Autenticación](/es/resources/fundamentals/authentication) | Contexto de usuario de OAuth 1.0a<br /><br />Solo aplicación | Contexto de usuario de OAuth 1.0a<br /><br />OAuth 2.0 Authorization Code with PKCE<br /><br />Solo aplicación |
| [Límites de frecuencia](/es/resources/fundamentals/rate-limits) de solicitud predeterminados | 75 solicitudes por 15 minutos con OAuth 1.0a<br /><br />75 solicitudes por 15 minutos con OAuth 2.0<br /><br />75 solicitudes por 15 minutos con solo aplicación | 75 solicitudes por 15 minutos con OAuth 1.0a<br /><br />75 solicitudes por 15 minutos con OAuth 2.0<br /><br />75 solicitudes por 15 minutos con solo aplicación |

**Búsqueda de listas propiedad del usuario**

|     |     |     |
| :--- | :--- | :--- |
| Descripción | v1.1 estándar | X API v2 |
| Métodos HTTP admitidos | `GET` | `GET` |
| Dominio del host | `https://api.x.com` | `https://api.x.com` |
| Ruta del endpoint | `/1.1/lists/ownerships.json` | `/2/users/:id/owned_lists` |
| [Autenticación](/es/resources/fundamentals/authentication) | Contexto de usuario de OAuth 1.0a<br /><br />Solo aplicación | Contexto de usuario de OAuth 1.0a<br /><br />OAuth 2.0 Authorization Code with PKCE<br /><br />Solo aplicación |
| [Límites de frecuencia](/es/resources/fundamentals/rate-limits) de solicitud predeterminados | 15 solicitudes por 15 minutos con OAuth 1.0a<br /><br />15 solicitudes por 15 minutos con solo aplicación | 15 solicitudes por 15 minutos con OAuth 1.0a<br /><br />15 solicitudes por 15 minutos con OAuth 2.0<br /><br />15 solicitudes por 15 minutos con solo aplicación |

Para acceder a los endpoints de X API v2, debes [registrarte para obtener una cuenta de desarrollador](https://developer.x.com/en/portal/petition/essential/basic-info). Al autenticarte, debes usar claves y tokens de una [aplicación de desarrollador](/es/resources/fundamentals/developer-apps) que se encuentra dentro de un [Proyecto](/es/resources/fundamentals/projects).

Obtén más información sobre cómo obtener acceso a los endpoints de X API v2 en nuestra página de [primeros pasos](/es/x-api/getting-started/getting-access).
