La consulta de Publicaciones de Listas tiene un endpoint disponible para recuperar Publicaciones de una Lista específica. Con este endpoint, puedes crear soluciones que permitan a los usuarios personalizar, organizar y priorizar las Publicaciones que ven en su cronología.

Hay un límite de 900 solicitudes cada 15 minutos para este endpoint y la respuesta permitirá consultar las últimas 800 Publicaciones de una determinada Lista.

Puedes usar [Contexto de usuario de OAuth 1.0a](/es/resources/fundamentals/authentication), [solo App](/es/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token) o [OAuth 2.0 Authorization Code with PKCE](resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2) para autenticar tus solicitudes a este endpoint. 

**Configuración de la cuenta**

Para acceder a estos endpoints, necesitarás:

* Una [cuenta de desarrollador](https://developer.x.com/en/portal/petition/essential/basic-info) aprobada.
* Autenticarte usando las claves y tokens de una [aplicación de desarrollador](/es/resources/fundamentals/developer-apps) que forme parte de un [Proyecto](/es/resources/fundamentals/projects). 

Obtén más información sobre cómo acceder a los endpoints de la X API v2 en nuestra [guía de primeros pasos](/es/x-api/getting-started/getting-access).
