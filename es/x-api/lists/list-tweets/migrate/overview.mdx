  ## Comparación de los endpoints de consulta de Publicaciones de Listas de X API
</div>

El endpoint de consulta de Publicaciones de Listas de la v2 sustituirá al endpoint estándar de la v1.1 [GET lists/statuses](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/get-lists-statuses). Si tienes código, aplicaciones o herramientas que usan esta versión del endpoint y estás considerando migrar al endpoint más reciente de X API v2, este conjunto de guías es para ti.

Las siguientes tablas comparan los endpoints estándar de la v1.1 y de X API v2 para Listas:

**Consulta de Lista por id**

|     |     |     |
| :--- | :--- | :--- |
| Descripción | Estándar v1.1 | X API v2 |
| Métodos HTTP compatibles | `GET` | `GET` |
| Dominio del host | `https://api.x.com` | `https://api.x.com` |
| Ruta del endpoint | `/1.1/lists/statuses.json` | `/2/lists/:id/tweets` |
| [Autenticación](/es/resources/fundamentals/authentication) | Contexto de usuario de OAuth 1.0a<br /><br />Solo aplicación | Contexto de usuario de OAuth 1.0a<br /><br />OAuth 2.0 Authorization Code with PKCE<br /><br />Solo aplicación |
| [Límites de frecuencia](/es/resources/fundamentals/rate-limits) de solicitud predeterminados | 900 solicitudes por 15 min con OAuth 1.0a<br /><br />900 solicitudes por 15 min con solo aplicación | 900 solicitudes por 15 min con OAuth 1.0a<br /><br />900 solicitudes por 15 min con OAuth 2.0<br /><br />900 solicitudes por 15 min con solo aplicación |

Para acceder a los endpoints de X API v2, debes [registrarte para obtener una cuenta de desarrollador](https://developer.x.com/en/portal/petition/essential/basic-info). Al autenticarte, debes usar claves y tokens de una [aplicación de desarrollador](/es/resources/fundamentals/developer-apps) que esté dentro de un [Proyecto](/es/resources/fundamentals/projects). 

Obtén más información sobre cómo obtener acceso a los endpoints de X API v2 en nuestra página de [primeros pasos](/es/x-api/getting-started/getting-access).
