<div id="steps-to-build-a-manage-pinned-list-request">
  ### Pasos para crear una solicitud para gestionar List fijadas
</div>

**Paso uno: Elige la colección de endpoints de List en Postman**

Una vez que tengas la colección de X API v2 cargada en Postman, navega a la carpeta “List”, selecciona la carpeta “Pinned Lists” y, a continuación, elige &quot;Pin a List&quot;.
 

**Paso dos: Especifica la List que quieres fijar**

Los endpoints de administración de List fijadas requieren dos IDs: una para el usuario (el usuario autenticado que va a fijar una List) y otra para la List de destino (la List que se va a fijar). El ID del usuario debe corresponder al ID del usuario que se autentica, lo que significa que debes pasar los Tokens de acceso asociados con el ID de usuario al autenticar tu solicitud. En este caso, puedes especificar el ID que pertenece a tu usuario. Puedes encontrar tu ID de dos maneras:

1. Utilizando el endpoint [users lookup](/es/x-api/users/lookup/introduction) por nombre de usuario, puedes pasar un nombre de usuario y recibir el campo id.
2. Al mirar tu Token de acceso, verás que la parte numérica es tu ID de usuario.

El ID de la List de destino puede ser cualquier List válida. En Postman, navega a la pestaña &quot;Params&quot; e introduce el ID de usuario en la columna &quot;Value&quot; de la variable de ruta id. Navega a la pestaña “Body” e introduce el ID de la List que deseas fijar como valor del parámetro list&#95;id. Asegúrate de no incluir espacios antes ni después de ningún ID.

|             |                                           |                    |
| :---------- | :---------------------------------------- | :----------------- |
| **Key**     | **Value**                                 | **Parameter type** |
| `id`        | El ID de usuario                          | path               |
| list&#95;id | El ID de la List de destino que se fijará | body               |

Ahora deberías ver una URL similar junto al botón &quot;Send&quot;:

```
      https://api.x.com/2/users/2244994945/pinned_lists
```

Paso tres: realiza tu solicitud y revisa tu respuesta

Una vez que tengas todo configurado, haz clic en el botón «Send» y recibirás una respuesta similar al siguiente ejemplo:

```
{
  "data": {
    "pinned": true
  }
}
```

Has fijado correctamente la Lista de destino si el objeto de respuesta devuelto contiene un valor `true` para la clave `pinned`. 

Para desfijar una Lista, selecciona la solicitud “Unpin List” que también se encuentra en la carpeta “Lists” de la colección de X API v2 cargada en Postman. Este endpoint requiere el ID del usuario autenticado y el ID de la Lista que se va a desfijar. En la pestaña “Params”, introduce el ID de usuario como valor para la columna id y el ID de la Lista que se va a desfijar como valor para la columna list&#95;id. 

Si la solicitud de eliminación se completa correctamente, recibirás una respuesta similar al siguiente ejemplo:

```
{
  "data": {
    "pinned": false
  }
}
```
