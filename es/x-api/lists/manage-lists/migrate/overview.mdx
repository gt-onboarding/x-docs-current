<div id="comparing-x-apis-lists-endpoints">
  ## Comparación de los endpoints de Listas de X API
</div>

Los endpoints de administración de Listas de X API v2 sustituirán eventualmente a [POST lists/create](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/post-lists-create), [POST lists/destroy](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/post-lists-destroy) y [POST lists/update](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/post-lists-update). Si tienes código, aplicaciones o herramientas que usan una versión anterior de estos endpoints y estás considerando migrar a la nueva X API v2, entonces esta guía es para ti. 

Las siguientes tablas comparan los endpoints de Lista estándar de v1.1 y de X API v2:

**Crear una Lista**

| Descripción | v1.1 estándar | X API v2 |
| :--- | :--- | :--- |
| Métodos HTTP compatibles | POST | POST |
| Dominio del host | https://api.x.com | https://api.x.com |
| Ruta del endpoint | /1.1/lists/create.json | /2/lists |
| [Autenticación](/es/resources/fundamentals/authentication) | Contexto de usuario de OAuth 1.0a | Contexto de usuario de OAuth 1.0a |
| [Límites de frecuencia](/es/resources/fundamentals/rate-limits) de solicitudes predeterminados | Ninguno | 300 solicitudes por 15 min (por usuario) |

**Eliminar una Lista**

| Descripción | v1.1 estándar | X API v2 |
| :--- | :--- | :--- |
| Métodos HTTP compatibles | POST | DELETE |
| Dominio del host | https://api.x.com | https://api.x.com |
| Ruta del endpoint | /1.1/lists/destroy.json | /2/lists/:id |
| [Autenticación](/es/resources/fundamentals/authentication) | Contexto de usuario de OAuth 1.0a | Contexto de usuario de OAuth 1.0a |
| [Límites de frecuencia](/es/resources/fundamentals/rate-limits) de solicitudes predeterminados | Ninguno | 300 solicitudes por 15 min (por usuario) |

**Actualizar una Lista**

| Descripción | v1.1 estándar | X API v2 |
| :--- | :--- | :--- |
| Métodos HTTP compatibles | POST | PUT |
| Dominio del host | https://api.x.com | https://api.x.com |
| Ruta del endpoint | /1.1/lists/update.json | /2/lists/:id |
| [Autenticación](/es/resources/fundamentals/authentication) | Contexto de usuario de OAuth 1.0a | Contexto de usuario de OAuth 1.0a |
| [Límites de frecuencia](/es/resources/fundamentals/rate-limits) de solicitudes predeterminados | Ninguno | 300 solicitudes por 15 min (por usuario) |

Para acceder a los endpoints de X API v2, debes [registrarte para obtener una cuenta de desarrollador](https://developer.x.com/en/portal/petition/essential/basic-info). Al autenticarte, debes usar claves y tokens de una [aplicación de desarrollador](/es/resources/fundamentals/developer-apps) que forme parte de un [Proyecto](/es/resources/fundamentals/projects). 

Obtén más información sobre cómo conseguir acceso a los endpoints de X API v2 en nuestra página de [primeros pasos](/es/x-api/getting-started/getting-access).
