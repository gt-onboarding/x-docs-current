[X Lists](https://help.x.com/en/using-twitter/twitter-lists) permite a los usuarios personalizar, organizar y priorizar las Publicaciones que ven en su cronología. Con los endpoints de Listas, puedes crear soluciones que permitan a las personas seleccionar y organizar Publicaciones según sus preferencias, intereses, grupos o temas.

Dado que estás realizando solicitudes en nombre de un usuario con los endpoints de administración de List, debes autenticar estos endpoints con el [Contexto de usuario de OAuth 1.0a](/es/resources/fundamentals/authentication) o con [OAuth 2.0 Authorization Code with PKCE](resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2), y usar los tokens de acceso de usuario asociados a un usuario que haya autorizado tu aplicación, los cuales se pueden generar usando el [flujo OAuth de tres patas](/es/resources/fundamentals/authentication#obtaining-access-tokens-using-3-legged-oauth-flow) (OAuth 1.0a) o el [Authorization Code with PKCE grant flow](/es/resources/fundamentals/authentication#how-to-connect-to-endpoints-using-oauth-2-0-authorization-code-flow-with-pkce) (OAuth 2.0).

<div id="manage-lists">
  ### Administrar Listas
</div>

Los endpoints para administrar Listas te permiten crear, eliminar y actualizar Listas en nombre de un usuario autenticado. Para estos endpoints, hay tres métodos disponibles: POST, DELETE y PUT. El método POST te permite crear una Lista, el método DELETE te permite eliminar una Lista y el método PUT te permite actualizar los metadatos de una Lista. Existe un límite de 300 solicitudes por usuario cada 15 minutos para los tres endpoints.

Ten en cuenta que puedes crear hasta 1000 Listas por cuenta.

**Configuración de la cuenta**

Para acceder a estos endpoints, necesitarás:

* Una [cuenta de desarrollador](https://developer.x.com/en/portal/petition/essential/basic-info) aprobada.
* Autenticarte usando las claves y tokens de una [aplicación de desarrollador](/es/resources/fundamentals/developer-apps) que se encuentre dentro de un [Proyecto](/es/resources/fundamentals/projects). 

Obtén más información sobre cómo acceder a los endpoints de X API v2 en nuestra [guía de primeros pasos](/es/x-api/getting-started/getting-access).
