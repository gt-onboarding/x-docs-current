<div id="manage-list-members-quick-start-guide">
  ### Guía de inicio rápido para administrar miembros de una Lista
</div>

<Note>
  **Ten en cuenta:** Esta guía asume que ya completaste los requisitos previos de la [descripción general de inicio rápido](/es/x-api/lists/list-members#getting-started-with-the-list-members-endpoint-group).
</Note>

<div id="steps-to-build-a-manage-list-member-request">
  ### Pasos para crear una solicitud para gestionar miembros de una Lista
</div>

<div id="step-one-choose-the-list-endpoint-collection-from-postman">
  #### Paso uno: selecciona la colección de endpoints List en Postman
</div>

Una vez que tengas cargada la colección de X API v2 en Postman, dirígete a la carpeta “List”, selecciona la carpeta “List members” y luego elige &quot;Add a member&quot;.
 

<div id="step-two-specify-the-user-to-add">
  #### Paso dos: especifica el usuario que quieres agregar
</div>

Los endpoints para administrar miembros de una Lista requieren dos ID: el ID de la Lista y el ID del usuario que se va a agregar. Puedes encontrar el ID del usuario usando la [búsqueda de usuarios](/es/x-api/users/lookup/introduction) y pasar un nombre de usuario para recibir el campo id.

El ID de destino puede ser cualquier ID de usuario válido. En Postman, ve a la pestaña &quot;Params&quot; e introduce tu ID en la columna &quot;Value&quot; de la variable de ruta id. Ve a la pestaña &quot;Body&quot; e introduce el ID del usuario que deseas agregar como valor para el parámetro user&#95;id. Asegúrate de no incluir espacios antes o después de ningún ID.

|             |                                                           |                    |
| :---------- | :-------------------------------------------------------- | :----------------- |
| **Key**     | **Value**                                                 | **Parameter type** |
| `id`        | ID de la Lista                                            | path               |
| user&#95;id | ID del usuario de destino que deseas agregar como miembro | body               |

Ahora deberías ver una URL similar junto al botón &quot;Send&quot;:

`https://api.x.com/2/lists/1441162269824405510/members`

#### Paso tres: envía tu solicitud y revisa la respuesta

Una vez que tengas todo configurado, haz clic en el botón &quot;Send&quot; y recibirás una respuesta similar al siguiente ejemplo de respuesta:

```
{
  "data": {
    "is_member": true
  }
}
```

Si el objeto de respuesta devuelto contiene un valor true para la clave is&#95;member, habrás añadido correctamente al usuario como miembro de la Lista. 

Para eliminar a un miembro de una Lista, selecciona la solicitud “Remove a member”, que también se encuentra en la carpeta “Lists” de la colección de X API v2 cargada en Postman. Este endpoint requiere el ID de la Lista y el ID de usuario del miembro que deseas eliminar. En la pestaña “Params”, introduce el ID de la Lista como valor de la columna id y el ID del usuario que se va a eliminar como valor de la columna user&#95;id. 

Si la solicitud de eliminación se completa correctamente, recibirás una respuesta similar al siguiente ejemplo:

```
{
  "data": {
    "is_member": false
  }
}
```
