<div id="manage-list-members-quick-start-guide">
  ### Guía de inicio rápido para gestionar los miembros de una Lista
</div>

<Note>
  **Ten en cuenta:** Esta guía asume que ya completaste los requisitos previos de la [descripción general de inicio rápido](/es/x-api/lists/list-members#getting-started-with-the-list-members-endpoint-group).
</Note>

<div id="steps-to-build-a-manage-list-member-request">
  ### Pasos para configurar una solicitud para gestionar miembros de una Lista
</div>

<div id="step-one-choose-the-list-endpoint-collection-from-postman">
  #### Paso uno: Elige la colección de endpoints de List en Postman
</div>

Una vez que tengas la colección de X API v2 cargada en Postman, dirígete a la carpeta “List”, selecciona la carpeta “List members” y luego elige “Add a member”.
 

<div id="step-two-specify-the-user-to-add">
  #### Paso dos: especifica el usuario que se va a agregar
</div>

Los endpoints para administrar miembros de una Lista requieren dos ID: el ID de la Lista y el ID del usuario que se va a agregar. Puedes encontrar el ID del usuario usando la [búsqueda de usuarios](/es/x-api/users/lookup/introduction) y pasando un nombre de usuario para recibir el campo id.

El ID de destino puede ser cualquier ID de usuario válido. En Postman, navega a la pestaña &quot;Params&quot; e introduce tu ID en la columna &quot;Value&quot; de la variable de ruta id. Navega a la pestaña “Body” e introduce el ID del usuario que deseas agregar como valor del parámetro user&#95;id. Asegúrate de no incluir espacios antes ni después de ningún ID.

|             |                                                              |                    |
| :---------- | :----------------------------------------------------------- | :----------------- |
| **Key**     | **Value**                                                    | **Parameter type** |
| `id`        | El ID de la Lista                                            | path               |
| user&#95;id | El ID del usuario de destino que deseas agregar como miembro | body               |

Ahora deberías ver una URL similar junto al botón &quot;Send&quot;:

`https://api.x.com/2/lists/1441162269824405510/members`

Paso tres: realiza tu solicitud y revisa tu respuesta

Una vez que tengas todo configurado, haz clic en el botón &quot;Send&quot; y recibirás una respuesta similar al siguiente ejemplo de respuesta:

```
{
  "data": {
    "is_member": true
  }
}
```

Si el objeto de respuesta devuelto contiene un valor true para la clave is&#95;member, habrás añadido correctamente al usuario como miembro de la Lista. 

Para eliminar a un miembro de una Lista, selecciona la solicitud “Remove a member” que también se encuentra en la carpeta “Lists” de la colección de X API v2 cargada en Postman. Este endpoint requiere el ID de la Lista y el ID de usuario del miembro que deseas eliminar. En la pestaña “Params”, introduce el ID de la Lista como valor de la columna id y el ID del usuario que se va a eliminar como valor de la columna user&#95;id. 

Si la solicitud de eliminación se procesa correctamente, recibirás una respuesta similar al siguiente ejemplo:

```
{
  "data": {
    "is_member": false
  }
}
```
