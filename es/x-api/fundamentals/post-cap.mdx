X API v2 tiene un límite de consumo de Publicaciones que restringe la cantidad de Publicaciones que se pueden recuperar desde determinados endpoints cada mes. Los límites de Publicaciones se aplican a nivel de [Proyecto](/es/resources/fundamentals/projects), lo que significa que cualquier solicitud a los endpoints enumerados a continuación que utilice las claves y tokens de las [aplicaciones de desarrollador](/es/resources/fundamentals/developer-apps) dentro de ese Proyecto contará para el límite de consumo de Publicaciones de ese Proyecto.

<div id="limited-v2-endpoints">
  ## Endpoints limitados de v2
</div>

Las Publicaciones provenientes de cualquiera de los siguientes endpoints cuentan para este límite mensual de Publicaciones:

* [Posts Lookup](/es/x-api/posts/lookup/introduction)
* [Búsqueda reciente](/es/x-api/posts/counts/introduction)
* [Búsqueda en el archivo completo](/es/x-api/posts/counts/introduction)
* [Filtered stream](/es/x-api/posts/filtered-stream)
* [Filtered Stream Webhooks](/es/x-api/webhooks/stream/introduction)
* [Cronología de Publicaciones de usuario](/es/x-api/posts/timelines)
* [Cronología de menciones de usuario](/es/x-api/posts/timelines)
* [Likes lookup - Publicaciones que le han gustado a un usuario](/es/x-api/posts/likes)
* [Bookmarks lookup - Publicaciones que un usuario ha guardado en marcadores](/es/x-api/posts/bookmarks/introduction)
* [Búsqueda de Publicaciones de Lista](/es/x-api/lists/list-tweets/introduction)
* [Spaces Lookup](/es/x-api/spaces/lookup/introduction)

<Note>
  **Nota:** Si la misma Publicación se devuelve en varias consultas durante un día, solo cuenta una vez para el límite de Publicaciones; los duplicados se eliminan diariamente.
</Note>

<div id="post-cap-volumes">
  ## Volúmenes del límite de Publicaciones
</div>

El volumen del límite de Publicaciones varía según tu nivel de acceso:

| Nivel de acceso | Límite mensual de Publicaciones |
|:---------------|:---------------------------------|
| **Free**       | 100 Publicaciones por mes        |
| **Basic**      | 15,000 Publicaciones por mes     |
| **Pro**        | 1,000,000 Publicaciones por mes  |
| **Enterprise** | Más de 50 millones de Publicaciones por mes |

Puedes consultar el uso que has hecho con respecto al límite mensual de Publicaciones en la [página principal del panel de control](https://developer.x.com/content/developer-twitter/en/portal/dashboard) en el [portal para desarrolladores](/es/resources/fundamentals/developer-portal). Debajo del nombre de tu Proyecto, verás una barra de estado que muestra el uso del mes actual en relación con el límite de Publicaciones, incluido el total de Publicaciones obtenidas, el porcentaje de Publicaciones utilizadas y la fecha de restablecimiento de tu uso.
