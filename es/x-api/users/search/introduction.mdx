Disponible para desarrolladores con acceso Pro y superior

El endpoint Users Search ofrece una interfaz de búsqueda sencilla, basada en relevancia, para cuentas de usuario públicas en X. Prueba a hacer consultas por intereses, nombre completo, nombre de la empresa, ubicación u otros criterios.

Este endpoint actualmente admite [Autenticación de usuario](/es/resources/fundamentals/authentication). De forma predeterminada, se devuelven 100 usuarios por solicitud y puedes especificar hasta 1000 usuarios por solicitud. Tiene un límite de 10.000 solicitudes cada 24 horas (además de las 300 solicitudes cada 15 minutos)

<Note>
  **Configuración de la cuenta**

  Para acceder a estos endpoints, necesitarás:

  * Una [cuenta de desarrollador](https://developer.x.com/en/portal/petition/essential/basic-info) aprobada.
  * Autenticarte usando las claves y tokens de una [aplicación de desarrollador](/es/resources/fundamentals/developer-apps) que se encuentre dentro de un [Proyecto](/es/resources/fundamentals/projects). 

  Obtén más información sobre cómo obtener acceso a los endpoints de X API v2 en nuestra [guía de introducción](/es/x-api/getting-started/getting-access).
</Note>
