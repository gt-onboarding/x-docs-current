<div id="ways-to-build-for-people-on-x">
  ## 在 X 上为人们构建产品的方式
</div>

我们正在开放平台，鼓励你构建能对公共对话产生积极影响、并服务于 X 上人们的新工具和产品。具体而言，我们希望在以下几个方向激发新的创新和开发：

<CardGroup>
  <Card title="为健康与安全开展对话治理" icon="bullhorn" iconType="solid">
    通过构建内容审核工具和其他解决方案，帮助提升公共对话的健康与安全，让人们在 X 上参与对话时感到安全、乐于参与。构建互补或增值的技术，为人们提供多样而完善的工具选择，以便更好地掌控他们在 X 上的互动和体验。

    [**了解详情**](/zh/x-api/what-to-build#moderate-conversations-for-health-and-safety) <Icon icon="arrow-right" />
  </Card>

  <Card title="支持创作和个人表达" icon="bullhorn" iconType="solid">
    为创作者、品牌和个人构建工具，帮助他们扩大影响力、表达品牌，并与拥有相同热情的人建立联系。或者拓展人们在 X 上的参与方式，例如跨平台发布内容、游戏化机制或实用的机器人。

    [**了解详情**](/zh/x-api/what-to-build#enable-creation-and-personal-expression) <Icon icon="arrow-right" />
  </Card>

  <Card title="衡量和分析「正在发生的事情」" icon="bullhorn" iconType="solid">
    帮助人们对公共对话或他们自己的内容进行衡量、分析并提炼洞察。更好地了解你的受众，衡量内容的表现和影响，做出更明智的决策，等等。

    [**了解详情**](/zh/x-api/what-to-build#measure-and-analyze-whats-happening) <Icon icon="arrow-right" />
  </Card>

  <Card title="改善社区体验" icon="bullhorn" iconType="solid">
    X 面向所有人，我们希望鼓励并赋能你运用自己的专业能力，更好地满足 X 上不同社区的独特需求。无论你是在为新兴全球市场构建本地化解决方案，还是在为平台上不断成长的社区（例如游戏或金融）提升体验，我们都希望每个人都能在 X 上找到自己的归属。

    [**了解详情**](/zh/x-api/what-to-build#improve-community-experiences) <Icon icon="arrow-right" />
  </Card>

  <Card title="策划和推荐内容" icon="bullhorn" iconType="solid">
    构建应用、机器人或其他工具，帮助人们发现新内容或建立新的联系。或者构建经过策划且可自定义的体验，让人们看到更多他们最关心的内容。

    [**了解详情**](/zh/x-api/what-to-build#curate-and-recommend-content) <Icon icon="arrow-right" />
  </Card>

  <Card title="造福公共利益" icon="bullhorn" iconType="solid">
    开展突破性的研究，帮助推进对公共健康和气候危机的认知，并以非商业目的开展构建，造福公共利益。

    [**了解详情**](/zh/x-api/what-to-build#impact-the-greater-good) <Icon icon="arrow-right" />
  </Card>
</CardGroup>
