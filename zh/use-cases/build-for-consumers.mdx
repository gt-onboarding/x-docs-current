<div id="ways-to-build-for-people-on-x">
  ## 在 X 上为人们构建产品的方式
</div>

我们正在开放平台，鼓励你构建对公共对话产生积极影响、并服务于 X 上人们的新工具和产品。具体来说，我们希望在以下领域激发新的创新和开发：

<CardGroup>
  <Card title="为健康与安全进行对话治理" icon="bullhorn" iconType="solid">
    通过构建内容审核工具和其他解决方案，帮助提升公共对话的健康和安全，让人们在 X 上参与对话时感到安全、兴奋。构建互补或增强型技术，为人们提供一整套工具选择，以便掌控自己的互动方式和使用体验。

    [**了解详情**](/zh/x-api/what-to-build#moderate-conversations-for-health-and-safety) <Icon icon="arrow-right" />
  </Card>

  <Card title="支持创作与个人表达" icon="bullhorn" iconType="solid">
    为创作者、品牌和希望扩大影响力、表达品牌形象并与志同道合人群建立联系的个人构建工具。或者拓展人们在 X 上参与互动的方式，例如跨平台发布内容、游戏化机制或实用机器人。

    [**了解详情**](/zh/x-api/what-to-build#enable-creation-and-personal-expression) <Icon icon="arrow-right" />
  </Card>

  <Card title="衡量与分析“正在发生的事情”" icon="bullhorn" iconType="solid">
    帮助人们衡量、分析并从公共对话或他们自己的内容中获取洞察。更好地理解你的受众，衡量内容的表现和影响力，做出更有依据的决策，等等。

    [**了解详情**](/zh/x-api/what-to-build#measure-and-analyze-whats-happening) <Icon icon="arrow-right" />
  </Card>

  <Card title="改进社区体验" icon="bullhorn" iconType="solid">
    X 面向所有人，我们希望鼓励并赋能你运用自身专长，更好地满足 X 上不同社区的独特需求。无论你是在为新兴全球市场构建本地化解决方案，还是在为平台上不断壮大的社区（如游戏或金融）提升体验，我们都希望每个人都能在 X 上找到自己的位置。

    [**了解详情**](/zh/x-api/what-to-build#improve-community-experiences) <Icon icon="arrow-right" />
  </Card>

  <Card title="策划与推荐内容" icon="bullhorn" iconType="solid">
    构建应用、机器人或其他工具，帮助人们发现新内容或建立新的联系。或者构建经过策划且可自定义的体验，让人们能看到更多他们最关心的内容。

    [**了解详情**](/zh/x-api/what-to-build#curate-and-recommend-content) <Icon icon="arrow-right" />
  </Card>

  <Card title="服务公共利益" icon="bullhorn" iconType="solid">
    开展突破性研究，推动公共健康和气候危机相关知识的进步，并以非商业方式构建产品以服务更广泛的公共利益。

    [**了解详情**](/zh/x-api/what-to-build#impact-the-greater-good) <Icon icon="arrow-right" />
  </Card>
</CardGroup>
