<div id="educators">
  ## 教育工作者
</div>

将 API 引入课堂，用它来教授学生编程基础和实用的数据处理技能。

**资源：**

你是一名想在课程中使用 X API 的教育工作者吗？这篇
[使用 X API 教学指南](https://dev.to/suhemparack/a-guide-to-teaching-with-the-twitter-api-v2-3n08)
将帮助你了解 X API、为什么它会被教育工作者使用、使用该 API 正在教授哪些类型的课程，以及入门所需了解的其他一切内容。

**精选课程：**

以下课程均基于 X API v2 设计和开展：

- [Social Media Data Mining and Analysis（社交媒体数据挖掘与分析）](https://canvas.uw.edu/courses/1434897)
  课程负责人：Tanu Mitra 博士

- [Data Science & Society（数据科学与社会）](https://dssoc.github.io/) 课程负责人：Chris Bail 博士

- [Introduction to Cultural Analytics & Python（文化分析与 Python 导论）](https://melaniewalsh.github.io/Intro-Cultural-Analytics/welcome.html)
  课程负责人：Melanie Walsh 博士

- [Oxford Spring School in Advanced Research Methods, Analysing X Data（牛津高级研究方法春季学校：X 数据分析）](https://github.com/cjbarrie/oxspsch)
  课程负责人：Chris Barrie 博士
