<div id="basic-authentication">
  ## 基本身份验证
</div>

许多 X 的 Enterprise API 需要使用 HTTP 基本身份验证（HTTP Basic Authentication）。要成功向需要基本身份验证的 API 发出请求，你必须在每个请求中，将有效的电子邮件地址和密码组合作为授权 HTTP 头部传递。该电子邮件和密码组合与用于访问 [Enterprise API 控制台](https://console.gnip.com/) 的账号相同，并且可以在该控制台中进行修改。 

使用基本身份验证构建请求时，请确保通过 HTTPS 添加带有编码凭据的 `Authentication: Basic` HTTP 头部。

在下面的 cURL 请求示例中，你需要在发送请求之前，将 `<email_address>` 和 `<password>` 替换为你的凭据：

```bash
curl -v --compressed -u<email_address>:<password>
    "https://gnip-api.x.com/search/30day/accounts/<account-name>/prod/counts.json?query=from%3Axdevelopers"
```

* [30-Day Search API](/zh/x-api/enterprise-gnip-2.0/fundamentals/search-api) Enterprise
