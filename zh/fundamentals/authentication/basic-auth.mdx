<div id="basic-authentication">
  ## 基本身份验证
</div>

X 的许多 Enterprise API 需要使用 HTTP Basic Authentication。要成功向需要 Basic Authentication 的 API 发起请求，你必须在每个请求中，在授权请求头中传递一个有效的电子邮件地址和密码组合。这个电子邮件地址和密码组合与你用于访问 [enterprise API console](https://console.gnip.com/) 的凭证相同，并且可以在该控制台中进行编辑。 

在使用 Basic Authentication 构建请求时，请确保通过 HTTPS 添加带有编码后凭证的 `Authentication: Basic` HTTP 请求头。

在下面的 cURL 请求示例中，你需要在发送请求之前，将 `<email_address>` 和 `<password>` 替换为你的凭证：

```bash
curl -v --compressed -u<email_address>:<password>
    "https://gnip-api.x.com/search/30day/accounts/<account-name>/prod/counts.json?query=from%3Axdevelopers"
```
