**I. 定义**

在本协议中，适用以下定义：

1. “**Developer Site**”指 X 的开发者网站，位于 [**https://developer.x.com/**](https://developer.x.com/)。

2. “**Direct Message**”指一名用户通过 X 应用使用 X 的私信功能向一个或多个其他特定用户发送的私人消息。

3. “**Intellectual Property Rights**”指全球范围内的所有版权、精神权利、专利权、商标权以及任何其他知识产权或类似权利（无论是否注册）。

4. “**Licensed Material**”指 X API 和 X Content（单独或合称）。

5. “**Paid Service(s)**”指你通过支付本协议项下与相关功能或特性对应的周期性费用而得以访问的 Licensed Material 的功能或特性。除非本协议另有规定，本协议中对“Licensed Material”的所有引用均应视为包括 Paid Service(s)。

6. “**Payment Portal**”指向你提供的在线门户，你可通过该门户选择使用 Licensed Material、管理 Paid Services 的订阅、管理 Paid Services 的付款方式以及更新账户信息。

7. “**Services**”指你的服务、网站、应用以及其他产品或服务（包括研究），这些产品或服务展示 X Content 或以其他方式使用 Licensed Material。

8. “**Post**”指通过 X 应用分发的短格式文本和多媒体消息。

9. “**X**”指：(a) 如果你的主要营业地点在欧盟、EFTA 国家和英国之外，则指 X Corp.（865 FM 1209, Building 2, Bastrop, TX 78602, USA）；或 (b) 如果你的主要营业地点在欧盟、EFTA 国家或英国，则指 X Internet Unlimited Company（One Cumberland Place, Fenian Street, Dublin 2, D02 AX07, Ireland）。

10. “**X API**”指 X 的应用程序编程接口（各称为 **“API”**）、软件开发工具包（各称为 **“SDK”**），以及 X 通过 Developer Site 向你提供的相关工具、文档、数据、技术、代码和其他材料。

11. “**X Applications**”指 X 的实时信息服务，通常称为“X”，包括当前由 X 通过其网站以及 X 拥有、运营或控制的移动应用、社交插件和应用程序编程接口所提供的服务。

12. “**X Content**”指 Posts、为每个 Post 生成的唯一标识符、X 终端用户个人资料信息，以及通过 X API 或由 X 授权的任何其他方式向你提供的任何其他数据和信息，以及其任何副本和衍生作品。

13. “**X Marks**”指 X 的名称、商标和标识，这些是 X 向你提供的，包括通过 Developer Site 提供，并仅可根据本协议使用。

14. “**Users**”指你的 Services 的访问者、使用者或客户。
