**一、定义**

在本协议中，下列定义适用：

1. “**开发者网站（Developer Site）**”指 X 的开发者网站，网址为 [**https://developer.x.com/**](https://developer.x.com/)

2. “**私信（Direct Message）**”指通过 X 应用中 X 的私信功能，由一名用户向一个或多个其他特定用户发送的私密消息。

3. “**知识产权（Intellectual Property Rights）**”指全球范围内的所有版权、精神权利、专利权、商标权，以及任何其他知识产权或类似权利（已注册或未注册）。

4. “**许可材料（Licensed Material）**”单独或统称指 X API 和 X 内容（X Content）。

5. “**付费服务（Paid Service(s)）**”指你为获取相关功能或特性以及本协议项下适用内容而支付周期性费用后，方可访问的许可材料中的功能或特性。除非本协议中另有规定，对“许可材料”的所有引用均应视为包括付费服务。

6. “**支付门户（Payment Portal）**”指向你提供的在线门户，你可通过该门户选择使用许可材料、管理付费服务的订阅、管理付费服务的付款方式，以及更新账户信息。

7. “**服务（Services）**”指你的服务、网站、应用以及其他产品（包括研究），这些产品展示 X 内容或以其他方式使用许可材料。

8. “**帖子（Post）**”指通过 X 应用分发的短文本及多媒体消息。

9. “**X**”指 (a) 如果你的主要营业地位于欧盟、EFTA 国家和英国之外，则为 X Corp.（865 FM 1209, Building 2, Bastrop, TX 78602, USA）；或 (b) 如果你的主要营业地位于欧盟、EFTA 国家或英国之内，则为 X Internet Unlimited Company（One Cumberland Place, Fenian Street, Dublin 2, D02 AX07, Ireland）。

10. “**X API**”指 X 的应用程序编程接口（各称为“**API**”）、软件开发工具包（各称为“**SDK**”），以及 X 通过开发者网站提供的相关工具、文档、数据、技术、代码和其他材料。

11. “**X 应用（X Applications）**”指 X 的实时信息服务，通常称为“X”，其中包括 X 目前通过其网站以及 X 拥有、运营或控制的移动应用、社交插件和应用程序编程接口所提供的服务。

12. “**X 内容（X Content）**”指帖子、为每条帖子生成的唯一识别号码、X 终端用户的个人资料信息，以及通过 X API 或经 X 授权的任何其他方式向你提供的任何其他数据和信息，以及其任何副本和衍生作品。

13. “**X 标志（X Marks）**”指 X 向你提供的、仅可依据本协议使用的 X 名称、商标和标识，包括通过开发者网站向你提供的此类名称、商标和标识。

14. “**用户（Users）**”指你的服务的访问者、使用者或客户。
