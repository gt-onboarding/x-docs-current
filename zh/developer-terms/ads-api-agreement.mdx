<div id="2-definitions">
  ### 2. 定义。
</div>

**2.1** “**Campaign（广告活动）**”系指在桌面端、移动网站或移动应用程序上开展的广告活动，或其任何部分。

**2.2** “**Client（客户）**”系指使用一项（1）或多项 Ads Products、并在每一情形下均经 X 批准的 X 广告主（同时亦为 Company 的客户）。

**2.3** “**Company Marks（Company 标识）**”系指 Company 的名称以及 Company 向 X 提供以供根据本协议使用的任何 Company 标识、商标和/或服务标记（或 Company 以其他方式授权 X 根据本协议使用的此类标识、商标和/或服务标记）。

**2.4** “**Company Materials（Company 资料）**”系指 Company 通过 Ads Products 向 X 提供的任何 Company 数据、交付成果或用户内容。

**2.5** “**Company Service（Company 服务）**”系指展示或以其他方式使用 X Materials 或其任何部分的 Company 网站、应用程序及其他产品或服务。

**2.6** “**Data（数据）**”系指 X 向 Company 提供的指标、数据以及任何其他信息和/或内容，包括通过分析或使用上述任何内容（不论以汇总形式或其他形式）而产生的任何结果、使用统计、数据或其他信息。

**2.7** “**End Users（终端用户）**”系指代表该 Client 使用 Ads Products 的 Client 员工或获授权代理人。

**2.8** “**Marks（标识）**”系指视情形而定（基于上下文）之 X Marks 和/或 Company Marks。

**2.9** “**Materials（资料）**”系指视情形而定（基于上下文）之 X Materials 和/或 Company Materials。

**2.10** “**Personal Data（个人数据）**”系指与在世自然人有关的数据，该自然人可从该等数据本身，或将该等数据与数据控制者持有或可能获得的其他信息相结合而被识别或可被识别。

**2.11** “**X Ads**”系指 X 的广告产品和服务，包括在 X Network 上投放广告。

**2.12** “**X Code**”系指由 X 开发并提供给 Company 的计算机代码，该代码至少将调用一个 X 服务器。

**2.13** “**X Marks（X 标识）**”系指 X 的名称以及 X 向 Company 提供以供根据本协议使用的任何 X 标识、商标和/或服务标记（或 X 以其他方式授权 Company 根据本协议使用的此类标识、商标和/或服务标记）。

**2.14** “**X Materials（X 资料）**”系指由 X 生成的任何 X 数据、交付成果或用户内容（包括其衍生成果），以及任何由 X 生成、收集或开发的，或依据本协议由 X 向 Company 提供或使其可获得的 X 数据、交付成果或用户内容（包括其衍生成果），包括源自、基于或因 Company 与 X 的合作关系而产生的任何数据，任何设备级数据，以及 Company 或任何经 X 批准的第三方从 Company 的 X 广告活动中或与之相关而收集、推断、演绎或获取的任何转化、互动、跟踪或定向数据。为避免疑义且不作限制，“X Materials”包括 X Ads API（定义见[附件 A](#Exhibit-A)）、X Code、Data、Ads Products，以及用于创建 Custom Audience（定义见下文）的任何 X 用户 ID。

**2.15** “**X Network**”系指 X 可通过其分发广告的广告渠道网络，包括所有形式的媒体、应用程序和设备，无论是当前已知或今后开发的任何媒介。

**2.16** “**X Marketing Partners Program**”系指由 X 提供的合作与品牌计划，如[附件 C](#Exhibit-C)中进一步所述。

**2.17** “**X Service**”系指：(i) 由 X 运营的实时内容和信息服务；(ii) 促成该等服务的技术和系统，其统称包括 X 当前在其网站以及其拥有、运营和/或控制的移动应用程序和社交插件及 API 上所提供的面向消费者和广告主的产品和服务，包括但不限于 Measurement Code、任何 HTML 标签或其他代码、互联网门户、控制面板、系统和分析工具及相关服务；以及 (iii) 由 X 及其 Affiliates 提供的所有其他产品或服务，包括但不限于（仅为示例）X Audience Platform 以及 Data &amp; Enterprise Solutions。
