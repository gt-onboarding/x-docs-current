在你的项目中开始使用 X API 的 TypeScript SDK。

<div id="install">
  ## 安装
</div>

<CodeGroup>
  ```bash npm theme={null}
  npm install @xdevplatform/xdk
  ```

  ```bash yarn theme={null}
  yarn add @xdevplatform/xdk
  ```

  ```bash pnpm theme={null}
  pnpm add @xdevplatform/xdk
  ```
</CodeGroup>

<div id="typescript-support">
  ## TypeScript 支持
</div>

该 SDK 采用 TypeScript 编写，并内置完整的类型定义。无需额外安装任何类型相关的包。

<div id="requirements">
  ## 前提条件
</div>

* Node.js 16+
* TypeScript 4.5+（如果使用 TypeScript）
