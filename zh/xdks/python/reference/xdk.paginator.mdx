适用于 X API SDK 的基于游标的分页工具。

此模块提供一个 Cursor 类，为所有 API 客户端提供优雅的分页支持。通过 .pages() 和 .items() 方法，Cursor 能以类型安全的方式简化对分页结果的迭代。

<div id="xdkpaginatorcursor">
  ### `xdk.paginator.cursor`
</div>

创建一个具有正确类型推断和验证功能的游标。
这个工厂函数有助于进行类型推断，从而为响应类型提供正确的
类型提示，并在静态分析阶段和运行时验证该方法是否支持分页。

</ParamField>
