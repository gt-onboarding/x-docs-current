适用于 X API SDK 的基于游标的分页工具。

此模块提供了一个 Cursor 类，为所有 API 客户端提供优雅的分页支持。Cursor 通过 .pages() 和 .items() 方法，实现对分页结果的便捷迭代，并具备完备的类型安全。

<div id="xdkpaginatorcursor">
  ### `xdk.paginator.cursor`
</div>

创建一个具备正确类型推断和校验的游标。
这个工厂函数有助于进行类型推断，使你能够为响应类型获得正确的类型提示，并在静态分析阶段和运行时验证该方法是否支持分页。
