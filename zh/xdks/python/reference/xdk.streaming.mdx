适用于 X API SDK 的可靠流式传输工具库。

此模块提供流式连接管理，具备自动重连、指数退避以及全面的错误处理能力。客户端可以在无需关心连接管理的情况下使用流式端点，SDK 会自动完成所有恢复操作。

<div id="xdkstreamingstream_with_retry">
  ### `xdk.streaming.stream_with_retry`
</div>

从端点流式获取数据，并提供自动重连和指数退避重试。
此函数负责所有连接管理，包括：

- 在连接断开时自动重连
- 带抖动的指数退避重试等待时间
- 将错误分类为可重试错误与致命错误
- 用于监控连接状态的生命周期回调
