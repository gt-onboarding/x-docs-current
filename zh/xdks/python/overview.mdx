Python XDK（X Developer Kit）是我们面向 Python 与 X API v2 交互提供的官方客户端库。它可以帮助开发者快速上手我们的 API，并基于此构建应用程序。它是基于我们的官方 [OpenAPI 规范](https://api.x.com/2/openapi.json) 自动生成的。它在屏蔽底层 HTTP 细节的同时，在需要时又提供精细的控制能力。

<div id="key-features">
  ## 关键特性
</div>

- 🔐 **OAuth 支持**：完整支持 Bearer Token（仅应用模式）认证、带有 PKCE 的 OAuth 2.0（用户上下文）以及 OAuth 1.0。
- 🔄 **分页**：自动处理大型结果集的分页。XDK 会代为管理分页逻辑，你无需手动多次调用 API 并传递 `next_token`。
- 📡 **流式传输**：为诸如 `filtered stream` 这类需要持久 HTTP 连接的端点提供实时数据流支持。
- 🎯 **全面覆盖**：支持包括搜索、时间线、`filtered stream` 等在内的所有 X API v2 端点。  
**版本兼容性**：Python 3.8+。已在 CPython 和 PyPy 上测试。
