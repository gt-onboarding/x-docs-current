Python XDK（X Developer Kit）是我们用于通过 Python 与 X API v2 交互的官方客户端库。它使开发者能够快速上手我们的 API，并在此基础上构建应用程序。它基于我们的官方 [OpenAPI 规范](https://api.x.com/2/openapi.json) 生成。它屏蔽了底层 HTTP 细节，同时在需要时提供精细的控制能力。

<div id="key-features">
  ## 关键特性
</div>

- 🔐 **OAuth 支持**：完整支持 Bearer Token（仅应用）认证、带 PKCE 的 OAuth 2.0（用户上下文）以及 OAuth 1.0。
- 🔄 **分页**：自动处理大规模结果集的分页。XDK 会负责所有分页逻辑，你无需为携带 `next_token` 而手动多次调用 API。
- 📡 **流式传输**：支持对需要持久 HTTP 连接的接口（如 filtered stream）的实时数据流。
- 🎯 **全面覆盖**：支持所有 X API v2 端点，包括搜索、时间线、filtered stream 等。
**版本兼容性**：Python 3.8+。已在 CPython 和 PyPy 上测试。
