XDK Python SDK 可直接从 GitHub 仓库获取，并通过 `pip` 安装。

<div id="prerequisites">
  ## 前提条件
</div>

- Python 3.8 或更高版本。
- 使用 `pip` 和 `venv` 管理虚拟环境（推荐）。

<div id="quick-install">
  ## 快速安装
</div>

从 GitHub 仓库的子目录中安装 XDK：

```bash
pip install xdk
```

这将从 `main` 分支获取最新生成的版本。


<div id="development-install">
  ## 开发环境安装
</div>

用于开发或参与贡献：

1. 克隆仓库：
   ```bash
   git clone https://github.com/xdevplatform/xdk.git
   cd xdk/python
   ```
2. 以可编辑模式安装依赖：
   ```bash
   pip install -e .
   ```
   这会安装 SDK 及其运行时依赖。
3. （可选）安装用于测试/代码检查的开发依赖：
   ```bash
   pip install -e .[dev]
   ```

<div id="verification">
  ## 验证
</div>

测试安装：

```python
import xdk
print(xdk.__version__)  # 应打印 XDK 版本号
```

**注意：** 由于 XDK 是使用 OpenAPI 规范生成的，请务必查看 [X API 更新日志](https://docs.x.com/changelog) 和代码仓库中的 XDK 发行说明，以了解最新变更。
若要查看使用 Python XDK 的详细代码示例，请访问我们的 [代码示例 GitHub 仓库](https://github.com/xdevplatform/samples/tree/main/python)。
