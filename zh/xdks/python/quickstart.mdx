此示例演示如何使用 XDK 和 Bearer Token 认证快速搜索帖子。

<div id="step-1-install-the-sdk">
  ## 步骤 1：安装 SDK
</div>

```bash
pip install xdk
```

<div id="step-2-get-your-bearer-token">
  ## 步骤 2：获取你的 Bearer Token
</div>

1. 登录 [X 开发者门户](https://developer.x.com/en/portal/dashboard)。
2. 创建或选择一个应用。
3. 在 &quot;Keys and Tokens&quot; 下生成一个 Bearer Token（仅应用级认证）。

<div id="step-3-write-and-run-your-first-script">
  ## 步骤 3：编写并运行你的第一个脚本
</div>

创建一个名为 `quickstart.py` 的文件：

```python
# 导入 client
from xdk import Client
# 将此处替换为您的实际 Bearer Token
client = Client(bearer_token="YOUR_BEARER_TOKEN_HERE")
# 获取提及 "api" 的近期帖子
# search_recent 返回一个迭代器,因此需要对其进行迭代
for page in client.posts.search_recent(query="api", max_results=10):
    if page.data and len(page.data) > 0:
        # 访问第一个帖子 - Pydantic 模型同时支持属性访问和字典访问
        first_post = page.data[0]
        post_text = first_post.text if hasattr(first_post, 'text') else first_post.get('text', '')
        print(f"Latest Post: {post_text}")
        break
    else:
        print("未找到帖子。")
        break
```

运行：

```bash
python quickstart.py
```

**预期结果**:

```
Latest Post: XDK Python SDK 的重要更新！
```

**故障排查**：如果你收到 401 错误，请仔细检查你的 Bearer Token。对于速率限制错误（429），请等待一段时间后再重试。
