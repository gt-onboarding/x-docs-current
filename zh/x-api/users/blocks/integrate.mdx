<div id="authentication">
  #### 身份验证
</div>

所有 X API v2 端点都要求你使用一组凭证（也称为密钥和令牌）对请求进行身份验证。你可以使用 OAuth 1.0a 用户上下文或 OAuth 2.0 Authorization Code with PKCE 来对发送到这些端点的请求进行身份验证。 

[OAuth 1.0a 用户上下文](/zh/resources/fundamentals/authentication) 要求你使用 API Keys、用户访问令牌以及少量其他参数来[创建授权头](/zh/resources/fundamentals/authentication#authorizing-a-request)，然后将其随请求一同发送。访问令牌必须与你代表其发起请求的用户关联。如果你希望为另一位用户生成一组访问令牌，该用户必须通过[三方 OAuth 流程](https://developer-staging.x.com/resources/fundamentals/authentication/obtaining-user-access-tokens)授权你的 App。 

请注意，OAuth 1.0a 可能比较难用。如果你不熟悉这种身份验证方式，建议你使用[库](/zh/x-api/tools-and-libraries/overview)、使用 Postman 之类的工具，或使用 OAuth 2.0 来对请求进行身份验证。

[OAuth 2.0 Authorization Code with PKCE](/zh/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-3) 允许你对应用的权限范围（scope）以及跨多设备的授权流程进行更精细的控制。OAuth 2.0 允许你选择特定的细粒度权限范围（scope），从而代表用户获得相应的具体权限。 

要在你的应用中启用 OAuth 2.0，你必须在开发者门户中该 App 的设置部分找到其身份验证设置，并在其中启用它。

<div id="developer-portal-projects-and-developer-apps">
  #### 开发者门户、项目和开发者应用
</div>

要获取一组可用于 X API v2 端点的身份验证凭证，你必须先[注册开发者账号](https://developer.x.com/en/portal/petition/essential/basic-info)，在该账号下设置一个[项目](/zh/resources/fundamentals/projects)，然后在该项目中创建一个[开发者应用](/zh/resources/fundamentals/developer-apps)。接下来，你可以在开发者应用中找到你的密钥和令牌。
 

<div id="rate-limits">
  #### 速率限制
</div>

每天都有成千上万的开发者向 X API 发起请求。为了帮助管理如此巨大的请求量，每个端点都会设置 [速率限制](/zh/x-api/fundamentals/rate-limits)，限制你代表应用或代表已认证用户所能发起的请求次数。 

这些端点采用用户级别的速率限制，这意味着，你代表其发起请求的已认证用户，在任何开发者应用中对该端点的调用次数都有上限。对于同时支持 POST 和 DELETE 方法的每个端点，用户级别的速率限制为每 15 分钟 50 次请求。而对于使用 GET 方法时，速率限制仅为每 15 分钟 15 次请求。
 

<div id="fields-and-expansions">
  #### 字段和扩展字段
</div>

X API v2 的 GET 端点允许用户通过一组称为字段和扩展字段的工具，精确选择希望从 API 返回的哪些数据。`expansions` 参数允许你将响应负载中引用的对象展开为完整对象。例如，此端点允许你获取以下[扩展字段](/zh/x-api/fundamentals/expansions)：

* pinned&#95;tweet&#95;id

**fields** 参数允许你在不同的数据对象中，精确选择希望接收的哪些[字段](/zh/x-api/fundamentals/fields)。这些端点主要返回帖子（Post）对象。默认情况下，Post 对象仅返回 **id** 和 **text** 字段。要接收其他字段，例如 **tweet.created&#95;at** 或 **tweet.entities**，你必须使用 **fields** 参数显式请求这些字段。在你的集成中，你可能希望考虑使用的一些重要字段包括投票数据、指标数据、帖子标注以及会话 ID 字段。

我们在 [X API v2 数据字典](/zh/x-api/fundamentals/data-dictionary)中增加了一篇指南，介绍如何[结合使用字段和扩展字段](/zh/x-api/fundamentals/data-dictionary#how-to-use-fields-and-expansions)。
