X API v2 具有帖子消耗上限，用于限制每月可从特定端点检索的帖子数量。帖子上限适用于 [项目](/zh/resources/fundamentals/projects) 级别，这意味着使用该项目中 [开发者应用](/zh/resources/fundamentals/developer-apps) 的密钥和令牌向下文列出的端点发出的任何请求，都会计入该项目的帖子消耗上限。

<div id="limited-v2-endpoints">
  ## 受限的 v2 端点
</div>

从以下任一端点获取的帖子都会计入每月帖子额度：

* [帖子查找](/zh/x-api/posts/lookup/introduction)
* [最近搜索](/zh/x-api/posts/counts/introduction)
* [完整归档搜索](/zh/x-api/posts/counts/introduction)
* [过滤流](/zh/x-api/posts/filtered-stream)
* [过滤流 Webhook](/zh/x-api/webhooks/stream/introduction)
* [用户帖子时间线](/zh/x-api/posts/timelines)
* [用户提及时间线](/zh/x-api/posts/timelines)
* [点赞查找 - 用户点赞的帖子](/zh/x-api/posts/likes)
* [书签查找 - 用户添加书签的帖子](/zh/x-api/posts/bookmarks/introduction)
* [列表内帖子查找](/zh/x-api/lists/list-tweets/introduction)
* [Spaces 查找](/zh/x-api/spaces/lookup/introduction)

<Note>
  **注意：** 如果同一条帖子在某一天内通过多个查询被返回，它在帖子额度中只计一次；帖子会按日去重。
</Note>

<div id="post-cap-volumes">
  ## 帖子上限
</div>

帖子上限会根据你的访问级别而变化：

| Access Tier   | Monthly Post Cap                |
|:---------------|:---------------------------------|
| **Free**      | 每月 100 条帖子           |
| **Basic**     | 每月 15,000 条帖子          |
| **Pro**       | 每月 1,000,000 条帖子       |
| **Enterprise**| 每月超过 5,000 万条帖子     |

你可以通过在[主控制面板页面](https://developer.x.com/content/developer-twitter/en/portal/dashboard)中查看[开发者门户](/zh/resources/fundamentals/developer-portal)，来检查你本月的帖子用量及距离每月帖子上限的情况。在你的项目名称下方，你会看到一个状态条，用于显示你本月的用量与帖子上限之间的对比，包括已拉取帖子总数、已使用帖子的百分比，以及用量重置日期。
