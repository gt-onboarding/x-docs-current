`news lookup` 端点允许开发者获取在 X 上最新发布的新闻和头条。

此端点支持使用 app-auth 和 user-auth 方式的 OAuth1 和 OAuth2 PKCE 身份验证。

<div id="getting-started">
  ## 入门
</div>

要使用此端点，你需要从[开发者门户](https://developer.x.com/en/portal/dashboard)获取一个[bearer token](/zh/fundamentals/authentication/oauth-2-0/application-only)。

获得 bearer token 之后，你可以按如下方式调用新闻 API：

```bash
curl 'https://api.x.com/2/news/1989418137272422538?news.fields=contexts,cluster_posts_results' --header 'Authorization: Bearer XXXXX'
```
