<div id="community-tools-and-libraries-for-v2">
  ## 面向 v2 的社区工具与库
</div>

此处列出的库由开发者社区成员构建。请注意，它们对 API 的覆盖程度和支持进度可能各不相同。

如果你构建了自己的 X API 库或实用工具，请[告诉我们](https://devcommunity.x.com/c/libraries-and-sdks/63)，我们会将其添加到此列表中，方便其他人发现和使用。我们还提供了一些可借用的[版本徽章](https://twbadges.glitch.me/)，你可以用于自己的 README 文件中。

需要灵感？你可以在 GitHub 上浏览和搜索 [X](https://github.com/topics/twitter?o=desc&s=updated) 和 [X-api-v2](https://github.com/topics/twitter-api-v2?o=desc&s=updated) 主题，以查找其他开发者提供的有用代码示例。

<div id="c-net">
  ### C# / .NET
</div>

- [**CoreTweet**](https://github.com/CoreTweet/CoreTweet) 又一款 .NET X 库

- [**SocialOpinion**](https://github.com/jamiemaguiredotnet/SocialOpinion-Public) 使用 C# 编写、用于连接 X API 的 API
- [**Tweetinvi**](https://github.com/linvi/tweetinvi) 直观易用的 X C# 库
- [**TwitterSharp**](https://github.com/Xwilarg/TwitterSharp) 基于 X API v2 的 C# 封装库

<div id="go">
  ### Go
</div>

- [**ctw**](https://github.com/0dayfall/ctw) 一个用于 X API 的库
- [**go-twitter**](https://github.com/g8rswimmer/go-twitter) 一个用于集成 X API v2 的 Go 库
- [**gotwi**](https://github.com/michimani/gotwi) 一个用于 X API v2 的 Go 库
- [**gotwtr**](https://github.com/sivchari/gotwtr) 一个用于 X API 的库
- [**twitter-stream**](https://github.com/Fallenstedt/twitter-stream) 一个用于 X 的 V2 过滤流 API 的 Go 封装库
- [**twitter**](https://github.com/creachadair/twitter) 一个用于 X API 的 Go 客户端

<div id="java">
  ### Java
</div>

- [**twittered**](https://github.com/redouane59/twittered) 面向 Java 开发者的 X API 客户端
- [**twitter4j-v2**](https://github.com/takke/twitter4j-v2) 一个针对 X API v2 的简单封装，设计用于与 Twitter4J 一起使用

- [**JTW**](https://github.com/uakihir0/jtw) 面向 Java 的 X API v2 客户端库

<div id="javascript-nodejs-typescript">
  ### JavaScript（Node.js）/TypeScript
</div>

- [**node-twitter-api-v2**](https://github.com/PLhery/node-twitter-api-v2) 强类型、功能完备、轻量、灵活而强大的 Node.js X API 客户端库
- [**twitter.js**](https://github.com/twitterjs/twitter.js) 面向对象的 Node.js 和 TypeScript 库，用于与 X API v2 交互
- [**twitter-types**](https://github.com/twitterjs/twitter-types) X API 的类型定义
- [**twitter-v2**](https://github.com/HunterLarco/twitter-v2) 用于 X API 的异步客户端库
- [**tweet-json-to-html**](https://github.com/wdl/tweet-json-to-html) 将 X API v2 帖子 JSON 对象转换为 HTML 格式

<div id="kotlin">

- [**TwitterApiKit**](https://github.com/kojofosu/TwitterApiKit) 帮助你在创建用于访问 X API v2 的数据对象时节省时间。该库支持 Java、Kotlin 和 Android。

<div id="community-tools-and-libraries">
  ## 社区工具和库
</div>

以下是一些由社区维护的库，适用于多个编程语言和平台上的 X API。请注意，Twitter 团队可能尚未对所有这些资源进行测试。

除非特别说明，这里列出的库应当实现标准版 API **v1.1** 的大部分功能——有关详细信息和额外支持，请与各库作者联系。\
如果你构建了支持 X API **v2** 的库，请通过我们的[社区论坛](https://devcommunity.x.com/c/libraries-and-sdks/63)告知我们，以便有机会将其添加到本页面。你也可以使用论坛来报告这些列表的任何变更。

如果你常用的编程语言还没有对应的库或工具，请通过[反馈平台](https://twitterdevfeedback.uservoice.com/)告诉我们，你也可以在那里为其他想法投票，或获得灵感来构建并提交新的项目。
