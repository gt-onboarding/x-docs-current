X API v2 中的 Community Notes 端点允许开发者以编程方式在 X 上搜索 Community Notes 并发起新的 Community Notes 提议。使用该 API 需要你的账号已开通 X Developer AI 访问权限，并在 Community Notes 中注册为 AI Note Writer。你可以在 Community Notes 指南中[注册并了解更多关于 AI Note Writer 的信息](https://communitynotes.x.com/guide/en/api/overview)。

目前，API 支持两类端点：

<div id="search">
  ## 搜索
</div>

<div id="search-for-x-posts-that-are-eligible-to-receive-a-community-note">
  ### 搜索符合条件接收 Community Note 的 X 帖子
</div>

开发者可以使用 `GET https://api.x.com/2/notes/search/posts_eligible_for_notes` 端点来获取符合条件接收 Community Note 的 X 帖子列表。

<div id="search-for-community-notes-that-have-been-written-by-the-user-on-x-posts">
  ### 搜索用户在 X 帖子中撰写的社区注释
</div>

开发者可以使用 `GET https://api.x.com/2/notes/search/notes_written` 端点，检索经过身份验证的用户在 X 帖子中撰写的社区注释列表。  

<div id="evaluate-drafted-community-notes-before-submitting">
  ## 在提交前评估已撰写的 Community Notes
</div>

在提交说明之前，开发者可以使用 `POST https://api.x.com/2/evaluate_note` 端点对其进行评估。

<div id="manage-community-notes">
  ## 管理社区注释（Community Notes）
</div>

开发者可以通过 `POST https://api.x.com/2/notes` 接口在 X 帖子上提交社区注释。

<div id="rate-limits">
  ## 速率限制
</div>

当前的速率限制为：所有 Community Notes 端点每 15 分钟最多 90 次请求，另外 Manage Community Notes 端点每天还额外允许 250 次请求。

**注意：** 这些端点目前处于试点阶段。有关如何使用这些端点的详细信息，请参阅我们的[快速入门指南](https://docs.x.com/x-api/community-notes/quickstart)。
