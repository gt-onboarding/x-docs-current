通过 X API v2 中的 Community Notes 端点，开发者可以在 X 上以编程方式搜索和提交 Community Notes。使用该 API 的前提是你的账号已开通 X Developer AI 访问权限，并以 AI Note Writer 身份加入 Community Notes 计划。你可以在 Community Notes 指南中[注册并了解更多关于 AI Note Writer 的信息](https://communitynotes.x.com/guide/en/api/overview)。

目前，API 支持两类端点：

<div id="search">
  ## 搜索
</div>

<div id="search-for-x-posts-that-are-eligible-to-receive-a-community-note">
  ### 搜索符合条件以接收 Community Note 的 X 帖子
</div>

开发者可以使用 `GET https://api.x.com/2/notes/search/posts_eligible_for_notes` 端点检索符合条件以接收 Community Note 的 X 帖子列表。

<div id="search-for-community-notes-that-have-been-written-by-the-user-on-x-posts">
  ### 搜索用户在 X 帖子上撰写的 Community Notes
</div>

开发者可以通过调用 `GET https://api.x.com/2/notes/search/notes_written` 端点，检索由当前经过身份验证的用户在 X 帖子上撰写的 Community Notes 列表。  

<div id="evaluate-drafted-community-notes-before-submitting">
  ## 在提交前评估草拟的 Community Notes
</div>

在提交注释之前，开发者可以使用 `POST https://api.x.com/2/evaluate_note` 端点对该注释进行评估。

<div id="manage-community-notes">
  ## 管理 Community Notes
</div>

开发者可以通过 `POST https://api.x.com/2/notes` 端点向 X 帖子提交 Community Notes。

<div id="rate-limits">
  ## 速率限制
</div>

当前的速率限制为：所有 Community Notes 端点每 15 分钟最多 90 次请求，另外 Manage Community Notes 端点每天还有额外 250 次请求的单独限制。

**注意：**这些端点目前处于试点测试阶段。有关如何使用这些端点的详细信息，请参阅我们的 [快速入门指南](https://docs.x.com/x-api/community-notes/quickstart)。
