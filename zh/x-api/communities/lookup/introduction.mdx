**开始使用**

<div id="authentication">
  ### 身份验证
</div>

你可以使用 [OAuth 1.0a 用户上下文](/zh/resources/fundamentals/authentication)、[OAuth 2.0 应用仅模式](https://developer.x.com(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token)，或 [OAuth 2.0 Authorization Code with PKCE](resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2) 来对该端点进行身份验证。
 

<div id="making-a-request">
  ### 发起请求
</div>

你可以像下面这样，通过提供一个 ID 来调用 Communities 查找端点（这里使用的 ID 为 `1758747817642700922`）：

```
      curl --location 'https://api.x.com/2/communities/1758747817642700922' --header 'Authorization: ••••••'
```

如果请求成功，你应该会看到如下所示的 JSON 响应：

```
{
    "data": {
        "access": "Public",
        "description": "欢迎来到动漫社区!动漫爱好者在这里聚集,分享他们最喜爱的动漫作品,讨论一切与动漫相关的话题。",
        "join_policy": "Open",
        "name": "Anime Community"
    }
}
```

<div id="community-fields">
  ### Community 字段
</div>

通过指定 Community 字段，你可以获取与某个社区相关的更多数据。

可用的 Community 字段包括：

| 字段值 | 类型 | 说明 |     |
| :--- | :--- | :--- | :--- |
| created&#95;at | date (ISO 8601) | Community 的创建时间。 |     |
| id  | string | Community 的唯一标识符。 |     |
| name | string | Community 的名称。 |     |
| description | string | Community 的描述文本（如果已提供）。 |     |
| access | string | Community 的访问级别。  <br />  <br />可能的取值为：<br /><br />* `Public`<br />* `Closed` |     |
| join&#95;policy | string | Community 的加入策略。  <br />  <br />可能的取值为：<br /><br />* `Open`<br />* `RestrictedJoinRequestsDisabled`<br />* `RestrictedJoinRequestsRequireAdminApproval`<br />* `RestrictedJoinRequestsRequireModeratorApproval`<br />* `SuperFollowRequired` |     |
| member&#95;count | integer | 已加入该 Community 的成员数量。 |     |

<div id="example-request">
  #### 示例请求
</div>

你可以像下面这样调用 Communities Lookup 端点，并指定额外的 `community.fields` 参数（使用 ID `1758747817642700922`）：

```
curl --location 'https://api.x.com/2/communities/1758747817642700922?community.fields=access,created_at,description,id,join_policy,member_count,name' --header 'Authorization: ••••••'
```

**示例响应**

如果请求成功，你就会看到如下所示的 JSON 响应：

```
{
    "data": {
        "description": "Welcome to the Anime Community! Where anime fans gather to share their favorite shows and discuss everything anime-related.",
        "access": "Public",
        "member_count": 40369,
        "created_at": "2024-02-17T06:58:50.000Z",
        "join_policy": "Open",
        "name": "Anime Community",
        "id": "Q29tbXVuaXR5OjE3NTg3NDc4MTc2NDI3MDA5MjI="
    }
}
```
