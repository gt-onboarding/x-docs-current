**开始使用**

<div id="authentication">
  ### 认证
</div>

你可以使用 [OAuth 1.0a 用户上下文](/zh/resources/fundamentals/authentication)、[OAuth 2.0 应用仅模式](https://developer.x.com(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token)，或 [OAuth 2.0 Authorization Code with PKCE](resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2) 对该端点进行身份验证。
 

<div id="making-a-request">
  ### 发起请求
</div>

您可以按如下方式通过提供一个 ID 来调用 Communities Lookup 端点（使用 ID `1758747817642700922`）：

```
      curl --location 'https://api.x.com/2/communities/1758747817642700922' --header 'Authorization: ••••••'
```

如果请求成功，你将会看到如下所示的 JSON 响应：

```
{
    "data": {
        "access": "Public",
        "description": "欢迎来到动漫社区!动漫爱好者在这里聚集,分享他们最喜爱的动漫作品,讨论一切与动漫相关的话题。",
        "join_policy": "Open",
        "name": "Anime Community"
    }
}
```

<div id="community-fields">
  ### Community 字段
</div>

你可以通过指定 Community 字段来获取与社区关联的更多数据。

可用的 Community 字段如下：

| 字段值 | 类型 | 描述 |     |
| :--- | :--- | :--- | :--- |
| created&#95;at | date (ISO 8601) | 社区的创建时间。 |     |
| id  | string | 社区的唯一标识符。 |     |
| name | string | 社区的名称。 |     |
| description | string | 社区的描述文本（如果已提供）。 |     |
| access | string | 社区的访问级别。  <br />  <br />可以是以下之一：<br /><br />* `Public`<br />* `Closed` |     |
| join&#95;policy | string | 社区的加入策略。  <br />  <br />可以是以下之一：<br /><br />* `Open`<br />* `RestrictedJoinRequestsDisabled`<br />* `RestrictedJoinRequestsRequireAdminApproval`<br />* `RestrictedJoinRequestsRequireModeratorApproval`<br />* `SuperFollowRequired` |     |
| member&#95;count | integer | 已加入该社区的成员数量。 |     |

<div id="example-request">
  #### 示例请求
</div>

您可以在调用 Communities Lookup 端点时传入更多 community 字段，如下所示（使用 ID `1758747817642700922`）：

```
curl --location 'https://api.x.com/2/communities/1758747817642700922?community.fields=access,created_at,description,id,join_policy,member_count,name' --header 'Authorization: ••••••'
```

**示例响应**

如果请求成功，你将看到如下所示的 JSON 响应：

```
{
    "data": {
        "description": "Welcome to the Anime Community! Where anime fans gather to share their favorite shows and discuss everything anime-related.",
        "access": "Public",
        "member_count": 40369,
        "created_at": "2024-02-17T06:58:50.000Z",
        "join_policy": "Open",
        "name": "Anime Community",
        "id": "Q29tbXVuaXR5OjE3NTg3NDc4MTc2NDI3MDA5MjI="
    }
}
```
