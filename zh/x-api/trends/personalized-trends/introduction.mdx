个性化趋势端点允许你为已通过身份验证的用户账号获取个性化趋势。

<Note>
  **注意：** 只有订阅了 X Premium 的用户账号，此端点才会返回个性化趋势。
</Note>

<Note>
  **账号设置**

  若要访问此端点，你需要：

  * 一个已通过审核的[开发者账号](https://developer.x.com/en/portal/petition/essential/basic-info)。
  * 使用位于某个[项目](/zh/resources/fundamentals/projects)中的[开发者应用](/zh/resources/fundamentals/developer-apps)的密钥和令牌进行身份验证。 

  请参阅我们的[入门指南](/zh/x-api/getting-started/getting-access)，了解如何获取对 X API v2 端点的访问权限。
</Note>

<div id="getting-started">
  ### 开始使用
</div>

<div id="authentication">
  #### 身份验证
</div>

你可以使用 [OAuth 1.0a 用户上下文](/zh/resources/fundamentals/authentication) 或 [OAuth 2.0 Authorization Code with PKCE](resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2) 来对此端点的请求进行身份验证。
 

<div id="making-a-request">
  #### 发起请求
</div>

你可以按如下方式调用 Personalized Trends 端点：

`curl 'https://api.x.com/2/users/personalized_trends' --header 'Authorization: ••••••'`

如果请求成功，你将会看到如下所示的 JSON 响应：

```
{
    "data": [
        {
            "category": "Bitcoin",
            "post_count": "4.2万条帖子",
            "trend_name": "MicroStrategy 420亿美元比特币投资",
            "trending_since": "5小时前"
        },
        {
            "category": "Music",
            "post_count": "2千条帖子",
            "trend_name": "Drake《No Face》音乐视频发布",
            "trending_since": "当前热门"
        },
        {
            "category": "Weather",
            "post_count": "1.2千条帖子",
            "trend_name": "超强台风Leon影响台湾和巴丹群岛",
            "trending_since": "当前热门"
        }
    ]
}
```

**注意：** 如果发起该请求的已通过身份验证的用户未订阅 X Premium，则会收到如下所示的响应：

```
{
    "data": [
        {
            "category": "Unknown",
            "post_count": "Unknown",
            "trending_since": ""
        }
    ]
}
```
