趋势查询端点允许开发者获取某个位置的趋势，该位置使用 where-on-earth id (WOEID) 指定。

<Note>
  **注意**：WOEID 是由 Yahoo 创建的遗留标识符，现已被弃用。X API 使用其数值来标识城镇和国家的趋势位置。请参考我们早期的 [博客文章](https://blog.x.com/engineering/en_us/a/2010/woeids-in-twitters-trends.html)，或 [归档数据](https://archive.org/details/geoplanet_data_7.10.0.zip0.)
</Note>

如果可用，许多趋势还会返回其过去 24 小时内的 `tweet_count`。

此端点支持 app-auth 认证，并且在每个 15 分钟窗口内的速率限制为 75 个请求。

<div id="getting-started">
  ### 入门
</div>

要使用该端点，你需要从 [developer portal](https://developer.x.com/en/portal/dashboard) 获取一个 [bearer token]([https://developer.x.com(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token)](https://developer.x.com\(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token\))。获得 bearer token 后，你可以按如下所示调用 Usage API：

```
      curl 'https://api.x.com/2/trends/by/woeid/26062' --header 'Authorization: Bearer XXXXX'
```

如果请求成功，你应当会看到如下所示的 JSON 响应：

**账号设置**

要访问这些端点，你需要：

* 一个已获批准的[开发者账号](https://developer.x.com/en/portal/petition/essential/basic-info)。
* 使用位于某个[项目](/zh/resources/fundamentals/projects)中的[开发者应用](/zh/resources/fundamentals/developer-apps)提供的密钥和令牌进行身份验证。 

在我们的[入门指南](/zh/x-api/getting-started/getting-access)中了解更多关于如何获取对 X API v2 端点访问权限的信息。
