<div id="introduction">
  ## 简介
</div>

我们的开发者社区拥有独特的技能、经验和视角，能够填补空白、解决问题，并在 X 上催生远超我们自身能力范围的全新创新成果。我们鼓励你构建工具和产品，让 X 变得更好、更健康，并拓展公共讨论。特别是，我们希望在以下类别中看到创新：

<div id="moderate-conversations-for-health-and-safety">
  ### 为健康与安全适度管理对话
</div>

我们希望每个人在 X 上参与对话时都能感到舒适、安全，并乐于参与。我们一直在努力提供工具，让人们对自己的体验拥有更多控制权，例如帖子回复设置、屏蔽和静音功能。不过，总还会有更多工作要做，因此我们鼓励你构建互补或扩展性的解决方案，帮助提升平台的健康度与安全性。请查看下方的一些资源，帮助你入门：

<div id="improve-community-experiences">
  ### 改善社区体验
</div>

随着新的社区在 X 上不断涌现和壮大，我们希望开发者能运用他们的专业技能，帮助满足这些社区独特的需求。X 是全球对话发生的地方，我们希望每个人都能参与其中，而不受语言、时区、文化差异或其他本地化需求的限制。此外，我们也希望围绕金融或游戏等主题不断壮大的社区，能够拥有专为其独特体验和需求打造的工具。
