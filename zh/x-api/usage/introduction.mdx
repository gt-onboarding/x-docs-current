X API v2 中的 Usage API 允许开发者通过编程方式检索其项目用量。使用此端点，开发者可以跟踪并监控在指定计费周期内拉取的帖子数量。

开发者可以使用该 GET 端点获取最长追溯 90 天的每日项目用量。用量数据也可以按连接到你项目的每个客户端应用进行汇总。

此 GET 端点的应用级速率限制为每 15 分钟 50 个请求。

<div id="getting-started">
  ### 入门
</div>

要使用此端点，你需要从[开发者门户](https://developer.x.com/en/portal/dashboard)获取一个[bearer token](https://developer.x.com/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token)。获取 bearer token 后，你可以按下述方式调用 Usage API：

```
      curl 'https://api.x.com/2/usage/tweets' --header 'Authorization: Bearer XXXXX'
```

如果请求成功，您应该会看到如下所示的 JSON 响应：

```
{
    "data": {
        "cap_reset_day": 10, // 配额重置日
        "project_cap": "5000000000", // 项目配额
        "project_id": "1369785403853424", // 项目 ID
        "project_usage": "43435" // 项目使用量
    }
}
```

<Note>
  **账户设置**

  若要访问这些端点，你需要：

  * 一个已获批准的 [开发者账号](https://developer.x.com/en/portal/petition/essential/basic-info)。
  * 使用位于某个 [项目](/zh/resources/fundamentals/projects) 中的 [开发者应用](/zh/resources/fundamentals/developer-apps) 所提供的密钥和令牌进行身份验证。 

  请参阅我们的 [入门指南](/zh/x-api/getting-started/getting-access)，了解如何获取 X API v2 端点的访问权限。
</Note>
