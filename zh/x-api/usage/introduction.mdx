X API v2 中的 Usage API 允许开发者以编程方式获取其项目用量。通过使用此 endpoint，开发者可以跟踪和监控在特定计费周期内已获取的帖子数量。

开发者可以使用该 GET endpoint 获取最近最多 90 天的项目每日用量。用量也可以按连接到你项目的各个客户端应用进行汇总。

此 GET endpoint 的应用速率限制为每 15 分钟最多 50 次请求。

<div id="getting-started">
  ### 入门
</div>

要使用此端点，你需要从[开发者门户](https://developer.x.com/en/portal/dashboard)获取一个[Bearer Token](https://developer.x.com/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token)。获得 Bearer Token 后，你就可以按照下面的示例调用 usage API：

```
      curl 'https://api.x.com/2/usage/tweets' --header 'Authorization: Bearer XXXXX'
```

如果请求成功，你应该会看到如下所示的 JSON 响应：

```
{
    "data": {
        "cap_reset_day": 10, // 配额重置日
        "project_cap": "5000000000", // 项目配额
        "project_id": "1369785403853424", // 项目 ID
        "project_usage": "43435" // 项目使用量
    }
}
```

<Note>
  **账号设置**

  要访问这些端点，你需要：

  * 一个已获批准的 [开发者账号](https://developer.x.com/en/portal/petition/essential/basic-info)。
  * 使用位于某个 [项目](/zh/resources/fundamentals/projects) 中的 [开发者应用](/zh/resources/fundamentals/developer-apps) 所提供的密钥和令牌进行身份验证。 

  在我们的 [入门指南](/zh/x-api/getting-started/getting-access) 中了解如何获取 X API v2 端点的访问权限。
</Note>
