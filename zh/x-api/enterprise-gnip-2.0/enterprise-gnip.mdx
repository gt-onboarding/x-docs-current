我们的 Enterprise API 为依赖 X 数据的用户提供最高级别的访问权限和可靠性。非常适合在使用规模超出高级方案、需要更可靠的访问、自定义套餐或更长期合同的场景。获得 Enterprise API 访问权限后，你还将拥有专属客户经理和个性化技术支持。

如果你已经确定 Basic 和 Pro 方案不足以满足你的需求，可以通过以下链接申请访问 Enterprise 产品：

</Button>

<div id="available-endpoints">
  ## 可用的端点
</div>

| 实时 | [PowerTrack API](/zh/x-api/enterprise-gnip-2.0/powertrack-api) | 为实时发布的公开帖子提供完整覆盖，并可使用多种运算符和规则进行过滤。 |
| :--- | :--- | :--- |
| [Decahose API](/zh/x-api/enterprise-gnip-2.0/fundamentals/decahose-api) | 以流式方式传输所有公开帖子中随机抽取的 10% 及其点赞。 |
| :--- | :--- | :--- |
| [Account Activity API](/zh/x-api/enterprise-gnip-2.0/fundamentals/account-activity) | 订阅用户，并接收多种不同类型的活动，包括帖子、私信、点赞、关注等。 |
| :--- | :--- | :--- |
| 历史 | [Search API](/zh/x-api/enterprise-gnip-2.0/fundamentals/search-api) | 使用高级过滤工具搜索过去 30 天或完整归档中的公开帖子。 |
| [Historical PowerTrack API](/zh/x-api/enterprise-gnip-2.0/powertrack-api) | 通过经济高效的批量访问流程过滤完整归档中的公开帖子。 |
| 洞察 | [Engagement API](/zh/x-api/enterprise-gnip-2.0/fundamentals/engagement-api) | 衡量公开帖子的展示量和互动量。 |
| 合规 | [Compliance Firehose API](/zh/x-api/enterprise-gnip-2.0/fundamentals/firehouse) | 以流式方式传输所有合规事件。 |
| 用量 | [Usage API](/zh/x-api/enterprise-gnip-2.0/fundamentals/usage) | 提供对各 Enterprise 产品活动使用量的程序化访问。 |

<div id="important-concepts">
  ## 关键概念
</div>

<div id="gnip-console">
  ### Gnip Console
</div>

您可以使用 [Gnip Console](/zh/x-api/enterprise-gnip-2.0/fundamentals/overview) 管理您的 Enterprise 产品和账户访问。 

<div id="enrichments">
  ### 数据富化
</div>

对于使用付费层级的开发者，我们在 premium API 的响应负载中添加了元数据。请访问我们的[数据富化部分](/zh/x-api/enterprise-gnip-2.0/fundamentals/data-enrichments)，以进一步了解展开后的 URL、投票数据以及个人资料地理位置信息。
 

<div id="advanced-filtering-tools">
  ### 高级过滤工具
</div>

Premium Search API 提供了一组在标准搜索 endpoint 中不可用的高级运算符。许多此类运算符对免费 sandbox 用户也可用，但如果你使用的是付费层级，将能获得更完整、更强大的运算符集合。你可以访问我们的[规则和过滤章节](/zh/x-api/enterprise-gnip-2.0/fundamentals/rules-filtering#getting-started-with-enterprise-rules-and-queries)了解更多信息。
