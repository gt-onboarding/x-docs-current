我们的 Enterprise API 为依赖 X 数据的用户提供最高级别的访问权限和可靠性。非常适合在使用规模超出高级版、需要更可靠的访问、定制套餐或长期合同时使用。获得 Enterprise API 访问权限还将配备专属客户经理和个性化技术支持。

如果你发现 Basic 和 Pro 方案不足以满足你的需求，可以通过以下链接申请使用 Enterprise 产品：

<Button href="https://developer.x.com/en/products/twitter-api/enterprise/enterprise-api-interest-form">
  申请访问权限
</Button>

<div id="available-endpoints">
  ## 可用的端点
</div>

| 实时 | [PowerTrack API](/zh/x-api/enterprise-gnip-2.0/powertrack-api) | 对实时发布的公共帖子提供完整覆盖，可使用广泛的操作符和规则进行过滤。 |
| :--- | :--- | :--- |
| [Decahose API](/zh/x-api/enterprise-gnip-2.0/fundamentals/decahose-api) | 流式传输所有公共帖子及其点赞中随机抽样的 10%。 |
| :--- | :--- | :--- |
| [Account Activity API](/zh/x-api/enterprise-gnip-2.0/fundamentals/account-activity) | 订阅用户并接收多种不同类型的活动，包括帖子、私信、点赞、关注等。 |
| :--- | :--- | :--- |
| 历史 | [Search API](/zh/x-api/enterprise-gnip-2.0/fundamentals/search-api) | 使用高级过滤工具搜索最近 30 天或公共帖子完整存档。 |
| [Historical PowerTrack API](/zh/x-api/enterprise-gnip-2.0/powertrack-api) | 使用经济高效的批量访问流程过滤整个公共帖子完整存档。 |
| 洞察 | [Engagement API](/zh/x-api/enterprise-gnip-2.0/fundamentals/engagement-api) | 衡量公共帖子的展示量和互动量。 |
| 合规 | [Compliance Firehose API](/zh/x-api/enterprise-gnip-2.0/fundamentals/firehouse) | 流式传输所有合规事件。 |
| 使用情况 | [Usage API](/zh/x-api/enterprise-gnip-2.0/fundamentals/usage) | 提供对各 Enterprise 产品活动使用量的程序化访问。 |

<div id="important-concepts">
  ## 重要概念
</div>

<div id="gnip-console">
  ### Gnip 控制台
</div>

您可以通过 [Gnip 控制台](/zh/x-api/enterprise-gnip-2.0/fundamentals/overview) 来管理 Enterprise 产品和账户访问权限。 

<div id="enrichments">
  ### 数据富化
</div>

对于使用付费级别的开发者，我们在 Premium API 的响应中添加了元数据。请访问我们的[数据富化部分](/zh/x-api/enterprise-gnip-2.0/fundamentals/data-enrichments)，了解有关扩展 URL、投票数据以及个人资料地理位置信息的更多内容。
 

<div id="advanced-filtering-tools">
  ### 高级过滤工具
</div>

Premium Search API 提供了一组标准搜索端点不具备的高级运算符。许多此类运算符对免费 Sandbox 用户也可用，但如果你使用的是付费层级，将能使用更丰富、更强大的运算符集合。要了解更多，请访问我们的[规则和过滤章节](/zh/x-api/enterprise-gnip-2.0/fundamentals/rules-filtering#getting-started-with-enterprise-rules-and-queries)。
