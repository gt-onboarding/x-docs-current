<div id="overview">
  ## 概览
</div>

每天都有成千上万的开发者向 X API 发起请求。为了管理如此庞大的请求量，我们对可发起的请求次数设置了限制。这些限制有助于我们为开发者社区提供可靠、可扩展的 API。 

允许的最大请求次数是基于时间间隔来确定的，即某个特定的时间周期或时间窗口。例如，如果某个端点的频率限制是每 15 分钟 900 次请求，那么在任意 15 分钟窗口内最多允许 900 次请求。
