<div id="overview">
  ## 概览
</div>

每天都有成千上万的开发者向 X API 发起请求。为了管理如此庞大的请求量，我们对可发起的请求次数设置了限制。这些限制有助于我们为开发者社区提供可靠且可扩展的 API。 

允许的最大请求次数是基于一个时间区间（即指定的时间段或时间窗口）来计算的。如果某个 endpoint 的速率限制为 900 次请求/15 分钟，那么在任意 15 分钟内最多允许发出 900 次请求。
