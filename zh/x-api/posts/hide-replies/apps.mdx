  ## 使用隐藏回复端点的应用
</div>

以下是一些值得一提的应用，这些应用由集成了隐藏回复端点的开发者构建。通过将他们的技术与 X API 结合，他们打造出有助于所有用户提升公共对话质量的实用体验。

![Clarabridge](https://cdn.cms-twdigitalassets.com/content/dam/developer-twitter/hide-replies/clarabridge.jpg.twimg.1920.jpg)

**Clarabridge**

Clarabridge 集成了隐藏回复功能，使公司可以保持其 X 信息流整洁且更具可读性。回复可以手动隐藏，客服人员也可以利用 Clarabridge 的文本分析功能，自动检测不符合品牌政策的脏话和其他用语。

[访问网站 ▸](https://www.clarabridge.com/ "访问网站 ▸")

[Clarabridge 在 X 上](https://x.com/Clarabridge "Clarabridge 在 X 上")

![Perspective API template app](https://cdn.cms-twdigitalassets.com/content/dam/developer-twitter/hide-replies/perspective-logo.png.twimg.1920.png)

**Perspective API 模板应用**

Perspective 是由 Jigsaw（Alphabet 内的一个部门）训练的人工智能，用于检测含有攻击性的评论。该应用会检测某人收到的回复，并根据一个“评分”自动隐藏回复，该评分表示 Perspective 认为该评论与其过去见过的攻击性评论相似的可信度。这是由 X 开发的开源模板应用。我们鼓励你对其进行个性化定制，为你的用户构建工具。

[使用应用 ▸](https://quintessential-yoke.glitch.me "使用应用 ▸")

[Jigsaw 在 X 上](https://x.com/jigsaw "Jigsaw 在 X 上")

![Reshuffle](data:image/svg+xml,%3Csvg xmlns=%22http://www.w3.org/2000/svg%22 viewBox=%220 0 150 150%22%3E%3C/svg%3E)

**Reshuffle**

Reshuffle 是一个用于通过灵活脚本连接业务应用的平台。他们构建了一个集成脚本，用于检测并隐藏包含你所定义关键词的回复。要试用，请确保你有一个为“隐藏回复”启用的有效应用，然后按照代码仓库中的说明进行操作。

[访问应用 ▸](https://github.com/reshufflehq/reshuffle "访问应用 ▸")

[Reshuffle 在 X 上](https://x.com/reshufflehq "Reshuffle 在 X 上")

![Hide unwanted replies](data:image/svg+xml,%3Csvg xmlns=%22http://www.w3.org/2000/svg%22 viewBox=%220 0 396 397%22%3E%3C/svg%3E)

**Hide unwanted replies**

热门应用 QuotedReplies 背后的开发者 Dara Oladosu 构建了一个应用，会自动隐藏满足他所设定的某些条件、且更有可能表现出辱骂行为的回复，其中包括包含他过去已静音的某些关键词的回复。

[访问应用 ▸](https://hideunwantedreplies.com/ "访问应用 ▸")

[Dara Oladosu 在 X 上](https://x.com/dara_tobi "Dara Oladosu 在 X 上")

![Pandaflow](data:image/svg+xml,%3Csvg xmlns=%22http://www.w3.org/2000/svg%22 viewBox=%220 0 94 94%22%3E%3C/svg%3E)

**Pandaflow**

Pandaflow 是一个无需编写代码的自动化工作流平台。他们集成了该端点，以便用户可以基于自己定义的流程，以编程方式隐藏回复。

[访问应用 ▸](https://pandaflow.io/ "访问应用 ▸")

[Pandaflow 在 X 上](https://x.com/pandaflowio "Pandaflow 在 X 上")
