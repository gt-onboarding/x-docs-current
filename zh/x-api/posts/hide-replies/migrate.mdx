  ## 比较 X API 的隐藏回复端点
</div>

v2 隐藏回复端点正在取代 Labs 隐藏回复端点。如果你有使用该端点 Labs 版本的代码、应用或工具，并正在考虑迁移到更新的 X API v2 端点，那么本指南适合你。

要使用新的 X API v2（包括隐藏回复端点），你需要[选择加入新的开发者门户](https://developer.x.com/en/portal/opt-in)，创建一个[项目](/zh/resources/fundamentals/projects)，并向该项目添加一个应用。然后，你可以使用与该应用关联的凭证向隐藏回复端点发起请求。将已为 Labs v2 隐藏回复端点注册的同一个应用用于新端点，可以让你的用户保持已认证状态。

下表比较了 Labs 与更新的 X API v2 端点之间的差异：
 

| **说明** | **Labs v2** | **X API v2** |
| :--- | :--- | :--- |
| 主机域名 | https://api.x.com | https://api.x.com |
| 端点路径 | /labs/2/tweets/:id/hidden | /2/tweets/:id/hidden |
| 认证方式 | OAuth 1.0a 用户上下文 | OAuth 1.0a 用户上下文 |
| 支持的 HTTP 方法 | PUT | PUT |
| 默认请求速率限制 | 每 15 分钟 10 次请求（在所有已认证用户之间共享） | 每 15 分钟 50 次请求（针对每个已认证用户） |
| 可以隐藏回复 | ✔︎  | ✔︎  |
| 可以取消隐藏先前被隐藏的回复 | ✔︎  | ✔︎  |
| 可以多次隐藏或取消隐藏回复 | ✔︎  | ✔︎  |
| 需要使用与某个[项目](/zh/resources/fundamentals/projects)关联的[开发者应用](/zh/resources/fundamentals/developer-apps)的凭证 |     | ✔   |
