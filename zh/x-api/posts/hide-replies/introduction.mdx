此端点允许你根据自定义条件，以编程方式隐藏或取消隐藏回复。与 X 主产品中的相关功能相同，被隐藏的回复会从主对话视图中移除，但仍可在单独页面中查看。你可以使用此端点构建应用，帮助用户隐藏他们认为没有价值的回复。

隐藏回复端点使用 [OAuth 1.0a 用户上下文](/zh/resources/fundamentals/authentication#oauth-1-0a-2) 或 [OAuth 2.0 Authorization Code with PKCE](resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2) 进行认证。调用成功后，该端点会隐藏一条由已通过身份验证的用户发起的帖子对话中的单条回复。每个对话最多支持隐藏 725 条帖子。

<Note>
  **账号设置**

  要访问这些端点，你需要：

  * 一个已获批准的[开发者账号](https://developer.x.com/en/portal/petition/essential/basic-info)。
  * 使用位于某个[项目](/zh/resources/fundamentals/projects)中的[开发者应用](/zh/resources/fundamentals/developer-apps)的密钥和令牌进行认证。 

  在我们的[入门指南](/zh/x-api/getting-started/getting-access)中了解更多关于如何获取 X API v2 端点访问权限的信息。
</Note>
