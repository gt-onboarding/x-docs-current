<div id="comparing-x-apis-timelines-endpoints">
  ## 比较 X API 的时间线端点
</div>

v2 的按时间倒序主页时间线、用户帖子时间线和用户提及时间线端点分别取代了 [v1.1 statuses/home&#95;timeine](https://developer.x.com/en/docs/twitter-api/v1/tweets/timelines/api-reference/get-statuses-home_timeline)、[v1.1 statuses/user&#95;timeline](https://developer.x.com/en/docs/twitter-api/v1/tweets/timelines/api-reference/get-statuses-user_timeline.html) 和 [v1.1 statuses/mentions&#95;timeline](https://developer.x.com/en/docs/twitter-api/v1/tweets/timelines/api-reference/get-statuses-mentions_timeline.html) 端点。如果你已有使用这些旧版端点的代码、应用或工具，并且正在考虑迁移到新版的 X API v2 端点，那么本指南适合你。更深入的迁移说明请参阅 [Standard v1.1 迁移到 X API v2](/zh/x-api/posts/timelines/migrate/standard-to-twitter-api-v2)。

本页包含三张对比表：

* 按时间倒序的主页时间线
* 用户帖子时间线
* 用户提及时间线

<div id="reverse-chronological-home-timeline">
  ### 按时间倒序的主页时间线
</div>

下表比较了标准 v1.1 和 X API v2 的主页时间线端点：

|                                                                                                                     |                                                                                                                                                        |                                                                                                                                                                                                                                                                                                                                                                             |
| :------------------------------------------------------------------------------------------------------------------ | :----------------------------------------------------------------------------------------------------------------------------------------------------- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **说明**                                                                                                              | **标准版 v1.1**                                                                                                                                           | **X API v2**                                                                                                                                                                                                                                                                                                                                                                |

| 支持的 HTTP 方法                                                                                                         | `GET`                                                                                                                                                  | `GET`                                                                                                                                                                                                                                                                                                                                                                       |
| 主机域名                                                                                                                | `https://api.x.com`                                                                                                                                    | `https://api.x.com`                                                                                                                                                                                                                                                                                                                                                         |
| 接口路径                                                                                                                | `/1.1/statuses/home_timeline.json`                                                                                                                     | `/2/users/:id/timelines/reverse_chronological`                                                                                                                                                                                                                                                                                                                              |
| 必需参数                                                                                                                | `user_id` or `screen_name`                                                                                                                             | 用户 ID 作为路径参数 :id 传入                                                                                                                                                                                                                                                                                                                                                         |
| 认证                                                                                                                  | OAuth 1.0a 用户上下文                                                                                                                                       | OAuth 1.0a 用户上下文<br /><br />[带 PKCE 的 OAuth 2.0 授权码流程](/zh/resources/fundamentals/authenticationoauth-2-0/authorization-code)                                                                                                                                                                                                                                                  |

| 每个响应可返回的最多帖子数                                                                                                       | 800                                                                                                                                                    | 此端点会返回时间线上过去 7 天内创建的所有帖子，以及不论创建时间的最近 800 条帖子。                                                                                                                                                                                                                                                                                                                               |

| 可获取的历史帖子                                                                                                            | 最新的 800 条帖子（包括转发）                                                                                                                                      | 包括转帖在内的最新 3,200 条帖子                                                                                                                                                                                                                                                                                                                                                         |
| 时间线导航选项                                                                                                             | since&#95;id（不包含该值），用于轮询获取更新<br /><br />`max_id`（包含该值）                                                                                                 | `start_time`<br /><br />end&#95;time<br /><br />`since_id`（不包含该 id），用于轮询获取更新 <br /><br />`until_id`（不包含该 id）                                                                                                                                                                                                                                                                |
| 用于细化结果的可选参数                                                                                                         | `count`<br /><br />`exclude_replies`<br /><br />`include_rts`<br /><br />`trim_user`<br /><br />`tweet_mode`<br /><br />`since_id`<br /><br />`max_id` | `max_results`<br /><br />`exclude`(retweets,replies)<br /><br />`tweet.fields`<br /><br />`user.fields`<br /><br />`place.fields`<br /><br />`media.fields`<br /><br />`poll.fields`<br /><br />`expansions`<br /><br />`start_time`<br /><br />`end_time`<br /><br />`since_id`<br /><br />`until_id`                                                                      |
| 支持请求和接收[注解](/zh/x-api/fundamentals/post-annotations)                                                                   | 不支持                                                                                                                                                    | 如果在 tweet.fields 中包含注解，结果将带有基于帖子文本推断的注解数据，例如 &#39;Music Genre&#39; 与 &#39;Folk Music&#39;，或 &#39;Musician&#39; 与 &#39;Dolly Parton&#39;                                                                                                                                                                                                                                     |

| 支持请求并接收 [conversation&#95;id](/zh/x-api/fundamentals/conversation-id)                                                  | 不适用                                                                                                                                                    | 返回一个 conversation&#95;id 字段，其中的值表示该回复链中最早发布的帖子，帮助你跟踪会话。                                                                                                                                                                                                                                                                                                                     |
| 帖子 JSON 数据格式                                                                                                        | [Standard v1.1 数据格式](https://developer.x.com/en/docs/twitter-api/v1/data-dictionary/overview.html)                                                     | [X API v2](/zh/x-api/fundamentals/data-dictionary) 格式（由 fields 和 expansions 请求参数决定，与 v1.1 格式不向后兼容）<br /><br />要了解如何从 Standard v1.1 格式迁移到 X API v2 格式，请参阅我们的[数据格式迁移指南](/zh/x-api/migrate/data-format-migration)。                                                                                                                                                                   |
| 结果排序                                                                                                                | 按时间倒序（最新在前）                                                                                                                                            | 按时间倒序                                                                                                                                                                                                                                                                                                                                                                       |
| 结果分页方式                                                                                                              | 不适用；必须通过帖子 id 进行跳转                                                                                                                                     | 可以使用 `pagination_token` 向前或向后翻页查看结果                                                                                                                                                                                                                                                                                                                                         |
| 需要使用隶属于某个 [Project](/zh/resources/fundamentals/projects) 的 [developer App](/zh/resources/fundamentals/developer-apps) 的凭证 |                                                                                                                                                        | ✔                                                                                                                                                                                                                                                                                                                                                                           |
