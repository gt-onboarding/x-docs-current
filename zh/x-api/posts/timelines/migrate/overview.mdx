<div id="comparing-x-apis-timelines-endpoints">
  ## 比较 X API 的时间线端点
</div>

v2 的倒序时间线、用户帖子时间线和用户提及时间线端点分别替代了 [v1.1 statuses/home&#95;timeine,](https://developer.x.com/en/docs/twitter-api/v1/tweets/timelines/api-reference/get-statuses-home_timeline) [v1.1 statuses/user&#95;timeline](https://developer.x.com/en/docs/twitter-api/v1/tweets/timelines/api-reference/get-statuses-user_timeline.html) 和 [v1.1 statuses/mentions&#95;timeline](https://developer.x.com/en/docs/twitter-api/v1/tweets/timelines/api-reference/get-statuses-mentions_timeline.html) 端点。如果您有代码、应用或工具使用的是该端点的旧版本，并且正考虑迁移到较新的 X API v2 端点，那么本指南适合您。若需要更深入的迁移指南，请参阅 [Standard v1.1 migration to X API v2](/zh/x-api/posts/timelines/migrate/standard-to-twitter-api-v2)。

本页包含三张对比表：

* 倒序首页时间线
* 用户帖子时间线
* 用户提及时间线

<div id="reverse-chronological-home-timeline">
  ### 按时间倒序的主页时间线
</div>

| 支持的 HTTP 方法                                                                                                         | `GET`                                                                                                                                                  | `GET`                                                                                                                                                                                                                                                                                                                                                                     |
| 主机域名                                                                                                                | `https://api.x.com`                                                                                                                                    | `https://api.x.com`                                                                                                                                                                                                                                                                                                                                                       |
| 端点路径                                                                                                                | `/1.1/statuses/home_timeline.json`                                                                                                                     | `/2/users/:id/timelines/reverse_chronological`                                                                                                                                                                                                                                                                                                                            |
| 必需参数                                                                                                                | `user_id` or `screen_name`                                                                                                                             | 将用户 ID 设置为路径参数 :id                                                                                                                                                                                                                                                                                                                                                        |
| 身份验证                                                                                                                | OAuth 1.0a 用户上下文                                                                                                                                       | OAuth 1.0a 用户上下文<br /><br />[采用 PKCE 的 OAuth 2.0 授权码流程](/zh/resources/fundamentals/authenticationoauth-2-0/authorization-code)                                                                                                                                                                                                                                               |

| 每个响应的最大帖子数                                                                                                          | 800                                                                                                                                                    | 此端点会返回时间线上过去 7 天内创建的所有帖子，以及不论创建日期的最近 800 条帖子。                                                                                                                                                                                                                                                                                                                             |
| 提供帖子编辑历史                                                                                                            | ✔                                                                                                                                                      | ✔                                                                                                                                                                                                                                                                                                                                                                         |
| 可获取的历史帖子                                                                                                            | 最新的 800 条帖子（含转发）                                                                                                                                       | 最近 3,200 条帖子（包括转发）                                                                                                                                                                                                                                                                                                                                                        |
| 时间线导航方式                                                                                                             | since&#95;id（不包含）用于轮询获取更新<br /><br />`max_id`（包含）                                                                                                      | `start_time`<br /><br />end&#95;time<br /><br />`since_id`（不含该 id）用于轮询获取更新 <br /><br />`until_id`（不含该 id）                                                                                                                                                                                                                                                                 |
| 用于优化结果的可选参数                                                                                                         | `count`<br /><br />`exclude_replies`<br /><br />`include_rts`<br /><br />`trim_user`<br /><br />`tweet_mode`<br /><br />`since_id`<br /><br />`max_id` | `max_results`<br /><br />`exclude`(retweets,replies)<br /><br />`tweet.fields`<br /><br />`user.fields`<br /><br />`place.fields`<br /><br />`media.fields`<br /><br />`poll.fields`<br /><br />`expansions`<br /><br />`start_time`<br /><br />`end_time`<br /><br />`since_id`<br /><br />`until_id`                                                                    |
| 支持请求并接收[标注](/zh/x-api/fundamentals/post-annotations)                                                                   | 不支持                                                                                                                                                    | 如果在 tweet.fields 中包含注解，返回结果会基于帖子文本带有推断出的注解数据，例如 &#39;Music Genre&#39; 和 &#39;Folk Music&#39;，或 &#39;Musician&#39; 和 &#39;Dolly Parton&#39;                                                                                                                                                                                                                                |

| 帖子 JSON 数据格式                                                                                                        | [Standard v1.1 数据格式](https://developer.x.com/en/docs/twitter-api/v1/data-dictionary/overview.html)                                                     | [X API v2](/zh/x-api/fundamentals/data-dictionary) 格式（由 fields 和 expansions 请求参数决定，与 v1.1 格式不向后兼容）<br /><br />若要了解如何从 Standard v1.1 格式迁移到 X API v2 格式，请访问我们的[数据格式迁移指南](/zh/x-api/migrate/data-format-migration)。                                                                                                                                                                |
| 结果顺序                                                                                                                | 按时间倒序                                                                                                                                                  | 按时间倒序                                                                                                                                                                                                                                                                                                                                                                     |
| 结果分页方式                                                                                                              | 不适用，必须通过帖子 id 进行导航                                                                                                                                     | 可以使用 pagination&#95;token 向前或向后浏览结果                                                                                                                                                                                                                                                                                                                                       |
| 需要使用与某个 [Project](/zh/resources/fundamentals/projects) 关联的 [developer App](/zh/resources/fundamentals/developer-apps) 的凭证 |                                                                                                                                                        | ✔                                                                                                                                                                                                                                                                                                                                                                         |
