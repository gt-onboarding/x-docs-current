“Reposts of Me” 查询可让你查看哪些自己的帖子被其他用户转发或分享。

它能帮助你洞察哪些帖子正在被分享并引起受众共鸣，从而开展更有针对性的互动并相应调整内容策略。

<div id="authentication">
  #### 身份验证
</div>

由于你是在代表某个用户发起请求，因此必须使用 [OAuth 1.0a 用户上下文](/zh/resources/fundamentals/authentication) 或 [OAuth 2.0 Authorization Code with PKCE](/zh/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2) 对这些端点进行身份验证，并使用与你所代表的该用户关联的用户访问令牌。你可以通过 [三方 OAuth 流程](https://developer.x.com/resources/fundamentals/authentication/obtaining-user-access-tokens)（OAuth 1.0a）或 [Authorization Code with PKCE 授权流程](/zh/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-3)（OAuth 2.0）生成此用户访问令牌。

<div id="making-a-request">
  #### 发起请求
</div>

当用户在你的应用中完成身份验证后，你就可以代表该用户调用 Retweets of Me API，如下所示：

```bash
      curl 'https://api.x.com/2/users/reposts_of_me' --header 'Authorization: ••••••'
```
