<div id="authentication">
  #### 认证
</div>

所有 X API v2 端点都要求你使用一组凭据（也称为密钥和令牌）对请求进行认证。

你可以使用 OAuth 1.0a 用户上下文或 OAuth 2.0 Bearer Token 来对发往 Retweets 查询端点的请求进行认证。 

管理 Retweets 的端点需要使用 OAuth 1.0a 用户上下文，这意味着你必须使用一组 API 密钥和用户访问令牌才能成功发起请求。访问令牌必须与你代表其发起请求的用户关联。如果你想为另一位用户生成一组访问令牌，该用户必须通过三方 OAuth 流程授权或认证你的 App。

请注意，OAuth 1.0a 使用起来可能比较棘手。如果你对这种认证方式不熟悉，建议你使用[库](/zh/x-api/tools-and-libraries/overview)来正确地对请求进行认证。

**请注意**

如果你要请求以下字段，则需要使用 OAuth 1.0a 用户上下文： 

* tweet.fields.non&#95;public&#95;metrics
* tweet.fields.promoted&#95;metrics
* tweet.fields.organic&#95;metrics
* media.fields.non&#95;public&#95;metrics
* media.fields.promoted&#95;metrics
* media.fields.organic&#95;metrics

<div id="developer-portal-projects-and-developer-apps">
  #### 开发者门户、项目和开发者应用
</div>

要获取可用于 X API v2 端点的一组认证凭证，你必须先[注册开发者账号](https://developer.x.com/en/portal/petition/essential/basic-info)，在该账号中设置一个[项目](/zh/resources/fundamentals/projects)，并在该项目中创建一个[开发者应用](/zh/resources/fundamentals/developer-apps)。然后，你可以在该开发者应用中找到你的密钥和令牌。 
 

<div id="rate-limits">
  #### 速率限制
</div>

每天都有成千上万的开发者向 X API 发起请求。为了管理如此庞大的请求量，我们在每个端点上都设置了[速率限制](/zh/x-api/fundamentals/rate-limits)，以限制你代表你的应用或代表已认证用户所能发起的请求次数。

用于管理转发（Retweet）的端点限制为每位用户每 15 分钟最多 50 次请求。此外，对于 POST 端点，你在每个 3 小时时间窗口内（每用户、每应用）最多可以发起 300 次请求。 

使用 Retweets 查询（lookup）端点时，你在每个 15 分钟窗口内最多可以发起 75 次请求。此外，该端点只会返回最近的 100 位转发用户。

<div id="fields-and-expansions">
  #### 字段与扩展字段
</div>

X API v2 允许用户使用一组称为 `fields` 与 `expansions` 的工具，精确选择他们希望从 API 返回的哪些数据。`expansions` 参数允许你展开在有效负载中被引用的对象。例如，此端点允许你获取以下[扩展字段](/zh/x-api/fundamentals/expansions)：

* attachments.poll&#95;ids
* attachments.media&#95;keys
* author&#95;id, entities.mentions.username
* geo.place&#95;id
* in&#95;reply&#95;to&#95;user&#95;id,
* referenced&#95;tweets.id,
* referenced&#95;tweets.id.author&#95;id

`fields` 参数允许你在不同的数据对象中精确选择你希望接收的哪些[字段](/zh/x-api/fundamentals/fields)。这些端点主要返回帖子对象。默认情况下，帖子对象仅返回 `id` 和 `text` 字段。若要接收诸如 `tweet.created_at` 或 `tweet.entities` 等其他字段，你必须通过 `fields` 参数显式请求这些字段。在集成中你可能会考虑使用的一些重要字段包括投票数据、指标、帖子注释以及会话 ID 字段。

我们在 [X API v2 数据字典](/zh/x-api/fundamentals/data-dictionary)中添加了一篇关于[如何同时使用字段与扩展字段](/zh/x-api/fundamentals/data-dictionary#how-to-use-fields-and-expansions)的指南。
