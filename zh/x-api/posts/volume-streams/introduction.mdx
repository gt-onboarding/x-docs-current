* 1% 抽样流。
* 10% 抽样流，通常称为 “Decahose”。

这些被称为 “volume streams（高流量流）”，因为它们都会传送大量数据。即使是 1% 流，每秒也可以输出数十条帖子。通过这些流，你可以识别和追踪趋势、监控整体情绪、监控全球事件等。

这些流式端点通过持久的 HTTP GET 连接传送 [Post objects](/zh/x-api/fundamentals/data-dictionary#tweet)，并使用 [OAuth 2.0 App-Only](/zh/resources/fundamentals/authentication#oauth-2-0) 认证。在 Essential 访问权限级别下，你一次可以拥有一个连接。在所有访问级别下，每 15 分钟窗口内最多可以发起 50 次连接请求。

这些 volume stream 端点支持已编辑的帖子。这些端点会传送已编辑的帖子及其编辑历史记录，其中包括一个帖子 ID 数组。对于没有编辑历史的帖子，该数组只包含一个 ID。对于已被编辑的帖子，该数组包含多个 ID，按编辑顺序升序排列，数组最后一个元素为最新版本。要了解帖子编辑的工作方式的更多信息，请参阅 [Post edits fundamentals](/zh/x-api/fundamentals/edit-posts) 页面。

*要使用这些 API，你必须先通过我们的 Enterprise 销售团队开通账号。*
