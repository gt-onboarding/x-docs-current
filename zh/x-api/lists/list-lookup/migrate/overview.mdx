  ## 比较 X API 的列表查找端点
</div>

v2 列表查找端点组将取代标准 v1.1 的 [GET lists/show](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/get-lists-show) 和 [GET lists/ownership](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/get-lists-ownerships) 端点。如果你已有代码、应用或工具在使用这些版本的列表查找端点，并且正在考虑迁移到更新的 X API v2 端点，那么本系列指南适合你参考。

下列表格对比了标准 v1.1 和 X API v2 的列表端点：

**按 ID 查找列表**

|     |     |     |
| :--- | :--- | :--- |
| Description | Standard v1.1 | X API v2 |
| HTTP methods supported | `GET` | `GET` |
| Host domain | `https://api.x.com` | `https://api.x.com` |
| Endpoint path | `/1.1/lists/show.json` | `/2/lists/:id` |
| [Authentication](/zh/resources/fundamentals/authentication) | OAuth 1.0a 用户上下文<br /><br />仅应用 | OAuth 1.0a 用户上下文<br /><br />OAuth 2.0 Authorization Code with PKCE<br /><br />仅应用 |
| Default request [rate limits](/zh/resources/fundamentals/rate-limits) | 使用 OAuth 1.0a 时每 15 分钟 75 次请求<br /><br />使用 OAuth 2.0 时每 15 分钟 75 次请求<br /><br />仅应用时每 15 分钟 75 次请求 | 使用 OAuth 1.0a 时每 15 分钟 75 次请求<br /><br />使用 OAuth 2.0 时每 15 分钟 75 次请求<br /><br />仅应用时每 15 分钟 75 次请求 |

**用户拥有的列表查找**

|     |     |     |
| :--- | :--- | :--- |
| Description | Standard v1.1 | X API v2 |
| HTTP methods supported | `GET` | `GET` |
| Host domain | `https://api.x.com` | `https://api.x.com` |
| Endpoint path | `/1.1/lists/ownerships.json` | `/2/users/:id/owned_lists` |
| [Authentication](/zh/resources/fundamentals/authentication) | OAuth 1.0a 用户上下文<br /><br />仅应用 | OAuth 1.0a 用户上下文<br /><br />OAuth 2.0 Authorization Code with PKCE<br /><br />仅应用 |
| Default request [rate limits](/zh/resources/fundamentals/rate-limits) | 使用 OAuth 1.0a 时每 15 分钟 15 次请求<br /><br />仅应用时每 15 分钟 15 次请求 | 使用 OAuth 1.0a 时每 15 分钟 15 次请求<br /><br />使用 OAuth 2.0 时每 15 分钟 15 次请求<br /><br />仅应用时每 15 分钟 15 次请求 |

要访问 X API v2 端点，你必须先[注册开发者账号](https://developer.x.com/en/portal/petition/essential/basic-info)。进行认证时，你必须使用某个[项目](/zh/resources/fundamentals/projects)中[开发者应用](/zh/resources/fundamentals/developer-apps)的密钥和令牌。

请在我们的[入门](/zh/x-api/getting-started/getting-access)页面了解更多关于获取 X API v2 端点访问权限的内容。
