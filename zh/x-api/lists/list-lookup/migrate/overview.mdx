<div id="comparing-x-apis-list-lookup-endpoints">
  ## 比较 X API 的 List 查找端点
</div>

X API v2 的 List 查找端点组将取代 v1.1 标准版的 [GET lists/show](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/get-lists-show) 和 [GET lists/ownership](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/get-lists-ownerships) 端点。如果你有使用上述任一版本 List 查找端点的代码、应用或工具，并且正在考虑迁移到新的 X API v2 端点，那么这一组指南就是为你准备的。

下表对比了 v1.1 标准版和 X API v2 的 List 端点：

**根据 ID 查找 List**

|     |     |     |
| :--- | :--- | :--- |
| Description | Standard v1.1 | X API v2 |
| HTTP methods supported | `GET` | `GET` |
| Host domain | `https://api.x.com` | `https://api.x.com` |
| Endpoint path | `/1.1/lists/show.json` | `/2/lists/:id` |
| [Authentication](/zh/resources/fundamentals/authentication) | OAuth 1.0a 用户上下文<br /><br />仅应用 | OAuth 1.0a 用户上下文<br /><br />OAuth 2.0 Authorization Code with PKCE<br /><br />仅应用 |
| Default request [rate limits](/zh/resources/fundamentals/rate-limits) | 使用 OAuth 1.0a 时每 15 分钟 75 次请求<br /><br />使用 OAuth 2.0 时每 15 分钟 75 次请求<br /><br />仅应用模式下每 15 分钟 75 次请求 | 使用 OAuth 1.0a 时每 15 分钟 75 次请求<br /><br />使用 OAuth 2.0 时每 15 分钟 75 次请求<br /><br />仅应用模式下每 15 分钟 75 次请求 |

**查找用户拥有的 List**

|     |     |     |
| :--- | :--- | :--- |
| Description | Standard v1.1 | X API v2 |
| HTTP methods supported | `GET` | `GET` |
| Host domain | `https://api.x.com` | `https://api.x.com` |
| Endpoint path | `/1.1/lists/ownerships.json` | `/2/users/:id/owned_lists` |
| [Authentication](/zh/resources/fundamentals/authentication) | OAuth 1.0a 用户上下文<br /><br />仅应用 | OAuth 1.0a 用户上下文<br /><br />OAuth 2.0 Authorization Code with PKCE<br /><br />仅应用 |
| Default request [rate limits](/zh/resources/fundamentals/rate-limits) | 使用 OAuth 1.0a 时每 15 分钟 15 次请求<br /><br />仅应用模式下每 15 分钟 15 次请求 | 使用 OAuth 1.0a 时每 15 分钟 15 次请求<br /><br />使用 OAuth 2.0 时每 15 分钟 15 次请求<br /><br />仅应用模式下每 15 分钟 15 次请求 |

要访问 X API v2 端点，你必须先[注册开发者账号](https://developer.x.com/en/portal/petition/essential/basic-info)。在进行认证时，你必须使用位于某个[项目](/zh/resources/fundamentals/projects)中的[开发者应用](/zh/resources/fundamentals/developer-apps)所提供的密钥和令牌。

在我们的[入门](/zh/x-api/getting-started/getting-access)页面中了解更多关于获取 X API v2 端点访问权限的信息。
