X API 可用于以编程方式检索和分析 X 数据，并构建围绕 X 上对话的应用。

多年来，X API 通过为开发者提供更多访问级别不断发展，使他们能够扩大访问规模，以增强和研究公共对话。

最近，我们发布了 X API v2。X API v2 具备现代化的基础架构、全新的高级功能，并支持快速获取 [Basic access](https://developer.x.com/en/portal/products/basic) 访问权限。

以下三个选项卡介绍了 X API 的不同版本和访问级别、v2 的新功能，以及你可以使用该 API 检索、创建、删除和修改的 X 资源。

  <Tab title="v2 有哪些新内容">
    ## v2 有哪些新变化

    X API v2 是自 2012 年以来对 X API 最大的一次升级。它带来了大量全新和高级功能，并提供快速且免费的 API 访问。

    v2 中提供的一些功能包括：

    ### 新的端点

    我们已经为 X API v2 发布了一组全新的端点。你可以在我们的 X API 端点映射指南中查看 v2 端点的完整列表，包括其中新增的端点。

    [访问 X API 端点映射 &gt;](/zh/x-api/migrate/x-api-endpoint-map)

    ### 全新且更详尽的数据对象

    我们通过多项改进对数据对象进行了现代化升级，使你能够更轻松地浏览和解析数据。

    [访问数据字典 &gt;](/zh/x-api/fundamentals/data-dictionary)

    ### 新的参数，帮助你仅检索所需的对象和字段

    我们在数据端点中添加了 `fields` 和 `expansions` 参数，让你可以请求默认返回字段之外的关联对象和字段。

    [了解如何使用 fields 和 expansions &gt;](/zh/x-api/fundamentals/data-dictionary#how-to-use-fields-and-expansions)

    ### 高级指标

    通过请求包括展示次数、视频观看次数、用户主页访问和 URL 点击在内的公共和私有指标（其中一些还区分自然与推广场景），你可以直接在响应负载中更轻松地了解帖子、用户、媒体和投票的表现。

    [进一步了解指标 &gt;](/zh/x-api/fundamentals/metrics)

    ### 筛选并识别包含不同主题的帖子

    在使用搜索帖子或过滤流时，你现在可以使用我们的实体和上下文运算符按主题进行过滤。我们还在帖子负载中提供了这些主题，以帮助你进行分析。

    [进一步了解 Post 注解 &gt;](/zh/x-api/fundamentals/post-annotations)

    ### 筛选并识别属于回复线程的帖子

    在使用搜索帖子、过滤流和帖子查询时，可以更轻松地识别某个帖子是否属于会话线程的一部分。我们还新增了能力，可以通过 Post 字段 `reply_settings` 判断帖子是否设置了会话回复选项。

    [进一步了解会话追踪 &gt;](/zh/x-api/fundamentals/conversation-id)

    ### 远不止于此……

    * 高置信度的垃圾信息过滤
    * 短链接将被完全展开，便于进行 URL 分析
    * 通过移除已弃用字段并现代化标签来简化 JSON 响应对象
    * 为我们的流式端点提供恢复和冗余功能
    * 在搜索查询中返回 100% 匹配的公开且可用的帖子
    * 流式“规则”，让你无需断开连接即可进行更改
    * 为过滤流和搜索提供更具表达力的查询语言
    * 使用 OpenAPI 规范构建新库并更透明地跟踪变更
    * 随着我们的平台不断发展以满足开发者、研究人员、企业和使用 X 的用户的需求，更快速地为新功能和端点提供 API 支持
  </Tab>

  <Tab title="X API 资源">
    ## X API 平台资源

    在 API 设计中，资源是一个实体，拥有相关数据、与其他资源的关系，以及一组可对其进行操作的方法。比如，帖子就是一种资源，你可以使用多种不同的工具来创建、删除或检索它，例如在历史数据中搜索它们，或实时获取它们。

    X API 允许你在平台上创建、删除、接收或调整多种不同的资源，包括：

    | Resource            | Description                                                                         |
    | :------------------ | :---------------------------------------------------------------------------------- |
    | **Posts**           | 利用数以百万计的帖子来洞察公共对话，或创建你自己的帖子参与其中。 当前可用性： - X API v2  - Enterprise  - Standard v1.1   |
    | **Users**           | 管理或查询 X 用户，以分析网络、了解你的受众或促进积极的在线关系。 当前可用性： - X API v2  - Enterprise  - Standard v1.1 |
    | **Spaces**          | 查询和搜索 X Spaces 及其参与者，帮助人们发现有趣且相关的音频对话。 当前可用性： - X API v2                            |
    | **Direct Messages** | 发送和接收私信，以分类处理客户问题、发送欢迎消息或营造积极的人际互动。 当前可用性： - Standard v1.1                          |
    | **Lists**           | 策划和管理账号列表，以密切关注行业专家、有影响力的声音，或更好地组织你关注的账号。 当前可用性： - X API v2  - Standard v1.1        |
    | **Trends**          | 率先识别地域趋势，以定位行业动向、发现热门话题或抢先掌握最新潮流。 当前可用性： - Standard v1.1                            |
    | **Media**           | 上传媒体对象，分享你的创意灵感、打造交互式体验或构建无障碍工具。 当前可用性： - Standard v1.1                             |
    | **Places**          | 搜索地点，以了解你所在社区及全球范围内正在发生的事情。 当前可用性： - Standard v1.1                                  |
  </Tab>
