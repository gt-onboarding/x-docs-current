X API 可用于以编程方式获取和分析 X 数据，并围绕 X 上的对话进行构建和开发。

多年来，X API 通过为开发者增加更多访问级别不断拓展，使他们能够按需扩展访问规模，以更好地丰富和研究公共对话。

我们最近发布了 X API v2。X API v2 提供了现代化的基础、全新且更高级的功能，以及可快速上手的 [Basic 访问权限](https://developer.x.com/en/portal/products/basic)。

下面的三个选项卡介绍了 X API 的不同版本和访问级别、v2 的新特性，以及你可以使用该 API 检索、创建、删除和修改的 X 资源。

  <Tab title="v2 有哪些新变化">
    ## v2 有哪些新变化

    X API v2 是自 2012 年以来 X API 最大的一次升级。它带来了大量全新的高级功能，并提供快速且免费的 API 访问能力。

    部分在 v2 中可用的功能包括：

    ### 全新的 endpoints

    我们为 X API v2 发布了一组全新的 endpoints。你可以在我们的 X API endpoint 映射指南中查看包括这些新接口在内的完整 v2 endpoints 列表。

    [访问 X API endpoint 映射 &gt;](/zh/x-api/migrate/x-api-endpoint-map)

    ### 更新且更加详细的数据对象

    我们对数据对象进行了现代化改造，引入了多项新改进，帮助你更轻松地浏览和解析数据。

    [访问数据字典 &gt;](/zh/x-api/fundamentals/data-dictionary)

    ### 新增参数，帮助你仅检索所需的对象和字段

    我们在数据 endpoints 中添加了 `fields` 和 `expansions` 参数，使你可以请求默认返回字段之外的相关对象和字段。

    [了解如何使用 fields 和 expansions &gt;](/zh/x-api/fundamentals/data-dictionary#how-to-use-fields-and-expansions)

    ### 高级指标

    通过在 payload 中直接请求包括展示次数（impressions）、视频观看数（video views）、用户主页访问和 URL 点击在内的公开和私有指标（其中部分指标按自然和推广两种场景区分），你可以更轻松地了解帖子、用户、媒体和投票的表现。

    [了解更多有关指标的信息 &gt;](/zh/x-api/fundamentals/metrics)

    ### 按主题过滤并识别包含不同主题的帖子

    使用搜索帖子或过滤流时，你现在可以借助实体与上下文运算符按主题进行过滤。我们还在帖子 payload 中提供了这些主题，便于你进行分析。

    [了解更多关于 Post 注释的信息 &gt;](/zh/x-api/fundamentals/post-annotations)

    ### 针对回复线程过滤并识别属于该线程的帖子

    在使用搜索帖子、过滤流和帖子查询时，可以更容易地识别某个帖子是否属于会话线程的一部分。我们还新增了能力，可以通过 Post 字段 `reply_settings` 判断某个帖子是否设置了会话回复选项。

    [了解更多关于会话跟踪的信息 &gt;](/zh/x-api/fundamentals/conversation-id)

    ### 以及更多功能...

    * 高置信度的垃圾内容过滤
    * 缩短链接会被完全展开，便于 URL 分析
    * 通过移除已弃用字段并更新标签来简化 JSON 响应对象
    * 为我们的流式 endpoints 提供恢复与冗余功能
    * 在搜索查询中返回 100% 符合条件且可用的公开帖子
    * 流式 &quot;rules&quot;，让你无需断开连接即可修改规则
    * 针对过滤流和搜索提供更具表现力的查询语言
    * 使用 OpenAPI 规范构建新库并更透明地跟踪变更
    * 随着平台不断演进以满足开发者、研究人员、企业以及 X 用户的需求，更快速地为新功能和 endpoints 提供 API 支持
  </Tab>

  <Tab title="X API 资源">
    ## X API 平台资源

    在 API 设计领域中，资源是指具有相关数据、与其他资源的关系，以及一组可在其上执行的方法的实体。例如，一个 Post 就是一种资源，你可以通过多种不同的工具来创建、删除或检索它，例如在历史数据中搜索这些 Post，或者实时获取它们。

    X API 提供能力，使你能够在平台上创建、删除、接收或调整多种不同的资源，包括以下内容：

    | Resource            | Description                                                                         |
    | :------------------ | :---------------------------------------------------------------------------------- |
    | **Posts**           | 利用数以百万计的帖子来了解公共对话，或创建你自己的帖子参与其中。当前可用性：  - X API v2  - Enterprise  - Standard v1.1   |
    | **Users**           | 管理或查询 X 用户，以分析网络、了解你的受众或促进积极的线上关系。当前可用性：  - X API v2  - Enterprise  - Standard v1.1 |
    | **Spaces**          | 查询和搜索 X Spaces 及其参与者，帮助人们发现有趣且相关的音频对话。当前可用性：  - X API v2                            |
    | **Direct Messages** | 发送和接收 Direct Messages，用于处理客户问题、发送欢迎消息或创造积极的人际互动。当前可用性：  - Standard v1.1             |
    | **Lists**           | 策划和管理账号列表，以便紧跟行业专家、关键声音，或整理你所关注的对象。当前可用性：  - X API v2  - Standard v1.1              |
    | **Trends**          | 率先识别地理趋势，以定位行业动向、发现热门话题或抢先掌握最新潮流。当前可用性：  - Standard v1.1                            |
    | **Media**           | 上传媒体对象，以分享你的创意、创建交互式体验或构建无障碍工具。当前可用性：  - Standard v1.1                              |
    | **Places**          | 搜索地点，以了解在你所在社区以及世界各地正在发生的事情。当前可用性：  - Standard v1.1                                 |
  </Tab>
