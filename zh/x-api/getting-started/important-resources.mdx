<div id="learn-about-whats-possible">
  ## 了解平台能实现什么
</div>

从顶部导航可以访问的用例、解决方案和产品页面，是获取平台整体能力高层概览的绝佳途径。

我们还编写了多篇[教程](/zh/tutorials)，帮助你了解如何使用一个或多个平台工具来满足各类不同的用例需求。

在我们的文档体系中，最底层是端点和工具文档，这些文档都包含介绍性内容，用于简要概述该工具的功能和典型用法。例如，这里是我们的 X API v2 中用于[搜索帖子](/zh/x-api/posts/search/introduction)的端点部分。

<div id="integrate-an-endpoint-into-your-system">
  ### 将端点集成到你的系统中
</div>

为了让你更顺畅地集成我们的端点，我们提供了一套工具、库以及集成指南。最简单的集成方式是使用我们的[工具和库](/zh/x-api/tools-and-libraries/overview)，它们可以自动处理复杂功能，例如 [OAuth 1.0a User Context authentication](/zh/resources/fundamentals/authentication#oauth-1-0a-2)、分页以及[速率限制处理](/zh/resources/fundamentals/rate-limits)。

如果你更倾向于从头开始构建解决方案，我们的端点集成指南会介绍将我们的端点集成到你的系统时涉及的功能和最佳实践。这些指南可以在不同的端点和工具文档中找到，通常标记为“integrate”或“guides”。例如，这里有一篇关于[如何为 Filter stream 构建规则](/zh/x-api/posts/filtered-stream/integrate/build-a-rule)的集成指南。

<div id="troubleshoot-an-issue">
  ### 排查问题
</div>

如果你遇到错误或有任何疑问，请访问我们的[支持中心](https://developer.x.com/en/support/twitter-api)。在这里，你可以找到故障排查提示、常见问题解答及其他实用资源。

<div id="keep-up-to-date-with-the-latest-releases">
  ### 及时了解最新版本发布情况
</div>

我们经常发布平台更新，通过 API 解锁全新的 X 功能。除此之外，我们的目标是每年至少发布一次新的主版本升级（major version bump），其中会包含一些不兼容变更（breaking changes）。

要及时获知最新更新，请确保订阅我们 [论坛的 Announcements 分类](https://devcommunity.x.com/c/announcements/22)，在 X 上关注 [@API](https://x.com/api) 和 [@XDevelopers](https://x.com/XDevelopers) 并开启通知，并通过我们的 [Stay informed](https://developer.x.com/en/stay-informed) 页面查看各类资源。

此外，我们还构建了全新的 [迁移中心](/zh/x-api/migrate/overview)，帮助你完成升级到最新 API 版本的整个流程。
