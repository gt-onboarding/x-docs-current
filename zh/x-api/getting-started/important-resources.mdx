  ## 了解平台的可能性
</div>

顶部导航栏中可访问的用例、解决方案和产品页面，是从高层次了解平台能做什么的绝佳途径。

我们还编写了多篇[教程](/zh/tutorials)，帮助你确定如何使用一个或多个不同的平台工具来满足各种用例需求。

在我们文档的最底层，是各个端点和工具的文档，每篇文档都包含介绍性内容，简要概述该工具的功能和典型用例。例如，这里是我们的 X API v2 端点文档中用于[搜索帖子](/zh/x-api/posts/search/introduction)的部分。

<div id="integrate-an-endpoint-into-your-system">
  ### 将端点集成到你的系统中
</div>

为了让你与我们端点的集成更加顺利，我们构建了一系列工具、库和集成指南。集成我们端点的最简单方式是使用我们的[工具和库](/zh/x-api/tools-and-libraries/overview)，它们会自动处理诸如 [OAuth 1.0a 用户上下文身份验证](/zh/resources/fundamentals/authentication#oauth-1-0a-2)、分页以及[速率限制处理](/zh/resources/fundamentals/rate-limits)等复杂功能。

如果你更倾向于从零开始构建解决方案，我们的端点集成指南会介绍如何将我们的端点集成到你的系统中所涉及的功能和最佳实践。这些指南可以在不同的端点和工具文档中找到，通常标为 “integrate” 或简写为 “guides”。例如，这里有一篇关于[如何为 Filter stream 构建规则](/zh/x-api/posts/filtered-stream/integrate/build-a-rule)的集成指南。

<div id="troubleshoot-an-issue">
  ### 排查问题
</div>

如果你遇到错误或有任何疑问，请访问我们的[支持中心](https://developer.x.com/en/support/twitter-api)。在这里，你可以找到问题排查技巧、常见问题解答以及其他实用资源。

<div id="keep-up-to-date-with-the-latest-releases">
  ### 及时掌握最新版本发布情况
</div>

我们会经常发布平台更新，通过 API 解锁全新的 X 功能。除此之外，我们的目标是每年至少发布一个新的主版本更新，其中会包含一些不兼容变更。

若要获取最新动态，请务必订阅我们[论坛的 Announcements 公告板块](https://devcommunity.x.com/c/announcements/22)，在 X 上关注 [@API](https://x.com/api) 和 [@XDevelopers](https://x.com/XDevelopers) 并开启通知，同时通过我们的 [stay informed](https://developer.x.com/en/stay-informed) 页面查阅各类资源。

此外，我们还推出了全新的 [migration hub](/zh/x-api/migrate/overview)，帮助指导你完成更新到最新 API 版本的整个过程。
