  ## 官方资源
</div>

下面这些工具可以帮助你开始在网站上使用 X。

<CardGroup cols={3}>
<Card
  title="publish.x.com"
  icon="x-twitter"
  href="https://publish.x.com/"
>
创建你自己的嵌入式帖子、时间线或按钮。
</Card>
<Card
  title="Twitter for WordPress"
  icon="x-twitter"
  href="https://github.com/twitter/wordpress"
>
官方插件，用于将你的 WordPress 站点优化为适配 X，包括帖子嵌入和 X 卡片。
</Card>
<Card
  title="twemoji"
  icon="x-twitter"
  href="https://twemoji.x.com/"
>
X 提供的免费、开源 emoji 字符集，包含用于跨平台支持的 JavaScript 库。
</Card>
</CardGroup>
