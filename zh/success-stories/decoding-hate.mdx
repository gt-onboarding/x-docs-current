<div id="situation">
  ## 情况
</div>

在新冠疫情爆发之后，针对亚裔美国人和太平洋岛民（AAPI）的仇恨事件不断增加。虽然其中一些事件见诸报端，但仍有成千上万的人向全美性的举报中心 [Stop AAPI Hate](https://stopaapihate.org/) 进行报告。该中心通过研究、倡导和能力建设来回应并处理仇恨事件。就在这一时期，[The Asian American Foundation（TAAF）](https://www.taaf.org/) 成立。该组织成为这一团队背后的核心推动力量，这个团队决定利用 X，提供一个全新的窗口，展示这些事件如何在个人层面影响被针对的人群。

<div id="solution">
  ## 解决方案
</div>

由两名工程师组成的小团队将 X 作为他们的事件数据来源。他们选择这个平台有两个原因：首先，它包含丰富多样的信息，比如新闻、人们的想法、观点、日常生活事件，以及他们所在社区正在发生的事情。正因如此，X 提供了来自不同人群的独特且贴近现实的样本。其次，X 也是唯一一个真正开放 API、能够让 TAAF 的开发者访问数据，并且提供强大工具来收集和分析这些数据的平台。 

利用 X API v2 的搜索帖子和帖子查找端点，团队可以搜索帖子并识别人们何时报告了仇恨事件（甚至能识别他们会用哪些标签）。这帮助团队设计出一个 1023 字符的搜索查询，将数十亿条帖子缩小到一个团队可以处理的规模。 

接下来就是验证这些帖子是否确实与仇恨事件相关这一任务。团队开发了一个使用自然语言处理的工具，帮助从成千上万条帖子中筛选出与项目相关的内容。在应用他们训练好的机器学习模型之后，这些帖子还会再由人工审核，以进一步提升准确性。 

2021 年，TAAF 还与 Stop AAPI Hate 联盟共同撰写了 [Documenting Anti-AAPI Hate Codebook](https://stopaapihate.org/wp-content/uploads/2021/12/SAH-x-TAAF-Codebook-FINAL-2021-08-19.pdf)。该资源包含了基于社区的数据收集标准和实践草案，帮助团队对人们在 X 上分享的事件进行分类。 

在这之后，下一个挑战是如何以一种人们容易理解的方式呈现这些数据。TAAF 创建了一个数据可视化工具 [Decoding Hate](https://decodinghate.taaf.org/)，以交互式的方式展示他们海量的数据。在一家数据可视化工作室的帮助下，他们将标注过的 X 数据转化为数千条帖子背后富有洞察力的故事，从而揭示 AAPI 社群中正在发生的一切真相。

<div id="impact">
  ## 影响
</div>

Decoding Hate 项目帮助发现了数以千计此前未被举报的仇恨相关事件，并且这一工作至今仍在持续。该项目已经成为一项持续性工具，用于提升公众对亚裔美国人和太平洋岛裔（AAPI）社区中个体经历的认知。

在下一阶段的工作中，TAAF 将与 Stop AAPI Hate 合作，试点为在 X 上举报仇恨事件的个人提供一对一的人工回应。在此阶段，Stop AAPI Hate 将主动与许多发布帖子讨论其遭遇仇恨经历的人互动，并鼓励他们通过 [Stop AAPI Hate Reporting Center](https://stopaapihate.org/) 举报这些事件。向 Stop AAPI Hate 举报的人越多，该联盟及其合作伙伴就能越好地了解正在发生的仇恨事件的性质，并为最有效的政策解决方案发声。 

除了揭示 AAPI 社区中正在发生的仇恨相关事件外，该项目也凸显了 X API 的价值。这个案例向我们展示，借助合适的数据和工具，开发者可以通过 X 为公益目标进行创新。
