<div id="situation">
  ## 情况
</div>

在新冠疫情爆发之后，针对亚裔美国人和太平洋岛裔（AAPI）的仇恨事件有所增加。虽然其中一些事件登上了新闻，但仍有成千上万的人向全国性举报中心 [Stop AAPI Hate](https://stopaapihate.org/) 提交报告。该机构通过研究、倡导和能力建设来回应和处理仇恨事件。在此期间，[亚裔美国人基金会（The American Asian Foundation，TAAF）](https://www.taaf.org/) 成立。该组织成为团队背后的推动力量，团队决定借助 X 提供一个新的窗口，展现这些事件如何在个人层面影响受害者。

<div id="solution">
  ## 解决方案
</div>

这支由两名工程师组成的小团队将 X 作为他们的事件数据来源。他们选择这个平台有两个原因：首先，它包含丰富多样的信息，如新闻、人们的观点、想法、日常生活事件以及他们所在社区发生的事情。正因如此，X 提供了一个独特而贴近生活的多元人群样本。X 也是唯一一个真正提供开放 API、让 TAAF 的开发者能够访问数据的平台，并且还提供强大的工具来收集和分析这些数据。 

通过使用 X API v2 的 search Tweets 和 Tweets lookup 端点，团队能够搜索帖子并识别出人们何时报告了仇恨事件（甚至能识别他们会使用哪些标签）。这帮助团队设计出一个 1023 字符的搜索查询，把数十亿条帖子缩小到一个团队可以处理的规模。 

接下来要完成的任务是验证这些帖子是否确实与仇恨事件相关。团队开发了一个使用自然语言处理的工具，帮助从成千上万条帖子中筛选出与项目相关的内容。在应用他们训练好的机器学习模型之后，这些帖子还会再由人工进行核查，以进一步提高准确性。 

2021 年，TAAF 还与 Stop AAPI Hate 联盟共同编写了 [Documenting Anti-AAPI Hate Codebook](https://stopaapihate.org/wp-content/uploads/2021/12/SAH-x-TAAF-Codebook-FINAL-2021-08-19.pdf)。该资源包含基于社区的数据收集草案标准和规范，帮助团队对人们在 X 上分享的事件进行分类。 

在此基础上，下一个挑战是如何以一种人们容易理解的方式呈现这些数据。TAAF 创建了一个数据可视化工具 [Decoding Hate](https://decodinghate.taaf.org/)，以交互的方式展示他们庞大的数据量。在一家数据可视化工作室的帮助下，他们将带注释的 X 数据转化为以成千上万条帖子为基础的富有洞察力的故事，从而揭示 AAPI 社群中实际发生情况的真相。

<div id="impact">
  ## 影响
</div>

Decoding Hate 项目帮助检测到了成千上万原本不会被举报的仇恨相关事件，并且至今仍在持续发挥作用。这个项目已经成为一项长期工具，用于提升人们对发生在 AAPI 群体中个体经历的认知和关注。

在下一阶段的工作中，TAAF 将与 Stop AAPI Hate 合作，在 X 上为举报仇恨事件的个人试点提供一对一的人工回应。在此阶段，Stop AAPI Hate 将主动与许多在帖子中讲述自己遭遇仇恨的人互动，并鼓励他们通过 [Stop AAPI Hate Reporting Center](https://stopaapihate.org/) 举报这些事件。向 Stop AAPI Hate 举报的人越多，该联盟及其合作伙伴就能越好地理解正在发生的仇恨事件的性质，并为最有效的政策解决方案进行倡导。 

除了揭示 AAPI 群体中正在发生的仇恨相关事件之外，该项目还凸显了 X 的 API 的价值。这个案例向我们展示，通过将合适的工具与合适的数据结合起来，开发者可以利用 X 为公益进行创新。
