<Note>
**要点速览**

下面介绍开发者兼创作者 Aaron Ng 如何使用 X 的 API 构建出 Flocknet——一个为你的 X 人脉网络打造的搜索引擎。人们正在使用 Flocknet 来拓展人脉、招募人才以及寻找投资线索。Flocknet 最初只是为他个人使用而构建，如今已经在帮助人们在 X 上管理数以百万计的联系人。
</Note>

<div id="challenge">
  ## 挑战
</div>

当 Aaron Ng 参加在纽约举行的一场设计大会时，他希望能和业界的其他人建立联系并当面交流。但这时他意识到，即便是在 X 上与自己的设计师朋友们对接，也比他想象的要困难得多。他的大部分圈子都在 X 上，尤其是行业里的朋友。难道就没有办法知道哪些朋友会来纽约吗？如果不知道谁会到附近来，他又该如何和行业内的人脉见面叙旧呢？

Aaron 立刻把这一挑战视为一个机会。也正是出于他的创业精神，Flocknet 应运而生。 

<div id="solution">
  ## 解决方案
</div>

仅仅用了一个周末，Aaron 就构建出了一款产品，用来在他的 X 网络中搜索、排序，并持续关注人们、朋友及其他专业人士，即便看不到他们的帖子也可以做到这一点。他选择在这个项目中使用 X 的 API，“因为我知道总有可以提问的人，而且[知道 X 的]团队很在乎这件事，这让整个体验好得多。”他还补充说：“促使我不断继续构建的社区，大多都在 X 上——这也是为什么它对我来说变得如此重要，以及我为什么想要做一些工具来提升我对它的使用体验。”

凭借他自身的创造力、动力和经验，Flocknet 因此诞生——一款用于在你的 X 网络中查找人的搜索引擎和联系人管理工具。借助 X 的关注关系（follows）端点和 Python，Flocknet 允许用户按照工作、公司、地点以及许多其他属性来搜索他们的关注者。

虽然最初只是为了他个人使用而打造，Flocknet 却引发了巨大的公众兴趣，让他意识到这款工具应该被分享出去。因此，他很快把精力转向扩展 Flocknet，使之发展成今天的模样。

<div id="impact">
  ## 影响
</div>

短短几个月内，Flocknet 就已经代表用户管理了数以百万计的联系人。技术从业者发现它格外有用，从招聘到投资，以及两者之间的各种场景都会用到它。事实上，就连 Aaron 自己也通过 Flocknet 招聘过员工！

但 Aaron 还没有打算就此止步。通过专注于源源不断的功能需求，Aaron 不断寻找扩展产品的方式。

最终，Aaron 找到了一种方法，把 X 打造成一个元平台，从而提升那些主要在 X 上运营社群的用户的使用体验。他说：“我就是会做很多东西。我热爱做那些我自己觉得有用、别人也觉得有用的东西。”而他也确实做到了这一点。
