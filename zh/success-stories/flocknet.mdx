<Note>
**要点如下**

以下是开发者兼创作者 Aaron Ng 如何使用 X 的 API 构建 Flocknet：一款为你的 X 人脉网络打造的搜索引擎。人们正在使用 Flocknet 来扩展人脉、招聘人才以及寻找投资线索。Flocknet 最初是他为自己打造的工具，如今已经帮助人们在 X 上管理数以百万计的联系人。
</Note>

<div id="challenge">
  ## 挑战
</div>

当 Aaron Ng 前往纽约参加一场设计大会时，他希望能和行业里的其他人建立联系并见面。但这时他意识到，哪怕只是想在 X 上和自己那些设计师朋友建立联系，也比他想象中更具挑战性。他所在的社群大多活跃在 X 上，尤其是他的行业好友。难道就没有办法让他看到有哪些朋友会来纽约吗？如果不知道谁会在附近，他又该如何与行业内的联系人叙旧、交流近况？

Aaron 立刻把这个难题视为一个机会。也正是由于他的创业精神，Flocknet 由此诞生。 

<div id="solution">
  ## 解决方案
</div>

仅仅用了一个周末，Aaron 就构建出了一个产品，可以在他的 X 网络中搜索、排序并持续关注身边的人、朋友及其他职场人士，即便看不到他们的帖子也不受影响。他选择为这个项目使用 X 的 API，“因为我知道有可以提问的人，而且知道 X 团队很在意这些问题，这让整个体验好得多。”他补充道：“促使我一直坚持做下去的社区，很大一部分就在 X 上——这也是为什么它对我来说变得如此重要，以及我为什么想做一些工具来提升自己在上面的使用体验。”

凭借自身的创意、干劲和经验，Flocknet 诞生了——一个帮助你在自己的 X 网络中找到人的搜索引擎和联系人管理工具。借助 X 的 follows 端点和 Python，Flocknet 让用户可以按照职位、公司、所在地以及许多其他属性来搜索他们的关注者。

虽然最初只是为他个人使用而构建，Flocknet 却引发了广泛关注，他因此意识到这个产品应该被分享出去。于是，他迅速把精力转向将 Flocknet 进行扩展，打造成今天的模样。

<div id="impact">
  ## 影响
</div>

在短短几个月内，Flocknet 就已经代表用户管理了数百万个联系人。技术从业者对它尤为青睐，用它来做从招聘到投资以及各种其他用途的工作。事实上，就连 Aaron 也通过 Flocknet 招聘过员工！

但 Aaron 还没有停下脚步。通过专注于源源不断的新功能需求，Aaron 持续探索完善和扩展产品的方式。

最终，Aaron 找到了一种方法，将 X 打造成一个元平台，来提升那些主要在 X 上运营社区的用户的体验。他说：“我就是会不停地做各种东西。我热爱去做那些我觉得有用、而其他人也觉得有用的东西。”而这正是他所做到的。
