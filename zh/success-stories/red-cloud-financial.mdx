**要点速览**

乘势走向成功：当 Red Cloud 金融服务受托帮助一家矿业公司推广其全新的绿色技术时，他们借助基于 X 分析打造的 Audiense 定向工具来挖掘数据并触达全新的受众群体。
</Note>

<div id="situation">
  ## 情况
</div>

Red Cloud Financial Services 的任务是帮助其客户——一家上市矿业公司——在全新受众中提升其新开发、正在申请专利的技术的知名度。 

该公司必须找到一种方式，为这款新产品带来实际成果，以便进军新市场，并有效开展一场能够触达这些新客户的营销活动。 

这项需求带来了两个关键挑战： 

1. **触达全新受众**

    客户的新技术不再只面向其现有的矿业客户，而是吸引了更广泛的、对绿色创新和社会责任感兴趣的群体。对 Red Cloud 来说，这是全新的领域，与其此前开展的任何活动都截然不同。

1. **提供明确的投资回报率（ROI）**
    
    该客户并不习惯投放付费广告活动。Red Cloud 需要清晰说明以提升认知为目标的活动的 ROI，并在对这一受众几乎缺乏一手经验的情况下跑出效果，这是一项艰巨的任务。 
    
    因此，Red Cloud 求助于 Audiense，既触达现有客户，又研究潜在新客户。Audiense 的 X 营销产品专注于借助其强大的受众定向工具，投放具有影响力的 X 营销活动。

<div id="solution">
  ## 解决方案
</div>

Red Cloud 首先通过设定预期着手开展工作。借助 Audiense 提供的洞察来创建广告活动效果预测，Red Cloud 得以就广告活动的最终目标及相关 KPI 与客户达成一致预期。 

在 Audiense 的帮助下，Red Cloud 将其研究重点放在 X 上，以挖掘公司新闻和相关股票信息。Audiense 的平台让他们可以轻松发现广告活动与兴趣提升之间的潜在相关性，这些兴趣提升体现在视频观看量和网站流量的增长上。

一个三层策略：

1. 构建全新的受众群体。

    在客户的指导下，Red Cloud Financial 确定了两个主要受众群体作为广告活动的聚焦对象：第一类细分受众包括采矿和投资，第二类则包括清洁科技、电动汽车、ESG 和责任投资。X 的数据进一步帮助 Red Cloud 将这些受众与后续广告活动进行匹配。

1. 细分新的受众群体。 

    尽管这些新受众覆盖了广泛的行业领域，但它们之间仍有大量共同点。Audiense 帮助 Red Cloud Financial 深入挖掘这些新细分受众，以识别影响者、内容消费习惯等信息。这些洞察还帮助他们确定应关注哪些账号，以期对方回关，从而扩大 Red Cloud 及其客户的受众规模。

1. 确定哪些受众最有价值。 
    
    确定哪些受众最有价值。借助 Audiense 的 [Targeting Pack 功能](https://help.audiense.com/knowledge/what-is-a-targeting-pack)，Red Cloud 创建了极其精准定向的 X 付费广告活动，同时利用为 X 细分受众累积的数据来开展自然（非付费）推广活动。通过运用从 Targeting Pack 功能中获得的洞察，Audiense 和 Red Cloud 制作了两个分别面向这两类受众的独特视频。他们得以判断哪一个视频在广告活动中持续表现最佳，并据此确保定向始终保持在正确轨道上。

<div id="impact">
  ## 影响
</div>

Audiense 的平台以及他们基于 X 的研究，帮助 Red Cloud 找到合适的受众，对其进行细分，并用来扩大粉丝群体，同时为其有机和付费内容的整体社交媒体策略提供参考。

Red Cloud Financial 超出了其客户的预期。事实上，活动效果非常理想，以至于客户决定继续推进，并加大投入启动新一轮活动。 

- 合并后的活动带来了超过 350 万次曝光和 280 万次总观看量
- 与客户正在申请专利的新技术相关的讨论量有所上升 
- 他们注意到网站引荐流量有所提升，其中包括因有关其新技术的推广文章和新闻稿而带来的 7,000 名新访客

这是一个典型示例，说明在 X 上进行受众分析与定向如何帮助你触达新的受众、优化营销活动并交付真实可衡量的结果。这也可能是帮助打造你客户品牌并促进其业务增长的黄金机遇。
