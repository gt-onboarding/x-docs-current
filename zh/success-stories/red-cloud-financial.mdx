<Note>
**要点如下**

迈向成功：当 Red Cloud 金融服务公司受托帮助一家矿业公司推广其新型绿色技术时，它借助基于 X 分析构建的 Audiense 受众定向工具来挖掘数据并触达全新的目标受众。
</Note>

<div id="situation">
  ## 情况
</div>

Red Cloud Financial Services 受托帮助其客户（一家上市矿业公司）在全新的受众群体中提升其新开发且正在申请专利的技术的知名度。 

为了进入新市场，并有效开展能够触达这些新客户的营销活动，该公司必须找到一种方式，为这一新产品带来实实在在的效果。 

这一需求带来了两个关键挑战： 

1. **触达全新的受众**

    与其继续聚焦现有的矿业客户，客户推出的新技术更能吸引更广泛的、关注绿色创新和社会责任的群体。对 Red Cloud 来说，这是前所未有的领域，与此前开展过的任何活动都截然不同。

1. **提供可量化的投资回报率（ROI）**
    
    该客户过去并不习惯开展付费营销活动。Red Cloud 被期望能够清晰阐明以提升认知为目标的活动的 ROI，并在一个它并无太多一手经验的受众群体中拿出成果，这是一项艰巨的任务。 
    
    Red Cloud 求助于 Audiense，以便在研究潜在新客户的同时触达现有客户。Audiense 的 X 营销产品专注于利用其强大的受众定向工具，打造具有影响力的 X 营销活动。

<div id="solution">
  ## 解决方案
</div>

Red Cloud 首先设定了预期。借助 Audiense 提供的洞察来创建活动绩效预测，Red Cloud 得以就本次活动的最终目标及相关 KPI 与客户对齐预期。 

在 Audiense 的帮助下，Red Cloud 将研究重点放在 X 上，以挖掘公司新闻和相关股票信息。Audiense 的平台使他们能够轻松找出活动与兴趣提升之间的潜在关联，这些兴趣提升体现在视频观看次数和网站流量的增长上。

一个三步法：

1. 构建全新的受众群体。

    在客户的指导下，Red Cloud Financial 确定了两个主要受众群体来聚焦投放活动：第一类包括矿业和投资领域，第二类包括清洁科技、电动汽车、ESG 和负责任投资。X 的数据进一步帮助 Red Cloud 将这些受众与后续的活动进行匹配。

1. 细分新的受众群体。 

    尽管新的受众群体涵盖了广泛的行业，但它们之间仍有大量共同点。Audiense 帮助 Red Cloud Financial 深入挖掘这些新细分群体，以识别影响者、内容消费习惯等信息。这些洞察也帮助他们确定应该关注哪些账号，希望对方能够回关，从而扩大 Red Cloud 及其客户的受众群体。

1. 确定哪一部分价值最高。 
    
    为了确定哪一部分价值最高，借助 Audiense 的 [Targeting Pack 功能](https://help.audiense.com/knowledge/what-is-a-targeting-pack)，Red Cloud 创建了高度精准定向的付费 X 广告活动，同时利用针对 X 细分受众积累的数据来开展自然（非付费）推广活动。通过利用 Targeting Pack 功能所提供的洞察，Audiense 和 Red Cloud 为这两个受众分别制作了两个独特的视频。他们得以确定哪个视频在活动中始终表现最佳，并据此保持定向策略的准确性和一致性。

<div id="impact">
  ## 成效
</div>

Audiense 的平台以及其基于 X 的研究，帮助 Red Cloud 找到合适的受众，对其进行细分，并利用这些受众来增长粉丝数量，同时为其整体社交媒体策略（涵盖自然和付费内容）提供参考。

Red Cloud Financial 超出了客户的预期。事实上，这些营销活动的效果非常出色，以至于该客户决定继续推进并加大新一轮活动的投入。 

- 整体活动共获得了 350 万+ 次曝光和 280 万+ 次总观看量
- 与该客户新的待批专利技术相关的讨论量有所上升 
- 他们注意到网站引荐流量有所提升，其中包括来自关于其新技术的推广文章和新闻稿的 7,000 名新访客

这充分说明，借助 X 上的受众分析和定向功能，你可以挖掘新的受众群体，优化营销活动，并取得切实成果。这同样可能成为帮助你打造客户品牌、推动其业务增长的黄金机会。
