货币类型通过 [ISO-4217](http://en.wikipedia.org/wiki/ISO_4217) 标识。这是一个由三个字母组成的字符串，例如 `USD` 或 `EUR`。货币的数值以微单位（micros）表示。对于 USD，5.50 美元会被编码为 5.50*1e6，即 5,500,000。要表示“整数金额”，对于所有货币，你都需要将本地货币的微单位数值乘以 1e6（1&#95;000&#95;000）。

Ads API 会识别所有 X SMB Ads 产品可用国家/地区的法定货币。有关这些国家/地区的完整列表，请点击[此处](https://business.x.com/en/help/overview/about-eligibility-for-twitter-ads.html)。

大多数货币的小数位数为两位（例如 USD：$1.23）。

* `AED`, `VEF`, `PKR`, `IDR`, `HKD`, `QAR`, `PHP`, `NZD`, `INR`, `EUR`, `ARS`, `THB`, `MXN`, `EGP`, `DZD`, `SAR`, `LBP`, `COP`, `TRY`, `SGD`, `MAD`, `AUD`, `USD`, `MYR`, `CAD`, `BRL`, `GBP`.

以下货币不使用小数：

* `VND`, `KRW`, `CLP`, `JPY`

以下货币使用三位小数：

* `KWD`, `IQD`, `BHD`, `TND`

货币通过[资金工具](/zh/x-ads-api/introduction)进行使用。
