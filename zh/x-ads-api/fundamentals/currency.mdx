货币的类型使用 [ISO-4217](http://en.wikipedia.org/wiki/ISO_4217) 进行标识。这是一个类似 `USD` 或 `EUR` 的三字母字符串。货币的数值以微单位（micros）表示。以 USD 为例，$5.50 编码为 5.50*1e6，即 5,500,000。要表示“整数金额”，对于所有货币，你需要将本地货币单位乘以 1e6（1&#95;000&#95;000），转换为微单位。

Ads API 会识别所有 X SMB Ads 产品可用国家/地区的法定货币。要查看这些国家/地区的完整列表，请点击 [此处](https://business.x.com/en/help/overview/about-eligibility-for-twitter-ads.html)。

大多数货币的小数“精度”为 2 位（例如 USD：$1.23）。

* `AED`, `VEF`, `PKR`, `IDR`, `HKD`, `QAR`, `PHP`, `NZD`, `INR`, `EUR`, `ARS`, `THB`, `MXN`, `EGP`, `DZD`, `SAR`, `LBP`, `COP`, `TRY`, `SGD`, `MAD`, `AUD`, `USD`, `MYR`, `CAD`, `BRL`, `GBP`.

下列货币不使用小数：

* `VND`, `KRW`, `CLP`, `JPY`

以下这些使用三位小数：

* `KWD`, `IQD`, `BHD`, `TND`

货币的使用是通过 [funding instruments](/zh/x-ads-api/introduction) 来实现的。
