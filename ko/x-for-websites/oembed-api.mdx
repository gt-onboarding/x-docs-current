oEmbed API를 사용하면 [트윗](https://developer.x.com/en/docs/twitter-for-websites/embedded-tweets/overview)과 [타임라인](https://developer.x.com/en/docs/twitter-for-websites/timelines/overview)과 같은 임베드 콘텐츠를 프로그램을 통해 가져올 수 있습니다. 

oEmbed API의 응답으로 HTML 스니펫이 반환되며, 이는 [페이지에 X의 위젯 JavaScript를 포함](https://developer.x.com/web/javascript/loading)하면 자동으로 인식됩니다.

이 API는 대량 작업에 사용하는 것을 권장하며, 콘텐츠를 임베드할 때에는 강력한 [publish.x.com](https://publish.x.com/#) 도구를 사용하는 것이 좋습니다.

<Tabs>
  <Tab title="임베디드 타임라인">
    반환된 HTML 스니펫은 [임베드된 타임라인](https://developer.x.com/en/docs/twitter-for-websites/timelines/overview)으로 자동 인식되며, [페이지에 X 위젯 JavaScript가 포함되어 있는 경우](https://developer.x.com/web/javascript/loading)에 적용됩니다.

    oEmbed 엔드포인트를 사용하면 HTML 마크업 내의 해당 속성을 설정하여, 기본적으로 HTML 응답과 함께 번들되는 X JavaScript가 이를 해석해 최종 임베드 타임라인의 모양을 사용자 지정할 수 있습니다. 반환되는 마크업 형식은 X가 새 기능을 추가하거나 타임라인 표현을 조정함에 따라 시간이 지나면서 변경될 수 있습니다.

    타임라인 URL로 지정된 X 타임라인을 [oEmbed](https://oembed.com/) 호환 JSON 형식으로 반환합니다. 사용자 타임라인과 리스트 타임라인이 지원됩니다. 타임라인 마크업은 `cache&#95;age` 속성에 지정된 권장 캐시 수명 동안 서버에 캐시하는 것을 목적으로 합니다.

    ## Resource URL

    *[https://publish.x.com/oembed](https://publish.x.com/oembed)*

    ## Example Requests

    ```bash
    curl --request GET --url 'https://publish.x.com/oembed?url=https%3A%2F%2Ftwitter.com%2FInterior%2Fstatus%2F507185938620219395'
    twurl -H publish.x.com "/oembed?url=https://x.com/Interior/status/463440424141459456"
    ```

    ## 예시 응답

    ```json
      "provider_name": "Twitter",
      "provider_url": "https://x.com",

    }
    ```

  <Tab title="임베디드 트윗">
    반환된 HTML 스니펫은 [페이지에 X의 위젯 JavaScript가 포함](https://developer.x.com/web/javascript/loading)되어 있으면 자동으로 [임베디드 트윗](https://developer.x.com/web/embedded-tweets)으로 인식됩니다.

    oEmbed 엔드포인트를 사용하면 HTML 마크업에서 해당 속성을 설정하여 임베디드 트윗의 최종 표시 형태를 사용자 지정할 수 있으며, 이는 기본적으로 HTML 응답과 함께 번들로 제공되는 X의 JavaScript에 의해 해석됩니다. 반환되는 마크업 형식은 X가 새로운 기능을 추가하거나 트윗 표현 방식을 조정함에 따라 변경될 수 있습니다.

    트윗 폴백 마크업은 `cache_age` 속성에 지정된 권장 캐시 수명 동안 서버에 캐시되도록 설계되었습니다.

    ## 리소스 URL

    *[https://publish.x.com/oembed](https://publish.x.com/oembed)*

    |            |      |

    | 인증이 필요한가요? | 아니요  |

    ## 매개변수

    | 이름                                                                                                                               | 기본값     | 설명                                                                                                                                                                                                                                                                                                                                         |
    | :------------------------------------------------------------------------------------------------------------------------------- | :------ | :----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
    | `url` 필수  <br />문자열                                                                                                              |         | 임베드할 트윗의 URL                                                                                                                                                                                                                                                                                                                               |
    | `maxwidth`  <br />Int `[220..550]`                                                                                               | `325`   | 렌더링된 트윗의 최대 너비(픽셀 단위, 정수)입니다. 허용 범위를 벗어나는 값이 제공되면 지원되는 최소 또는 최대 너비로 자동 조정되며, 조정된 너비 값은 반환되는 `width` 속성에 반영됩니다. X는 oEmbed `maxheight` 파라미터를 지원하지 않습니다. 트윗은 본질적으로 텍스트 기반이므로, 이미지나 비디오처럼 비율을 조정할 수 없는 예측 불가능한 높이를 가집니다. 이와 관련하여, oEmbed 응답에는 `height` 값이 제공되지 않습니다. 트윗에 대해 일정한 높이가 필요한 구현에서는 아래의 `hide_thread` 및 `hide_media` 파라미터를 참고해야 합니다. |
    | `hide_media`  <br />Boolean, String 또는 Int                                                                                       | `false` | `true`, `"t"`, 또는 `1`로 설정하면 트윗에 포함된 링크는 사진, 동영상 또는 링크 미리보기로 확장되지 않습니다.                                                                                                                                                                                                                                                                     |
    | `hide_thread`  <br />Boolean, String 또는 Int                                                                                      | `false` | `true`, `"t"` 또는 `1`로 설정하면 요청한 트윗이 다른 트윗에 대한 답글인 경우 대화 스레드에서 이전 트윗의 축약된 버전이 표시되지 않습니다.                                                                                                                                                                                                                                                     |
    | `omit_script`  <br />Boolean, String 또는 Int                                                                                      | `false` | `true`, `"t"` 또는 `1`로 설정하면 `widgets.js`를 로드하는 `<script>` 요소는 제공되지 않습니다. 모든 X 위젯(예: [Embedded Tweets](https://developer.x.com/web/embedded-tweets))에서 사용할 수 있도록 웹 페이지에 `widgets.js`에 대한 참조를 직접 포함해야 합니다.                                                                                                                                    |
    | `align`  <br />열거형 `{left,right,center,none}`                                                                                    | `none`  | 상위 요소를 기준으로 페이지에서 포함된 트윗을 왼쪽, 오른쪽 또는 가운데에 정렬할지 여부를 지정합니다.                                                                                                                                                                                                                                                                                  |
    | `lang`  <br />Enum([언어](https://developer.x.com/en/docs/twitter-for-websites/twitter-for-websites-supported-languages/overview)) | `en`    | 요청 결과는 지정된 [임베드된 트윗이 지원하는 X 언어](https://developer.x.com/web/overview/languages)로 된 HTML과 렌더링된 트윗을 반환합니다.                                                                                                                                                                                                                                   |
    | `theme`  <br />열거형 `{light, dark}`                                                                                               | `light` | `dark`로 설정하면 트윗이 어두운 배경 위에 밝은 색상의 텍스트로 표시됩니다.                                                                                                                                                                                                                                                                                              |
    | `dnt`  <br />Boolean                                                                                                             | `false` | `true`로 설정하면 트윗과 귀하의 사이트에 임베드된 해당 페이지는 [맞춤형 추천](https://support.x.com/articles/20169421)이나 [맞춤형 광고](https://support.x.com/articles/20170405) 등의 목적으로 사용되지 않습니다.                                                                                                                                                                            |
