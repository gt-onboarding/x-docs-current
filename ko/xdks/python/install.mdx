XDK Python SDK는 GitHub 리포지토리에서 직접 받아 `pip`로 설치할 수 있습니다.

<div id="prerequisites">
  ## 사전 준비 사항
</div>

- Python 3.8 이상.
- 가상 환경 구성을 위한 `pip` 및 `venv`(권장).

<div id="quick-install">
  ## 빠른 설치
</div>

GitHub 저장소의 하위 디렉터리에서 XDK를 설치합니다:

```bash
pip install xdk
```

이는 `main` 브랜치에서 최신으로 생성된 버전을 가져오는 명령입니다.


<div id="development-install">
  ## 개발용 설치
</div>

개발 또는 기여를 위해:

1. 저장소를 클론합니다:
   ```bash
   git clone https://github.com/xdevplatform/xdk.git
   cd xdk/python
   ```
2. 의존성을 editable 모드로 설치합니다:
   ```bash
   pip install -e .
   ```
   이렇게 하면 SDK와 그 런타임 의존성이 설치됩니다.
3. (선택 사항) 테스트/린트를 위한 개발용 의존성을 설치합니다:
   ```bash
   pip install -e .[dev]
   ```

<div id="verification">
  ## 검증
</div>

설치를 테스트합니다:

```python
import xdk
print(xdk.__version__)  # XDK 버전을 출력해야 합니다
```

**참고:** XDK는 OpenAPI 스펙을 사용해 생성되므로, 항상 [X API 변경 로그](https://docs.x.com/changelog)와 리포지토리의 XDK 릴리스 노트를 확인해 변경 사항을 확인하세요.
Python XDK를 사용한 자세한 코드 예시는 [코드 샘플 GitHub 리포지토리](https://github.com/xdevplatform/samples/tree/main/python)를 참조하세요.
