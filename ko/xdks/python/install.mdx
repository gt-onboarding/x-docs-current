XDK Python SDK는 GitHub 저장소에서 직접 내려받을 수 있으며 `pip`를 통해 설치할 수 있습니다.

<div id="prerequisites">
  ## 사전 준비 사항
</div>

- Python 3.8 이상
- 가상 환경 구성을 위한 `pip` 및 `venv` (권장)

<div id="quick-install">
  ## 빠른 설치
</div>

GitHub의 하위 디렉터리에서 XDK를 설치합니다:

```bash
pip install xdk
```

이 명령은 `main` 브랜치에서 최신으로 생성된 버전을 가져옵니다.


<div id="development-install">
  ## 개발용 설치
</div>

개발 환경에서 사용하거나 기여하려는 경우:

1. 저장소를 복제합니다:
   ```bash
   git clone https://github.com/xdevplatform/xdk.git
   cd xdk/python
   ```
2. 의존성을 editable 모드로 설치합니다:
   ```bash
   pip install -e .
   ```
   이렇게 하면 SDK와 런타임 의존성이 설치됩니다.
3. (선택 사항) 테스트/린팅을 위한 개발용 의존성을 설치합니다:
   ```bash
   pip install -e .[dev]
   ```

<div id="verification">
  ## 검증
</div>

설치가 제대로 되었는지 확인합니다:

```python
import xdk
print(xdk.__version__)  # XDK 버전을 출력해야 합니다
```

**참고:** XDK는 OpenAPI 명세를 기반으로 생성되므로, 항상 [X API 변경 로그](https://docs.x.com/changelog)와 저장소의 XDK 릴리스 노트를 확인해 변경 사항을 파악하세요.
Python XDK를 사용하는 자세한 코드 예제는 [코드 샘플 GitHub 저장소](https://github.com/xdevplatform/samples/tree/main/python)를 참고하세요.
