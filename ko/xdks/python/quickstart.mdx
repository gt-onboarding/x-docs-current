이 예제는 Bearer Token 인증을 사용하여 XDK를 통해 게시물을 빠르게 검색하는 방법을 보여줍니다.

<div id="step-1-install-the-sdk">
  ## 1단계: SDK 설치하기
</div>

```bash
pip install xdk
```

<div id="step-2-get-your-bearer-token">
  ## 2단계: Bearer Token 받기
</div>

1. [X Developer Portal](https://developer.x.com/en/portal/dashboard)에 로그인합니다.
2. 앱을 생성하거나 선택합니다.
3. &quot;키와 토큰&quot; 섹션에서 Bearer Token(앱 전용 인증)을 생성합니다.

<div id="step-3-write-and-run-your-first-script">
  ## 3단계: 첫 스크립트 작성 및 실행
</div>

`quickstart.py` 파일을 만듭니다:

```python
# Client 가져오기
from xdk import Client
# 실제 Bearer Token으로 교체
client = Client(bearer_token="YOUR_BEARER_TOKEN_HERE")
# "api"를 언급하는 최근 게시물 가져오기
# search_recent는 Iterator를 반환하므로 반복 처리
for page in client.posts.search_recent(query="api", max_results=10):
    if page.data and len(page.data) > 0:
        # 첫 번째 게시물 접근 - Pydantic 모델은 속성 및 딕셔너리 접근 모두 지원
        first_post = page.data[0]
        post_text = first_post.text if hasattr(first_post, 'text') else first_post.get('text', '')
        print(f"Latest Post: {post_text}")
        break
    else:
        print("No Posts found.")
        break
```

실행해 보세요:

```bash
python quickstart.py
```

**예상 결과**:

```
Latest Post: XDK Python SDK의 흥미로운 업데이트!
```

**문제 해결**: 401 오류가 발생하면 Bearer Token을 다시 확인하세요. 요청 한도(429) 오류가 발생한 경우, 잠시 기다렸다가 다시 시도하세요.
