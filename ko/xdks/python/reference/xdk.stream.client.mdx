이 모듈은 X API의 스트림 엔드포인트와 상호작용하기 위한 클라이언트를 제공합니다.
실시간 스트리밍 작업은 데이터가 도착하는 즉시 이를 내보내는 제너레이터를 반환합니다.
스트리밍 연결은 안정적인 처리를 위해 지수 백오프 재시도 로직으로 자동 관리됩니다.
