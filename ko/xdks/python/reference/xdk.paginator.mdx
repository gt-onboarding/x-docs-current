X API SDK를 위한 커서 기반 페이지네이션 유틸리티입니다.

이 모듈은 모든 API 클라이언트에서 효율적인 페이지네이션을 지원하는 Cursor 클래스를 제공합니다. Cursor를 사용하면 적절한 타입 안정성을 유지하면서 .pages() 및 .items() 메서드로 페이지네이션 결과를 쉽게 순회할 수 있습니다.

<div id="xdkpaginatorcursor">
  ### `xdk.paginator.cursor`
</div>

적절한 타입 추론 및 검증이 적용된 커서를 생성합니다.
이 팩토리 함수는 타입 추론을 도와 응답 타입에 대한
정확한 타입 힌트를 제공하고, 메서드가 정적 분석 시점과 런타임
모두에서 페이지네이션을 지원하는지 검증합니다.

<div id="parameters">
  #### 매개변수
</div>

<ParamField path="path.method" type="PaginatableMethod">
래핑할 API 메서드(페이지네이션을 지원해야 함)
</ParamField>
