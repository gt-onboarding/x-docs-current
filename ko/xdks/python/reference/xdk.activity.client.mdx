이 모듈은 X API의 Activity 엔드포인트와 상호작용하기 위한 클라이언트를 제공합니다.
실시간 스트리밍 작업은 데이터가 도착하는 대로 값을 내보내는 제너레이터를 반환합니다.
스트리밍 연결은 신뢰성 있는 처리를 위해 지수적 백오프를 사용하는 재시도 로직으로 자동 관리됩니다.

<div id="stream">
  ### `stream`
</div>

Activity Stream(스트리밍)
X Activity 스트림
데이터가 생성되는 대로 실시간으로 전달하는 스트리밍 endpoint입니다.
각 수신 항목은 스트림에서 나온 단일 데이터 포인트를 나타냅니다.
연결 관리는 지수 백오프 재시도 로직으로 자동으로 처리됩니다.
스트림 연결이 끊어지면 SDK가 클라이언트 개입 없이 자동으로 재연결합니다.

</ParamField>

YYYY-MM-DDTHH:mm:ssZ. 게시물 라벨이 제공될 가장 이른 UTC 타임스탬프입니다.

<ParamField path="path.end_time" type="str or None" default="None">
YYYY-MM-DDTHH:mm:ssZ. 게시물 라벨이 제공될 가장 늦은 UTC 타임스탬프입니다.
</ParamField>

<ParamField path="path.stream_config" type="StreamConfig">
