이 모듈은 X API의 activity 엔드포인트와 상호작용하기 위한 클라이언트를 제공합니다.
실시간 스트리밍 작업은 데이터가 도착하는 대로 이를 내보내는 제너레이터를 반환합니다.
스트리밍 연결은 안정적인 처리를 위해 지수 백오프 재시도 로직으로 자동 관리됩니다.

<div id="stream">
  ### `stream`
</div>

Activity Stream (스트리밍)
X Activity 스트림
데이터가 생성되는 즉시 실시간으로 전달하는 스트리밍 endpoint입니다.
각각의 항목은 스트림에서 나온 개별 데이터 포인트 하나를 나타냅니다.
연결은 지수 백오프 재시도 로직으로 자동 관리됩니다.
스트림이 끊어지면 SDK가 클라이언트의 개입 없이 자동으로 다시 연결합니다.

</ParamField>

YYYY-MM-DDTHH:mm:ssZ. 게시물 레이블이 제공되는 가장 이른 UTC 타임스탬프입니다.
</ParamField>

<ParamField path="path.end_time" type="str or None" default="None">
YYYY-MM-DDTHH:mm:ssZ. 게시물 레이블이 제공되는 가장 늦은 UTC 타임스탬프입니다.
</ParamField>

<ParamField path="path.stream_config" type="StreamConfig">
재시도 동작, 타임아웃, 콜백을 사용자 지정하기 위한 선택적 StreamConfig입니다.
</ParamField>
