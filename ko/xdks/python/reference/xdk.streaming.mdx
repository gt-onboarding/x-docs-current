X API SDK를 위한 강력한 스트리밍 유틸리티입니다.

이 모듈은 자동 재연결, 지수 백오프, 포괄적인 오류 처리를 포함한 스트리밍 연결 관리를 제공합니다. 클라이언트는 연결 관리에 신경 쓰지 않고 스트리밍 endpoint를 사용할 수 있으며, SDK가 모든 복구를 자동으로 처리합니다.

<div id="xdkstreamingstream_with_retry">
  ### `xdk.streaming.stream_with_retry`
</div>

엔드포인트로부터 자동 재연결 및 지수적 백오프를 사용해 데이터를 스트리밍합니다.
이 함수는 다음을 포함한 모든 연결 관리를 처리합니다:

- 연결이 끊어질 경우 자동 재연결
- 재시도 지연을 위한 지터가 포함된 지수적 백오프
- 오류를 재시도 가능 오류와 치명적 오류로 분류
- 연결 상태 모니터링을 위한 라이프사이클 콜백

<div id="parameters">
  #### 매개변수
</div>

<ParamField path="path.session" type="Session">
HTTP 호출에 사용할 requests Session입니다.
</ParamField>

<ParamField path="path.method" type="str">
HTTP 메서드(일반적으로 “get”)입니다.
</ParamField>

<ParamField path="path.url" type="str">
스트리밍을 받을 전체 URL입니다.
</ParamField>

<ParamField path="path.config" type="StreamConfig">
재시도 및 콜백 설정을 포함하는 StreamConfig입니다.
</ParamField>
