X API SDK를 위한 강력한 스트리밍 유틸리티입니다.

이 모듈은 자동 재연결, 지수 백오프, 포괄적인 오류 처리를 포함한 스트리밍 연결 관리를 제공합니다.
클라이언트는 연결 관리를 신경 쓰지 않고 스트리밍 엔드포인트를 소비할 수 있으며, 모든 복구는 SDK가 자동으로 처리합니다.

<div id="xdkstreamingstream_with_retry">
  ### `xdk.streaming.stream_with_retry`
</div>

엔드포인트에서 자동 재연결 및 지수적 백오프를 사용해 데이터를 스트리밍합니다.
이 함수는 다음을 포함해 모든 연결 관리를 처리합니다.

- 연결 끊김 발생 시 자동 재연결
- 재시도 지연을 위한 지수적 백오프 및 지터
- 오류를 재시도 가능 오류와 치명적 오류로 분류
- 연결 상태 모니터링을 위한 라이프사이클 콜백
