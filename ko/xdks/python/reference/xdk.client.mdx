이 모듈은 X API와 상호 작용하기 위한 주요 Client 클래스를 제공합니다.
모든 하위 클라이언트를 관리하며 인증, 세션 관리, OAuth2 PKCE 플로우를 처리합니다.
모든 기능은 OpenAPI 사양을 기반으로 생성됩니다.

<div id="parameters">
  #### Parameters
</div>

<ParamField path="path.base_url" type="str" default="'https://api.x.com'">
X API의 기본 URL입니다(기본값은 [https://api.x.com](https://api.x.com)입니다).
</ParamField>

<ParamField path="path.bearer_token" type="str or None" default="None">
X API용 Bearer Token입니다(앱 전용 인증에 사용됩니다).
</ParamField>

<ParamField path="path.access_token" type="str or None" default="None">
사용자 컨텍스트용 OAuth2 액세스 토큰입니다(Bearer Token으로 직접 사용할 수 있습니다).
</ParamField>

<ParamField path="path.client_id" type="str or None" default="None">
X API용 클라이언트 ID입니다(OAuth2 PKCE 플로우에 필요합니다).
</ParamField>

<ParamField path="path.client_secret" type="str or None" default="None">
X API용 클라이언트 시크릿입니다.
</ParamField>

<ParamField path="path.redirect_uri" type="str or None" default="None">
OAuth2 인가에 사용할 리디렉트 URI입니다.
</ParamField>

<ParamField path="path.token" type="Dict[str, Any] or None" default="None">
기존 OAuth2 토큰 딕셔너리(있는 경우)입니다. 제공된 경우 access_token이 추출됩니다.
</ParamField>

<ParamField path="path.scope" type="str or List[str] or None" default="None">
OAuth2 인가 스코프를 위한 공백으로 구분된 문자열 또는 문자열 목록입니다.
</ParamField>

<ParamField path="path.authorization_base_url" type="str" default="'https://x.com/i'">
OAuth2 인가용 기본 URL입니다(기본값은 [https://x.com/i](https://x.com/i)입니다).
</ParamField>

<ParamField path="path.auth" type="OAuth1">
OAuth1.0a 인증에 사용할 OAuth1 인스턴스입니다.
</ParamField>

  ### `exchange_code`

</ParamField>

<div id="fetch_token">
  ### `fetch_token`
</div>

<ParamField path="path.authorization_response" type="Any">
인가가 완료된 후에 수신한 전체 콜백 URL입니다.
</ParamField>

  ### `get_authorization_url`

</ParamField>
