프로젝트에서 X API용 TypeScript SDK 사용을 시작해 보세요.

<div id="install">
  ## 설치
</div>

<CodeGroup>
  ```bash npm theme={null}
  npm install @xdevplatform/xdk
  ```

  ```bash yarn theme={null}
  yarn add @xdevplatform/xdk
  ```

  ```bash pnpm theme={null}
  pnpm add @xdevplatform/xdk
  ```
</CodeGroup>

<div id="typescript-support">
  ## TypeScript 지원
</div>

이 SDK는 TypeScript로 작성되었으며 전체 타입 정의를 제공합니다. 별도의 타입 패키지를 설치할 필요가 없습니다.

<div id="requirements">
  ## 요구 사항
</div>

* Node.js 16+
* TypeScript 4.5+ (TypeScript를 사용하는 경우)
