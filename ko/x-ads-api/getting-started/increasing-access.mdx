<div id="requesting-additional-app-level-permissions">
  ### 추가 앱 수준 권한 요청하기
</div>

애플리케이션에 아래 나열된 앱 수준 권한만 부여되어 있고 추가 엔드포인트에 액세스하고자 하는 경우, **Standard Access**를 신청하기 위해 X 담당자에게 문의하세요.

2023년 7월 이전에 Ads API 액세스를 요청한 개발자는 다음과 같은 앱 수준 권한 중 일부를 요청했을 수 있으며, 이제는 Standard Access 수준에 포함됩니다:

- **Analytics**(읽기 전용): 사용자는 [읽기 전용](https://developer.x.com/en/docs/apps/app-permissions) 권한으로 [Analytics](https://developer.x.com/en/docs/twitter-ads-api/analytics/overview) 엔드포인트에 액세스할 수 있습니다. 사용자는 Campaign Management, Creative 또는 Custom Audiences 엔드포인트에는 액세스할 수 없지만, Analytics 엔드포인트의 요청 파라미터에 크리에이티브 엔티티를 전달하여 메트릭을 조회할 수 있습니다. `media_id` 또는 `media_name` 수준의 세분화 데이터를 조회하려면 사용자 애플리케이션에 Ads API 쓰기 권한이 있어야 합니다.

- **Campaign Management & Creative**(읽기 및 쓰기): 사용자는 [읽기 및 쓰기](https://developer.x.com/en/docs/apps/app-permissions) 권한으로 [Campaign Management](https://developer.x.com/en/docs/twitter-ads-api/campaign-management/overview) 및 [Creative](https://developer.x.com/en/docs/twitter-ads-api/creatives/overview) 엔드포인트에 액세스할 수 있습니다. 또한 Analytics 엔드포인트에도 액세스할 수 있습니다.

- **Custom Audiences**(읽기 및 쓰기): 사용자는 [읽기 및 쓰기](https://developer.x.com/en/docs/apps/app-permissions) 권한으로 [Custom Audiences](https://developer.x.com/en/docs/twitter-ads-api/audiences/overview) 엔드포인트에 액세스할 수 있습니다. 또한 Campaign Management, Creative, Analytics 엔드포인트에도 액세스할 수 있습니다.

- **Conversion**(읽기 및 쓰기): 사용자는 [읽기 및 쓰기](https://developer.x.com/en/docs/apps/app-permissions) 권한으로 [Mobile](https://developer.x.com/en/docs/twitter-ads-api/measurement/mobile-conversions/overview) 및 [Web Conversion](https://developer.x.com/en/docs/twitter-ads-api/measurement/web-conversions/overview) 엔드포인트에 액세스할 수 있습니다. 사용자는 다른 Ads API 엔드포인트에 액세스 권한이 있을 수도, 없을 수도 있습니다. 

<div id="requesting-an-increase-to-user-oauth-token-limits">
  ### 사용자 OAuth 토큰 한도 상향 요청
</div>

2023년 7월 이전에 Ads API 액세스를 요청한 개발자의 경우, 애플리케이션에 사용자 OAuth 토큰 한도가 5개로 설정되어 있을 수 있습니다. 개발자 앱의 토큰 한도를 늘려야 하는 경우 X 담당자에게 문의해 주세요.

<Note>**참고:** 어떤 Ads API 엔드포인트에 대해서도 [요청 한도](https://developer.x.com/en/docs/twitter-ads-api/rate-limiting)는 상향되지 않습니다. 요청 한도 내에서 사용하는 방법은 [모범 사례](https://developer.x.com/en/docs/twitter-ads-api/rate-limiting)를 참고하세요.</Note>
