<div id="overview">
  ## 개요
</div>

Catalog API는 광고주가 상품 피드를 설정하고, 상품을 세트로 그룹화하며, 카탈로그 상품을 통합적으로 관리할 수 있게 해 주는 커머스 솔루션입니다. Catalog API는 프로그래매틱 카탈로그 관리를 가능하게 하는 핵심 도구로, 광고주가 카탈로그가 가져와지고 업데이트되는 방식에 대해 더 많은 제어권을 가질 수 있도록 합니다.

카탈로그에는 사용자의 상품과 상품 세트에 대한 모든 정보가 포함되며, 사용자의 핸들과 연결됩니다. 현재는 **하나의 사용자는 하나의 카탈로그만 생성할 수 있습니다**. 

Catalog API는 상품을 가져오는 두 가지 방식을 지원합니다.

1.  Scheduled Feed: 사용자는 주기적으로 가져올 피드 URL(CSV, TSV, XML)을 추가할 수 있습니다(현재 X는 최대 8GB 크기의 파일을 지원합니다).

2.  Batch Products API: 사용자는 배치(JSON) 요청을 통해 상품 속성을 조회, 생성, 업데이트 및 삭제할 수 있습니다.

X 판매자는 필터 규칙을 기반으로 상품 세트를 생성할 수 있습니다. 이를 통해 상품에 추가 메타데이터를 부여하고, 오가닉 커머스 기능이나 동적 상품 광고를 통해 이러한 상품을 노출하기 위한 특수 상품 세트를 생성할 수 있습니다. Catalog API와 X 쇼핑 매니저는 두 가지 유형의 상품 세트를 지원합니다. 

1.  Manual: 사용자는 최대 50개의 상품을 선택하고, 이름과 설명을 설정할 수 있습니다.

2.  Filter: 사용자는 최대 30개의 필터를 추가하여, 해당 필터를 충족하는 상품으로 세트를 자동 생성할 수 있습니다. 필터 유형에는 가격, Google 상품 카테고리, 상품 유형, 재고, 세일 가격, 커스텀 필드 등이 포함됩니다.

Catalog API에는 카탈로그를 관리하기 위한 4개의 엔드포인트가 포함되어 있습니다.

* [Product Catalogs](https://developer.x.com/en/docs/x-ads-api/catalog-management/api-reference/product-catalogs) \- 상품 정보를 위한 컨테이너 생성
* [Products](https://developer.x.com/en/docs/x-ads-api/catalog-management/api-reference/products) -  카탈로그에서 상품을 동기식으로 추가, 업데이트, 조회 및 제거
* [Product Sets](https://developer.x.com/en/docs/x-ads-api/catalog-management/api-reference/product-sets) -  상품 정보를 기준으로 상품을 그룹으로 분류
* [Schedule Feeds](https://developer.x.com/en/docs/twitter-ads-api/catalog-management/api-reference/scheduled-feeds) \- 파일 기반 상품 데이터를 카탈로그로 비동기식으로 가져오도록 설정

<div id="prerequisites-for-the-catalog-api">
  ### Catalog API를 사용하기 위한 사전 준비 사항
</div>

Catalog API 엔드포인트는 현재 조기 액세스(early access)로만 제공됩니다. 액세스 권한을 신청하려면 X 담당자에게 문의해 주세요. 

[X Shopping Manager](https://ads.x.com/shopping_manager)를 통해 이용약관에 동의해야 합니다.

Shopping Manager 및 상품 사양에 대한 자세한 내용은 [상품 사양 가이드](https://business.x.com/en/help/shopping-specs.html)를 참고하세요.

</div>

<div id="product-catalog">
  #### 제품 카탈로그
</div>

| HTTP 메서드 | 요청 한도 |
| :--- | :--- |
| GET /product_catalogs | 15분당 1,000회 |
| POST /product_catalogs | 15분당 20회 |
| PUT /product\_catalogs/:product\_catalog_id | 15분당 20회 |
| DELETE /product\_catalogs/:product\_catalog_id | 15분당 20회 |
