<div id="connect-to-the-x-ads-platform">
  ## X Ads 플랫폼에 연결
</div>

X Ads API는 개발자가 전 세계 X 광고주의 요구를 충족하는 솔루션을 구축할 수 있도록 X의 광고 플랫폼과 연결해 줍니다.

<CardGroup cols={2}>
  <Card title="캠페인 관리" icon="list-check" href="/ko/x-ads-api/campaign-management">X에서 사람들에게 도달할 수 있도록 광고 캠페인을 프로그래밍 방식으로 생성, 스케줄링 및 관리합니다.</Card>
  <Card title="맞춤 잠재고객" href="/ko/x-ads-api/audiences" icon="user-group">X, 웹 또는 모바일에서 제공한 데이터를 활용해 Tailored Audiences를 생성하고 관리합니다.</Card>
  <Card title="크리에이티브" href="/ko/x-ads-api/creatives" icon="paintbrush">게시물을 작성 및 게시하고, Cards로 게시물 기능을 확장하며, 이미지와 동영상을 관리 및 업로드합니다.</Card>
  <Card title="광고 분석" href="/ko/x-ads-api/analytics" icon="chart-pie">비동기 및 동기 API 엔드포인트를 사용해 다양한 메트릭 전반에 걸쳐 광고 캠페인의 세분화된 인사이트를 조회합니다.</Card>
  <Card title="도구 및 라이브러리" href="/ko/x-ads-api/tools-and-libraries" icon="screwdriver-wrench">X에서 제작하고 커뮤니티가 지원하는 클라이언트 라이브러리 모음을 확인하세요.</Card>
  <Card title="Postman 컬렉션" href="https://documenter.getpostman.com/view/7379786/SzRw1WTU" icon="code">v2 엔드포인트용 Postman 컬렉션을 제공하므로, Postman의 시각적 클라이언트를 사용해 API를 탐색해 볼 수 있습니다!</Card>
  <Card title="명령줄 도구" href="https://github.com/xdevplatform/xurl" icon="terminal">명령줄 도구를 사용해 엔드포인트를 다루는 방식을 선호한다면, xurl이 바로 여러분을 위해 만들어졌습니다!</Card>
</CardGroup>
