<div id="connect-to-the-x-ads-platform">
  ## X Ads 플랫폼에 연결
</div>

X Ads API는 개발자를 X의 광고 플랫폼에 연결해 전 세계 X 광고주의 요구를 충족하는 솔루션을 구축할 수 있게 해줍니다.

  <Card title="캠페인 관리" icon="list-check" href="/ko/x-ads-api/campaign-management">X에서 이용자에게 도달할 수 있도록 광고 캠페인을 프로그래밍 방식으로 생성하고, 일정을 설정하고, 관리하세요.</Card>
  <Card title="맞춤 잠재고객" href="/ko/x-ads-api/audiences" icon="user-group">X, 웹 또는 제공한 모바일 데이터를 활용해 Tailored Audiences를 생성하고 관리하세요.</Card>

  <Card title="광고 분석" href="/ko/x-ads-api/analytics" icon="chart-pie">비동기 및 동기 API 엔드포인트를 사용해 다양한 메트릭 전반에 걸친 광고 캠페인의 세분화된 인사이트를 조회하세요.</Card>
  <Card title="도구 및 라이브러리" href="/ko/x-ads-api/tools-and-libraries" icon="screwdriver-wrench">X에서 개발하고 커뮤니티에서 지원하는 클라이언트 라이브러리 선별 모음을 확인해 보세요.</Card>
  <Card title="Postman 컬렉션" href="https://documenter.getpostman.com/view/7379786/SzRw1WTU" icon="code">v2 엔드포인트를 위한 Postman 컬렉션을 제공하므로, Postman의 시각적 클라이언트를 사용해 API를 탐색할 수 있습니다!</Card>
