<div id="mact-overview">
  ## MACT 개요
</div>

X [모바일 앱 프로모션](https://biz.x.com/mobile-app-promotion) 측정 기능을 사용하면 광고주는 설치 유도 또는 기타 인앱 전환을 목표로 설계된, X에서 진행하는 광고 캠페인의 성과를 추적할 수 있습니다. X 모바일 측정 파트너는 광고주가 X에서 프로모션 중인 앱에서 어떤 전환을 추적할지 관리할 수 있는 기능을 제공합니다. 광고주가 추적하려는 이벤트의 유형과 시간 범위(윈도우)를 설정하면, 모바일 측정 파트너는 해당하는 모든 이벤트를 X로 전송하며, 전환이 발생하도록 사용자가 어떤 캠페인과 크리에이티브에 참여했는지를 나타내는 귀속 데이터(attribution data)를 즉시 반환받습니다.

<div id="conversion-events">
  ### 전환 이벤트
</div>

[전환 이벤트](https://developer.x.com/en/docs/x-ads-api/measurement/api-reference/conversion-event)를 X에 보고하고 어트리뷰션 데이터를 받으려면 파트너는 모바일 앱 ID, 전환의 유형과 타임스탬프, 그리고 광고/디바이스 ID의 해시값을 전송해야 합니다. 이 데이터를 제공하면 X 플랫폼에 전환 이벤트가 기록되며, 어트리뷰션이 가능한 경우 해당 어트리뷰션 정보도 반환됩니다. 또한 광고주는 구매 금액, 상품 ID, 가입(등록) 방식과 같은 전환 이벤트에 대한 추가 메타데이터를 제공할 수도 있습니다.

<div id="post-conversion_event">
  #### POST conversion&#95;event
</div>

모바일 측정 전환 이벤트를 기록합니다. 응답에는 X 또는 X Audience Platform(TAP) 기여도가 표시됩니다.

이는 [GET conversion&#95;attribution](https://developer.x.com/en/docs/x-ads-api/measurement/api-reference/conversion-attribution#get-conversion-attribution) 엔드포인트와 관련이 있습니다.

응답에는 X 기여도, TAP 기여도, 또는 기여도 없음 중 하나가 표시됩니다. X 기여도가 없는 경우 `twitter_attribution` 노드는 항상 존재하며 값이 `null`입니다(아래 예시 응답 참조). TAP 기여도가 부여된 경우 `tpn_attribution` 노드가 존재하며 해당 값이 설정됩니다. 자세한 내용은 [TAP overview](https://developer.x.com/en/docs/x-ads-api/measurement/overview/twitter-audience-platform)를 참조하십시오.

각 전환 이벤트와 연결된 메타데이터를 설정하기 위한 여러 선택적 매개변수가 있습니다. 이 메타데이터는 기여도 계산에는 아무런 영향을 주지 않습니다.

<div id="resource-url">
  ##### 리소스 URL
</div>

`https://ads-api.x.com/12/conversion_event`

| user&#95;payment&#95;info  <br />*optional* | 이 이벤트와 연관된 앱에 사용자의 결제 정보가 저장되어 있는지를 나타내는 불리언 값입니다.<br /><br />Type: bool<br /><br />Possible values: `true` or `false` |
| view&#95;through&#95;window  <br />*optional* | 이 이벤트에 대한 뷰스루 윈도우 기간(일 단위)입니다.<br /><br />**Note**:<br /><br />`click_window` 값은 view&#95;through&#95;window 이상이어야 합니다.<br /><br />Type: int<br /><br />Default: `1`  <br />Possible values: `0`, `1`, `7`, `14`, `30` |

  #### GET accounts/:account&#95;id/app&#95;event&#95;tags
</div>

  ##### 리소스 URL
</div>

`https://ads-api.x.com/11/accounts/:account_id/app_event_provider_configurations`

##### 매개변수

| Name | Description |
| :--- | :--- |
| account&#95;id  <br />*required* | 해당 광고 계정을 식별하는 값입니다. 리소스의 경로에 포함되며, [GET accounts](/ko/x-ads-api/campaign-management#accounts)를 제외한 모든 Advertiser API 요청에서 일반적으로 필수 매개변수입니다. 지정한 계정은 인증된 사용자와 연결되어 있어야 합니다.<br /><br />Type: string<br /><br />Example: `18ce54d4x5t` |
| provider&#95;advertiser&#95;id  <br />*required* | 제공자 사이트에서의 광고주 식별자입니다.<br /><br />Type: string<br /><br />Example: `client1` |

<div id="example-request">
  ##### 요청 예시
</div>

`POST https://ads-api.x.com/11/accounts/18ce54d4x5t/app_event_provider_configurations?provider_advertiser_id=client1`

<div id="get-conversion_attribution">
  #### GET conversion&#95;attribution
</div>

전환 이벤트를 기록하지 않고 전환 기여도를 확인하기 위해 X에 쿼리를 수행합니다. 응답에는 X 기여 여부가 표시됩니다.

이는 [POST conversion&#95;event](https://developer.x.com/en/docs/x-ads-api/measurement/api-reference/conversion-event#post-conversion-event) 엔드포인트와 관련이 있습니다.

응답에서는 X, TAP, 또는 기여 없음 중 하나만 기여로 표시됩니다. `twitter_attribution` 노드는 항상 존재하며, X 기여도가 없을 경우 값이 `null`이며, 아래 예시 응답에서 볼 수 있듯이 X 기여도가 있을 경우 해당 값으로 채워집니다. TAP 기여도가 표시되는 경우 `tpn_attribution` 노드가 존재하며 그에 맞게 채워집니다. 자세한 내용은 [TAP overview](https://developer.x.com/en/docs/twitter-ads-api/measurement/twitter-audience-platform/overview)를 참조하십시오.

<div id="resource-url">
  ##### 리소스 URL
</div>

`https://ads-api.x.com/12/conversion_attribution`

<div id="parameters">
  ##### Parameters
</div>

| Name | Description |

| conversion&#95;time  <br />*required* | 밀리초까지 포함한 ISO-8601 타임스탬프 형식의 전환 이벤트 발생 시각입니다.<br /><br />Type: string<br /><br />Example: `2014-05-22T02:38:28.103Z` |
| conversion&#95;type  <br />*required* | 전환 이벤트 유형입니다.<br /><br />Type: enum<br /><br />Possible values: `ACHIEVEMENT_UNLOCKED`, `ADDED_PAYMENT_INFO`, `ADD_TO_CART`, `ADD_TO_WISHLIST`, `CHECKOUT_INITIATED`, `CONTENT_VIEW`, `INSTALL`, `INVITE`, `LEVEL_ACHIEVED`, `LOGIN`, `PURCHASE`, `RATED`, `RESERVATION`, `RE_ENGAGE`, `SEARCH`, `SHARE`, `SIGN_UP`, `SPENT_CREDITS`, `TUTORIAL_COMPLETE`, `UPDATE` |

| click&#95;window  <br />*optional* | 이 이벤트에 대한 클릭 윈도우(일 단위)입니다.<br /><br />Type: int<br /><br />Default: `14`  <br />Possible values: `1`, `7`, `14`, `30` |

| view&#95;through&#95;window  <br />*optional* | 이 이벤트에 대한 뷰스루(노출 기반) 윈도우(일 단위)입니다.<br /><br />Type: int<br /><br />Default: `1`  <br />Possible values: `0`, `1`, `7`, `14`, `30` |
