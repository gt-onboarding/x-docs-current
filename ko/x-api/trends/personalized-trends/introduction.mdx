Personalized Trends endpoint를 사용하면 인증 중인 사용자 계정에 대한 맞춤형 트렌드를 가져올 수 있습니다.

<Note>
  **참고:** Personalized trends는 X Premium에 가입된 사용자 계정에 대해서만 이 endpoint에서 반환됩니다.
</Note>

<Note>
  **계정 설정**

  이 endpoint에 액세스하려면 다음이 필요합니다.

  * 승인된 [개발자 계정](https://developer.x.com/en/portal/petition/essential/basic-info)
  * [프로젝트](/ko/resources/fundamentals/projects) 내에 있는 [개발자 앱](/ko/resources/fundamentals/developer-apps)의 키와 토큰을 사용한 인증 

  X API v2 endpoint에 대한 액세스 방법을 더 알아보려면 [시작 가이드](/ko/x-api/getting-started/getting-access)를 참고하세요.
</Note>

<div id="getting-started">
  ### 시작하기
</div>

<div id="authentication">
  #### 인증
</div>

이 엔드포인트는 [OAuth 1.0a 사용자 컨텍스트](/ko/resources/fundamentals/authentication) 또는 [OAuth 2.0 Authorization Code with PKCE](resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2)를 사용해 인증할 수 있습니다.
 

<div id="making-a-request">
  #### 요청 보내기
</div>

아래와 같이 Personalized Trends 엔드포인트를 호출할 수 있습니다:

`curl 'https://api.x.com/2/users/personalized_trends' --header 'Authorization: ••••••'`

요청이 성공하면 아래와 같은 JSON 응답을 확인할 수 있습니다:

```
{
    "data": [
        {
            "category": "Bitcoin",
            "post_count": "42K posts",
            "trend_name": "MicroStrategy's $42 Billion Bitcoin Investment",
            "trending_since": "5 hours ago"
        },
        {
            "category": "Music",
            "post_count": "2K posts",
            "trend_name": "Drake's 'No Face' Music Video Release",
            "trending_since": "Trending now"
        },
        {
            "category": "Weather",
            "post_count": "1.2K posts",
            "trend_name": "Super Typhoon Leon Impacts Taiwan and Batanes",
            "trending_since": "Trending now"
        }
    ]
}
```

**참고:** 요청을 보낸 인증 사용자가 X Premium을 구독하지 않은 경우, 대신 다음과 같은 응답이 반환됩니다.

```
{
    "data": [
        {
            "category": "Unknown",
            "post_count": "Unknown",
            "trending_since": ""
        }
    ]
}
```
