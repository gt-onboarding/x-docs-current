Trends 조회 엔드포인트는 개발자가 where-on-earth id (WOEID)로 지정된 위치에 대한 Trends를 조회할 수 있도록 합니다.

<Note>
  **참고**: WOEID는 Yahoo에서 만든 레거시 식별자로, 현재는 사용이 중단되었습니다. X API는 도시 및 국가별 트렌드 위치를 식별하기 위해 이 숫자 값을 사용합니다. 레거시 [블로그 게시물](https://blog.x.com/engineering/en_us/a/2010/woeids-in-twitters-trends.html) 또는 [보관된 데이터](https://archive.org/details/geoplanet_data_7.10.0.zip0.)를 참조하세요.
</Note>

가능한 경우, 많은 트렌드에 대해 최근 24시간 동안의 `tweet_count`도 함께 반환됩니다.

이 엔드포인트는 app-auth 인증을 지원하며, 15분당 75회의 요청 한도가 적용됩니다.

<div id="getting-started">
  ### 시작하기
</div>

이 엔드포인트를 사용하려면 [bearer token](https://developer.x.com\(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token\))과 [developer portal](https://developer.x.com/en/portal/dashboard)에서 발급받은 Bearer Token이 필요합니다. Bearer Token을 발급받은 후에는 아래와 같이 usage API를 호출할 수 있습니다:

```
      curl 'https://api.x.com/2/trends/by/woeid/26062' --header 'Authorization: Bearer XXXXX'
```

요청이 성공하면 아래와 같은 JSON 응답을 볼 수 있습니다.

**계정 설정**

이 엔드포인트에 액세스하려면 다음이 필요합니다:

* 승인된 [개발자 계정](https://developer.x.com/en/portal/petition/essential/basic-info)
