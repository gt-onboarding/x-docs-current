<div id="overview">
  ## 개요
</div>

매일 수천 명의 개발자가 X API에 요청을 보냅니다. 이러한 요청의 방대한 양을 관리하기 위해, 일정 기간 동안 보낼 수 있는 요청 수에 제한을 두고 있습니다. 이러한 제한은 개발자 커뮤니티가 신뢰하는 안정적이고 확장 가능한 API를 제공하는 데 도움이 됩니다. 

허용되는 최대 요청 수는 특정 기간 또는 시간 창과 같은 시간 간격을 기준으로 합니다. 예를 들어, 어떤 엔드포인트의 rate limit이 15분당 요청 900건이라면, 임의의 15분 간격 동안 최대 900건의 요청을 보낼 수 있습니다.
