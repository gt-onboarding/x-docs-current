당사의 Enterprise API는 X 데이터에 의존하는 고객에게 최고 수준의 액세스와 안정성을 제공합니다. Premium 단계를 넘어 규모를 확장하면서 더 안정적인 액세스, 맞춤형 패키지 또는 장기 계약이 필요한 경우에 적합합니다. Enterprise API 액세스에는 전담 계정 관리자와 맞춤형 기술 지원이 제공됩니다.

Basic 및 Pro 솔루션만으로는 요구 사항을 충족하기에 충분하지 않다고 판단되는 경우, 아래 링크에서 Enterprise 제품에 대한 액세스를 신청할 수 있습니다:

<div id="available-endpoints">
  ## 사용 가능한 엔드포인트
</div>

| 실시간 | [PowerTrack API](/ko/x-api/enterprise-gnip-2.0/powertrack-api) | 다양한 연산자와 규칙을 사용해 필터링할 수 있는, 실시간 전체 공개 게시물에 대한 완전한 커버리지를 제공합니다. |
| :--- | :--- | :--- |
| [Decahose API](/ko/x-api/enterprise-gnip-2.0/fundamentals/decahose-api) | 전체 공개 게시물과 해당 좋아요 중 10%에 해당하는 무작위 샘플을 스트리밍합니다. |
| :--- | :--- | :--- |
| [Account Activity API](/ko/x-api/enterprise-gnip-2.0/fundamentals/account-activity) | 사용자를 구독하여 게시물, 다이렉트 메시지, 좋아요, 팔로우 등 다양한 활동을 수신합니다. |
| :--- | :--- | :--- |
| 과거 데이터 | [Search API](/ko/x-api/enterprise-gnip-2.0/fundamentals/search-api) | 고급 필터링 도구를 사용하여 지난 30일 또는 전체 공개 게시물 전체 아카이브를 검색합니다. |
| [Historical PowerTrack API](/ko/x-api/enterprise-gnip-2.0/powertrack-api) | 경제적인 배치 액세스 방식을 사용하여 전체 공개 게시물 아카이브를 필터링합니다. |
| 인사이트 | [Engagement API](/ko/x-api/enterprise-gnip-2.0/fundamentals/engagement-api) | 공개 게시물에 대한 노출 수와 참여 수를 측정합니다. |
| 컴플라이언스 | [Compliance Firehose API](/ko/x-api/enterprise-gnip-2.0/fundamentals/firehouse) | 모든 컴플라이언스 이벤트를 스트리밍합니다. |
| 사용량 | [Usage API](/ko/x-api/enterprise-gnip-2.0/fundamentals/usage) | Enterprise 제품 전반의 활동 사용량에 대한 프로그래매틱 액세스를 제공합니다. |

<div id="important-concepts">
  ## 핵심 개념
</div>

<div id="gnip-console">
  ### Gnip Console
</div>

[Gnip Console](/ko/x-api/enterprise-gnip-2.0/fundamentals/overview)를 통해 Enterprise 제품과 계정 접근 권한을 관리할 수 있습니다. 

<div id="enrichments">
  ### Enrichments
</div>

유료 티어를 사용하는 개발자를 위해 Premium API의 응답 페이로드에 메타데이터를 추가했습니다. 확장 URL, 투표 데이터, 프로필 위치 정보에 대해 더 알아보려면 [enrichments 섹션](/ko/x-api/enterprise-gnip-2.0/fundamentals/data-enrichments)을 방문하세요.
 

<div id="advanced-filtering-tools">
  ### 고급 필터링 도구
</div>

Premium Search API는 표준 검색 엔드포인트에서는 사용할 수 없는 고급 연산자 집합을 제공합니다. 이러한 연산자 중 상당수는 무료 샌드박스 사용자에게도 제공되지만, 유료 플랜을 사용하는 경우 더욱 강력한 연산자 집합을 사용할 수 있습니다. 자세한 내용은 [규칙 및 필터 섹션](/ko/x-api/enterprise-gnip-2.0/fundamentals/rules-filtering#getting-started-with-enterprise-rules-and-queries)에서 확인하세요.

<div id="explore-x-api-v2">

</div>
