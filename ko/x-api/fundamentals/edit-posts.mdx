<div id="introduction">
  # Introduction
</div>

X API v2 엔드포인트는 수정된 게시물 메타데이터를 제공합니다. *게시물 수정(Edit Posts)* 기능은 2022년 9월 1일 X 직원 대상 테스트로 처음 도입되었습니다. 해당 날짜부터 자격을 갖춘 게시물은 30분 동안 최대 5회까지 수정할 수 있습니다. &#95;2022년 9월 29일 이후에 생성된 모든 게시물 객체&#95;에는 해당 게시물이 한 번도 수정된 적이 없더라도 게시물 수정 메타데이터가 포함됩니다. 게시물이 수정될 때마다 새 게시물 ID가 생성됩니다. 게시물의 수정 내역은 원본 ID부터 시작하여 이러한 ID들을 연결해 설명할 수 있습니다. 또한 수정 체인에 속한 게시물 중 하나라도 삭제되면, 해당 체인에 속한 모든 게시물이 함께 삭제됩니다.

X API v2를 사용하면 개발자는 다음을 확인할 수 있습니다:

* 게시물이 생성 시점에 수정 가능한 상태였는지 여부. 예를 들어 설문조사가 포함된 게시물이나 예약된 게시물 등은 수정할 수 없습니다.
* 게시물은 30분 동안 수정할 수 있으며, 최대 5번까지 수정할 수 있습니다. 수정 가능한 게시물의 경우 남은 수정 가능 시간과 남은 수정 횟수를 확인할 수 있습니다.
* 현재 보고 있는 게시물이 수정된 버전인지 여부 (대부분의 경우 API는 특정 과거 버전이 게시물 ID로 요청되지 않는 한 게시물의 최신 버전을 반환합니다).
* 게시물의 전체 수정 내역.
* 게시물 각 버전에 귀속된 참여(engagement) 정보.

게시물의 수정 내역은 세 가지 구성 요소로 이루어집니다:

1. 기본적으로 게시물 페이로드에는 게시물 수정 내역에 포함되는 게시물 ID 배열이 포함됩니다. 이 정보는 게시물 페이로드의 기본 필드인 `edit_history_tweet_ids`로 지정됩니다. 이 배열에는 최소 하나의 ID, 즉 원본 미수정 게시물의 ID가 포함됩니다. ID가 하나만 있다면, 해당 게시물에는 수정 내역이 없다는 의미입니다. 
2. 게시물이 생성 시점에 수정 가능했는지 여부, 게시물을 수정할 수 있는 남은 시간(있는 경우), 그리고 남은 수정 횟수 등의 정보를 얻으려면 [tweet.fields](/ko/x-api/fundamentals/fields) 파라미터에 `edit_controls`를 지정하면 됩니다.
3. 마지막으로 [expansion](/ko/x-api/fundamentals/expansions) 파라미터를 사용하여 `edit_history_tweet_ids`를 지정함으로써, 게시물의 수정 내역에 속한 각 게시물에 대한 게시물 객체를 가져올 수 있습니다.

대부분의 게시물은 수정이 가능합니다. 그러나 다음 유형의 게시물은 수정할 수 없습니다: 

* 프로모션된 게시물
* 설문조사가 포함된 게시물
* 자신이 시작한 스레드가 아닌 답글(non-self-thread reply) 게시물
* 리트윗(인용 트윗은 수정 가능)
* nullcast 게시물
* 커뮤니티 게시물
* Superfollow 게시물
* 협업 게시물

Examples

***

아래 예제는 개발자가 X API v2를 사용하여 수정된 게시물 메타데이터를 요청하는 방법을 보여줍니다. 

**Note:** 아래 예제에서는 User Post Timeline 엔드포인트를 사용하지만, 게시물을 반환하는 모든 엔드포인트(예: 게시물 조회, 검색, filtered stream 등)에 대해 동일한 파라미터(fields 및 expansions)를 사용하여 이 메타데이터를 요청할 수 있습니다.

<div id="default-behavior">
  ### 기본 동작
</div>

기본적으로, X API v2에서 게시물 객체를 반환하는 endpoint에 API 요청을 보내면 다음 정보가 반환됩니다:

* 게시물 ID
* 게시물 텍스트
* 게시물의 편집 이력에 포함된 게시물 ID의 배열. ID가 하나만 제공된 경우, 해당 게시물은 아직 편집되지 않았다는 의미입니다.

**요청:**

`curl --request GET 'https://api.x.com/2/users/:id/tweets' --header 'Authorization: Bearer $BEARER_TOKEN'`

**응답 예시:**

```{
  "data": [
    {
      "id": "1514991667853602823",
      "text": "we have an edit button",
      "edit_history_tweet_ids": ["1514991667853602822", "1514991667853602823"]
    }
  ]
}
```

<div id="getting-additional-data-with-edit_controls">
  ### edit&#95;controls로 추가 데이터 가져오기
</div>

게시물이 생성될 때 편집이 가능했는지 여부와 게시물을 편집할 수 있는 남은 시간 등과 같은 추가 편집 메타데이터가 필요하다면, `tweet.fields` 파라미터를 사용하고 값을 `edit&#95;control`로 설정하여 이 정보를 요청할 수 있습니다.

**요청:**

`curl --request GET 'https://api.x.com/2/users/:id/tweets?tweet.fields=edit_control' --header 'Authorization: Bearer $BEARER_TOKEN'`

**예시 응답:**

```
  {
  "data": [
    {
      "id": "1514991667853602823",
      "text": "we have an edit button",
      "edit_history_tweet_ids": ["1514991667853602822", "1514991667853602823"],
      "edit_controls": {
        "is_edit_eligible": true,
        "editable_until": "2022-04-21 09:35:20.311",
        "edits_remaining": 4
      }
    }
  ]
}
```

<div id="getting-post-objects-for-all-posts-in-a-posts-edit-history">
  ### 특정 게시물의 수정 이력에 포함된 모든 게시물에 대한 Post 객체 가져오기
</div>

위의 예시에서는 특정 게시물의 수정 이력에 있는 게시물 ID 배열을 제공합니다. 이러한 각 게시물 ID에 대한 실제 Post 객체가 필요하다면, expansions 파라미터를 사용해 값을 `edit_history_tweet_ids`로 설정하면 됩니다. 수정 이력을 구성하는 Post 객체는 &quot;includes&quot; 객체에 제공됩니다. 

**요청:**

```bash
      curl --location --request GET 'https://api.x.com/2/users/:id/tweets?tweet.fields=edit_control&expansions=edit_history_tweet_ids' --header 'Authorization: Bearer $BEARER_TOKEN'`
```

**예시 응답:**

```
  {
  "data": [
    {
      "id": "1514991667853602823",
      "text": "we have an edit button",
      "edit_history_tweet_ids": ["1514991667853602822", "1514991667853602823"],
      "edit_controls": {
        "is_edit_eligible": true,
        "editable_until": "2022-04-21 09:35:20.311",
        "edits_remaining": 4
      }
    }
  ],
  "includes": {
    "tweets": [
      {
        "id": "1514991667853602822",
        "text": "we need an eidt button",
        "edit_history_tweet_ids": [
          "1514991667853602822",
          "1514991667853602823"
        ],
        "edit_controls": {
          "is_edit_eligible": true,
          "editable_until": "2022-04-21 09:35:20.311",
          "edits_remaining": 4
        }
      }
    ]
  }
}
```
