X API는 X 데이터를 프로그래밍 방식으로 조회·분석할 수 있을 뿐 아니라, X에서 이루어지는 대화를 기반으로 기능과 서비스를 구축하는 데 사용할 수 있습니다.

수년에 걸쳐 X API는 개발자가 공개 대화에 대한 접근을 확장하고, 이를 활용·연구할 수 있도록 추가적인 액세스 레벨을 제공하는 방향으로 발전해 왔습니다.

최근 우리는 X API v2를 출시했습니다. X API v2는 현대적인 기반과 새로운 고급 기능을 제공하며, [Basic access](https://developer.x.com/en/portal/products/basic)에 빠르게 가입해 시작할 수 있도록 설계되었습니다.

아래의 세 개 탭에서는 X API의 다양한 버전과 액세스 레벨, v2에서 새로 제공되는 기능, 그리고 API를 사용하여 가져오고 생성하며 삭제·조정할 수 있는 X 리소스에 대해 설명합니다.

  <Tab title="v2의 새로운 기능">
    ## v2의 새로운 기능

    X API v2는 2012년 이후 X API에 이루어진 가장 큰 업그레이드입니다. 이를 통해 새로운 고급 기능과 함께 빠르고 무료로 API에 접근할 수 있습니다.

    v2에서 사용할 수 있는 일부 기능은 다음과 같습니다.

    ### 새로운 엔드포인트

    X API v2에 완전히 새로운 엔드포인트 세트를 출시했습니다. 새로 추가된 엔드포인트를 포함한 v2 엔드포인트 전체 목록은 X API 엔드포인트 맵 가이드에서 확인할 수 있습니다.

    [X API 엔드포인트 맵 방문 &gt;](/ko/x-api/migrate/x-api-endpoint-map)

    ### 새롭고 더 자세한 데이터 객체

    데이터를 더 쉽게 탐색하고 파싱할 수 있도록 다양한 새로운 개선 사항을 적용해 데이터 객체를 현대화했습니다.

    [데이터 사전 방문 &gt;](/ko/x-api/fundamentals/data-dictionary)

    ### 원하는 객체와 필드만 가져오는 데 도움이 되는 새로운 파라미터

    데이터 엔드포인트에 fields 및 expansions 파라미터를 추가하여, 기본적으로 반환되는 필드를 넘어 연관된 객체와 필드를 요청할 수 있도록 했습니다.

    [fields 및 expansions 사용 방법 알아보기 &gt;](/ko/x-api/fundamentals/data-dictionary#how-to-use-fields-and-expansions)

    ### 고급 메트릭

    공개 및 비공개 메트릭을 모두 요청하여, 페이로드 내에서 바로 게시물, 사용자, 미디어, 투표의 성과를 더 쉽게 파악할 수 있습니다. 여기에는 노출 수, 동영상 조회 수, 사용자 프로필 및 URL 클릭 수 등이 포함되며, 일부는 유기적(organic) 및 프로모션(promoted) 환경으로 구분됩니다.

    [메트릭 자세히 알아보기 &gt;](/ko/x-api/fundamentals/metrics)

    ### 서로 다른 주제를 포함한 게시물 필터링 및 식별

    search Posts 또는 filtered stream을 사용할 때 엔티티 및 컨텍스트 연산자를 사용해 주제별로 필터링할 수 있습니다. 또한 분석에 도움이 되도록 이 토픽들을 게시물 페이로드 내에 함께 제공합니다.

    [게시물 주석(Post annotations) 자세히 알아보기 &gt;](/ko/x-api/fundamentals/post-annotations)

    ### 답글 스레드에 속한 게시물 필터링 및 식별

    search Posts, filtered stream, Post lookup을 사용할 때 특정 게시물이 대화 스레드의 일부인지 더 쉽게 식별할 수 있습니다. 또한 게시물 필드 `reply_settings` 를 통해 해당 게시물에 대화 답글 설정이 지정되었는지 판단할 수 있는 기능을 추가했습니다.

    [대화 추적(conversation tracking) 자세히 알아보기 &gt;](/ko/x-api/fundamentals/conversation-id)

    ### 그리고 이 외에도...

    * 높은 신뢰도의 스팸 필터링
    * URL 분석을 쉽게 하기 위해 단축 URL을 원래 URL로 완전 복원
    * 사용 중단된 필드를 제거하고 라벨을 현대화하여 단순화된 JSON 응답 객체
    * 스트리밍 엔드포인트를 위한 복구 및 이중화 기능
    * 검색 쿼리에서 일치하는 공개 및 사용 가능한 게시물을 100% 반환
    * 연결을 끊지 않고도 변경할 수 있는 스트리밍 &quot;rules&quot;
    * filtered stream 및 검색을 위한 더 강력한 쿼리 언어
    * 새로운 라이브러리를 빌드하고 변경 사항을 더 투명하게 추적할 수 있는 OpenAPI 스펙
    * 개발자, 연구자, 비즈니스, 그리고 X를 사용하는 사람들의 요구를 충족하기 위해 플랫폼이 발전함에 따라 새로운 기능 및 엔드포인트에 더 빠르게 대응하는 API 지원
  </Tab>

    ## X API 플랫폼 리소스

    API 설계에서 리소스란 연관된 데이터, 다른 리소스와의 관계, 그리고 그 리소스에 대해 동작하는 메서드 집합을 가진 개체를 의미합니다. 예를 들어, 게시물은 다양한 도구를 사용해 생성, 삭제 또는 조회할 수 있는 리소스로, 과거의 게시물을 검색하거나 실시간으로 가져오는 등의 방식으로 활용할 수 있습니다.

    X API는 다음과 같은 플랫폼의 다양한 리소스를 생성, 삭제, 조회 또는 수정할 수 있는 접근을 제공합니다:

    | Resource     | Description                                                                                                               |
    | :----------- | :------------------------------------------------------------------------------------------------------------------------ |
    | **게시물**      | 수백만 개의 게시물을 활용해 공개 대화를 이해하거나, 직접 게시물을 작성해 대화에 참여할 수 있습니다.  현재 제공 범위:  - X API v2  - Enterprise  - Standard v1.1           |
    | **사용자**      | X 사용자를 관리하거나 조회하여 네트워크를 분석하고, 오디언스를 파악하거나, 긍정적인 온라인 관계를 형성할 수 있습니다.  현재 제공 범위:  - X API v2  - Enterprise  - Standard v1.1 |
    | **Spaces**   | X Spaces와 참여자를 조회하고 검색해, 사람들이 흥미롭고 관련성 높은 오디오 대화를 찾을 수 있도록 도울 수 있습니다.  현재 제공 범위:  - X API v2                              |
    | **다이렉트 메시지** | 다이렉트 메시지를 보내고 받아 고객 이슈를 분류하고, 환영 메시지를 전송하거나, 긍정적인 인간적 상호작용을 만들 수 있습니다.  현재 제공 범위:  - Standard v1.1                        |
    | **리스트**      | 업계 전문가나 영향력 있는 계정의 동향을 파악하거나, 팔로우하는 계정을 정리하기 위해 계정 리스트를 큐레이션하고 관리할 수 있습니다.  현재 제공 범위:  - X API v2  - Standard v1.1        |
    | **트렌드**      | 지리적 트렌드를 먼저 파악해 업계 움직임을 짚어내고, 인기 주제를 발견하거나, 최신 유행을 앞서 파악할 수 있습니다.  현재 제공 범위:  - Standard v1.1                             |
    | **미디어**      | 미디어 객체를 업로드해 창의적인 콘텐츠를 공유하고, 인터랙티브한 경험을 만들거나, 접근성 도구를 구축할 수 있습니다.  현재 제공 범위:  - Standard v1.1                             |
    | **장소**       | 내 주변과 전 세계에서 무슨 일이 일어나고 있는지 이해하기 위해 장소를 검색할 수 있습니다.  현재 제공 범위:  - Standard v1.1                                           |
  </Tab>
