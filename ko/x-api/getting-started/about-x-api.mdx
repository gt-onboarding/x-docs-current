X API는 X 데이터에 프로그래밍 방식으로 접근하여 검색 및 분석할 수 있게 해주며, X에서 이루어지는 대화를 바탕으로 다양한 기능을 구현하는 데 사용할 수 있습니다.

수년에 걸쳐 X API는 개발자가 공개 대화에 대한 액세스를 확장하고 연구할 수 있도록 추가 액세스 수준을 제공하는 방향으로 발전해 왔습니다.

최근 우리는 X API v2를 출시했습니다. X API v2는 최신 기반, 새로운 고급 기능, 그리고 [Basic access](https://developer.x.com/en/portal/products/basic)에 대한 빠른 온보딩을 제공합니다.

아래의 세 개 탭에서는 X API의 서로 다른 버전과 액세스 수준, v2에서 새로워진 점, 그리고 API를 사용해 가져오고, 생성하고, 삭제하고, 관리할 수 있는 X 리소스에 대해 설명합니다.

  <Tab title="v2의 새로운 사항">
    ## v2의 새로운 점

    X API v2는 2012년 이후 X API에 이루어진 가장 큰 업그레이드입니다. 이를 통해 다양한 신규 및 고급 기능뿐 아니라 빠르고 무료인 API 액세스를 제공합니다.

    v2에서 사용 가능한 기능에는 다음과 같은 것들이 있습니다:

    ### 새로운 엔드포인트

    X API v2에 완전히 새로운 엔드포인트 세트를 추가했습니다. 신규 엔드포인트를 포함한 v2 엔드포인트 전체 목록은 X API 엔드포인트 맵 가이드에서 확인할 수 있습니다.

    [Visit the X API endpoint map &gt;](/ko/x-api/migrate/x-api-endpoint-map)

    ### 새롭고 더 상세한 데이터 객체

    데이터를 더 쉽게 탐색하고 파싱할 수 있도록 다양한 개선 사항을 적용해 데이터 객체를 현대화했습니다.

    [Visit the data dictionary &gt;](/ko/x-api/fundamentals/data-dictionary)

    ### 원하는 객체와 필드만 조회할 수 있도록 돕는 새로운 파라미터

    데이터 엔드포인트에 fields 및 expansions 파라미터를 추가하여, 기본으로 반환되는 필드를 넘어 연관된 객체와 필드를 요청할 수 있도록 했습니다.

    [Learn how to use fields and expansions &gt;](/ko/x-api/fundamentals/data-dictionary#how-to-use-fields-and-expansions)

    ### 고급 메트릭

    노출 수(impressions), 동영상 조회 수(video views), 사용자 프로필, URL 클릭 수 등 공개 및 비공개 메트릭을 모두 요청하여, 페이로드 내에서 직접 게시물, 사용자, 미디어, 투표의 성과를 더 쉽게 파악할 수 있습니다. 이 중 일부는 자연 노출(organic)과 프로모션(promoted) 컨텍스트로 구분되어 제공됩니다.

    [Learn more about metrics &gt;](/ko/x-api/fundamentals/metrics)

    ### 서로 다른 주제를 포함하는 게시물 필터링 및 식별

    search Posts 또는 filtered stream을 사용할 때 entity 및 context 연산자를 이용해 주제별로 필터링할 수 있습니다. 또한 분석에 도움이 되도록 이러한 주제 정보를 게시물 페이로드에 함께 제공합니다.

    [Learn more about Post annotations &gt;](/ko/x-api/fundamentals/post-annotations)

    ### 답글 스레드에 속한 게시물 필터링 및 식별

    search Posts, filtered stream, Post lookup을 사용할 때, 게시물이 대화 스레드의 일부인지 더 쉽게 식별할 수 있습니다. 또한 게시물 필드 `reply_settings`를 통해 해당 게시물에 대화 답글 설정이 적용되었는지 확인할 수 있는 기능도 추가했습니다.

    [Learn more about conversation tracking &gt;](/ko/x-api/fundamentals/conversation-id)

    ### 그리고 이 외에도 많은 것들이 있습니다...

    * 높은 신뢰도의 스팸 필터링
    * URL 분석을 더 쉽게 할 수 있도록 단축 URL을 완전히 해제(unwind)
    * 사용 중단(deprecated)된 필드를 제거하고 라벨을 현대화하여 단순해진 JSON 응답 객체
    * 스트리밍 엔드포인트를 위한 복구 및 이중화(redundancy) 기능
    * 검색 쿼리에서 일치하는 공개·사용 가능 게시물 100% 반환
    * 연결을 끊지 않고도 변경할 수 있는 스트리밍 &quot;rules&quot;
    * filtered stream 및 검색을 위한 더 표현력이 풍부한 쿼리 언어
    * 새로운 라이브러리를 빌드하고 변경 사항을 더 투명하게 추적할 수 있도록 하는 OpenAPI 명세
    * 개발자, 연구자, 비즈니스, 그리고 X를 사용하는 사람들의 요구에 맞춰 플랫폼이 발전함에 따라 새로운 기능과 엔드포인트를 더 빠르게 지원하는 API
  </Tab>

  <Tab title="X API 리소스">
    ## X API 플랫폼 리소스

    API 설계에서 리소스는 관련 데이터와 다른 리소스와의 관계, 그리고 그 리소스에 대해 동작하는 메서드 집합을 가진 엔터티를 의미합니다. 예를 들어, 게시물은 다양한 도구를 사용해 생성, 삭제 또는 조회할 수 있는 리소스로, 과거의 게시물을 검색하거나 실시간으로 받아오는 등 여러 방식으로 활용할 수 있습니다.

    X API는 플랫폼에서 다음과 같은 다양한 리소스를 생성, 삭제, 조회 또는 조작할 수 있는 기능을 제공합니다.

    | Resource            | Description                                                                                                            |
    | :------------------ | :--------------------------------------------------------------------------------------------------------------------- |
    | **Posts**           | 수백만 개의 게시물을 활용해 공개 대화를 이해하거나, 직접 게시물을 만들어 대화에 참여할 수 있습니다. 현재 제공 범위:  - X API v2  - Enterprise  - Standard v1.1         |
    | **Users**           | X 사용자를 관리하거나 조회해 네트워크를 분석하고, 오디언스를 파악하며, 긍정적인 온라인 관계를 구축할 수 있습니다. 현재 제공 범위:  - X API v2  - Enterprise  - Standard v1.1 |
    | **Spaces**          | 사람들이 흥미롭고 관련성 높은 오디오 대화를 찾을 수 있도록 X Spaces와 참가자를 조회하고 검색할 수 있습니다. 현재 제공 범위:  - X API v2                                |
    | **Direct Messages** | 고객 이슈를 분류하고, 환영 메시지를 보내거나, 긍정적인 인간 간 상호작용을 만들기 위해 다이렉트 메시지를 주고받을 수 있습니다. 현재 제공 범위:  - Standard v1.1                    |
    | **Lists**           | 업계 전문가나 영향력 있는 목소리를 파악하거나, 팔로우하는 계정을 정리하기 위해 계정 리스트를 큐레이션하고 관리할 수 있습니다. 현재 제공 범위:  - X API v2  - Standard v1.1         |
    | **Trends**          | 지리적 트렌드를 먼저 파악해 업계 변화를 짚어 보고, 인기 있는 주제를 발견하거나, 최신 유행을 앞서갈 수 있습니다. 현재 제공 범위:  - Standard v1.1                           |
    | **Media**           | 창의력을 공유하고, 인터랙티브한 경험을 제공하거나, 접근성 도구를 구축하기 위해 미디어 객체를 업로드할 수 있습니다. 현재 제공 범위:  - Standard v1.1                           |
    | **Places**          | 내 주변과 전 세계에서 무슨 일이 일어나고 있는지 이해하기 위해 장소를 검색할 수 있습니다. 현재 제공 범위:  - Standard v1.1                                         |
  </Tab>
