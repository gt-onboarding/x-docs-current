<div id="learn-about-whats-possible">
  ## 가능한 기능 살펴보기
</div>

상단 내비게이션에서 접근할 수 있는 활용 사례, 솔루션, 제품 페이지는 이 플랫폼에서 어떤 일을 할 수 있는지 높은 수준에서 파악하는 데 좋은 출발점입니다.

또한 다양한 활용 사례에 맞춰 하나 이상의 플랫폼 도구를 어떻게 조합해 사용할 수 있는지 파악하는 데 도움이 되는 [튜토리얼](/ko/tutorials)도 여러 개 제공하고 있습니다.

문서 구조에서 가장 하위 수준에는 엔드포인트 및 도구 문서가 있으며, 각각 해당 도구의 기능과 일반적인 활용 사례를 간략히 소개하는 도입부 내용을 포함하고 있습니다. 예를 들어, 다음은 [Search Posts](/ko/x-api/posts/search/introduction)에 대한 X API v2 엔드포인트 섹션입니다.

<div id="integrate-an-endpoint-into-your-system">
  ### 시스템에 엔드포인트 통합하기
</div>

엔드포인트를 손쉽게 통합할 수 있도록 다양한 도구, 라이브러리, 통합 가이드를 제공하고 있습니다. 가장 간단한 통합 방법은 [도구 및 라이브러리](/ko/x-api/tools-and-libraries/overview)를 사용하는 것으로, 이는 [OAuth 1.0a User Context 인증](/ko/resources/fundamentals/authentication#oauth-1-0a-2), 페이지네이션, [rate limit 처리](/ko/resources/fundamentals/rate-limits)와 같이 복잡한 기능을 자동으로 처리합니다.

처음부터 직접 솔루션을 구축하고 싶다면, 엔드포인트 통합 가이드에서 엔드포인트를 시스템에 통합하는 데 필요한 기능과 모범 사례를 설명합니다. 이러한 가이드는 다양한 엔드포인트 및 도구 문서 내에서 ‘integrate’ 또는 ‘guides’로 표시됩니다. 예를 들어, [Filtered 스트림을 위한 규칙을 만드는 방법](/ko/x-api/posts/filtered-stream/integrate/build-a-rule)에 대한 통합 가이드가 있습니다.

<div id="troubleshoot-an-issue">
  ### 문제 해결하기
</div>

오류가 발생했거나 궁금한 점이 있다면 [support hub](https://developer.x.com/en/support/twitter-api)를 확인하세요. 여기에서 문제 해결 팁, 자주 묻는 질문에 대한 답변, 기타 유용한 자료를 확인할 수 있습니다.

<div id="keep-up-to-date-with-the-latest-releases">
  ### 최신 릴리스로 계속 업데이트 상태 유지하기
</div>

우리는 API를 통해 새로운 X 기능을 활용할 수 있도록 플랫폼에 대한 업데이트를 자주 배포합니다. 또한 최소 연 1회의 주요 버전 업데이트를 목표로 하고 있으며, 여기에는 일부 호환성을 깨는(breaking) 변경 사항이 포함될 수 있습니다.

최신 업데이트를 받아보려면, 당사의 [포럼 Announcements 카테고리](https://devcommunity.x.com/c/announcements/22)를 구독하고 X에서 [@API](https://x.com/api) 및 [@XDevelopers](https://x.com/XDevelopers)를 팔로우한 뒤 알림을 켜고, [stay informed](https://developer.x.com/en/stay-informed) 페이지에서 다양한 리소스를 확인해 주세요.

또한 최신 API 버전으로 업데이트하는 과정을 안내하기 위해 새로운 [migration hub](/ko/x-api/migrate/overview)를 구축했습니다.
