  ## X API의 답글 숨기기 엔드포인트 비교
</div>

v2 답글 숨기기 엔드포인트는 Labs 답글 숨기기 엔드포인트를 대체합니다. 이 엔드포인트의 Labs 버전을 사용하는 코드, 앱 또는 도구가 있고 새 X API v2 엔드포인트로 마이그레이션하는 것을 고려하고 있다면 이 가이드가 도움이 될 것입니다.

새 X API v2(답글 숨기기 엔드포인트 포함)를 사용하려면 [새 developer portal에 옵트인](https://developer.x.com/en/portal/opt-in)하고, [프로젝트](/ko/resources/fundamentals/projects)를 생성한 다음 해당 프로젝트에 앱을 추가해야 합니다. 그런 다음 해당 앱과 연결된 자격 증명을 사용하여 답글 숨기기 엔드포인트에 요청을 보낼 수 있습니다. Labs v2 답글 숨기기 엔드포인트에 등록된 것과 동일한 앱을 추가하면 사용자가 계속 인증된 상태로 유지됩니다.

다음 표는 Labs와 새로운 X API v2 엔드포인트 간의 차이점을 비교합니다.

| **설명** | **Labs v2** | **X API v2** |
| :--- | :--- | :--- |
| 호스트 도메인 | https://api.x.com | https://api.x.com |
| 엔드포인트 경로 | /labs/2/tweets/:id/hidden | /2/tweets/:id/hidden |
| 인증 | OAuth 1.0a 사용자 컨텍스트 | OAuth 1.0a 사용자 컨텍스트 |
| 지원되는 HTTP 메서드 | PUT | PUT |
| 기본 요청 한도 | 15분당 10회 요청(인증된 모든 사용자가 공유) | 15분당 50회 요청(각 인증된 사용자 기준) |
| 답글을 숨길 수 있음 | ✔︎  | ✔︎  |
| 이전에 숨긴 답글을 다시 표시할 수 있음 | ✔︎  | ✔︎  |
| 여러 번 답글을 숨기거나 다시 표시할 수 있음 | ✔︎  | ✔︎  |
| [프로젝트](/ko/resources/fundamentals/projects)와 연결된 [developer App](/ko/resources/fundamentals/developer-apps)의 자격 증명 사용이 필요함 |     | ✔   |
