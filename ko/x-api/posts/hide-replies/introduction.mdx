이 엔드포인트를 사용하면 정의한 기준에 따라 답글을 프로그래밍 방식으로 숨기거나 다시 표시할 수 있습니다. 기본 X 환경의 기능과 마찬가지로, 숨겨진 답글은 메인 대화에서는 보이지 않지만 별도의 페이지에서 여전히 확인할 수 있습니다. 이 엔드포인트를 활용해, 사용자가 가치 없다고 판단하는 답글을 숨기는 데 도움을 주는 앱을 만들 수 있습니다.

답글 숨기기 엔드포인트는 [OAuth 1.0a 사용자 컨텍스트](/ko/resources/fundamentals/authentication#oauth-1-0a-2) 또는 [OAuth 2.0 Authorization Code with PKCE](resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2) 인증을 사용합니다. 요청이 성공하면, 인증된 사용자가 시작한 게시물 대화에 포함된 단일 답글 하나를 숨깁니다. 각 대화에서는 최대 725개의 게시물을 숨길 수 있습니다.

<Note>
  **계정 설정**

  이 엔드포인트들에 접근하려면 다음이 필요합니다:

  * 승인된 [개발자 계정](https://developer.x.com/en/portal/petition/essential/basic-info)
  * [프로젝트](/ko/resources/fundamentals/projects) 내에 위치한 [개발자 앱](/ko/resources/fundamentals/developer-apps)의 키와 토큰을 사용한 인증

  X API v2 엔드포인트에 대한 접근 권한을 얻는 방법은 [시작하기 가이드](/ko/x-api/getting-started/getting-access)를 참조하세요.
</Note>
