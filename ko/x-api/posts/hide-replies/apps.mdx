  ## 답글 숨기기 엔드포인트를 사용하는 앱
</div>

다음은 답글 숨기기 엔드포인트를 통합한 개발자들이 만든 주목할 만한 앱 모음입니다. 이들은 자신의 기술을 X API와 결합해, 모든 사용자가 공개 대화를 더 건강하게 만드는 데 도움이 되는 유용한 경험을 만들어 냈습니다.

![Clarabridge](https://cdn.cms-twdigitalassets.com/content/dam/developer-twitter/hide-replies/clarabridge.jpg.twimg.1920.jpg)

**Clarabridge**

Clarabridge는 답글 숨기기 기능을 통합하여, 기업이 자신의 X 피드를 깔끔하고 읽기 좋게 유지할 수 있도록 합니다. 답글은 수동으로 숨길 수 있으며, 담당자는 Clarabridge의 텍스트 분석 기능을 사용해 욕설이나 브랜드 정책에 맞지 않는 표현을 자동으로 감지할 수도 있습니다.

[웹사이트로 이동 ▸](https://www.clarabridge.com/ "웹사이트로 이동 ▸")

[X의 Clarabridge](https://x.com/Clarabridge "Clarabridge on X")

![Perspective API template app](https://cdn.cms-twdigitalassets.com/content/dam/developer-twitter/hide-replies/perspective-logo.png.twimg.1920.png)

**Perspective API 템플릿 앱**

Perspective는 독성 댓글을 탐지하도록 Jigsaw(Alphabet 산하 조직)에서 학습시킨 인공지능입니다. 이 앱은 사용자가 받은 답글을 감지하고, 과거에 본 독성 댓글과 얼마나 유사한지에 대해 Perspective가 갖는 신뢰 수준을 나타내는 “점수”에 따라 답글을 자동으로 숨깁니다. 이는 X에서 개발한 오픈 소스 템플릿 앱입니다. 사용자에게 제공할 도구로 맞춤화해 활용해 보시길 권장합니다.

[앱 사용하기 ▸](https://quintessential-yoke.glitch.me "앱 사용하기 ▸")

[X의 Jigsaw](https://x.com/jigsaw "Jigsaw on X")

![Reshuffle](data:image/svg+xml,%3Csvg xmlns=%22http://www.w3.org/2000/svg%22 viewBox=%220 0 150 150%22%3E%3C/svg%3E)

**Reshuffle**

Reshuffle는 비즈니스 애플리케이션을 유연한 스크립트로 연결하는 플랫폼입니다. 이들은 사용자가 정의한 키워드를 포함하는 답글을 감지하고 숨기는 통합 스크립트를 만들었습니다. 사용해 보려면, 먼저 답글 숨기기 기능이 활성화된 유효한 앱이 있는지 확인한 뒤, 리포지토리의 안내를 따르세요.

[앱으로 이동 ▸](https://github.com/reshufflehq/reshuffle "앱으로 이동 ▸")

[X의 Reshuffle](https://x.com/reshufflehq "Reshuffle on X")

**원치 않는 답글 숨기기**

인기 앱 QuotedReplies를 만든 개발자인 Dara Oladosu는, 자신이 정한 몇 가지 기준을 충족하는 답글을 자동으로 숨기는 앱을 만들었습니다. 여기에는 그가 과거에 뮤트했던 특정 키워드를 포함하는 답글 등, 공격적인 행태를 보일 가능성이 더 높은 답글이 포함됩니다.

**Pandaflow**

Pandaflow는 워크플로우를 자동화하는 노코드 플랫폼입니다. 이들은 엔드포인트를 통합하여, 사용자가 자신이 정의한 사용자 지정 플로우를 기반으로 프로그래밍 방식으로 답글을 숨길 수 있도록 했습니다.

[X의 Pandaflow](https://x.com/pandaflowio "Pandaflow on X")
