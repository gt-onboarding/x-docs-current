  ## 답글 숨기기 엔드포인트를 사용하는 앱
</div>

여기에는 답글 숨기기 엔드포인트를 통합한 개발자들이 만든 주목할 만한 앱들을 모았습니다. 이들은 자신의 기술을 X API와 결합해, 모든 사용자가 공개 대화를 더 잘 관리할 수 있는 유용한 경험을 만들어 냈습니다.

![Clarabridge](https://cdn.cms-twdigitalassets.com/content/dam/developer-twitter/hide-replies/clarabridge.jpg.twimg.1920.jpg)

**Clarabridge**

Clarabridge는 답글 숨기기 기능을 통합해 기업이 자신의 X 피드를 깔끔하고 읽기 좋게 유지할 수 있도록 합니다. 답글은 수동으로 숨길 수도 있고, 담당자가 Clarabridge의 텍스트 분석 기능을 사용해 비속어 및 브랜드 정책에 부합하지 않는 표현을 자동으로 감지해 숨길 수도 있습니다.

[웹사이트로 이동 ▸](https://www.clarabridge.com/ "웹사이트로 이동 ▸")

[X에서의 Clarabridge](https://x.com/Clarabridge "X에서의 Clarabridge")

![Perspective API template app](https://cdn.cms-twdigitalassets.com/content/dam/developer-twitter/hide-replies/perspective-logo.png.twimg.1920.png)

**Perspective API 템플릿 앱**

Perspective는 Jigsaw(Alphabet 산하 조직)에서 유해한 댓글을 감지하도록 학습시킨 인공지능입니다. 이 앱은 사용자가 받는 답글을 감지하고, Perspective가 과거에 보았던 유해한 댓글과 유사하다고 판단하는 정도를 나타내는 “점수”를 기준으로 답글을 자동으로 숨깁니다. 이는 X에서 개발한 오픈 소스 템플릿 앱입니다. 사용자들을 위한 도구를 구축할 수 있도록 자유롭게 개인화해 사용해 보시기 바랍니다.

[앱 사용하기 ▸](https://quintessential-yoke.glitch.me "앱 사용하기 ▸")

[X에서의 Jigsaw](https://x.com/jigsaw "X에서의 Jigsaw")

![Reshuffle](data:image/svg+xml,%3Csvg xmlns=%22http://www.w3.org/2000/svg%22 viewBox=%220 0 150 150%22%3E%3C/svg%3E)

**Reshuffle**

Reshuffle은 비즈니스 애플리케이션을 유연한 스크립트로 연결하는 플랫폼입니다. 이들은 사용자가 정의한 키워드를 포함하는 답글을 감지하고 숨기는 통합 스크립트를 개발했습니다. 사용해 보려면 먼저 답글 숨기기 기능이 활성화된 유효한 앱이 있는지 확인한 뒤, 리포지토리에 있는 안내를 따르세요.

[앱으로 이동 ▸](https://github.com/reshufflehq/reshuffle "앱으로 이동 ▸")

[X에서의 Reshuffle](https://x.com/reshufflehq "X에서의 Reshuffle")

![Hide unwanted replies](data:image/svg+xml,%3Csvg xmlns=%22http://www.w3.org/2000/svg%22 viewBox=%220 0 396 397%22%3E%3C/svg%3E)

**Hide unwanted replies**

인기 앱 QuotedReplies를 만든 개발자 Dara Oladosu는, 자신이 정한 몇 가지 기준을 충족하는 답글을 자동으로 숨기는 앱을 만들었습니다. 여기에는 그가 과거에 뮤트해 둔 특정 키워드를 포함하는 답글 등, 공격적인 행동을 보일 가능성이 더 높은 답글이 포함됩니다.

[앱으로 이동 ▸](https://hideunwantedreplies.com/ "앱으로 이동 ▸")

[X에서의 Dara Oladosu](https://x.com/dara_tobi "X에서의 Dara Oladosu")

![Pandaflow](data:image/svg+xml,%3Csvg xmlns=%22http://www.w3.org/2000/svg%22 viewBox=%220 0 94 94%22%3E%3C/svg%3E)

**Pandaflow**

Pandaflow는 워크플로를 자동화하기 위한 노코드 플랫폼입니다. 이들은 이 엔드포인트를 통합해, 사용자가 정의한 커스텀 플로우에 따라 답글을 프로그래밍 방식으로 숨길 수 있도록 했습니다.

[앱으로 이동 ▸](https://pandaflow.io/ "앱으로 이동 ▸")

[X에서의 Pandaflow](https://x.com/pandaflowio "X에서의 Pandaflow")
