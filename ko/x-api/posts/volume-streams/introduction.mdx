* 1% 샘플 스트림.
* 10% 샘플 스트림. 일반적으로 “Decahose”라고 불립니다.

이 스트림들은 모두 대량의 데이터를 전송하기 때문에 “볼륨 스트림(volume streams)”이라고 합니다. 1% 스트림만으로도 매초 수십 개의 게시물이 전송될 수 있습니다. 이러한 스트림을 사용하면 트렌드를 식별·추적하고, 전반적인 여론을 모니터링하며, 글로벌 이벤트를 모니터링하는 등 다양한 작업을 수행할 수 있습니다. 

이 스트리밍 엔드포인트는 지속적인 HTTP GET 연결을 통해 [Post objects](/ko/x-api/fundamentals/data-dictionary#tweet)를 전달하며 [OAuth 2.0 App-Only](/ko/resources/fundamentals/authentication#oauth-2-0) 인증을 사용합니다. Essential 액세스의 경우 한 번에 하나의 연결만 가질 수 있습니다. 모든 액세스 레벨에서 연결 요청은 15분 단위로 최대 50회까지 수행할 수 있습니다.

이 볼륨 스트림 엔드포인트는 수정된 게시물을 지원합니다. 이 엔드포인트는 해당 게시물의 수정 이력과 함께, 게시물 ID 배열을 포함하여 수정된 게시물을 전달합니다. 수정 이력이 없는 게시물의 경우 이 배열에는 단일 ID만 포함됩니다. 수정된 게시물의 경우 이 배열에는 여러 개의 ID가 포함되며, 수정 순서를 반영하는 오름차순으로 정렬되고, 가장 최근 버전은 배열의 마지막 위치에 있습니다. 게시물 수정이 어떻게 작동하는지 자세히 알아보려면 [Post edits fundamentals](/ko/x-api/fundamentals/edit-posts) 페이지를 참조하세요. 

*이 API를 사용하려면 먼저 Enterprise 영업팀을 통해 계정을 설정해야 합니다.*

<Note>
  **계정 설정**

  이 엔드포인트에 액세스하려면 다음이 필요합니다:

  * 승인이 완료된 [developer account](https://developer.x.com/en/portal/petition/essential/basic-info).
  * [Project](/ko/resources/fundamentals/projects) 내에 있는 [developer App](/ko/resources/fundamentals/developer-apps)의 키와 토큰을 사용한 인증. 

  X API v2 엔드포인트 액세스에 대한 자세한 내용은 [시작 가이드](/ko/x-api/getting-started/getting-access)를 참고하세요.
</Note>
