* 1% 샘플링 스트림.
* 10% 샘플링 스트림. 일반적으로 &quot;Decahose&quot;라고 불립니다.

이들은 모두 대량의 데이터를 전송하기 때문에 &quot;볼륨 스트림&quot;이라고 부릅니다. 1% 스트림만으로도 매초 수십 개의 게시물이 전송될 수 있습니다. 이러한 스트림을 사용하면 트렌드를 식별·추적하고, 전반적인 감성을 모니터링하고, 전 세계 이벤트를 모니터링하는 등 다양한 작업을 수행할 수 있습니다. 

이 스트리밍 엔드포인트는 지속적인 HTTP GET 연결을 통해 [Post 객체](/ko/x-api/fundamentals/data-dictionary#tweet)를 전달하며 [OAuth 2.0 App-Only](/ko/resources/fundamentals/authentication#oauth-2-0) 인증을 사용합니다. Essential 액세스의 경우 한 번에 하나의 연결만 유지할 수 있습니다. 모든 액세스 레벨에서 15분 동안 최대 50번까지 연결 요청을 보낼 수 있습니다.

이 볼륨 스트림 엔드포인트는 수정된 게시물을 지원합니다. 이 엔드포인트는 게시물의 수정 이력과 함께 수정된 게시물을 전달하며, 여기에는 게시물 ID 배열이 포함됩니다. 수정 이력이 없는 게시물의 경우 이 배열에는 단일 ID가 포함됩니다. 수정된 게시물의 경우 이 배열에는 여러 개의 ID가 포함되며, 수정 순서를 반영하도록 오름차순으로 정렬되고, 배열의 마지막 위치에 가장 최신 버전이 위치합니다. 게시물 수정이 어떻게 동작하는지에 대한 자세한 내용은 [Post edits fundamentals](/ko/x-api/fundamentals/edit-posts) 페이지를 참고하세요. 

*이 API를 사용하려면 먼저 Enterprise 영업팀을 통해 계정을 설정해야 합니다.*

<Note>
  **계정 설정**

  이 엔드포인트에 액세스하려면 다음이 필요합니다:

  * 승인된 [개발자 계정](https://developer.x.com/en/portal/petition/essential/basic-info).
  * [Project](/ko/resources/fundamentals/projects) 내에 있는 [developer App](/ko/resources/fundamentals/developer-apps)의 키와 토큰을 사용해 인증해야 합니다. 

  X API v2 엔드포인트에 대한 액세스 권한을 얻는 방법은 [시작하기 가이드](/ko/x-api/getting-started/getting-access)를 참고하세요.
</Note>
