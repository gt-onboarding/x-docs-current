  ## X API의 필터링 스트림 엔드포인트 비교
</div>

v2 필터링 스트림 엔드포인트 그룹은 [standard v1.1 statuses/filter](https://developer.x.com/en/docs/x-api/v1/tweets/filter-realtime/api-reference/post-statuses-filter) 및 [PowerTrack API](/ko/x-api/enterprise-gnip-2.0/powertrack-api)를 대체합니다. 기존 버전의 필터링 스트림 엔드포인트를 사용하는 코드, 앱, 도구가 있고 새로운 X API v2 엔드포인트로 마이그레이션하는 것을 고려 중이라면, 아래 비교 내용이 마이그레이션을 시작하는 데 도움이 될 수 있습니다. 

보다 심층적인 마이그레이션 가이드는 다음을 참고하세요:

[Standard v1.1에서 X API v2로 마이그레이션](x-api/posts/filtered-stream#standard-v1-1-compared-to-x-api-v2)

[PowerTrack API에서 X API v2로 마이그레이션](/ko/x-api/enterprise-gnip-2.0/powertrack-api#powertrack-operators)
