<div id="comparing-x-apis-timelines-endpoints">
  ## X API의 타임라인 엔드포인트 비교
</div>

v2 시간 역순(최신순) 타임라인, 사용자 게시물 타임라인, 사용자 멘션 타임라인 엔드포인트는 각각 [v1.1 statuses/home&#95;timeine,](https://developer.x.com/en/docs/twitter-api/v1/tweets/timelines/api-reference/get-statuses-home_timeline) [v1.1 statuses/user&#95;timeline](https://developer.x.com/en/docs/twitter-api/v1/tweets/timelines/api-reference/get-statuses-user_timeline.html), [v1.1 statuses/mentions&#95;timeline](https://developer.x.com/en/docs/twitter-api/v1/tweets/timelines/api-reference/get-statuses-mentions_timeline.html) 엔드포인트를 대체합니다. 기존 버전의 이 엔드포인트를 사용하는 코드, 앱 또는 도구가 있고 최신 X API v2 엔드포인트로 마이그레이션을 고려하고 있다면 이 가이드를 참고하십시오. 보다 심층적인 마이그레이션 가이드는 [Standard v1.1 migration to X API v2](/ko/x-api/posts/timelines/migrate/standard-to-twitter-api-v2)를 참고하세요.

이 페이지에는 다음과 같은 세 개의 비교 표가 포함되어 있습니다.

* 시간 역순(최신순) 홈 타임라인
* 사용자 게시물 타임라인
* 사용자 멘션 타임라인
