news lookup 엔드포인트를 사용하면 개발자가 X에서 실시간으로 올라오는 뉴스와 속보 헤드라인을 조회할 수 있습니다.

이 엔드포인트는 OAuth1 및 OAuth2 PKCE에 대해 app-auth 및 user-auth 방식의 인증을 지원합니다.

<div id="getting-started">
  ## 시작하기
</div>

엔드포인트를 사용하려면 [developer portal](https://developer.x.com/en/portal/dashboard)에서 [bearer token](/ko/fundamentals/authentication/oauth-2-0/application-only)이 필요합니다.

bearer token을 얻은 후에는 아래와 같이 news API를 호출할 수 있습니다:

```bash
curl 'https://api.x.com/2/news/1989418137272422538?news.fields=contexts,cluster_posts_results' --header 'Authorization: Bearer XXXXX'
```
