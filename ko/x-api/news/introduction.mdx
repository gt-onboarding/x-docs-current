뉴스 조회 엔드포인트를 사용하면 개발자는 X에서 속보로 게시되는 뉴스와 헤드라인을 가져올 수 있습니다.

이 엔드포인트는 OAuth1 및 OAuth2 PKCE에서 app-auth 및 user-auth 방식의 인증을 모두 지원합니다.

<div id="getting-started">
  ## 시작하기
</div>

이 엔드포인트를 사용하려면 [developer portal](https://developer.x.com/en/portal/dashboard)에서 [Bearer Token](/ko/fundamentals/authentication/oauth-2-0/application-only)을 발급받아야 합니다.

Bearer Token을 발급받았다면, 아래와 같이 뉴스 API를 호출할 수 있습니다.

```bash
curl 'https://api.x.com/2/news/1989418137272422538?news.fields=contexts,cluster_posts_results' --header 'Authorization: Bearer XXXXX'
```
