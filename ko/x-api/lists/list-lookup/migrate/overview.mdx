  ## X API의 리스트 조회(List lookup) 엔드포인트 비교
</div>

v2 리스트 조회(List lookup) 엔드포인트 그룹은 기존 v1.1의 표준 엔드포인트인 [GET lists/show](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/get-lists-show)와 [GET lists/ownership](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/get-lists-ownerships) 엔드포인트를 대체합니다. 이들 리스트 조회 엔드포인트 중 하나를 사용하는 코드, 앱 또는 도구를 이미 운영 중이며 최신 X API v2 엔드포인트로 마이그레이션하는 것을 고려하고 있다면, 이 가이드 모음이 도움이 됩니다.

아래 표는 표준 v1.1과 X API v2 리스트 엔드포인트를 비교한 것입니다.

**ID로 리스트 조회**

|     |     |     |
| :--- | :--- | :--- |
| 설명 | 표준 v1.1 | X API v2 |
| 지원되는 HTTP 메서드 | `GET` | `GET` |
| 호스트 도메인 | `https://api.x.com` | `https://api.x.com` |
| 엔드포인트 경로 | `/1.1/lists/show.json` | `/2/lists/:id` |
| [Authentication](/ko/resources/fundamentals/authentication) | OAuth 1.0a 사용자 컨텍스트<br /><br />앱 전용 | OAuth 1.0a 사용자 컨텍스트<br /><br />OAuth 2.0 Authorization Code with PKCE<br /><br />앱 전용 |
| 기본 [요청 한도](/ko/resources/fundamentals/rate-limits) | OAuth 1.0a 사용 시 15분당 75회 요청<br /><br />OAuth 2.0 사용 시 15분당 75회 요청<br /><br />앱 전용 사용 시 15분당 75회 요청 | OAuth 1.0a 사용 시 15분당 75회 요청<br /><br />OAuth 2.0 사용 시 15분당 75회 요청<br /><br />앱 전용 사용 시 15분당 75회 요청 |

**사용자가 소유한 리스트 조회**

|     |     |     |
| :--- | :--- | :--- |
| 설명 | 표준 v1.1 | X API v2 |
| 지원되는 HTTP 메서드 | `GET` | `GET` |
| 호스트 도메인 | `https://api.x.com` | `https://api.x.com` |
| 엔드포인트 경로 | `/1.1/lists/ownerships.json` | `/2/users/:id/owned_lists` |
| [Authentication](/ko/resources/fundamentals/authentication) | OAuth 1.0a 사용자 컨텍스트<br /><br />앱 전용 | OAuth 1.0a 사용자 컨텍스트<br /><br />OAuth 2.0 Authorization Code with PKCE<br /><br />앱 전용 |
| 기본 [요청 한도](/ko/resources/fundamentals/rate-limits) | OAuth 1.0a 사용 시 15분당 15회 요청<br /><br />앱 전용 사용 시 15분당 15회 요청 | OAuth 1.0a 사용 시 15분당 15회 요청<br /><br />OAuth 2.0 사용 시 15분당 15회 요청<br /><br />앱 전용 사용 시 15분당 15회 요청 |

X API v2 엔드포인트에 접근하려면 [개발자 계정에 가입](https://developer.x.com/en/portal/petition/essential/basic-info)해야 합니다. 인증을 수행할 때는 [Project](/ko/resources/fundamentals/projects) 내에 있는 [developer App](/ko/resources/fundamentals/developer-apps)의 키와 토큰을 사용해야 합니다. 

X API v2 엔드포인트에 대한 접근 권한을 얻는 방법은 [시작하기](/ko/x-api/getting-started/getting-access) 페이지에서 자세히 알아볼 수 있습니다.
