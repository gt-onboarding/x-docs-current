지정한 List를 id로 조회할 때는 15분당 최대 75회, 사용자가 소유한 List를 조회할 때는 15분당 최대 15회의 요청 한도가 있습니다.

이 엔드포인트에 대한 요청을 인증할 때는 [OAuth 1.0a User Context](/ko/resources/fundamentals/authentication), [앱 전용(App only)]([https://developer.x.com(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token)](https://developer.x.com\(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token\)), 또는 [OAuth 2.0 Authorization Code with PKCE](resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2)를 사용할 수 있습니다. 

<Note>
  **계정 설정**

  이 엔드포인트에 접근하려면 다음이 필요합니다:

  * 승인된 [개발자 계정](https://developer.x.com/en/portal/petition/essential/basic-info)
  * [프로젝트](/ko/resources/fundamentals/projects) 내에 있는 [개발자 앱](/ko/resources/fundamentals/developer-apps)의 키와 토큰을 사용해 인증할 수 있어야 합니다. 

  X API v2 엔드포인트에 대한 접근 권한을 얻는 방법은 [시작하기 가이드](/ko/x-api/getting-started/getting-access)를 참고하세요.
</Note>
