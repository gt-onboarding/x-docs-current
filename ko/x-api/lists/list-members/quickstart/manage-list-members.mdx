<div id="steps-to-build-a-manage-list-member-request">
  ### 리스트 멤버 관리 요청을 구성하는 단계
</div>

<div id="step-one-choose-the-list-endpoint-collection-from-postman">
  #### 1단계: Postman에서 리스트 엔드포인트 컬렉션 선택
</div>

Postman에 X API v2 컬렉션을 로드했다면, “List” 폴더로 이동한 다음 “List members” 폴더를 선택하고, &quot;Add a member&quot; 요청을 선택합니다.
 

<div id="step-two-specify-the-user-to-add">
  #### Step two: Specify the user to add
</div>

List 멤버 관리 endpoint에는 두 개의 ID가 필요합니다. 리스트의 ID와 추가할 사용자의 ID입니다. [users lookup](/ko/x-api/users/lookup/introduction)을 사용해 사용자의 ID를 찾은 뒤, username을 전달하여 `id` 필드를 포함한 응답을 받을 수 있습니다.

대상 ID는 유효한 사용자 ID라면 무엇이든 사용할 수 있습니다. Postman에서 &quot;Params&quot; 탭으로 이동한 후, id path 변수의 &quot;Value&quot; 열에 리스트 ID를 입력합니다. 그런 다음 &quot;Body&quot; 탭으로 이동하여 추가하려는 사용자의 ID를 `user_id` 파라미터의 값으로 입력합니다. 어떤 ID 앞이나 뒤에도 공백이 포함되지 않도록 주의하세요.

|             |                     |                    |
| :---------- | :------------------ | :----------------- |
| **Key**     | **Value**           | **Parameter type** |
| `id`        | 리스트 ID              | path               |
| user&#95;id | 멤버로 추가하려는 대상 사용자 ID | body               |

이제 &quot;Send&quot; 버튼 옆에서 다음과 비슷한 URL을 볼 수 있습니다:

`https://api.x.com/2/lists/1441162269824405510/members`

Step three: 요청 보내기 및 응답 검토

모든 설정이 완료되면 &quot;Send&quot; 버튼을 눌러 요청을 전송하면, 다음 예시 응답과 유사한 응답을 받게 됩니다:

```
{
  "data": {
    "is_member": true
  }
}
```

반환된 response 객체에서 is&#95;member 키의 값이 true라면, 해당 사용자가 리스트의 멤버로 성공적으로 추가된 것입니다. 

리스트에서 멤버를 제거하려면, Postman에 로드된 X API v2 컬렉션의 “Lists” 폴더에 있는 “Remove a member” 요청을 선택합니다. 이 endpoint에는 리스트 ID와 제거하려는 멤버의 user ID가 필요합니다. “Params” 탭에서 리스트 ID를 id 열의 값으로 입력하고, 제거하려는 사용자의 ID를 user&#95;id 열의 값으로 입력합니다. 

삭제 요청이 성공하면, 다음 예와 유사한 response를 받게 됩니다:

```
{
  "data": {
    "is_member": false
  }
}
```
