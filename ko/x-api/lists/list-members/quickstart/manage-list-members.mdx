  ### 리스트 멤버 관리 빠른 시작 가이드
</div>

<Note>
  **참고:** 이 가이드는 [빠른 시작 개요](/ko/x-api/lists/list-members#getting-started-with-the-list-members-endpoint-group)에 있는 사전 요구 사항을 이미 완료했다고 가정합니다.
</Note>

<div id="steps-to-build-a-manage-list-member-request">
  ### 리스트 멤버 관리 요청을 생성하는 단계
</div>

<div id="step-one-choose-the-list-endpoint-collection-from-postman">
  #### 1단계: Postman에서 List 엔드포인트 컬렉션 선택
</div>

Postman에서 X API v2 컬렉션을 로드한 다음 “List” 폴더로 이동해 “List members” 폴더를 선택하고, 이어서 &quot;Add a member&quot; 요청을 선택합니다.
 

<div id="step-two-specify-the-user-to-add">
  #### 2단계: 추가할 사용자 지정하기
</div>

리스트 멤버 관리 엔드포인트에는 두 개의 ID가 필요합니다. 리스트의 ID와 추가할 사용자의 ID입니다. [users lookup](/ko/x-api/users/lookup/introduction)을 사용해 사용자의 ID를 찾고, username을 전달하여 id 필드를 받을 수 있습니다.

대상 ID는 어떤 유효한 사용자 ID라도 될 수 있습니다. Postman에서 &quot;Params&quot; 탭으로 이동한 다음 id 경로 변수의 &quot;Value&quot; 열에 자신의 ID를 입력합니다. 이어서 &quot;Body&quot; 탭으로 이동해, 추가하려는 사용자의 ID를 user&#95;id 파라미터의 값으로 입력합니다. 어떤 ID 앞뒤에도 공백이 포함되지 않도록 주의하십시오.

|             |                     |                    |
| :---------- | :------------------ | :----------------- |
| **Key**     | **Value**           | **Parameter type** |
| `id`        | 리스트 ID              | path               |
| user&#95;id | 멤버로 추가하려는 대상 사용자 ID | body               |

이제 &quot;Send&quot; 버튼 옆에 다음과 유사한 URL이 표시됩니다:

`https://api.x.com/2/lists/1441162269824405510/members`

3단계: 요청을 보내고 응답을 검토합니다

모든 설정을 마쳤다면 &quot;Send&quot; 버튼을 눌러 요청을 보내면, 아래 예시 응답과 유사한 응답을 받게 됩니다:

```
{
  "data": {
    "is_member": true
  }
}
```

반환된 response 객체에 is&#95;member 키의 값이 true라면, 해당 사용자를 리스트 멤버로 성공적으로 추가한 것입니다. 

리스트에서 멤버를 제거하려면, Postman에 로드된 X API v2 컬렉션의 “Lists” 폴더에 있는 “Remove a member” 요청을 선택합니다. 이 endpoint에는 리스트 ID와 제거하려는 멤버의 사용자 ID가 필요합니다. “Params” 탭에서 리스트 ID를 id 열의 값으로 입력하고, 제거할 사용자의 ID를 user&#95;id 열의 값으로 입력합니다. 

삭제 요청이 성공적으로 처리되면, 다음 예시와 유사한 response를 받게 됩니다:

```
{
  "data": {
    "is_member": false
  }
}
```
