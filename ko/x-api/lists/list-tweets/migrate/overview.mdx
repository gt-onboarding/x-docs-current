  ## X API의 리스트 게시물 조회 엔드포인트 비교
</div>

v2 리스트 게시물 조회 엔드포인트는 표준 v1.1 [GET lists/statuses](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/get-lists-statuses) 엔드포인트를 대체할 예정입니다. 이 버전의 엔드포인트를 사용하는 코드, 앱 또는 도구가 있고 최신 X API v2 엔드포인트로 마이그레이션을 고려 중이라면 이 가이드 모음이 도움이 될 것입니다.

다음 표는 표준 v1.1과 X API v2 리스트 엔드포인트를 비교한 것입니다:

**ID로 리스트 조회**

|     |     |     |
| :--- | :--- | :--- |
| Description | Standard v1.1 | X API v2 |
| HTTP methods supported | `GET` | `GET` |
| Host domain | `https://api.x.com` | `https://api.x.com` |
| Endpoint path | `/1.1/lists/statuses.json` | `/2/lists/:id/tweets` |
| [Authentication](/ko/resources/fundamentals/authentication) | OAuth 1.0a User Context<br /><br />App only | OAuth 1.0a User Context<br /><br />OAuth 2.0 Authorization Code with PKCE<br /><br />App only |
| Default request [rate limits](/ko/resources/fundamentals/rate-limits) | 900 requests per 15 min with OAuth 1.0a<br /><br />900 requests per 15min with App only | 900 requests per 15 min with OAuth 1.0a<br /><br />900 requests per 15 min with OAuth 2.0<br /><br />900 requests per 15 min with App only |

X API v2 엔드포인트에 액세스하려면 먼저 [개발자 계정](https://developer.x.com/en/portal/petition/essential/basic-info)에 가입해야 합니다. 인증할 때는 [프로젝트](/ko/resources/fundamentals/projects) 내에 있는 [개발자 앱](/ko/resources/fundamentals/developer-apps)의 키와 토큰을 사용해야 합니다. 

X API v2 엔드포인트에 대한 액세스 권한을 얻는 방법은 [시작하기](/ko/x-api/getting-started/getting-access) 페이지를 참조하세요.
