리스트 게시물 조회에는 지정된 리스트의 게시물을 가져오기 위한 단일 엔드포인트가 제공됩니다. 이 엔드포인트를 사용하면 사용자가 타임라인에서 보게 되는 게시물을 사용자 지정하고 정리하며, 우선순위를 정할 수 있는 솔루션을 구축할 수 있습니다.

이 엔드포인트에는 15분당 900개의 요청이라는 레이트 리밋이 있으며, 응답에서는 지정된 리스트에 대해 최신 800개의 게시물까지 조회할 수 있습니다.

이 엔드포인트에 대한 요청을 인증하려면 [OAuth 1.0a 사용자 컨텍스트](/ko/resources/fundamentals/authentication), [앱 전용](/ko/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token), 또는 [OAuth 2.0 Authorization Code with PKCE](resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2)를 사용할 수 있습니다. 

**계정 설정**

이 엔드포인트에 액세스하려면 다음이 필요합니다:

* 승인된 [개발자 계정](https://developer.x.com/en/portal/petition/essential/basic-info)
* [프로젝트](/ko/resources/fundamentals/projects) 내에 있는 [개발자 앱](/ko/resources/fundamentals/developer-apps)의 키와 토큰을 사용한 인증

X API v2 엔드포인트에 액세스하는 방법에 대해서는 [시작하기 가이드](/ko/x-api/getting-started/getting-access)를 참고하세요.
