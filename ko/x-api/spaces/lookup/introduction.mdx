ID 조회 엔드포인트는 사용자에게 Spaces 정보를 표시하는 기능을 구현하거나 하나 이상의 Space에 대한 세부 정보를 반환하는 데 사용할 수 있는 데이터를 제공합니다. 하나의 요청으로 하나 이상의 Space를 조회할 수 있습니다.

또한 생성자 ID로 Spaces를 조회할 수도 있습니다. 이 엔드포인트는 리스트에 있는 사용자들의 Spaces를 조회하거나, 여러 정기 진행 호스트가 현재 라이브 중인지 또는 예정된 Space가 있는지 확인하는 데 유용합니다. 하나의 요청으로 최대 100명의 사용자를 조회할 수 있습니다.

**계정 설정**

이 엔드포인트에 액세스하려면 다음이 필요합니다.

* 승인된 [개발자 계정](https://developer.x.com/en/portal/petition/essential/basic-info)
* [프로젝트](/ko/resources/fundamentals/projects) 내에 위치한 [개발자 App](/ko/resources/fundamentals/developer-apps)의 키와 토큰을 사용한 인증

X API v2 엔드포인트에 대한 액세스 권한을 얻는 방법은 [시작하기 가이드](/ko/x-api/getting-started/getting-access)를 참조하세요.
