**시작하기**

<div id="authentication">
  ### 인증
</div>

이 엔드포인트는 [OAuth 1.0a 사용자 컨텍스트](/ko/resources/fundamentals/authentication), [OAuth 2.0 앱 전용(App-Only)](https://developer.x.com(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token), 또는 [OAuth 2.0 Authorization Code with PKCE](resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2) 중 하나로 인증할 수 있습니다.
 

<div id="making-a-request">
  ### 요청 보내기
</div>

다음과 같이 ID를 지정하여 Communities Lookup 엔드포인트를 호출할 수 있습니다 (ID `1758747817642700922` 사용):

```
      curl --location 'https://api.x.com/2/communities/1758747817642700922' --header 'Authorization: ••••••'
```

요청이 성공하면 아래와 같은 JSON 응답이 표시됩니다:

```
{
    "data": {
        "access": "Public",
        "description": "애니메이션 커뮤니티에 오신 것을 환영합니다! 애니메이션 팬들이 모여 좋아하는 작품을 공유하고 애니메이션 관련 모든 것을 이야기하는 곳입니다.",
        "join_policy": "Open",
        "name": "Anime Community"
    }
}
```

<div id="community-fields">
  ### 커뮤니티 필드
</div>

커뮤니티 필드를 지정하여 커뮤니티와 관련된 추가 데이터를 가져올 수 있습니다.

다음 커뮤니티 필드를 사용할 수 있습니다:

| Field value | Type | Description |     |
| :--- | :--- | :--- | :--- |
| created&#95;at | date (ISO 8601) | 커뮤니티가 생성된 시간입니다. |     |
| id  | string | 커뮤니티의 고유 식별자입니다. |     |
| name | string | 커뮤니티의 이름입니다. |     |
| description | string | 제공된 경우, 커뮤니티 설명 텍스트입니다. |     |
| access | string | 커뮤니티의 접근 수준입니다.  <br />  <br />다음 값 중 하나일 수 있습니다:<br /><br />* `Public`<br />* `Closed` |     |
| join&#95;policy | string | 커뮤니티의 가입 정책입니다.  <br />  <br />다음 값 중 하나일 수 있습니다:<br /><br />* `Open`<br />* `RestrictedJoinRequestsDisabled`<br />* `RestrictedJoinRequestsRequireAdminApproval`<br />* `RestrictedJoinRequestsRequireModeratorApproval`<br />* `SuperFollowRequired` |     |
| member&#95;count | integer | 커뮤니티에 가입한 회원 수입니다. |     |

<div id="example-request">
  #### 예시 요청
</div>

다음과 같이 추가 community 필드를 함께 지정하여 Communities Lookup 엔드포인트를 호출할 수 있습니다 (`1758747817642700922` id 사용):

```
curl --location 'https://api.x.com/2/communities/1758747817642700922?community.fields=access,created_at,description,id,join_policy,member_count,name' --header 'Authorization: ••••••'
```

요청이 성공적으로 처리되면 아래와 같은 JSON 응답이 반환됩니다:

```
{
    "data": {
        "description": "애니메이션 커뮤니티에 오신 것을 환영합니다! 애니메이션 팬들이 모여 좋아하는 작품을 공유하고 애니메이션 관련 모든 것을 논의하는 곳입니다.",
        "access": "Public",
        "member_count": 40369,
        "created_at": "2024-02-17T06:58:50.000Z",
        "join_policy": "Open",
        "name": "Anime Community",
        "id": "Q29tbXVuaXR5OjE3NTg3NDc4MTc2NDI3MDA5MjI="
    }
}
```
