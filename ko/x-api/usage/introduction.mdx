X API v2의 Usage API를 사용하면 개발자가 프로젝트 사용량을 프로그래밍 방식으로 조회할 수 있습니다. 이 엔드포인트를 사용하면 특정 과금 주기 동안 가져온 게시물 수를 추적하고 모니터링할 수 있습니다.

개발자는 GET 엔드포인트를 사용하여 최대 지난 90일간의 일별 프로젝트 사용량을 조회할 수 있습니다. 사용량은 프로젝트에 연결된 각 클라이언트 앱별로 집계할 수도 있습니다.

이 GET 엔드포인트에는 앱 기준으로 15분당 50회의 요청 제한이 적용됩니다.

<div id="getting-started">
  ### 시작하기
</div>

이 엔드포인트를 사용하려면 [developer portal](https://developer.x.com/en/portal/dashboard)에서 발급받은 [Bearer Token](https://developer.x.com/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token)이 필요합니다. Bearer Token을 보유하게 되면 아래와 같이 usage API를 호출할 수 있습니다:

```
      curl 'https://api.x.com/2/usage/tweets' --header 'Authorization: Bearer XXXXX'
```

요청이 성공적으로 처리되면 아래와 같은 JSON 응답이 표시됩니다.

```
{
    "data": {
        "cap_reset_day": 10,
        "project_cap": "5000000000",
        "project_id": "1369785403853424",
        "project_usage": "43435"
    }
}
```

<Note>
  **계정 설정**

  이 엔드포인트에 액세스하려면 다음이 필요합니다.

  * 승인된 [개발자 계정](https://developer.x.com/en/portal/petition/essential/basic-info)
  * [프로젝트](/ko/resources/fundamentals/projects) 내에 있는 [개발자 앱](/ko/resources/fundamentals/developer-apps)의 키와 토큰을 사용한 인증

  X API v2 엔드포인트에 대한 액세스 권한을 얻는 방법에 대해서는 [시작 가이드](/ko/x-api/getting-started/getting-access)를 참고하세요.
</Note>
