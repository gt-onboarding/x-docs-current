<div id="comparing-x-apis-mutes-endpoints">
  ## X API의 뮤트 엔드포인트 비교
</div>

<div id="mutes-lookup">
  #### 뮤트 조회
</div>

v2 뮤트 조회 엔드포인트는 기존 [v1.1 GET mutes/users/ids](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/mute-block-report-users/api-reference/get-mutes-users-ids) 및 [GET mutes/users/list](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/mute-block-report-users/api-reference/get-mutes-users-list) 엔드포인트를 대체합니다.

다음 표는 기존 v1.1과 X API v2 뮤트 엔드포인트를 비교합니다:
 

| 설명 | Standard v1.1 | X API v2 |
| :--- | :--- | :--- |
| 지원되는 HTTP 메서드 | ******GET****** | ******GET****** |
| 호스트 도메인 | ******https://api.x.com****** | ******https://api.x.com****** |
| 엔드포인트 경로 | ******/1.1/mutes/users/ids.json******<br /><br />/1.1/mutes/users/list.json | ******/2/users/:id/muting****** |
| [인증](/ko/resources/fundamentals/authentication) | OAuth 1.0a 사용자 컨텍스트 | OAuth 1.0a 사용자 컨텍스트<br /><br />OAuth 2.0 Authorization Code with PKCE |
| 기본 요청 [요청 한도](/ko/resources/fundamentals/rate-limits) | 15분당 15회 요청 (사용자당) | 15분당 15회 요청 (사용자당) |
| 데이터 포맷 | Standard v1.1 포맷 | [X API v2 포맷](/ko/x-api/fundamentals/data-dictionary) (fields 및 expansions 요청 파라미터에 의해 결정되며, v1.1 포맷과는 하위 호환되지 않음)<br /><br />Standard v1.1 포맷에서 X API v2 포맷으로 마이그레이션하는 방법에 대해 자세히 알아보려면 [데이터 포맷 마이그레이션 가이드](/ko/x-api/migrate/data-format-migration)를 참조하세요. |
| [Project](/ko/resources/fundamentals/projects)와 연결된 [developer App](/ko/resources/fundamentals/developer-apps)의 자격 증명 사용 필요 |     | ✔️  |

<div id="manage-mutes">
  #### 뮤트 관리
</div>

v2 뮤트 관리 엔드포인트는 기존 v1.1의 표준 [POST mutes/users/create](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/mute-block-report-users/api-reference/post-mutes-users-create) 및 [POST mutes/users/destroy](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/mute-block-report-users/api-reference/post-mutes-users-destroy) 엔드포인트를 대체합니다.

아래 표에서는 v1.1 표준 엔드포인트와 X API v2 뮤트 엔드포인트를 비교합니다.

**사용자 뮤트하기**

| 설명 | v1.1 표준 | X API v2 |
| :--- | :--- | :--- |
| 지원되는 HTTP 메서드 | POST | POST |
| 호스트 도메인 | https://api.x.com | https://api.x.com |
| 엔드포인트 경로 | /1.1/mutes/users/create.json | /2/users/:id/muting |
| [Authentication](/ko/resources/fundamentals/authentication) | OAuth 1.0a User Context | OAuth 1.0a User Context<br /><br />OAuth 2.0 Authorization Code with PKCE |
| 기본 요청 [rate limits](/ko/resources/fundamentals/rate-limits) | 15분당 50회 요청 | 15분당 50회 요청 |
| [Project](/ko/resources/fundamentals/projects)에 연결된 [developer App](/ko/resources/fundamentals/developer-apps)의 자격 증명 사용 필요 여부 |     | ✔️  |

**사용자 뮤트 해제하기**

아래 표에서는 v1.1 표준 엔드포인트와 X API v2 뮤트 해제 엔드포인트를 비교합니다.

| 설명 | v1.1 표준 | X API v2 |
| :--- | :--- | :--- |
| 지원되는 HTTP 메서드 | POST | DELETE |
| 호스트 도메인 | https://api.x.com | https://api.x.com |
| 엔드포인트 경로 | /1.1/mutes/users/destroy.json | /2/users/:source&#95;user&#95;id/muting/:target&#95;user&#95;id |
| [Authentication](/ko/resources/fundamentals/authentication) | OAuth 1.0a User Context | OAuth 1.0a User Context<br /><br />OAuth 2.0 Authorization Code with PKCE |
| 기본 요청 [rate limits](/ko/resources/fundamentals/rate-limits) | 15분당 50회 요청 | 15분당 50회 요청 |
| [Project](/ko/resources/fundamentals/projects)에 연결된 [developer App](/ko/resources/fundamentals/developer-apps)의 자격 증명 사용 필요 여부 |     | ✔️  |
