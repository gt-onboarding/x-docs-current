Pro 액세스 이상을 보유한 개발자가 사용할 수 있습니다.

Users Search 엔드포인트는 X의 공개 사용자 계정을 대상으로 단순하고 관련성 기반의 검색 인터페이스를 제공합니다. 관심 주제, 전체 이름, 회사 이름, 위치 또는 기타 기준으로 쿼리해 보세요.

이 엔드포인트는 현재 [User Auth](/ko/resources/fundamentals/authentication)를 지원합니다. 기본적으로 요청당 100명의 사용자를 반환하며, 요청당 최대 1000명의 사용자를 지정할 수 있습니다. 24시간당 10,000회 요청 한도가 있으며(15분당 300회 요청 한도에 추가로 적용됨)입니다.

<Note>
  **계정 설정**

  이 엔드포인트에 액세스하려면 다음이 필요합니다.

  * 승인된 [개발자 계정](https://developer.x.com/en/portal/petition/essential/basic-info)
  * [프로젝트](/ko/resources/fundamentals/projects) 내에 있는 [개발자 앱](/ko/resources/fundamentals/developer-apps)의 키와 토큰을 사용한 인증

  X API v2 엔드포인트에 대한 액세스 권한을 얻는 방법은 [시작하기 가이드](/ko/x-api/getting-started/getting-access)를 참조하세요.
</Note>
