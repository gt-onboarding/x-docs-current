  ## X API의 사용자 조회 엔드포인트 비교
</div>

v2 사용자 조회 엔드포인트는 기존 v1.1 표준 [GET users/lookup](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/follow-search-get-users/api-reference/get-users-lookup.html) 및 [GET users/show](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/follow-search-get-users/api-reference/get-users-show.html) 엔드포인트를 대체합니다. 현재 이러한 버전의 사용자 조회 엔드포인트 중 하나를 사용하는 코드, 앱 또는 도구가 있고 최신 X API v2 엔드포인트로의 마이그레이션을 고려하고 있다면, 이 일련의 가이드를 참고하면 좋습니다. 
 

다음 표는 다양한 유형의 사용자 조회 엔드포인트를 비교한 것입니다:
 

|     |     |     |
| :--- | :--- | :--- |
| **설명** | **Standard v1.1** | **X API v2** |
| 지원 HTTP 메서드 | `GET` | `GET` |
| 호스트 도메인 | `https://api.x.com` | `https://api.x.com` |
| 엔드포인트 경로 | `/1.1/users/show.json` `/1.1/users/lookup.json` | `/2/users`<br /><br />`/2/users/:id`<br /><br />`/2/users/by`<br /><br />`/2/users/by/:username` |
| [인증](/ko/resources/fundamentals/authentication) | OAuth 1.0a User Context | OAuth 1.0a User Context<br /><br />앱 전용<br /><br />OAuth 2.0 Authorization Code with PKCE |
| 기본 [요청 한도](/ko/resources/fundamentals/rate-limits) | 15분당 900회 요청(사용자 기준)<br /><br />/show - 15분당 900회 요청(앱 기준)  <br />/lookup - 15분당 300회 요청(앱 기준) | 15분당 900회 요청(사용자 기준)<br /><br />15분당 300회 요청(앱 기준) |
| 응답당 최대 사용자 수 | /show -  1명<br /><br />/lookup - 100명 | 100명 |
| JSON 응답 객체 형식 | Standard v1.1 형식 | [X API v2 형식](/ko/x-api/fundamentals/data-dictionary) (fields 및 expansions 요청 매개변수에 의해 결정되며, v1.1 형식과 하위 호환되지 않음)<br /><br />Standard v1.1 형식에서 X API v2 형식으로 마이그레이션하는 방법에 대해 더 알아보려면 [데이터 형식 마이그레이션 가이드](/ko/x-api/migrate/data-format-migration)를 참조하세요. |
| 페이로드에 어떤 [fields](/ko/x-api/fundamentals/data-dictionary)를 반환할지 선택 가능 |     | ✔   |
| 고정된 게시물에서 [annotations](/ko/x-api/fundamentals/post-annotations) 필드 지원 |     | ✔   |
| 고정된 게시물에서 새로운 [metrics](/ko/x-api/fundamentals/metrics) 필드 요청 지원 |     | ✔   |
| 고정된 게시물에서 [conversation&#95;id](/ko/x-api/fundamentals/conversation-id) 필드 지원 |     | ✔   |
| [project](/ko/resources/fundamentals/projects)와 연결된 [developer App](/ko/resources/fundamentals/developer-apps)의 자격 증명 사용 필수 |     | ✔   |
