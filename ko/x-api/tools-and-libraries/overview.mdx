이 페이지 아래쪽에는 [커뮤니티에서 지원하는 라이브러리](#community-libraries) 목록도 제공됩니다.

<div id="community-tools-and-libraries-for-v2">
  ## v2용 커뮤니티 도구 및 라이브러리
</div>

여기에 나열된 라이브러리는 개발자 커뮤니티 구성원들이 만든 것입니다. 각 라이브러리는 지원하는 API 범위나 완성도가 서로 다를 수 있습니다.

직접 만든 X API 라이브러리나 유용한 도구가 있다면 [알려주세요](https://devcommunity.x.com/c/libraries-and-sdks/63). 다른 사람들이 쉽게 찾을 수 있도록 이 목록에 추가하겠습니다. README 파일에서 사용할 수 있도록 가져다 쓸 수 있는 [버전 배지](https://twbadges.glitch.me/)도 제공하고 있습니다.

아이디어가 필요하신가요? GitHub에서 [X](https://github.com/topics/twitter?o=desc&s=updated) 및 [X-api-v2](https://github.com/topics/twitter-api-v2?o=desc&s=updated) 토픽을 둘러보고 검색하여 다른 개발자들의 유용한 코드 예제를 찾아볼 수 있습니다.

**바로가기:** [C# / .NET](#csharp), [Dart / Flutter](#dart), [Go](#go), [Java](#java), [JavaScript (Node.JS) / TypeScript](#nodejs), [Kotlin](#kotlin), [PHP](#php), [PowerShell](#powershell), [Python](#python), [R](#r), [Ruby](#ruby), [Rust](#rust), [Swift](#swift)

<div id="community-tools-and-libraries">
  ## 커뮤니티 도구 및 라이브러리
</div>

다음은 여러 프로그래밍 언어와 플랫폼에서 X API를 다룰 수 있게 해 주는 커뮤니티 지원 라이브러리들 중 일부입니다. 이 리소스가 모두 Twitter 팀에 의해 테스트된 것은 아니라는 점에 유의하세요.

여기에 나열된 라이브러리는 별도 언급이 없는 한 Standard API **v1.1**의 대부분 기능을 구현합니다. 자세한 내용과 추가 지원은 각 라이브러리 작성자에게 문의하세요.\
X API **v2**를 지원하는 라이브러리를 개발했다면, 이 페이지에 추가될 수 있도록 [커뮤니티 포럼](https://devcommunity.x.com/c/libraries-and-sdks/63)을 통해 알려 주세요. 또한 이 목록의 변경 사항을 보고하는 용도로도 포럼을 사용할 수 있습니다.

선호하는 프로그래밍 언어용 라이브러리나 도구가 없다면, [피드백 플랫폼](https://twitterdevfeedback.uservoice.com/)을 통해 알려 주세요. 이 플랫폼에서 아이디어에 투표하거나, 새로운 것을 만들어 제출하는 데 영감을 얻을 수도 있습니다.
