<div id="community-tools-and-libraries-for-v2">
  ## v2용 커뮤니티 도구 및 라이브러리
</div>

여기에 나열된 라이브러리는 개발자 커뮤니티 구성원들이 만든 것입니다. 각 라이브러리는 X API 지원 범위 수준이 제각각일 수 있습니다.

직접 만든 X API 라이브러리나 유용한 도구가 있다면 [여기에서 알려 주세요](https://devcommunity.x.com/c/libraries-and-sdks/63). 다른 사람들이 쉽게 찾을 수 있도록 이 리스트에 추가하겠습니다. 또한 README 파일에서 사용할 수 있도록 가져다 쓸 수 있는 [버전 배지](https://twbadges.glitch.me/)도 제공하고 있습니다.

아이디어가 필요하다면 GitHub의 [X](https://github.com/topics/twitter?o=desc&s=updated) 및 [X-api-v2](https://github.com/topics/twitter-api-v2?o=desc&s=updated) 토픽을 둘러보고 검색해 보면서 다른 개발자들의 유용한 코드 예제를 찾아보세요.

**바로 가기:** [C# / .NET](#csharp), [Dart / Flutter](#dart), [Go](#go), [Java](#java), [JavaScript (Node.JS) / TypeScript](#nodejs), [Kotlin](#kotlin), [PHP](#php), [PowerShell](#powershell), [Python](#python), [R](#r), [Ruby](#ruby), [Rust](#rust), [Swift](#swift)

<div id="community-tools-and-libraries">
  ## 커뮤니티 도구 및 라이브러리
</div>

여기에는 여러 프로그래밍 언어와 플랫폼 전반에서 X API를 다루는 커뮤니티 제공 라이브러리 중 일부가 나와 있습니다. 이 리소스들이 모두 Twitter 팀에 의해 테스트된 것은 아닐 수 있다는 점에 유의하세요.

여기에 나열된 라이브러리는 별도의 언급이 없는 한 Standard API **v1.1**의 대부분 기능을 구현하고 있어야 합니다. 자세한 내용과 추가 지원은 각 라이브러리 작성자에게 문의하세요.\
X API **v2**를 지원하는 라이브러리를 개발했다면, 이 페이지에 추가될 수 있도록 [커뮤니티 포럼](https://devcommunity.x.com/c/libraries-and-sdks/63)을 통해 알려주세요. 또한 포럼을 사용해 이 목록의 변경 사항을 보고할 수도 있습니다.

선호하는 프로그래밍 언어용 라이브러리나 도구가 없다면 [피드백 플랫폼](https://twitterdevfeedback.uservoice.com/)을 통해 알려주세요. 이곳에서 아이디어에 투표하거나, 새로운 것을 만들어 제출할 영감을 얻을 수도 있습니다.
