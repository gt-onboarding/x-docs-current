X는 X API를 활용해 구축하는 개발자 커뮤니티에 헌신하고 있습니다. 이 약속의 일환으로, 우리는 API를 개발자에게 개방적이고 공정하게 제공하는 동시에, X 상의 이용자에게 안전하고 X 플랫폼 전체에 이로운 방향으로 운영하는 것을 목표로 합니다. X 콘텐츠를 오프라인에 저장하는 모든 개발자는, 그 데이터가 이용자의 의도와 X 상의 콘텐츠의 최신 상태를 반영하도록 하는 것이 매우 중요합니다. 예를 들어, X 이용자가 게시물을 삭제하거나 계정을 삭제할 때, 게시물을 보호 대상으로 전환할 때, 또는 게시물을 수정할 때, 그 사람의 기대와 의도를 X와 개발자 모두가 존중하는 것이 무엇보다 중요합니다.

실시간 규정 준수(compliance) 이벤트 스트림은 개발자가 [X Developer Agreement and Policy](https://developer.x.com/en/developer-terms/policy)에 따라 X 데이터를 유지 관리할 수 있도록 도구를 제공합니다. 

두 가지 c&#95;ompliance event&#95; 스트림이 있으며, 하나는 Post&#95; compliance&#95; 이벤트용이고, 다른 하나는 *User compliance* 이벤트용입니다. 이 스트림은 Enterprise 액세스에서 이용할 수 있으며, 대량의 데이터를 수집하는 파트너가 게시물 수정 이벤트(Post edit events)와 같은 규정 준수 이벤트를 &#39;수신(listen)&#39;할 수 있도록 설계되었습니다.

이 스트림은 다음과 같은 이벤트를 제공합니다: 

**Post compliance stream: **

* delete - 게시물이 삭제되었음을 나타냅니다.

* tweet&#95;edit - 게시물이 수정되었음을 나타내며, 수정된 게시물의 ID를 제공합니다. 

* withheld - 게시물이 하나 이상의 국가에서 보류(withhold)되었음을 나타냅니다. 

* drop - 게시물을 공개 화면에서 제거해야 함을 나타냅니다.

* undrop - 게시물을 다시 표시할 수 있으며 공개 상태로 취급할 수 있음을 나타냅니다.

**User compliance stream: **

* **user&#95;delete** - 사용자 계정이 삭제되었음을 나타냅니다.

* **user&#95;undelete** - 사용자 계정 삭제가 취소되었음을 나타냅니다.

* **user&#95;protect** - 사용자 계정이 비공개로 전환되었음을 나타냅니다.

* **user&#95;unprotect** - 사용자 계정이 공개로 전환되었음을 나타냅니다.

* **user&#95;withheld** - 사용자 계정이 하나 이상의 국가에서 보류(withhold)되었음을 나타냅니다. 

* **user&#95;suspend** - 사용자 계정이 정지되었음을 나타냅니다.

* **user&#95;unsuspend** - 사용자 계정 정지가 해제되었음을 나타냅니다.

* **user&#95;profile&#95;modification** - 사용자 프로필이 업데이트되었음을 나타냅니다. 여기에는 설명, 이름, 위치, URL 업데이트가 포함됩니다.

**계정 설정**

이 엔드포인트에 액세스하려면 다음이 필요합니다:

* 승인된 [developer account](https://developer.x.com/en/portal/petition/essential/basic-info).
* [Project](/ko/resources/fundamentals/projects) 내에 위치한 [developer App](/ko/resources/fundamentals/developer-apps)의 키와 토큰을 사용하여 인증해야 합니다. 

X API v2 엔드포인트에 액세스하는 방법은 [시작하기 안내서](/ko/x-api/getting-started/getting-access)에서 자세히 알아보세요.
