X는 X API를 활용해 서비스를 구축하는 개발자 커뮤니티를 위해 노력하고 있습니다. 이러한 노력의 일환으로, 우리는 API가 개발자에게 개방적이고 공정하며, X를 이용하는 사람들에게 안전하고, X 플랫폼 전체에 이로울 수 있도록 하는 것을 목표로 합니다. X 콘텐츠를 오프라인으로 저장하는 모든 개발자는 해당 데이터가 사용자의 의도와 X 상의 현재 콘텐츠 상태를 반영하도록 하는 것이 매우 중요합니다. 예를 들어, 누군가가 X에서 게시물이나 계정을 삭제하거나, 자신의 게시물을 보호 대상으로 전환하거나, 게시물을 수정하는 경우, 그 사람의 기대와 의도를 X와 우리 개발자 모두가 존중하는 것이 매우 중요합니다.

실시간 컴플라이언스 이벤트 스트림은 개발자가 [X Developer Agreement and Policy](https://developer.x.com/en/developer-terms/policy)에 따라 X 데이터를 유지할 수 있도록 하는 도구를 제공합니다. 

두 개의 c&#95;ompliance event&#95; 스트림이 있으며, 하나는 게시물&#95; 컴플라이언스&#95; 이벤트용이고, 다른 하나는 *사용자 컴플라이언스* 이벤트용입니다. 이러한 스트림은 Enterprise 액세스 보유 시 이용할 수 있으며, 대량의 데이터를 수집하면서 게시물 수정 이벤트와 같은 컴플라이언스 이벤트를 수신해야 하는 파트너를 지원하도록 설계되었습니다.

이 스트림은 다음과 같은 이벤트를 제공합니다: 

**게시물 컴플라이언스 스트림: **

* delete - 게시물이 삭제되었음을 나타냅니다.

* tweet&#95;edit - 게시물이 수정되었음을 나타내며, 수정된 게시물의 ID를 제공합니다. 

* withheld - 게시물이 하나 이상의 국가에서 차단되었음을 나타냅니다. 

* drop - 게시물을 공개 보기에서 제거해야 함을 나타냅니다.

* undrop - 게시물을 다시 표시할 수 있으며 공개로 취급할 수 있음을 나타냅니다.

**사용자 컴플라이언스 스트림: **

* **user&#95;delete** - 사용자 계정이 삭제되었음을 나타냅니다.

* **user&#95;undelete** - 사용자 계정의 삭제가 취소되었음을 나타냅니다.

* **user&#95;protect** - 사용자 계정이 비공개로 전환되었음을 나타냅니다.

* **user&#95;unprotect** - 사용자 계정이 공개로 전환되었음을 나타냅니다.

* **user&#95;withheld** - 사용자 계정이 하나 이상의 국가에서 차단되었음을 나타냅니다. 

* **user&#95;suspend** - 사용자 계정이 정지되었음을 나타냅니다.

* **user&#95;unsuspend** - 사용자 계정의 정지가 해제되었음을 나타냅니다.

* **user&#95;profile&#95;modification** - 사용자 프로필이 업데이트되었음을 나타냅니다. 여기에는 설명, 이름, 위치, URL 업데이트가 포함됩니다.

**계정 설정**

이 엔드포인트에 액세스하려면 다음이 필요합니다.

* 승인된 [개발자 계정](https://developer.x.com/en/portal/petition/essential/basic-info)
* [프로젝트](/ko/resources/fundamentals/projects) 내에 있는 [개발자 앱](/ko/resources/fundamentals/developer-apps)의 키와 토큰을 사용해 인증해야 합니다. 

X API v2 엔드포인트에 대한 액세스 권한을 얻는 방법에 대해 더 알아보려면 [시작 안내서](/ko/x-api/getting-started/getting-access)를 확인하세요.
