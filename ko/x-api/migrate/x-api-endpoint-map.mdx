다음 표는 X API v2 엔드포인트를 해당하는 표준 v1.1 및 Enterprise 엔드포인트에 매핑한 것입니다. 각 버전과 티어에 대해 자세히 알아보려면 [X API 시작하기 가이드](/ko/x-api/getting-started/about-x-api)를 참조하세요.

여러 항목이 아직 **[Coming Soon]**으로 표시되어 있는 것을 확인하실 수 있습니다. 이 표에서 **[Replacement Under Consideration]** 또는 **[No Replacement Planned]**로 표시된 엔드포인트에 대해 v2 버전이 제공되기를 원하신다면, [커뮤니티 포럼](https://devcommunity.x.com/) 또는 Enterprise 계정 담당자를 통해 알려주시기 바랍니다.
