<div id="basic-authentication">
  ## 기본 인증
</div>

X의 많은 Enterprise API는 HTTP Basic Authentication 사용을 필요로 합니다. Basic Authentication이 필요한 API에 성공적으로 요청하려면 각 요청마다 유효한 이메일 주소와 비밀번호 조합을 Authorization 헤더에 포함해 전달해야 합니다. 이 이메일 주소와 비밀번호 조합은 [enterprise API console](https://console.gnip.com/)에 접속할 때 사용하는 것과 동일하며, 이 콘솔 내에서 수정할 수 있습니다. 

Basic Authentication을 사용해 요청을 작성할 때는 HTTPS를 통해 인코딩된 자격 증명을 포함한 `Authentication: Basic` HTTP 헤더를 추가해야 합니다.

다음 cURL 요청 예시에서, 요청을 보내기 전에 `<email_address>`와 `<password>`를 자신의 자격 증명으로 바꾸면 됩니다:

```bash
curl -v --compressed -u<email_address>:<password>
    "https://gnip-api.x.com/search/30day/accounts/<account-name>/prod/counts.json?query=from%3Axdevelopers"
```
