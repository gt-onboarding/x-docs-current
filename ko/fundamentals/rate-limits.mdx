매일 수많은 개발자가 X 개발자 플랫폼에 요청을 보냅니다. 이러한 요청의 막대한 양을 관리하기 위해 전송할 수 있는 요청 수에는 한도가 설정되어 있습니다. 이러한 한도는 개발자 커뮤니티가 의존하는 안정적이고 확장 가능한 API를 제공하는 데 도움이 됩니다.

각 API에는 서로 다른 방식의 요청 한도가 적용됩니다. 플랫폼별 차이점을 더 자세히 알아보려면 각 API 섹션에 있는 다음 요청 한도 페이지를 참고하세요.

* [X API v2](/ko/x-api/fundamentals/rate-limits)
* [X API: Enterprise](/ko/x-api/enterprise-gnip-2.0/fundamentals/rate-limits)
* [X Ads API](/ko/x-ads-api/fundamentals/rate-limiting)
