<div id="products">
  ## 製品
</div>

X Developer Platform の主要な製品を紹介します。各製品は、さまざまな方法で X と連携した構築・分析・統合が行えるように設計されています。

<CardGroup cols={3}>
  <Card title="X API" icon="square-x-twitter" iconType="solid" href="/ja/x-api/introduction">
    X の中核となるデータ（ポスト、ユーザー、スペース、DM、リスト、トレンド、メディアなど）へのプログラムによるアクセスを提供します。
  </Card>

  <Card title="X Ads API" icon="rectangle-ad" iconType="solid" href="/ja/x-ads-api/introduction">
    X Ads プラットフォーム上の広告キャンペーン、ターゲティング、クリエイティブ、アナリティクスを自動化・管理します。
  </Card>

  <Card title="X for Websites" icon="laptop-code" iconType="solid" href="https://developer.x.com/en/docs/twitter-for-websites.html">
    X のコンテンツ、タイムライン、エンゲージメントツールを、ウェブサイトやアプリに直接埋め込むことができます。
  </Card>
</CardGroup>
