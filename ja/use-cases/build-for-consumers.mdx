<div id="ways-to-build-for-people-on-x">
  ## X 上の人々のために構築する方法
</div>

私たちは、公共の会話に良い影響を与え、X 上の人々に貢献する新しいツールやプロダクトをあなたが構築できるよう、プラットフォームを開放しています。とくに、次の分野における新たなイノベーションと開発を奨励したいと考えています。

<CardGroup>
  <Card title="健全性と安全性のために会話をモデレートする" icon="bullhorn" iconType="solid">
    人々が X 上の会話に安心して前向きな気持ちで参加できるようにするコンテンツモデレーションツールやその他のソリューションを構築することで、公共の会話の健全性と安全性の向上に貢献してください。人々が自分の意図どおりに会話をコントロールできるよう、補完的・付加的なテクノロジーを構築し、幅広いツールの選択肢を提供しましょう。

    [**詳細を見る**](/ja/x-api/what-to-build#moderate-conversations-for-health-and-safety) <Icon icon="arrow-right" />
  </Card>

  <Card title="創造と自己表現を可能にする" icon="bullhorn" iconType="solid">
    クリエイター、ブランド、そしてリーチを広げ、自身のブランドを表現し、情熱を共有する人たちとつながりたい個人のためのツールを構築しましょう。あるいは、クロスポストされたコンテンツ、ゲーミフィケーション、有用なボットなどを通じて、人々が X 上で関わる方法を広げましょう。

    [**詳細を見る**](/ja/x-api/what-to-build#enable-creation-and-personal-expression) <Icon icon="arrow-right" />
  </Card>

  <Card title="「いま起きていること」を計測・分析する" icon="bullhorn" iconType="solid">
    公共の会話や自分自身のコンテンツを計測・分析し、インサイトを得られるように支援してください。オーディエンスをより深く理解し、コンテンツの成果と影響を測定し、より多くの情報に基づいた意思決定を行うなど、さまざまな活用が可能です。

    [**詳細を見る**](/ja/x-api/what-to-build#measure-and-analyze-whats-happening) <Icon icon="arrow-right" />
  </Card>

  <Card title="コミュニティ体験を向上させる" icon="bullhorn" iconType="solid">
    X はすべての人のためのものであり、X 上の多様なコミュニティ固有のニーズにより良く応えられるよう、あなたの専門性を活かしてもらいたいと考えています。新興のグローバル市場向けにローカライズされたソリューションを構築する場合でも、ゲームや金融など、プラットフォーム上で成長するコミュニティ向けの体験を強化する場合でも、誰もが X 上に自分の居場所を見つけられることを目指しています。

    [**詳細を見る**](/ja/x-api/what-to-build#improve-community-experiences) <Icon icon="arrow-right" />
  </Card>

  <Card title="コンテンツをキュレーションし、レコメンドする" icon="bullhorn" iconType="solid">
    人々が新しいコンテンツを発見したり、新たなつながりを見つけたりするのに役立つアプリやボット、その他のツールを構築してください。また、人々が最も関心のあるコンテンツをより多く見られるよう、キュレーションされ、カスタマイズ可能な体験を構築することもできます。

    [**詳細を見る**](/ja/x-api/what-to-build#curate-and-recommend-content) <Icon icon="arrow-right" />
  </Card>

  <Card title="より大きな公益に貢献する" icon="bullhorn" iconType="solid">
    先駆的な研究を実施し、公衆衛生や気候危機に関する知見の発展に貢献し、非営利目的で構築することで、より大きな公益の実現に資することができます。

    [**詳細を見る**](/ja/x-api/what-to-build#impact-the-greater-good) <Icon icon="arrow-right" />
  </Card>
</CardGroup>
