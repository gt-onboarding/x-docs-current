<div id="ways-to-build-for-people-on-x">
  ## X を利用する人々のために構築する方法
</div>

私たちは、公共の会話にポジティブな影響を与え、X 上の人々に役立つ新しいツールやプロダクトを構築していただけるよう、プラットフォームを開放しています。特に、次の分野での新たなイノベーションと開発を奨励したいと考えています。

<CardGroup>
  <Card title="健全性と安全性のために会話をモデレートする" icon="bullhorn" iconType="solid">
    コンテンツモデレーションツールやその他のソリューションを構築し、公共の会話の健全性と安全性を高めてください。人々が X 上の会話に安心して、そして意欲的に参加できるようにしましょう。人々が自分のインタラクションや体験を細かくコントロールできるようにする、包括的なツールの選択肢を提供してください。

    [**詳細を見る**](/ja/x-api/what-to-build#moderate-conversations-for-health-and-safety) <Icon icon="arrow-right" />
  </Card>

  <Card title="創造と個人表現を可能にする" icon="bullhorn" iconType="solid">
    クリエイターやブランド、リーチを広げて自分たちのブランドを表現し、情熱を共有する人々とつながりたい個人のためのツールを構築してください。あるいは、クロスポストされたコンテンツ、ゲーミフィケーション、便利なボットなどを通じて、X 上でのエンゲージのあり方を広げてください。

    [**詳細を見る**](/ja/x-api/what-to-build#enable-creation-and-personal-expression) <Icon icon="arrow-right" />
  </Card>

  <Card title="「いま起きていること」を計測・分析する" icon="bullhorn" iconType="solid">
    公共の会話や自分自身のコンテンツを計測・分析し、インサイトを得られるよう人々を支援してください。オーディエンスをより深く理解し、コンテンツのパフォーマンスや影響を測定し、より良い意思決定を行うことなどに役立てることができます。

    [**詳細を見る**](/ja/x-api/what-to-build#measure-and-analyze-whats-happening) <Icon icon="arrow-right" />
  </Card>

  <Card title="コミュニティ体験を向上させる" icon="bullhorn" iconType="solid">
    X はすべての人のための場であり、X 上のさまざまなコミュニティそれぞれのニーズにより良く応えられるよう、皆さんの専門性を生かしていただきたいと考えています。新興のグローバル市場向けにローカライズされたソリューションを構築する場合でも、ゲームや金融のようにプラットフォーム上で成長しているコミュニティの体験を強化する場合でも、誰もが X 上に自分の居場所を見つけられるようにしたいと考えています。

    [**詳細を見る**](/ja/x-api/what-to-build#improve-community-experiences) <Icon icon="arrow-right" />
  </Card>

  <Card title="コンテンツをキュレーション・推薦する" icon="bullhorn" iconType="solid">
    人々が新しいコンテンツを見つけたり、新たなつながりを得たりできるようにするアプリやボット、その他のツールを構築してください。あるいは、キュレーションされカスタマイズ可能な体験を構築し、人々が最も関心のあるコンテンツをより多く見られるようにしてください。

    [**詳細を見る**](/ja/x-api/what-to-build#curate-and-recommend-content) <Icon icon="arrow-right" />
  </Card>

  <Card title="より大きな公益に貢献する" icon="bullhorn" iconType="solid">
    画期的な研究を行い、公衆衛生や気候危機に関する知識の発展を支援し、商業目的ではなく、より大きな公益のために構築してください。

    [**詳細を見る**](/ja/x-api/what-to-build#impact-the-greater-good) <Icon icon="arrow-right" />
  </Card>
</CardGroup>
