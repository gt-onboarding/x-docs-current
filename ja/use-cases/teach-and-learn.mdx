<div id="educators">
  ## 教育者向け
</div>

API を授業に取り入れて、学生にプログラミングの基礎と実践的なデータ処理スキルを教えましょう。

**リソース:**

授業で X API を利用したい教育者の方は、この
[X API を使った教育ガイド](https://dev.to/suhemparack/a-guide-to-teaching-with-the-twitter-api-v2-3n08)
を参照してください。X API とは何か、なぜ教育現場で使われているのか、この API を用いてどのような授業が行われているのか、そして始めるにあたって知っておくべきことを一通り把握できます。

**注目の事例:**

以下のカリキュラムは、X API v2 を活用して設計・実施されています。

- [ソーシャルメディアのデータマイニングと分析](https://canvas.uw.edu/courses/1434897)
  （Tanu Mitra 博士）

- [データサイエンスと社会](https://dssoc.github.io/)（Chris Bail 博士）

- [文化分析と Python 入門](https://melaniewalsh.github.io/Intro-Cultural-Analytics/welcome.html)
  （Melanie Walsh 博士）

- [高度な研究手法のためのオックスフォード・スプリングスクール：X データの分析](https://github.com/cjbarrie/oxspsch)
  （Chris Barrie 博士）
