<div id="educators">
  ## 教育者向け
</div>

プログラミングの基礎や実践的なデータ処理スキルを教えるために、API を授業に取り入れましょう。

**リソース:**

授業で X API を活用したいと考えている教育者の方には、
[「X API を使った授業」のガイド](https://dev.to/suhemparack/a-guide-to-teaching-with-the-twitter-api-v2-3n08)
が役立ちます。このガイドでは、X API の概要、教育現場で利用されている理由、
どのような授業で API が使われているか、そして始めるにあたって知っておくべき
ポイントを理解できます。

**スポットライト:**

以下のカリキュラムは、X API v2 を用いて構築および実施されています。

- [Social Media Data Mining and Analysis](https://canvas.uw.edu/courses/1434897)
  by Dr. Tanu Mitra

- [Data Science & Society](https://dssoc.github.io/) by Dr. Chris Bail

- [Introduction to Cultural Analytics & Python](https://melaniewalsh.github.io/Intro-Cultural-Analytics/welcome.html)
  by Dr. Melanie Walsh

- [Oxford Spring School in Advanced Research Methods, Analysing X Data](https://github.com/cjbarrie/oxspsch)
  by Dr. Chris Barrie
