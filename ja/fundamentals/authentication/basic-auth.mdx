<div id="basic-authentication">
  ## Basic authentication
</div>

多くの X の Enterprise API では、HTTP Basic Authentication を使用する必要があります。Basic Authentication を必要とする API へ正常にリクエストを送信するには、有効なメールアドレスとパスワードの組み合わせを、各リクエストの認証ヘッダーとして渡す必要があります。メールアドレスとパスワードの組み合わせは、[enterprise API console](https://console.gnip.com/) にアクセスする際に使用するものと同じであり、このコンソール内から編集できます。

Basic Authentication を使ってリクエストを構築する際は、資格情報をエンコードした `Authentication: Basic` の HTTP ヘッダーを、HTTPS 経由で必ず追加してください。

次の cURL リクエスト例では、リクエストを送信する前に `<email_address>` と `<password>` を自身の認証情報に置き換えてください。

```bash
curl -v --compressed -u<email_address>:<password>
    "https://gnip-api.x.com/search/30day/accounts/<account-name>/prod/counts.json?query=from%3Axdevelopers"
```
