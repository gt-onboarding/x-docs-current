<div id="basic-authentication">
  ## Basic authentication
</div>

多くの X Enterprise API では、HTTP Basic Authentication の利用が必要です。Basic Authentication を必要とする API に正常にリクエストを行うには、有効なメールアドレスとパスワードの組み合わせを、各リクエストの Authorization ヘッダーとして渡す必要があります。メールアドレスとパスワードの組み合わせは、[Enterprise API コンソール](https://console.gnip.com/) にアクセスする際に使用するものと同じであり、このコンソール内から編集できます。 

Basic Authentication を使用してリクエストを作成する際は、HTTPS 経由でエンコード済みの認証情報を含んだ Authentication: Basic HTTP ヘッダーを必ず追加してください。

次の cURL リクエスト例では、リクエストを送信する前に `<email_address>` と `<password>` を自分の認証情報に置き換えます。

```bash
curl -v --compressed -u<email_address>:<password>
    "https://gnip-api.x.com/search/30day/accounts/<account-name>/prod/counts.json?query=from%3Axdevelopers"
```
