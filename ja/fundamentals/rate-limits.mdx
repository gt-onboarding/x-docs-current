毎日、何千人もの開発者が X デベロッパープラットフォームにリクエストを送信しています。これらの膨大な数のリクエストを管理するために、送信できるリクエスト数には制限が設けられています。これらの制限によって、私たちのデベロッパーコミュニティが依存している、信頼性が高くスケーラブルな API を提供し続けることができます。

各 API では、レート制限の適用方法がそれぞれ異なります。プラットフォーム間の違いについて詳しく確認するには、各 API セクション内にある、それぞれのレート制限のページを参照してください。

* [X API v2](/ja/x-api/fundamentals/rate-limits)
* [X API: Enterprise](/ja/x-api/enterprise-gnip-2.0/fundamentals/rate-limits)
* [X Ads API](/ja/x-ads-api/fundamentals/rate-limiting)
