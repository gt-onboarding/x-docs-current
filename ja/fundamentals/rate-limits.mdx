毎日、何千人もの開発者が X Developer Platform にリクエストを送信しています。こうした膨大な数のリクエストを管理するため、実行できるリクエスト数には上限が設けられています。これらの上限により、開発者コミュニティが日々利用している、信頼性が高くスケーラブルな API を提供し続けることができます。

各 API は、それぞれ異なる方法でレート制限を適用しています。プラットフォーム間の違いについて詳しく知るには、各 API セクション内のレート制限に関するページを参照してください。

* [X API v2](/ja/x-api/fundamentals/rate-limits)
* [X API: Enterprise](/ja/x-api/enterprise-gnip-2.0/fundamentals/rate-limits)
* [X Ads API](/ja/x-ads-api/fundamentals/rate-limiting)
