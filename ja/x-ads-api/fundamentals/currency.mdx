通貨のタイプは [ISO-4217](http://en.wikipedia.org/wiki/ISO_4217) によって識別されます。これは `USD` や `EUR` のような 3 文字の文字列です。通貨の金額はマイクロ（micros）単位で表現されます。USD の場合、$5.50 は 5.50*1e6、つまり 5,500,000 としてエンコードされます。「整数値」をマイクロ単位で表現するには、すべての通貨においてローカル通貨の金額に 1e6 (1&#95;000&#95;000) を掛ける必要があります。

Ads API は、X SMB Ads プロダクトが提供されているすべての国の公式通貨を認識します。これらの国の完全な一覧は[こちら](https://business.x.com/en/help/overview/about-eligibility-for-twitter-ads.html)をご覧ください。

ほとんどの通貨の「精度」（小数点以下の桁数）は 2 です (USD の例: $1.23)。

* `AED`, `VEF`, `PKR`, `IDR`, `HKD`, `QAR`, `PHP`, `NZD`, `INR`, `EUR`, `ARS`, `THB`, `MXN`, `EGP`, `DZD`, `SAR`, `LBP`, `COP`, `TRY`, `SGD`, `MAD`, `AUD`, `USD`, `MYR`, `CAD`, `BRL`, `GBP`.

次の通貨は小数部を使用しません:

* `VND`, `KRW`, `CLP`, `JPY`

また、次の通貨は小数点以下 3 桁を使用します:

* `KWD`, `IQD`, `BHD`, `TND`

通貨の利用は [funding instruments](/ja/x-ads-api/introduction) を通じて行われます。
