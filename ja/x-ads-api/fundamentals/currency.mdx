通貨の種類は [ISO-4217](http://en.wikipedia.org/wiki/ISO_4217) を使って識別されます。これは `USD` や `EUR` のような 3 文字の文字列です。通貨の値はマイクロ単位で表現されます。USD の場合、$5.50 は 5.50*1e6、つまり 5,500,000 としてエンコードされます。「整数値」を表現するには、すべての通貨でローカル通貨のマイクロ値に 1e6 (1&#95;000&#95;000) を掛ける必要があります。

Ads API は、X SMB Ads プロダクトが利用可能なすべての国の公式通貨を認識します。これらの国の完全な一覧については、[こちら](https://business.x.com/en/help/overview/about-eligibility-for-twitter-ads.html)をご覧ください。

ほとんどの通貨の「精度」は小数点以下 2 桁です (USD の場合: $1.23)。

* `AED`, `VEF`, `PKR`, `IDR`, `HKD`, `QAR`, `PHP`, `NZD`, `INR`, `EUR`, `ARS`, `THB`, `MXN`, `EGP`, `DZD`, `SAR`, `LBP`, `COP`, `TRY`, `SGD`, `MAD`, `AUD`, `USD`, `MYR`, `CAD`, `BRL`, `GBP`.

次の通貨は小数を使用しません:

* `VND`, `KRW`, `CLP`, `JPY`

また、これらは小数第 3 位まで使用します:

* `KWD`, `IQD`, `BHD`, `TND`

通貨の取り扱いは、[funding instruments](/ja/x-ads-api/introduction) を通じて行われます。
