サンドボックスは、Ads API 実装のテストと構築を行うための環境です。

サンドボックス経由で作成されたキャンペーンはユーザーに配信されず、Funding Instruments などのオブジェクト用のテストレコードを動的に作成できます。

サンドボックスは、ads-api-sandbox.x.com の適切なホスト名を呼び出すだけで利用できますが、サンドボックスにのみ存在する固有の機能も備えています。

新しい広告主アカウントを作成するには、まず [POST accounts](/ja/x-ads-api/campaign-management#post-accounts) にリクエストを送信します。この呼び出しのレスポンスには、その後のリクエストで使用できる account&#95;id が含まれます。これらのエンドポイントにより、複数の広告主アカウントを作成し、それぞれに対してテストを行うことができます。

<div id="sandbox-only-endpoint-list">
  ### サンドボックス専用のエンドポイント一覧:
</div>

* [POST accounts](/ja/x-ads-api/campaign-management#post-accounts) サンドボックス環境でアカウントを作成
* [POST accounts/:account&#95;id/features](/ja/x-ads-api/campaign-management#post-accounts-account-id-features) サンドボックス環境のアカウントにアカウント機能を追加
* [POST accounts/:account&#95;id/funding&#95;instruments](/ja/x-ads-api/campaign-management#post-accounts-account-id-funding-instruments) サンドボックス環境のアカウントに資金手段を追加
* [DELETE accounts/:account&#95;id](/ja/x-ads-api/campaign-management#delete-accounts-account-id) サンドボックス環境のアカウントを削除
* [DELETE accounts/:account&#95;id/features](/ja/x-ads-api/campaign-management#delete-accounts-account-id-features) サンドボックス環境のアカウントからアカウント機能を削除
* [DELETE accounts/:account&#95;id/funding&#95;instruments/:funding&#95;instrument&#95;id](/ja/x-ads-api/campaign-management#delete-accounts-account-id-funding-instruments-funding-instrument-id) サンドボックス環境のアカウントから資金手段を削除
