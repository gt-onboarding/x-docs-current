サンドボックスは、Ads API 実装のテストおよび構築を行うための環境です。

サンドボックス経由で作成されたキャンペーンはユーザーには配信されず、Funding Instruments などのオブジェクトに対するテストレコードを動的に作成できます。

サンドボックスの利用は、ads-api-sandbox.x.com という適切なホスト名にリクエストを送るだけとシンプルですが、サンドボックスでのみ利用できる固有の機能も含まれています。

新しい広告主アカウントを作成するには、まず [POST accounts](/ja/x-ads-api/campaign-management#post-accounts) にリクエストを送信してください。この呼び出しからのレスポンスには、その後のリクエストで使用できる account&#95;id が含まれます。これらのエンドポイントを使用することで、複数の広告主アカウントを作成し、それらに対してテストを行うことができます。

<div id="sandbox-only-endpoint-list">
  ### サンドボックス専用エンドポイント一覧:
</div>

* [POST accounts](/ja/x-ads-api/campaign-management#post-accounts) サンドボックス環境でアカウントを作成
* [POST accounts/:account&#95;id/features](/ja/x-ads-api/campaign-management#post-accounts-account-id-features) サンドボックス環境のアカウントにアカウント機能を追加
* [POST accounts/:account&#95;id/funding&#95;instruments](/ja/x-ads-api/campaign-management#post-accounts-account-id-funding-instruments) サンドボックス環境のアカウントに支払い手段を追加
* [DELETE accounts/:account&#95;id](/ja/x-ads-api/campaign-management#delete-accounts-account-id) サンドボックス環境のアカウントを削除
* [DELETE accounts/:account&#95;id/features](/ja/x-ads-api/campaign-management#delete-accounts-account-id-features) サンドボックス環境のアカウントからアカウント機能を削除
* [DELETE accounts/:account&#95;id/funding&#95;instruments/:funding&#95;instrument&#95;id](/ja/x-ads-api/campaign-management#delete-accounts-account-id-funding-instruments-funding-instrument-id) サンドボックス環境のアカウントから支払い手段を削除
