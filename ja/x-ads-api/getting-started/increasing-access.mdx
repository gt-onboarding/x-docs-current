<div id="requesting-additional-app-level-permissions">
  ### 追加のアプリレベル権限のリクエスト
</div>

アプリケーションが以下のいずれかのアプリレベル権限に制限されていて、追加のエンドポイントへのアクセスを希望する場合は、**Standard Access** を申請するために担当の X 窓口までご連絡ください。

2023 年 7 月以前に Ads API へのアクセスを申請した開発者の中には、現在は Standard Access レベルに含まれている以下のいずれかのアプリレベル権限を申請していた方もいる可能性があります。

- **Analytics**（読み取り専用）：ユーザーは [Analytics](https://developer.x.com/en/docs/twitter-ads-api/analytics/overview) エンドポイントに対する[読み取り専用](https://developer.x.com/en/docs/apps/app-permissions)アクセス権を有します。ユーザーは Campaign Management、Creative、Custom Audiences エンドポイントにはアクセスできませんが、Analytics エンドポイントのリクエストパラメータでクリエイティブエンティティを指定することでメトリクスを取得できます。`media_id` または `media_name` の粒度で取得するには、ユーザーのアプリケーションに Ads API の書き込み権限が必要です。

- **Campaign Management & Creative**（読み取りと書き込み）：ユーザーは [Campaign Management](https://developer.x.com/en/docs/twitter-ads-api/campaign-management/overview) および [Creative](https://developer.x.com/en/docs/twitter-ads-api/creatives/overview) エンドポイントに対する[読み取りと書き込み](https://developer.x.com/en/docs/apps/app-permissions)アクセス権を有します。ユーザーは Analytics エンドポイントにもアクセスできます。

- **Custom Audiences**（読み取りと書き込み）：ユーザーは [Custom Audiences](https://developer.x.com/en/docs/twitter-ads-api/audiences/overview) エンドポイントに対する[読み取りと書き込み](https://developer.x.com/en/docs/apps/app-permissions)アクセス権を有します。ユーザーは Campaign Management、Creative、および Analytics エンドポイントにもアクセスできます。

- **Conversion**（読み取りと書き込み）：ユーザーは [Mobile](https://developer.x.com/en/docs/twitter-ads-api/measurement/mobile-conversions/overview) および [Web Conversion](https://developer.x.com/en/docs/twitter-ads-api/measurement/web-conversions/overview) エンドポイントに対する[読み取りと書き込み](https://developer.x.com/en/docs/apps/app-permissions)アクセス権を有します。ユーザーが他の Ads API エンドポイントにアクセスできる場合も、できない場合もあります。 

<div id="requesting-an-increase-to-user-oauth-token-limits">
  ### ユーザー OAuth トークン上限引き上げのリクエスト
</div>

2023 年 7 月より前に Ads API へのアクセスをリクエストした開発者は、アプリケーションに対してユーザー OAuth トークン数が 5 個に制限されている場合があります。開発者アプリのトークン上限を引き上げる必要がある場合は、X の担当者にお問い合わせください。

<Note>**注:** いかなる Ads API エンドポイントについても、[レート制限](https://developer.x.com/en/docs/twitter-ads-api/rate-limiting) を引き上げることはありません。[ベストプラクティス](https://developer.x.com/en/docs/twitter-ads-api/rate-limiting) を参照して、レート制限の範囲内で運用する方法を確認してください。</Note>
