<div id="requesting-additional-app-level-permissions">
  ### 追加のアプリレベル権限のリクエスト
</div>

ご利用のアプリケーションが以下のいずれかのアプリレベル権限に限定されており、追加のエンドポイントへのアクセスを希望する場合は、**Standard Access** を申請するために X の担当者までご連絡ください。

2023 年 7 月より前に Ads API アクセスを申請した開発者は、以下のいずれかのアプリレベル権限を申請している場合がありますが、これらは現在 Standard Access レベルに含まれています。

- **Analytics**（読み取り専用）：ユーザーは [Analytics](https://developer.x.com/en/docs/twitter-ads-api/analytics/overview) エンドポイントに [読み取り専用](https://developer.x.com/en/docs/apps/app-permissions) でアクセスできます。ユーザーは Campaign Management、Creative、Custom Audiences の各エンドポイントにはアクセスできませんが、Analytics エンドポイントのリクエストパラメータでクリエイティブエンティティを指定することによりメトリクスを取得できます。`media_id` または `media_name` の粒度で取得するには、ユーザーのアプリケーションに Ads API の書き込みアクセス権が必要です。

- **Campaign Management & Creative**（読み取りおよび書き込み）：ユーザーは [Campaign Management](https://developer.x.com/en/docs/twitter-ads-api/campaign-management/overview) および [Creative](https://developer.x.com/en/docs/twitter-ads-api/creatives/overview) エンドポイントに [読み取りおよび書き込み](https://developer.x.com/en/docs/apps/app-permissions) でアクセスできます。ユーザーは Analytics エンドポイントにもアクセスできます。

- **Custom Audiences**（読み取りおよび書き込み）：ユーザーは [Custom Audiences](https://developer.x.com/en/docs/twitter-ads-api/audiences/overview) エンドポイントに [読み取りおよび書き込み](https://developer.x.com/en/docs/apps/app-permissions) でアクセスできます。ユーザーは Campaign Management、Creative、Analytics の各エンドポイントにもアクセスできます。

- **Conversion**（読み取りおよび書き込み）：ユーザーは [Mobile](https://developer.x.com/en/docs/twitter-ads-api/measurement/mobile-conversions/overview) および [Web Conversion](https://developer.x.com/en/docs/twitter-ads-api/measurement/web-conversions/overview) エンドポイントに [読み取りおよび書き込み](https://developer.x.com/en/docs/apps/app-permissions) でアクセスできます。ユーザーが他の Ads API エンドポイントにアクセスできる場合もあれば、できない場合もあります。 

<div id="requesting-an-increase-to-user-oauth-token-limits">
  ### ユーザー OAuth トークン上限の引き上げを申請する
</div>

2023 年 7 月以前に Ads API へのアクセスを申請した開発者は、アプリケーションのユーザー OAuth トークン上限が 5 に設定されている場合があります。開発者アプリのトークン上限を引き上げる必要がある場合は、X の担当者までご連絡ください。

<Note>**注意:** どの Ads API エンドポイントについても、[レート制限](https://developer.x.com/en/docs/twitter-ads-api/rate-limiting) を引き上げることはありません。レート制限内で運用する方法については、[ベストプラクティス](https://developer.x.com/en/docs/twitter-ads-api/rate-limiting) をご覧ください。</Note>
