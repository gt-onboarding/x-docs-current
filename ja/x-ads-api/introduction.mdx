<div id="connect-to-the-x-ads-platform">
  ## X 広告プラットフォームに接続する
</div>

X Ads API を利用すると、開発者は X の広告プラットフォームに接続し、世界中の X 広告主のニーズに応えるソリューションを構築できます。

  はじめる
</Button>

***

<CardGroup cols={2}>
  <Card title="Campaign management" icon="list-check" href="/ja/x-ads-api/campaign-management">X 上のユーザーとエンゲージする広告キャンペーンを、プログラムから作成・スケジュール設定・管理できます。</Card>
  <Card title="Custom audiences" href="/ja/x-ads-api/audiences" icon="user-group">お客様が提供する X 上のデータやウェブ／モバイルのデータを使って、Tailored Audiences を作成および管理できます。</Card>
  <Card title="Creatives" href="/ja/x-ads-api/creatives" icon="paintbrush">ポストの下書きと公開を行い、Card を使ってポストの機能を拡張し、画像や動画を管理・アップロードできます。</Card>
  <Card title="Advertising analytics" href="/ja/x-ads-api/analytics" icon="chart-pie">非同期および同期 API エンドポイントを使用して、幅広いメトリクスに基づく広告キャンペーンの詳細なインサイトを取得できます。</Card>
  <Card title="Tools and libraries" href="/ja/x-ads-api/tools-and-libraries" icon="screwdriver-wrench">X 製およびコミュニティがサポートするクライアントライブラリを厳選したコレクションをご覧ください。</Card>
  <Card title="Postman collection" href="https://documenter.getpostman.com/view/7379786/SzRw1WTU" icon="code">v2 エンドポイント用の Postman コレクションを用意しており、Postman のビジュアルクライアントを使って API を探索できます。</Card>
  <Card title="Command line tool" href="https://github.com/xdevplatform/xurl" icon="terminal">コマンドラインツールからエンドポイントを扱いたい場合は、xurl が最適です。</Card>
</CardGroup>
