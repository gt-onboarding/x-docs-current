<div id="mact-overview">
  ## MACT の概要
</div>

X の[モバイルアプリプロモーション](https://biz.x.com/mobile-app-promotion)計測機能により、広告主は、インストールやその他のアプリ内コンバージョンの促進を目的として X 上で実施する広告キャンペーンの成果をトラッキングできます。X のモバイル計測パートナーは、広告主が X 上でプロモーションしているアプリから、どのコンバージョンを計測対象とするかを管理できるようにします。広告主がトラッキングしたいイベントの種類と計測ウィンドウを設定すると、モバイル計測パートナーはそれらのイベントをすべて X に送信し、ユーザーのコンバージョンにつながったキャンペーンやクリエイティブを示すアトリビューションデータを即座に受け取ります。

<div id="conversion-events">
  ### コンバージョンイベント
</div>

[コンバージョンイベント](https://developer.x.com/en/docs/x-ads-api/measurement/api-reference/conversion-event) を X に送信してアトリビューションデータを取得するためには、パートナーはモバイルアプリ ID、コンバージョンのタイプとタイムスタンプに加えて、広告 ID / デバイス ID をハッシュ化した値を送信する必要があります。これらのデータを提供することで、X のプラットフォーム上でコンバージョンイベントが記録され、該当する場合にはアトリビューションも返されます。さらに、広告主は購入金額、商品の ID、登録方法といったコンバージョンイベントに関する追加メタデータを任意で提供することもできます。

<div id="post-conversion_event">
  #### POST conversion&#95;event
</div>

モバイル計測のコンバージョンイベントを記録します。レスポンスには、X または X Audience Platform (TAP) のアトリビューションが示されます。

これは [GET conversion&#95;attribution](https://developer.x.com/en/docs/x-ads-api/measurement/api-reference/conversion-attribution#get-conversion-attribution) エンドポイントに関連します。

レスポンスでは、X、TAP、またはいずれのアトリビューションもない、いずれかの結果が返されます。`twitter_attribution` ノードは常に含まれ、X のアトリビューションがない場合は、その値は `null` になります（以下のレスポンス例を参照してください）。TAP のアトリビューションが付与される場合は、`tpn_attribution` ノードが含まれ、適切に値が設定されます。詳細については [TAP overview](https://developer.x.com/en/docs/x-ads-api/measurement/overview/twitter-audience-platform) を参照してください。

各コンバージョンイベントに関連付けるメタデータを設定するために使用できる、いくつかの任意のパラメータがあります。これらのメタデータは、アトリビューションの計算には影響しません。

##### リソースURL

`https://ads-api.x.com/12/conversion_event`

<div id="parameters">
  ##### Parameters
</div>

| Name | Description |
| :--- | :--- |
| app&#95;id  <br />*required* | 対応するアプリストアでの一意の識別子。<br /><br />Type: int, string<br /><br />Example: `333903271`, `com.vine.android` |
| conversion&#95;time  <br />*required* | コンバージョンイベントの時刻（ミリ秒付きの ISO-8601 タイムスタンプ形式）。<br /><br />Type: string<br /><br />Example: `2014-05-22T02:38:28.103Z` |
| conversion&#95;type  <br />*required* | コンバージョンイベントの種類。<br /><br />Type: enum<br /><br />Possible values: `PURCHASE`, `SIGN_UP`, `INSTALL`, `RE_ENGAGE`, `UPDATE`, `TUTORIAL_COMPLETE`, `RESERVATION`, `ADD_TO_CART`, `ADD_TO_WISHLIST`, `LOGIN`, `CHECKOUT_INITIATED`, `SEARCH`, `LEVEL_ACHIEVED`, `ACHIEVEMENT_UNLOCKED`, `CONTENT_VIEW`, `SHARE`, `INVITE`, `ADDED_PAYMENT_INFO`, `SPENT_CREDITS`, `RATED` |
| hashed&#95;device&#95;id  <br />*required* | HMAC&#95;SHA-256 でハッシュ化された IDFA または AdID。<br /><br />Type: string<br /><br />Example: `ABCD1234XYZ` |
| os&#95;type  <br />*required* | アプリの OS の種類。<br /><br />Type: enum<br /><br />Possible values: `IOS`, `ANDROID` |
| click&#95;window  <br />*optional* | このイベントに対するクリック計測期間（日数）。<br /><br />Type: int<br /><br />**Note**: `click_window` は view&#95;through&#95;window 以上である必要があります。<br /><br />Default: `14`  <br />Possible values: `1`, `7`, `14`, `30` |
| device&#95;ip&#95;address  <br />*optional* | コンバージョンイベント発生時のデバイスの IPv4 または IPv6 アドレス。<br /><br />Type: string<br /><br />Example: `192.133.78.1` |
| level  <br />*optional* | このイベントに関連付けられたレベル。<br /><br />Type: int<br /><br />Example: `2` |
| non&#95;twitter&#95;engagement&#95;time  <br />*optional* | コンバージョン前の直近の、Twitter 以外でのエンゲージメントの時刻。<br /><br />Type: string<br /><br />Example: `2014-05-22T02:38:28.103Z` |
| non&#95;twitter&#95;engagement&#95;type  <br />*optional* | コンバージョンイベント前の、Twitter 以外でのエンゲージメントの種類。<br /><br />Type: enum<br /><br />Possible values: `CLICK`, `VIEW` |
| number&#95;items  <br />*optional* | このイベントに関連付けられたアイテム数。<br /><br />Type: int<br /><br />Example: `2` |
| price&#95;currency  <br />*optional* | このイベントに関連付けられた通貨を示す ISO 4217 コード。<br /><br />Type: String<br /><br />Examples: `EUR`,`USD`,`JPY` |
| price&#95;micro  <br />*optional* | このイベントに関連付けられた価格（金額）をマイクロ単位で表します。<br /><br />Type: int<br /><br />Example: `123450000` |
| user&#95;payment&#95;info  <br />*optional* | このイベントに関連するアプリに、ユーザーの支払い情報が保存されているかどうかを示すブール値。<br /><br />Type: bool<br /><br />Possible values: `true` or `false` |
| view&#95;through&#95;window  <br />*optional* | このイベントに対するビュースルー計測期間（日数）。<br /><br />**Note**:<br /><br />`click_window` は view&#95;through&#95;window 以上である必要があります。<br /><br />Type: int<br /><br />Default: `1`  <br />Possible values: `0`, `1`, `7`, `14`, `30` |

<div id="get-accountsaccount_idapp_event_tags">
  #### GET accounts/:account&#95;id/app&#95;event&#95;tags
</div>

現在のアカウントに関連付けられている一部またはすべてのアプリイベントタグの詳細を取得します。

これらは、Mobile App Conversion Tracking において、各コンバージョンタイプごとに設定されるコンバージョンウィンドウを定義します。

<div id="resource-url">
  ##### リソース URL
</div>

`https://ads-api.x.com/12/accounts/:account_id/app_event_tags`

##### パラメーター

| Name | Description |
| :--- | :--- |
| account&#95;id  <br />*required* | 対象アカウントの識別子です。リソースのパス内に含まれ、[GET accounts](/ja/x-ads-api/campaign-management#accounts) を除くすべての Advertiser API リクエストで一般的に必須パラメーターです。指定されたアカウントは、認証済みユーザーに関連付けられている必要があります。<br /><br />Type: string<br /><br />Example: `18ce54d4x5t` |
| app&#95;event&#95;tag&#95;ids  <br />*optional* | 識別子のカンマ区切りリストを指定することで、レスポンスを目的のアプリイベントタグのみに絞り込みます。最大 200 個の ID を指定できます。<br /><br />Type: string<br /><br />Example: `jhp` |
| count  <br />*optional* | 各リクエストごとに取得するレコード数を指定します。<br /><br />Type: int<br /><br />Default: `200`  <br />Min, Max: `1`, `1000` |
| cursor  <br />*optional* | 次のページの結果を取得するためのカーソルを指定します。詳細については [Pagination](/ja/x-ads-api/introduction) を参照してください。<br /><br />Type: string<br /><br />Example: `8x7v00oow` |
| sort&#95;by  <br />*optional* | サポートされている属性で、昇順または降順にソートします。詳細については [Sorting](/ja/x-ads-api/fundamentals/sorting) を参照してください。<br /><br />Type: string<br /><br />Example: `created_at-asc` |
| with&#95;deleted  <br />*optional* | リクエストに削除済みの結果を含めます。<br /><br />Type: boolean<br /><br />Default: `false`  <br />Possible values: `true`, `false` |
| with&#95;total&#95;count  <br />*optional* | レスポンスに `total_count` 属性を含めます。<br /><br />**Note**: このパラメーターと `cursor` は同時に指定できません。<br /><br />**Note**: `total_count` を含むリクエストには、現在 15 分あたり 200 に設定されている、より低いレート制限が適用されます。<br /><br />Type: boolean<br /><br />Default: `false`  <br />Possible values: `true`, `false` |
