Python XDK（X Developer Kit）は、Python を使用して X API v2 と連携するための公式クライアントライブラリです。これにより、開発者は API を手早く使い始め、それを利用したアプリケーションを構築できます。公式の [OpenAPI specification](https://api.x.com/2/openapi.json) に基づいて生成されています。低レベルな HTTP の処理を抽象化しつつ、必要に応じてきめ細かな制御も行えるようにします。

<div id="key-features">
  ## 主な機能
</div>

- 🔐 **OAuth 対応**: Bearer Token（アプリ専用）認証、PKCE を用いた OAuth 2.0（ユーザーコンテキスト）、OAuth 1.0 をフルサポートします。
- 🔄 **ページネーション**: 大量の結果を自動でページングします。XDK が内部でページネーションを処理するため、開発者が `next_token` を使って複数回 API を呼び出す必要はありません。
- 📡 **ストリーミング**: 永続的な HTTP 接続を必要とする filtered stream などのエンドポイント向けに、リアルタイムのデータストリーミングをサポートします。
- 🎯 **包括的なカバレッジ**: 検索、タイムライン、filtered stream などを含む、すべての X API v2 エンドポイントをサポートします。
**対応バージョン**: Python 3.8 以降。CPython および PyPy でテスト済みです。
