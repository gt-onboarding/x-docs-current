Python XDK（X Developer Kit）は、Python を使用して X API v2 と連携するための公式クライアントライブラリです。開発者が API の利用をすばやく開始し、X API v2 を利用したアプリケーションを構築できるようにします。このライブラリは、公式の [OpenAPI 仕様](https://api.x.com/2/openapi.json) に基づいて生成されています。HTTP の低レベルな詳細を抽象化しつつ、必要に応じてきめ細かな制御も提供します。

<div id="key-features">
  ## 主な機能
</div>

- 🔐 **OAuth サポート**: Bearer Token（アプリのみ）認証、PKCE を用いた OAuth 2.0（ユーザーコンテキスト）、OAuth 1.0 を完全にサポートします。
- 🔄 **ページネーション**: 大量の結果を自動的にページングします。XDK が `next_token` を使ったページネーションを処理するため、複数回 API を呼び出す必要はありません。
- 📡 **ストリーミング**: filtered stream のような、永続的な HTTP 接続を必要とするエンドポイント向けにリアルタイムのデータストリーミングをサポートします。
- 🎯 **包括的なカバレッジ**: search、timelines、filtered-stream などを含む、すべての X API v2 エンドポイントをサポートします。
**バージョン互換性**: Python 3.8+。CPython および PyPy でテスト済み。
