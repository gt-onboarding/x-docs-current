XDK Python SDK は GitHub リポジトリから直接取得でき、`pip` を使ってインストールできます。

<div id="prerequisites">
  ## 前提条件
</div>

- Python 3.8 以降。
- 仮想環境用の `pip` および `venv`（推奨）。

<div id="quick-install">
  ## クイックインストール
</div>

GitHub のサブディレクトリから XDK をインストールします。

```bash
pip install xdk
```

これにより `main` ブランチから最新の生成済みバージョンが取得されます。


<div id="development-install">
  ## 開発インストール
</div>

開発やコントリビュートを行う場合:

1. リポジトリをクローンします:
   ```bash
   git clone https://github.com/xdevplatform/xdk.git
   cd xdk/python
   ```
2. SDK を編集可能モードでインストールします:
   ```bash
   pip install -e .
   ```
   これにより、SDK とその実行時依存関係がインストールされます。
3. (任意) テストや lint 用の開発用依存関係をインストールします:
   ```bash
   pip install -e .[dev]
   ```

<div id="verification">
  ## 検証
</div>

インストールをテストする:

```python
import xdk
print(xdk.__version__)  # XDKのバージョンを表示します
```

**注意:** XDK は OpenAPI 仕様から生成されているため、変更がないか必ず [X API の変更履歴](https://docs.x.com/changelog) およびリポジトリ内の XDK のリリースノートを確認してください。
Python XDK を使用した詳細なコード例については、[コードサンプルの GitHub リポジトリ](https://github.com/xdevplatform/samples/tree/main/python) を参照してください。
