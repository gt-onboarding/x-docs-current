XDK Python SDK は GitHub のリポジトリから直接入手でき、`pip` を使ってインストールできます。

<div id="prerequisites">
  ## 前提条件
</div>

- Python 3.8 以降
- 仮想環境用の `pip` と `venv`（推奨）

<div id="quick-install">
  ## クイックインストール
</div>

GitHub リポジトリ内のサブディレクトリから XDK をインストールします。

```bash
pip install xdk
```

`main` ブランチから最新の生成済みバージョンを取得します。


<div id="development-install">
  ## 開発用インストール
</div>

開発やコントリビューションを行う場合:

1. リポジトリをクローンします:
   ```bash
   git clone https://github.com/xdevplatform/xdk.git
   cd xdk/python
   ```
2. パッケージを編集可能モードでインストールします:
   ```bash
   pip install -e .
   ```
   これにより、SDK 本体とその実行時の依存関係がインストールされます。
3. （任意）テスト／lint 用の開発依存関係をインストールします:
   ```bash
   pip install -e .[dev]
   ```

<div id="verification">
  ## 検証
</div>

インストールを確認します:

```python
import xdk
print(xdk.__version__)  # XDKのバージョンを表示します
```

**注意:** XDK は OpenAPI 仕様から自動生成されているため、[X API 変更履歴](https://docs.x.com/changelog) とリポジトリ内の XDK リリースノートを常に確認し、変更がないかチェックしてください。
Python XDK を使用した詳細なコード例については、[コードサンプルの GitHub リポジトリ](https://github.com/xdevplatform/samples/tree/main/python) を参照してください。
