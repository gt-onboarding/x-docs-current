このクイックスタートでは、Bearer Token 認証を使用して XDK でポストをすばやく検索する方法を説明します。

<div id="step-1-install-the-sdk">
  ## ステップ 1：SDK をインストールする
</div>

```bash
pip install xdk
```

<div id="step-2-get-your-bearer-token">
  ## ステップ 2：Bearer Token を取得する
</div>

1. [X Developer Portal](https://developer.x.com/en/portal/dashboard) にログインします。
2. アプリを新規作成するか、既存のアプリを選択します。
3. 「キーとトークン」セクションで、Bearer Token（アプリのみの認証）を生成します。

<div id="step-3-write-and-run-your-first-script">
  ## ステップ3：最初のスクリプトを書いて実行する
</div>

`quickstart.py` というファイルを作成します。

```python
# Clientをインポート
from xdk import Client
# 実際のBearer Tokenに置き換えてください
client = Client(bearer_token="YOUR_BEARER_TOKEN_HERE")
# "api"に言及している最近のポストを取得
# search_recentはIteratorを返すので、それを反復処理します
for page in client.posts.search_recent(query="api", max_results=10):
    if page.data and len(page.data) > 0:
        # 最初のポストにアクセス - Pydanticモデルは属性とdict両方のアクセスをサポート
        first_post = page.data[0]
        post_text = first_post.text if hasattr(first_post, 'text') else first_post.get('text', '')
        print(f"Latest Post: {post_text}")
        break
    else:
        print("No Posts found.")
        break
```

実行してみましょう:

```bash
python quickstart.py
```

**想定される出力**:

```
Latest Post: XDK Python SDKのエキサイティングなアップデート！
```

**トラブルシューティング**: 401 エラーが発生した場合は、Bearer Token が正しいか再確認してください。レート制限 (429) の場合は、しばらく待ってから再試行してください。
