X API SDK 向けのカーソルベースのページネーションユーティリティです。

このモジュールは、すべての API クライアントでページネーションを簡潔に扱うための Cursor クラスを提供します。Cursor を使用すると、適切な型安全性を維持したまま、.pages() メソッドと .items() メソッドの両方を使ってページネーションされた結果を容易に反復処理できます。

<div id="xdkpaginatorcursor">
  ### `xdk.paginator.cursor`
</div>

適切な型推論とバリデーションを備えたカーソルを作成します。
このファクトリ関数は型推論を行うことでレスポンス型に対する
適切な型ヒントが得られるようにし、メソッドが
静的解析時と実行時の両方でページネーションをサポートしているかどうかを検証します。

#### パラメータ

<ParamField path="path.method" type="PaginatableMethod">
ラップ対象の API メソッド（ページネーションをサポートしている必要があります）
</ParamField>
