X API SDK 向けのカーソルベースのページネーションユーティリティ。

このモジュールは、すべての API クライアントで便利にページネーションを扱うための Cursor クラスを提供します。Cursor を使うことで、.pages() と .items() の両メソッドを用いて、型安全性を確保しながらページネーションされた結果を簡単に反復処理できます。

<div id="xdkpaginatorcursor">
  ### `xdk.paginator.cursor`
</div>

適切な型推論とバリデーションを備えたカーソルを作成します。
このファクトリ関数は型推論を支援し、レスポンス型に対して適切な
型ヒントが得られるようにするとともに、指定したメソッドが
静的解析時と実行時の両方でページネーションをサポートしているか検証します。

#### パラメータ

<ParamField path="path.method" type="PaginatableMethod">
ラップ対象の API メソッド（ページネーションに対応している必要があります）
</ParamField>
