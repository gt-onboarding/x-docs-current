<div id="get_counts_all">
  ### `get_counts_all`
</div>

すべてのポストの件数を取得する
フルアーカイブから検索クエリにマッチするポストの件数を取得します。

<div id="parameters">
  #### Parameters
</div>

<ParamField path="path.query" type="str">
ポストにマッチさせるためのクエリ / ルール / フィルターを1つ指定します。最大クエリ長を確認するには [https://t.co/rulelength](https://t.co/rulelength) を参照してください。
</ParamField>

<ParamField path="path.start_time" type="str or None" default="None">
YYYY-MM-DDTHH:mm:ssZ。ポストが提供される、最も古い UTC タイムスタンプ（直近7日間の中で）を指定します。タイムスタンプは秒精度で、かつ両端を含みます（例: 12:00:01 はその分の最初の1秒を含みます）。
</ParamField>

<ParamField path="path.end_time" type="str or None" default="None">
YYYY-MM-DDTHH:mm:ssZ。ポストが提供される、最も新しい（最新の）UTC タイムスタンプを指定します。タイムスタンプは秒精度で、かつ片側排他的です（例: 12:00:01 はその分の最初の1秒を含みません）。
</ParamField>

<ParamField path="path.since_id" type="Any or None" default="None">
指定した ID より大きい（つまり、より新しい）ポスト ID を持つ結果を返します。
</ParamField>

<ParamField path="path.until_id" type="Any or None" default="None">
指定した ID より小さい（つまり、より古い）ポスト ID を持つ結果を返します。
</ParamField>

<ParamField path="path.next_token" type="Any or None" default="None">
このパラメータは結果の次の「ページ」を取得するために使用されます。パラメータに使用する値は API から返されたレスポンスから直接取得され、変更してはいけません。
</ParamField>

<ParamField path="path.pagination_token" type="Any or None" default="None">
このパラメータは結果の次の「ページ」を取得するために使用されます。パラメータに使用する値は API から返されたレスポンスから直接取得され、変更してはいけません。
</ParamField>

<ParamField path="path.granularity" type="str or None" default="None">
検索件数結果の集計粒度を指定します。
</ParamField>

<ParamField path="path.search_count_fields" type="List or None" default="None">
表示する SearchCount フィールドをカンマ区切りのリストで指定します。
</ParamField>
