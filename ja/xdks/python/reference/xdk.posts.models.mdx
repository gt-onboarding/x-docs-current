このモジュールは、X API のポストエンドポイント向けのリクエストおよびレスポンスのデータ構造を表現する Pydantic モデルを提供します。すべてのモデルは OpenAPI 仕様から生成されており、型安全性とバリデーションを提供します。

<div id="class-xdkpostsmodelscreaterequest">
  ### class xdk.posts.models.CreateRequest
</div>

作成用のリクエストモデルです。

<ResponseField name="model_config" type="ConfigDict">
Default: `{'populate_by_name': True, 'validate_by_alias': True, 'validate_by_name': True}`

モデルの設定であり、[ConfigDict][pydantic.config.ConfigDict] に準拠した辞書型である必要があります。
</ResponseField>
