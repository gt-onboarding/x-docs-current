このモジュールは、X API の activity エンドポイントを操作するためのクライアントを提供します。
リアルタイムのストリーミング処理では、到着したデータを逐次生成するジェネレーターを返します。
ストリーミング接続は、自動的に指数バックオフによる再試行ロジックで管理され、堅牢に動作するようになっています。

<div id="stream">
  ### `stream`
</div>

アクティビティストリーム（ストリーミング）
X アクティビティのストリーム
これは、利用可能になり次第リアルタイムでデータを返すストリーミングエンドポイントです。
返される各アイテムは、ストリーム内の 1 つのデータポイントを表します。
接続は、指数バックオフ付きの再試行ロジックによって自動的に管理されます。
ストリームが切断された場合でも、SDK はクライアント側の操作なしに自動的に再接続します。
