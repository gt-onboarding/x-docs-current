<div id="get_analytics">
  ### `get_analytics`
</div>

メディアのアナリティクスを取得
メディアに関するアナリティクスデータを取得します。

<div id="parameters">
  #### パラメーター
</div>

<ParamField path="path.media_keys" type="List">
カンマ区切りの Media Key のリストです。1 回のリクエストで最大 100 個まで指定できます。
</ParamField>

<ParamField path="path.end_time" type="str">
YYYY-MM-DDTHH:mm:ssZ。時間範囲の終了を表す UTC タイムスタンプです。
</ParamField>

<ParamField path="path.start_time" type="str">
YYYY-MM-DDTHH:mm:ssZ。時間範囲の開始を表す UTC タイムスタンプです。
</ParamField>

<ParamField path="path.granularity" type="str">
Search Counts の結果に対する集計粒度です。
</ParamField>

<ParamField path="path.media_analytics_fields" type="List or None" default="None">
表示する MediaAnalytics のフィールドのカンマ区切りリストです。
</ParamField>

<div id="returns">
  #### 戻り値
</div>

`GetAnalyticsResponse` - レスポンスデータ
