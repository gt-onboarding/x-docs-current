<div id="get_analytics">
  ### `get_analytics`
</div>

メディアのアナリティクスを取得
メディアのアナリティクスデータを取得します。

<div id="parameters">
  #### パラメーター
</div>

<ParamField path="path.media_keys" type="List">
Media Key をカンマ区切りで指定するリスト。1 回のリクエストで最大 100 個まで指定できます。
</ParamField>

<ParamField path="path.end_time" type="str">
YYYY-MM-DDTHH:mm:ssZ。時間範囲の終了を表す UTC タイムスタンプ。
</ParamField>

<ParamField path="path.start_time" type="str">
YYYY-MM-DDTHH:mm:ssZ。時間範囲の開始を表す UTC タイムスタンプ。
</ParamField>

<ParamField path="path.granularity" type="str">
検索件数結果の集計粒度。
</ParamField>

<ParamField path="path.media_analytics_fields" type="List or None" default="None">
表示する MediaAnalytics フィールドをカンマ区切りで指定するリスト。
</ParamField>
