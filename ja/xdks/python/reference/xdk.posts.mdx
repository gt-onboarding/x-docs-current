<div id="get_counts_all">
  ### `get_counts_all`
</div>

すべてのポストの件数を取得します
フルアーカイブから検索クエリに一致するポストの件数を取得します。

#### パラメーター

<ParamField path="path.query" type="str">
  ポストにマッチさせるためのクエリ／ルール／フィルターを 1 つ指定します。最大クエリ長を確認するには [https://t.co/rulelength](https://t.co/rulelength) を参照してください。
</ParamField>

<ParamField path="path.start_time" type="str or None" default="None">
  YYYY-MM-DDTHH:mm:ssZ。ポストが提供される期間における、直近 7 日間以内で最も古い UTC タイムスタンプです。タイムスタンプは秒単位の粒度で、かつ境界値を含みます（例: 12:00:01 はその分の最初の 1 秒を含みます）。
</ParamField>

<ParamField path="path.end_time" type="str or None" default="None">
  YYYY-MM-DDTHH:mm:ssZ。ポストが提供される期間における、最も新しい UTC タイムスタンプです。タイムスタンプは秒単位の粒度で、かつ境界値を含みません（例: 12:00:01 はその分の最初の 1 秒を含みません）。
</ParamField>

<ParamField path="path.since_id" type="Any or None" default="None">
  指定した ID より大きい（すなわち、より新しい）ポスト ID の結果を返します。
</ParamField>

<ParamField path="path.until_id" type="Any or None" default="None">
  指定した ID より小さい（すなわち、より古い）ポスト ID の結果を返します。
</ParamField>

<ParamField path="path.next_token" type="Any or None" default="None">
  このパラメーターは、結果の次の「ページ」を取得するために使用されます。パラメーターに指定する値は API が返したレスポンスから直接取得されるものであり、変更してはなりません。
</ParamField>

<ParamField path="path.pagination_token" type="Any or None" default="None">
  このパラメーターは、結果の次の「ページ」を取得するために使用されます。パラメーターに指定する値は API が返したレスポンスから直接取得されるものであり、変更してはなりません。
</ParamField>

<ParamField path="path.granularity" type="str or None" default="None">
  検索件数の結果の粒度を指定します。
</ParamField>

<ParamField path="path.search_count_fields" type="List or None" default="None">
  表示する SearchCount フィールドをカンマ区切りのリストで指定します。
</ParamField>

<div id="parameters">

<ParamField path="path.query" type="str">
  ポストをマッチさせるためのクエリ / ルール / フィルターを 1 つ指定します。最大クエリ長を確認するには [https://t.co/rulelength](https://t.co/rulelength) を参照してください。

  YYYY-MM-DDTHH:mm:ssZ。ポストが返される最も新しい（最新の）UTC タイムスタンプを指定します。タイムスタンプは秒単位の精度で、かつ排他的です（例: 12:00:01 はその分の最初の 1 秒を含みません）。
</ParamField>

<ParamField path="path.since_id" type="Any or None" default="None">
  指定した ID より大きい（つまり、より新しい）ポスト ID を持つ結果を返します。
</ParamField>

</ParamField>

<ParamField path="path.pagination_token" type="Any or None" default="None">
