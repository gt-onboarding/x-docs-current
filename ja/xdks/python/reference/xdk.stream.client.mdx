このモジュールは、X API のストリームエンドポイントを操作するためのクライアントを提供します。
リアルタイムのストリーミング処理は、到着したデータを逐次生成するジェネレーターを返します。
ストリーミング接続は自動的に管理され、堅牢な処理のために指数バックオフを用いたリトライロジックが適用されます。

<div id="labels_compliance">

ポストに適用されたすべてのラベリングイベントをストリーミングします。
