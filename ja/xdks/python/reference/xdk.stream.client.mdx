このモジュールは、X API のストリームエンドポイントとやり取りするためのクライアントを提供します。
リアルタイムのストリーミング処理は、データの到着に応じて順次データを生成するジェネレーターを返します。
ストリーミング接続は、自動的に指数バックオフのリトライロジックで管理され、堅牢に動作します。

<div id="labels_compliance">
  ### `labels_compliance`
</div>

ポストラベルのストリーミング（Streaming）
ポストに適用されたすべてのラベル付けイベントをストリーミングします。
このエンドポイントはストリーミングエンドポイントであり、利用可能になり次第リアルタイムでデータを配信します。
ストリームから順次出力される各アイテムは、単一のデータポイントを表します。
接続は、指数バックオフを用いた再試行ロジックによって自動的に管理されます。
ストリームが切断された場合でも、SDK はクライアント側の操作なしに自動的に再接続します。

  #### パラメーター
</div>

YYYY-MM-DDTHH:mm:ssZ。ポストのラベルが提供される最も早い UTC タイムスタンプ。
</ParamField>

<ParamField path="path.end_time" type="str or None" default="None">
YYYY-MM-DDTHH:mm:ssZ。ポストのラベルが提供される最も遅い UTC タイムスタンプ。
</ParamField>

<ParamField path="path.stream_config" type="StreamConfig">
リトライ動作、タイムアウト、コールバックをカスタマイズするためのオプションの StreamConfig。
</ParamField>
