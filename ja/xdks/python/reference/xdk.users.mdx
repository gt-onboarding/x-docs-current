<div id="block_dms">
  ### `block_dms`
</div>

DM をブロックする
認証済みユーザーに対して、特定のユーザーの id を指定して、そのユーザーとの間で送受信されるダイレクトメッセージをブロックします。

#### パラメーター

<ParamField path="path.id" type="Any">
  認証済みユーザーが、そのユーザーからのDMをブロックするよう要求している対象ユーザーのID。
</ParamField>
