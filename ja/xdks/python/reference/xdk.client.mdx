このモジュールは、X API とやり取りするための主要な Client クラスを提供します。
すべてのサブクライアントを統合し、認証、セッション管理、
および OAuth2 PKCE フローを処理します。このクライアントの機能はすべて OpenAPI 仕様から生成されています。

<div id="parameters">
  #### Parameters
</div>

<ParamField path="path.base_url" type="str" default="'https://api.x.com'">
X API のベースURL（デフォルトは [https://api.x.com](https://api.x.com)）。
</ParamField>

<ParamField path="path.bearer_token" type="str or None" default="None">
X API 用の Bearer Token（アプリケーション専用認証）。
</ParamField>

<ParamField path="path.access_token" type="str or None" default="None">
ユーザーコンテキスト用の OAuth2 アクセストークン（Bearer Token として直接使用可能）。
</ParamField>

<ParamField path="path.client_id" type="str or None" default="None">
X API 用のクライアントID（OAuth2 PKCE フローで必須）。
</ParamField>

<ParamField path="path.client_secret" type="str or None" default="None">
X API 用のクライアントシークレット。
</ParamField>

<ParamField path="path.redirect_uri" type="str or None" default="None">
OAuth2 認可用のリダイレクトURI。
</ParamField>

<ParamField path="path.token" type="Dict[str, Any] or None" default="None">
既存の OAuth2 トークンディクショナリ（利用可能な場合）。指定された場合は、access_token が取り出されます。
</ParamField>

<ParamField path="path.scope" type="str or List[str] or None" default="None">
OAuth2 認可スコープ用の、スペース区切りの文字列または文字列のリスト。
</ParamField>

<ParamField path="path.authorization_base_url" type="str" default="'https://x.com/i'">
OAuth2 認可用のベースURL（デフォルトは [https://x.com/i](https://x.com/i)）。
</ParamField>

<ParamField path="path.auth" type="OAuth1">
OAuth 1.0a 認証用の OAuth1 インスタンス。
</ParamField>

<div id="exchange_code">
  ### `exchange_code`
</div>

認可コードをトークンに交換します（TypeScript API と同じです）。

#### パラメーター

<ParamField path="path.code" type="Any">
コールバックで受け取る認可コード。
</ParamField>

<ParamField path="path.code_verifier" type="Any" default="None">
省略可能なコードベリファイア（指定しない場合は保存済みのベリファイアを使用します）。
</ParamField>

<div id="fetch_token">
  ### `fetch_token`
</div>

認可レスポンス URL からトークンを取得します（レガシー方式）。

#### パラメータ

<ParamField path="path.authorization_response" type="Any">
認可後に受信したコールバック URL の完全な文字列。
</ParamField>

<div id="get_authorization_url">
  ### `get_authorization_url`
</div>

OAuth2 PKCE フロー用の認可 URL を取得します。

#### パラメーター

<ParamField path="path.state" type="Any" default="None">
セキュリティ向上のために使用する、省略可能な state パラメーター。
</ParamField>

<div id="is_token_expired">
  ### `is_token_expired`
</div>

OAuth2 トークンが有効期限切れかどうかを確認します。

<div id="refresh_token">
  ### `refresh_token`
</div>

OAuth2 トークンを更新します。
