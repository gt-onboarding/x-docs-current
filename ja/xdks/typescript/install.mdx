プロジェクトで X API 向け TypeScript SDK の利用を開始しましょう。

<div id="install">
  ## インストール
</div>

<CodeGroup>
  ```bash npm theme={null}
  npm install @xdevplatform/xdk
  ```

  ```bash yarn theme={null}
  yarn add @xdevplatform/xdk
  ```

  ```bash pnpm theme={null}
  pnpm add @xdevplatform/xdk
  ```
</CodeGroup>

<div id="typescript-support">
  ## TypeScript サポート
</div>

この SDK は TypeScript で実装されており、完全な型定義を備えています。追加の型パッケージは不要です。

<div id="requirements">
  ## 要件
</div>

* Node.js 16以上
* TypeScript 4.5以上（TypeScript を使用する場合）
