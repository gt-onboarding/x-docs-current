プロジェクトで X API 用 TypeScript SDK の利用を開始しましょう。

<div id="install">
  ## インストール
</div>

<CodeGroup>
  ```bash npm theme={null}
  npm install @xdevplatform/xdk
  ```

  ```bash yarn theme={null}
  yarn add @xdevplatform/xdk
  ```

  ```bash pnpm theme={null}
  pnpm add @xdevplatform/xdk
  ```
</CodeGroup>

<div id="typescript-support">
  ## TypeScript サポート
</div>

この SDK は TypeScript で実装されており、完全な型定義を備えています。追加の型パッケージをインストールする必要はありません。

<div id="requirements">
  ## 前提条件
</div>

* Node.js 16 以上
* TypeScript 4.5 以上（TypeScript を使用する場合）
