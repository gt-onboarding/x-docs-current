I. **定義**

本契約では、以下の定義が適用されます。

1. **「Credits」**とは、Paid Services の利用を可能にするためにあなたが購入する仮想単位をいい、各 API コールまたはアクションごとに、料金表に基づき特定数のクレジットが差し引かれます。クレジットは、法律で義務付けられる場合を除き払い戻し不可であり、譲渡不可であり、別途定めがない限り有効期限はありません。

2\. **「Developer Site」**とは、[**https://developer.x.com**](https://developer.x.com/) に所在する X の開発者サイトをいいます。

3\. **「Direct Message」**とは、X のダイレクトメッセージ機能を使用して、X アプリケーション上で 1 人のユーザーから 1 人以上の特定の他のユーザーに送信される非公開メッセージをいいます。

4\. **「Intellectual Property Rights」**とは、すべての著作権、著作者人格権、特許権、商標権、その他あらゆる知的財産権またはこれに類する権利（登録の有無を問わず）を世界中で意味します。

5\. **「Licensed Material」**とは、単体または集合としての X API および X Content をいいます。

6\. **「Paid Service(s)」**とは、X から購入したクレジットが、料金ページに詳述されるコールおよび機能の利用状況に基づきあなたのアカウントから差し引かれることと引き換えに、あなたがアクセスする Licensed Material の機能または機能群をいいます。本契約に別段の定めがない限り、「Licensed Material」へのすべての言及は Paid Service(s) を含むものとします。

7\. **「Payment Portal」**とは、Licensed Material の利用へのオプトイン、Paid Services のためのクレジット購入の管理、Paid Services の支払方法の管理、「Auto-Recharge」の有効化（第 VII.F 条参照）、およびアカウント情報の更新を行うことができるオンラインポータルをいいます。 

8\. **「Services」**とは、X Content を表示するか、その他 Licensed Material を使用するあなたのサービス、ウェブサイト、アプリケーション、その他の提供物（調査を含みます）をいいます。

9\. **「Post」**とは、X アプリケーションを介して配信される、短文テキストおよびマルチメディアを基盤とするメッセージをいいます。

10\. **「Pricing Schedule」**とは、[https://console.x.com/pricing](https://console.x.com/pricing) で入手可能な、API コールまたはアクションごとの最新の料金表をいい、X は通知を行った上でこれを更新する場合があります。

11\. **“X”** とは、（a）あなたの主たる事業所が欧州連合、EFTA 加盟国、および英国の外にある場合には X Corp.（865 FM 1209, Building 2, Bastrop, TX 78602, USA）を、または（b）あなたの主たる事業所が欧州連合、EFTA 加盟国、または英国にある場合には X Internet Unlimited Company（One Cumberland Place, Fenian Street, Dublin 2, D02 AX07, Ireland）を意味します。

12\. **“X API”** とは、X の Application Programming Interface（それぞれ「API」）および Software Development Kit（それぞれ「SDK」）、ならびに Developer Site を通じて X によって提供される関連するツール、ドキュメント、データ、テクノロジー、コード、その他の資料を意味します。

13\. **“X Applications”** とは、一般に「X」と呼ばれる X のリアルタイム情報サービスを意味し、X がそのウェブサイトおよび X が所有、運営、もしくは管理するモバイルアプリケーション、ソーシャルプラグイン、アプリケーションプログラミングインターフェースにおいて現在提供しているサービスを含みます。

14\. **“X Content”** とは、ポスト、各ポストに対して生成される一意の識別番号、X エンドユーザーのプロフィール情報、および X API を通じて、または X によって認可されたその他の手段によりあなたに提供されるその他のデータおよび情報、ならびにそれらの複製および派生物を意味します。

15\. **“X Marks”** とは、本契約に従った使用のためにのみ、Developer Site を通じたものを含め、X があなたに提供する X の名称、商標、およびロゴを意味します。

16\. **“Users”** とは、あなたの Services の訪問者、ユーザー、または顧客を意味します。
