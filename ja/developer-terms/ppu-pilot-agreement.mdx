I. **定義**

本契約において、以下の定義が適用されます。

1. **「クレジット」**とは、あなたが有料サービスの利用を可能にするために購入する仮想単位を意味し、各 API コールまたはアクションごとに、料金表に基づいて特定の数のクレジットが差し引かれます。クレジットは、法律で要求される場合を除き払い戻し不可であり、譲渡不可であり、別途定めのない限り有効期限はありません。

2\. **「Developer Site」**とは、[**https://developer.x.com**](https://developer.x.com/) に所在する X の開発者向けサイトを意味します。

3\. **「ダイレクトメッセージ」**とは、X のダイレクトメッセージ機能を使用して、X アプリケーション上であるユーザーから 1 人以上の特定の他のユーザーに送信される非公開メッセージを意味します。

4\. **「知的財産権」**とは、世界中のすべての著作権、著作者人格権、特許権、商標権、およびその他の知的財産権またはこれに類似する権利（登録済みか未登録かを問いません）を意味します。

5\. **「ライセンス対象資料」**とは、個別または集合的に、X API および X コンテンツを意味します。

6\. **「有料サービス」**とは、あなたが X から購入したクレジットを差し引くことと引き換えにアクセスするライセンス対象資料の機能を意味し、そのクレジットは、当社の料金ページに詳述されているとおり、あなたによるコールおよび機能の利用状況に基づきあなたのアカウントから差し引かれます。本契約に別段の定めがない限り、「ライセンス対象資料」へのすべての言及には有料サービスが含まれるものとします。

7\. **「Payment Portal」**とは、あなたに提供されるオンラインポータルであって、ライセンス対象資料の利用へのオプトイン、有料サービス向けのクレジット購入の管理、有料サービス向け支払方法の管理、「自動リチャージ」（第 VII.F 条参照）の有効化、およびアカウント情報の更新を行うことができるものを意味します。 

8\. **「サービス」**とは、X コンテンツを表示するか、またはその他の方法でライセンス対象資料を使用する、あなたのサービス、ウェブサイト、アプリケーションその他の提供物（調査を含みます）を意味します。

9\. **「Post」**とは、X アプリケーションを介して配信される短文テキストおよびマルチメディアベースのメッセージを意味します。

10\. **「料金表」**とは、[https://console.x.com/pricing](https://console.x.com/pricing) で入手可能な、API コールまたはアクションごとの最新の価格一覧を意味し、X は通知のうえこれを更新することがあります。

11\. **「X」**とは、(a) あなたの主たる事業所が欧州連合、EFTA 加盟国、および英国の外にある場合には X Corp.（865 FM 1209, Building 2, Bastrop, TX 78602, USA）を意味し、(b) あなたの主たる事業所が欧州連合、EFTA 加盟国、または英国にある場合には X Internet Unlimited Company（One Cumberland Place, Fenian Street, Dublin 2, D02 AX07, Ireland）を意味します。

12\. **「X API」**とは、X Application Programming Interfaces（それぞれ「API」）、Software Development Kits（それぞれ「SDK」）、および X が Developer Site を通じて提供する関連ツール、ドキュメント、データ、技術、コードその他の資料を意味します。

13\. **「X Applications」**とは、一般に「X」と呼ばれる X のリアルタイム情報サービスを意味し、これには、現在 X がそのウェブサイトおよび X が所有、運営、または管理するモバイルアプリケーション、ソーシャルプラグイン、およびアプリケーションプログラミングインターフェースにおいて提供しているサービスが含まれます。

14\. **「X Content」**とは、ポスト、各ポストに対して生成される一意の識別番号、X エンドユーザープロフィール情報、および X API を通じて、または X により承認されたその他の手段によりあなたに提供されるその他のデータおよび情報、ならびにそれらの複製物および派生物を意味します。

15\. **「X Marks」**とは、本契約に従った利用のために、Developer Site 経由を含め、X があなたに利用可能にする X の名称、商標、およびロゴを意味します。

16\. **「Users」**とは、あなたのサービスの訪問者、ユーザー、または顧客を意味します。
