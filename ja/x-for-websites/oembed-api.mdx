oEmbed API を使用すると、[ポスト](https://developer.x.com/en/docs/twitter-for-websites/embedded-tweets/overview)や[タイムライン](https://developer.x.com/en/docs/twitter-for-websites/timelines/overview)などの埋め込みコンテンツをプログラムから取得できます。 

oEmbed API のレスポンスは HTML スニペットを返し、[ページに X のウィジェット JavaScript が読み込まれている場合](https://developer.x.com/web/javascript/loading) に自動的に認識されます。

この API は大量の処理を行う用途での利用を推奨しており、コンテンツを埋め込む場合は、高機能な [publish.x.com](https://publish.x.com/#) ツールの使用をお勧めします。

<Tabs>
  <Tab title="埋め込みタイムライン">
    返される HTML スニペットは、[埋め込みタイムライン](https://developer.x.com/en/docs/twitter-for-websites/timelines/overview) として自動的に認識されます（[ページに X のウィジェット JavaScript が読み込まれている場合](https://developer.x.com/web/javascript/loading)）。

    oEmbed エンドポイントでは、HTML マークアップ内の対応するプロパティを設定することで、埋め込みタイムラインの最終的な見た目をカスタマイズできます。これらのプロパティは、既定で HTML レスポンスにバンドルされている X の JavaScript によって解釈されます。返されるマークアップの形式は、X が新機能を追加したりタイムラインの表現を調整したりするのに伴い、時間とともに変更される可能性があります。

    タイムライン URL で指定された X タイムラインを、[oEmbed](https://oembed.com/) 互換の JSON 形式で取得します。ユーザータイムラインおよびリストタイムラインがサポートされています。タイムラインのマークアップは、`cache_age` プロパティで指定される推奨キャッシュ存続期間の範囲内で、あなたのサーバー上にキャッシュされることを想定しています。

    ## リソース URL

    *[https://publish.x.com/oembed](https://publish.x.com/oembed)*

    ## リクエスト例

    ```bash
    curl --request GET --url 'https://publish.x.com/oembed?url=https%3A%2F%2Ftwitter.com%2FInterior%2Fstatus%2F507185938620219395'
    twurl -H publish.x.com "/oembed?url=https://x.com/Interior/status/463440424141459456"
    ```

  <Tab title="ポストの埋め込み">
    返されたHTMLスニペットは、[ページにXのウィジェットJavaScriptが含まれている](https://developer.x.com/web/javascript/loading)場合、[埋め込みポスト](https://developer.x.com/web/embedded-tweets)として自動的に認識されます。

    oEmbedエンドポイントでは、HTMLマークアップ内の対応するプロパティを設定することで、埋め込みポストの最終的な表示をカスタマイズできます。これらのプロパティは、デフォルトでHTMLレスポンスにバンドルされているXのJavaScriptによって解釈されます。返されるマークアップの形式は、Xが新機能を追加したり、ポストの表現方法を調整したりする際に、時間の経過とともに変更される場合があります。

    ポストのフォールバックマークアップは、`cache_age` プロパティで指定された推奨キャッシュ有効期間まで、お客様のサーバーにキャッシュすることを想定しています。

    | 名前                                                                                                                               | デフォルト   | 説明                                                                                                                                                                                                                                                                                                                                                   |
    | :------------------------------------------------------------------------------------------------------------------------------- | :------ | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
    | `url`（必須）  <br />String                                                                                                          |         | 埋め込むポストのURL                                                                                                                                                                                                                                                                                                                                          |
    | `maxwidth`  <br />Int `[220..550]`                                                                                               | `325`   | レンダリングされるポストの最大幅（ピクセル単位の整数値）です。許可された範囲を下回る、または上回る値を指定した場合、それぞれサポートされる最小幅または最大幅として扱われ、その値が返されます。調整後の幅の値は、返される `width` プロパティに反映されます。なお、X では oEmbed の `maxheight` パラメータはサポートされていません。ポストは本質的にテキストであり、高さは予測不能で、画像や動画のようにスケーリングすることはできません。このため、oEmbed レスポンスでは `height` の値は提供されません。ポストの高さを一定に保つ必要がある実装では、下記の `hide_thread` および `hide_media` パラメータを参照してください。 |
    | `hide_media`  <br />ブール値、文字列または Int                                                                                              | `false` | `true`、`"t"`、または `1` に設定されている場合、ポスト内のリンクは写真、動画、またはリンクのプレビューに展開されません。                                                                                                                                                                                                                                                                                 |
    | `hide_thread`  <br />Boolean、String または Int                                                                                      | `false` | `true`、`"t"`、`1` のいずれかに設定すると、リクエストされたポストが別のポストへの返信である場合、その会話スレッド内の直前のポストは折りたたみ表示されません。                                                                                                                                                                                                                                                               |
    | `omit_script`  <br />Boolean、String、または Int                                                                                      | `false` | `true`、`"t"`、または `1` に設定すると、`widgets.js` の読み込みを行う `<script>` は返されなくなります。すべての X ウィジェット（[埋め込みポスト](https://developer.x.com/web/embedded-tweets) を含む）で使用するために、Web ページ側で `widgets.js` への参照を含めておく必要があります。                                                                                                                                                  |
    | `align`  <br />Enum `{left,right,center,none}`                                                                                   | `none`  | ページ内で親要素に対して埋め込みポストを左寄せ、右寄せ、中央寄せのいずれに配置するかを指定します。                                                                                                                                                                                                                                                                                                    |
    | `lang`  <br />Enum（[言語](https://developer.x.com/en/docs/twitter-for-websites/twitter-for-websites-supported-languages/overview)） | `en`    | リクエストは、指定した[埋め込みポストでサポートされている X の言語](https://developer.x.com/web/overview/languages)で HTML とレンダリング済みのポストを返します。                                                                                                                                                                                                                                       |
    | `theme`  <br />列挙型 `{light, dark}`                                                                                               | `light` | `dark` に設定すると、ポストは暗い背景に明るいテキストで表示されます。                                                                                                                                                                                                                                                                                                               |
    | `dnt`  <br />Boolean                                                                                                             | `false` | `true` に設定すると、サイト上のポストおよびその埋め込みページは、[パーソナライズされたおすすめ](https://support.x.com/articles/20169421)や[パーソナライズされた広告](https://support.x.com/articles/20170405)などの目的には使用されなくなります。                                                                                                                                                                              |
