oEmbed API を使用すると、[ポスト](https://developer.x.com/en/docs/twitter-for-websites/embedded-tweets/overview) や [タイムライン](https://developer.x.com/en/docs/twitter-for-websites/timelines/overview) などの埋め込みコンテンツをプログラムで取得できます。 

oEmbed API のレスポンスでは、[ページに X の widget JavaScript が読み込まれている場合](https://developer.x.com/web/javascript/loading) に自動的に認識される HTML スニペットが返されます。

なお、この API は一括処理での利用を推奨しており、コンテンツの埋め込みには高機能なツールである [publish.x.com](https://publish.x.com/#) のご利用をおすすめします。

<Tabs>
  <Tab title="埋め込みタイムライン">
    返される HTML スニペットは、[ページに X のウィジェット JavaScript が含まれている場合](https://developer.x.com/web/javascript/loading)、自動的に[埋め込みタイムライン](https://developer.x.com/en/docs/twitter-for-websites/timelines/overview)として認識されます。

    oEmbed エンドポイントでは、HTML レスポンスに同梱されている X の JavaScript によって解釈される HTML マークアップ内の対応するプロパティを設定することで、埋め込みタイムラインの最終的な表示をカスタマイズできます。返されるマークアップの形式は、X が新機能を追加したり、タイムラインの表現を調整したりするにつれて、時間の経過とともに変わる可能性があります。

    タイムライン URL で指定された X のタイムラインを対象に、[oEmbed](https://oembed.com/) 互換の JSON 形式を返します。ユーザータイムラインおよびリストタイムラインがサポートされています。タイムラインのマークアップは、`cache_age` プロパティで指定される推奨キャッシュ有効期間まで、お使いのサーバー上にキャッシュされることを想定しています。

    ## Resource URL

    *[https://publish.x.com/oembed](https://publish.x.com/oembed)*

    ## Example Requests

    ```bash
    curl --request GET --url 'https://publish.x.com/oembed?url=https%3A%2F%2Ftwitter.com%2FInterior%2Fstatus%2F507185938620219395'
    twurl -H publish.x.com "/oembed?url=https://x.com/Interior/status/463440424141459456"
    ```

  <Tab title="埋め込みポスト">
    返されたHTMLスニペットは、[ページにXのウィジェットJavaScriptが含まれている](https://developer.x.com/web/javascript/loading)場合、[埋め込みポスト](https://developer.x.com/web/embedded-tweets)として自動的に認識されます。

    oEmbedエンドポイントでは、HTMLマークアップ内の対応するプロパティを設定することで、埋め込みポストの最終的な表示をカスタマイズできます。これらのプロパティは、デフォルトでHTMLレスポンスにバンドルされているXのJavaScriptによって解釈されます。返されるマークアップの形式は、Xが新機能を追加したり、ポストの表現方法を調整したりする際に、時間の経過とともに変更される場合があります。

    ポストのフォールバックマークアップは、`cache_age` プロパティで指定された推奨キャッシュ有効期間まで、お客様のサーバーにキャッシュすることを想定しています。

    ## パラメータ

    | 名前                                                                                                                               | デフォルト値  | 説明                                                                                                                                                                                                                                                                                                                                            |
    | :------------------------------------------------------------------------------------------------------------------------------- | :------ | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
    | `url`（必須）  <br />文字列                                                                                                             |         | 埋め込むポストのURL                                                                                                                                                                                                                                                                                                                                   |
    | `maxwidth`  <br />Int `[220..550]`                                                                                               | `325`   | レンダリングされるポストの最大幅をピクセル単位（整数）で指定します。許容範囲を下回る、または上回る値を指定した場合、それぞれサポートされる最小幅または最大幅として解釈され、その調整後の幅の値が返される `width` プロパティに反映されます。なお、X は oEmbed の `maxheight` パラメータをサポートしていません。ポストは本質的にテキストであるため、高さは予測がつかず、画像や動画のようにスケーリングすることはできません。そのため、oEmbed レスポンスでは `height` の値も提供されません。ポストの高さを一定に保つ必要がある実装では、以下の `hide_thread` および `hide_media` パラメータを参照してください。 |
    | `hide_media`  <br />ブール値、文字列、または Int                                                                                             | `false` | `true`、`"t"`、または `1` に設定した場合、ポスト内のリンクは写真、動画、またはリンクプレビューに展開されません。                                                                                                                                                                                                                                                                              |
    | `hide_thread`  <br />Boolean、String または Int                                                                                      | `false` | `true`、`"t"`、または `1` に設定すると、リクエストされたポストが別のポストへの返信である場合、その会話スレッド内で返信先ポストの折りたたみ表示は行われません。                                                                                                                                                                                                                                                       |
    | `omit_script`  <br />Boolean、String または Int                                                                                      | `false` | `true`、`"t"`、または `1` に設定すると、`widgets.js` の読み込みを担う `<script>` は返されません。すべての X ウィジェット（[埋め込みポスト](https://developer.x.com/web/embedded-tweets) を含む）で利用できるように、各ウェブページに独自の `widgets.js` への参照を含める必要があります。                                                                                                                                             |
    | `align`  <br />列挙型 `{left,right,center,none}`                                                                                    | `none`  | 埋め込みポストを、ページ内で親要素に対して左寄せ・右寄せ・中央寄せのいずれで表示するかを指定します。                                                                                                                                                                                                                                                                                            |
    | `lang`  <br />Enum([言語](https://developer.x.com/en/docs/twitter-for-websites/twitter-for-websites-supported-languages/overview)) | `en`    | リクエストで指定した[埋め込みポストでサポートされているXの言語](https://developer.x.com/web/overview/languages)で、HTMLとレンダリング済みのポストが返されます。                                                                                                                                                                                                                                   |
    | `theme`  <br />Enum `{light, dark}`                                                                                              | `light` | `dark` に設定すると、ポストは暗い背景に明るい文字色で表示されます。                                                                                                                                                                                                                                                                                                         |
    | `dnt`  <br />Boolean                                                                                                             | `false` | `true` に設定すると、ポストおよびそのポストを埋め込んだサイト上のページは、[パーソナライズされたおすすめ](https://support.x.com/articles/20169421) や [パーソナライズされた広告](https://support.x.com/articles/20170405) などの目的には使用されません。                                                                                                                                                                  |
