<div id="official-resources">
  ## 公式リソース
</div>

以下のツールは、X for Websites を使い始める際に役立ちます。

<CardGroup cols={3}>
<Card
  title="publish.x.com"
  icon="x-twitter"
  href="https://publish.x.com/"
>
自分のサイト向けに埋め込みポスト、タイムライン、ボタンを作成できます。
</Card>
<Card
  title="Twitter for WordPress"
  icon="x-twitter"
  href="https://github.com/twitter/wordpress"
>
WordPress サイトを X 向けに最適化するための公式プラグインで、埋め込みポストや X Cards に対応しています。
</Card>
<Card
  title="twemoji"
  icon="x-twitter"
  href="https://twemoji.x.com/"
>
X が提供する無料・オープンソースの絵文字セットで、クロスプラットフォーム対応のための JavaScript ライブラリも含まれています。
</Card>
</CardGroup>
