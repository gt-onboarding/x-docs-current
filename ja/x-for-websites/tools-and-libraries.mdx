<div id="official-resources">
  ## 公式リソース
</div>

以下のツールを使うと、X for Websites を使い始めることができます。

<CardGroup cols={3}>
<Card
  title="publish.x.com"
  icon="x-twitter"
  href="https://publish.x.com/"
>
埋め込みポスト、タイムライン、ボタンを自在に作成できます。
</Card>
<Card
  title="Twitter for WordPress"
  icon="x-twitter"
  href="https://github.com/twitter/wordpress"
>
WordPress サイトを X 向けに最適化するための公式プラグインで、ポストの埋め込みや X Cards に対応しています。
</Card>
<Card
  title="twemoji"
  icon="x-twitter"
  href="https://twemoji.x.com/"
>
X が提供する無料のオープンソース絵文字セットで、クロスプラットフォーム対応の JavaScript ライブラリも含まれます。
</Card>
</CardGroup>
