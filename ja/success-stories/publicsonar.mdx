<Note>
**要点**

ハーグ市は、AI を活用したインサイトで組織の意思決定を支援する企業 PublicSonar とのパートナーシップにより、道路交通の円滑化と安全性を高めました。ソーシャルインサイトを活用することで、PublicSonar はハーグ市の交通管制センターに追加情報を提供し、変化する道路状況に迅速かつ効果的に対応できるよう支援しています。
</Note>

<div id="challenge">
  ## 課題
</div>

大陸やタイムゾーンをまたいで、交通は日常生活における予測不能で混沌とした要素となっています。これは世界共通の体験であり、多くの人々の生活に影響を及ぼしています。道路交通を再びコントロール下に置こうとする取り組みは常に行われていますが、その中でも特にこの問題に真剣に取り組んでいる自治体の一つがハーグ市です。

ハーグの交通管制センターは、CCTV（監視カメラ）や地下センサーといったテクノロジーやインテリジェンスツールに投資し、大都市では避けがたい交通関連インシデントに対して、迅速かつ安全な解決策と対応を提供することを目標としていました。しかし、依然として一つの課題が残っていました。最善の判断を下すために、どうすれば適切な情報をリアルタイムで入手できるのか、という点です。

先進的な技術を備えていたにもかかわらず、ハーグの交通管制センターにはなお、いくつかの重要な情報が欠けていました。そうした情報のギャップを埋めるために彼らが頼ったのが PublicSonar です。PublicSonar は、AI によるインサイトを用いて、組織が十分な情報に基づいた意思決定を行えるよう支援する企業です。

<div id="solution">
  ## Solution
</div>

モビリティを高め、安全を確保するうえで最大の障壁となるのは、現場から情報を得ることです。幸いなことに、PublicSonar と提携することで、ハーグ市は交通管制における欠けていたパズルのピース、すなわちソーシャルメディアを見つけ出すことができました。

そこで登場するのが X です。X 上のリアルタイムな会話は、PublicSonar が情報のギャップを埋めるためのインサイトを得るうえで、極めて重要な役割を果たしました。

PublicSonar にとって、X データの提供形態や X API の安定性・サポート体制は、業務に不可欠です。キーワードを活用することで、交通関連のトピックに関する情報を絞り込み、それを基にハーグ市が活用できる、分かりやすいオンラインダッシュボードを作成しました。彼らの価値は、交通管制センターを過負荷にしてしまう過剰な誤検知を避けつつ、重要な情報を取りこぼさないというバランスを見いだすことにあります。

<div id="results">
  ## 結果
</div>

ソーシャルメディアのモニタリングを通じて、PublicSonar はハーグ市に対し、計画されたイベント（例：コンサート）、日常的な状況（例：海岸が混雑する日）、インシデント（例：交通事故）の際に適切な判断を下すためのインサイトを提供しています。

最終的には、テクノロジーとソーシャルメディアから得られるインサイトが連携し合うことで、状況をより鮮明に把握できるようになり、ソーシャルメディアは「道路上のさらなる目」の役割を果たします。こうした情報の流れにより、交通管制センターは現場で何が起きているかを常に的確に把握でき、その結果として、ハーグ市により安全でスムーズな走行環境を継続的に提供することができます。
