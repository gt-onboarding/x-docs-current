<Note>
**要点**

Ryerson University（現 Toronto Metropolitan University）の Migration and Integration 分野の Canada Excellence Research Chair（CERC）プログラムにおけるリサーチフェローである Stein Monteiro 氏が、Twitter データを活用し、インドの農民抗議運動の際におけるインド系ディアスポラとインド本国との関係性に関する研究をどのように前進させたのかを紹介します。

Stein Monteiro 氏にとって、インドで起きた農民抗議運動は興味深い問いを投げかけました。それは、「インドで大きな出来事が起きたとき、インド系ディアスポラはどのように行動するのか」というものです。Twitter の API の助けにより、彼はその答えを明らかにすることができました。詳しくは本ストーリーをご覧ください。
</Note>

<div id="challenge">
  ## 課題
</div>

ライアソン大学のカナダ移民・統合分野カナダ卓越研究チェア（CERC）のリサーチフェローである Stein Monteiro にとって、仕事は統計学、文化、そして時事問題の交差点にある。経済学の専門性によってアカデミアでの成功への土台はできていたが、移民研究への情熱をかき立てたのは、彼自身の経験だった。

インドで生まれ育った Stein は、高校時代をドバイで過ごし、その後カナダに渡って学部課程の学位を取得した。彼にとって「新しい国の新参者である」という感覚は、とてもなじみ深いものだった。新しい社会制度や気候、生活様式の中を切り抜けていく経験は、彼にとって決して珍しいことではなかった。新たな文化や環境へと適応していくプロセスは、移民統合の旅路への関心を呼び起こした。経済学の学習を続ける中でも、こうしたより個人的な経験からくる関心は衰えることなく、そのため両者を結びつけようと考えるに至った。経済学の博士号を取得するにあたり、彼は移民研究の文献を博士論文のテーマとして据えた。

この博士論文に取り組むなかで、Stein は独学でコーディングを身につけた。のちに Humber College で統計学者として働いた際には、そのスキルを実務で活用することができた。コーディングによってプロセスの自動化が可能になり、データを整理して処理することで、彼の仕事は大幅に単純化された。

自らの業務プロセスをコードという形に落とし込むことで、Stein は問題点をより容易に特定できるようになり、自身の作業や思考プロセスを他者に対して段階的に説明できるようになった。これは、問題の見方や解決策の構築方法に、まったく新しい視点をもたらしたのだった。彼はこのスキルセットが持つ無限の可能性を少しずつ実感し、その知見を Canada Excellence Research Chair (CERC) に持ち込んだ。

CERC 在籍中、Stein の関心を強く引いたのが、ソーシャルメディア上で目にしたオンライン・アクティビズムだった。とりわけ（2020 年 8 月に導入された 3 つの物議を醸す法律をきっかけとした）インドの農民抗議運動にまつわる動きである。彼には、これらの抗議と自身の研究との間に関連性が見て取れた。この瞬間は重要だった。リアルタイムで起きている出来事が、移民やディアスポラの経験全体についての情報を明らかにしうると気づいたからだ。

議論が一気に盛り上がるなかで、彼は、オンライン・アクティビズムがカナダ、英国、オーストラリア、そして湾岸諸国に住むインド系ディアスポラの人々の行動にどのような影響を与えているのかを考えた。ソーシャルネットワークは、移民をどのようにインド国内の議論と結びつけているのだろうか。

いよいよ、取り組むべき時が来た。 

<div id="solution">
  ## 解決策
</div>

インド系ディアスポラと農民抗議運動との関係性をより深く理解するためには、オンライン上の会話に耳を傾ける必要があると考えた Stein は、Twitter に注目しました。

Stein によれば、「こうしたローカルな出来事が国際的なオーディエンスに届けられたのは、単に Twitter がグローバルな存在だからというだけでなく、人々、つまりごく普通の人々がオンラインで互いに語り合い、ハッシュタグを通じてつながることができたからです」。彼にとって Twitter は、「インドで起きている農民抗議運動に、インド人ではなくても、あるいはインド人であっても世界の別の場所にいながら参加できる場」でした。

Stein は、抗議参加者、抗議に反対する言説、政府寄りの主体、そして現場で実際に何が起きているかというフィルターのかかっていない体験が集まる場所として Twitter を捉えていました。そして、彼自身もその流れの中に身を置きたいと考えました。 

オンライン上の議論を追いながら、Stein はいくつかの点に焦点を絞る必要がありました。インドで誰かがオンラインに何かをポストしたとき、そのポストはどのように世界の他の地域へ届くのか。情報はネットワークやハッシュタグなどを通じてどのように流れていくのか。そして、インドから届くポストに対してディアスポラはどのように反応しているのか。 

Stein は、Twitter の API の助けを借りて、こうした点を明らかにすることができました。API には Academic Research 製品トラックが導入されており、履歴データに加えて大量のポストおよび投稿者情報にアクセスできるため、さらなる分析が可能になりました。 

Stein にとって、API の強みは、長期間にわたるデータにアクセスできる点にあり、それによって会話の時間的な推移——誰が何をリポストしているのか、誰と誰が話しているのか——を描き出すことができました。 

こうした情報をすべて手にした結果、彼は自らの発見に衝撃を受けました。インド系ディアスポラがインドの状況に反応して行っている活動は、非常に活発だったのです。Stein は言います。「しかも、その活動は遅れて現れたわけではなく、ほとんど即時的でした。1 日のうちに、インドからのポストに対するリポストが世界各地で行われているのが見て取れたのです。」

データが示していたのは明快な結論でした。ソーシャルメディアは、移民やディアスポラが母国とのつながりを維持し、重要な議論に能動的に参加することを可能にしているのです。

<div id="impact">
  ## インパクト
</div>

Twitter のオープン API を使用することで、Stein は移民としての経験をめぐるより大きな議論全体への洞察をもたらす、タイムリーな出来事を研究することができました。データへのアクセスと、そのデータをフィルタリングおよびソートするための機能は、このプロジェクトにとって極めて重要な要素となりました。

さらに重要な点として、Stein は学術研究全般における研究ツールとしての Twitter の重要性を認識しました。彼にとって研究における最大のハードルの一つは、データを世界中から収集しなければならないことです。幸いなことに、Twitter の国際的な普及により、世界各地で高額なアンケート調査やフォーカスグループ調査を行うことなく、グローバルな情報へのアクセスが可能になります。

Stein は次のように述べています。「Twitter は、研究者や学術コミュニティが、自身の研究目的のためにデータにアクセスするための新しい手段を提供し、新しい方法論を開発することを可能にしてくれますが、同時にコスト効率の高い手段でもあります。」彼はオンライン上で交わされる会話が有用であった理由について、「それらはほとんど加工されておらず、また偏りのない会話であり、研究者がその会話に介入することなく、人々同士の間で交わされていたからです」と述べています。

Stein の研究は、Twitter が世界中の人々をつなぐツールであり続けていること、そして Twitter の API がそのつながりから生まれる大規模な洞察を収集するのに役立つことを証明しました。
