**要点まとめ（tl;dr）**

Ryerson University の Migration and Integration 分野の Canada Excellence Research Chair（CERC）でリサーチフェローを務める Stein Monteiro 氏が、農民抗議運動におけるインド・ディアスポラ（海外在住インド人社会）とインド本国との関係に関する研究を、どのように Twitter データを用いて深めたのかを紹介します。

Stein Monteiro 氏にとって、インドで起きた農民抗議運動は興味深い問いを投げかけるものでした。それは、「インドで大きな出来事が起きたとき、インド・ディアスポラはどのように行動するのか」というものです。Twitter の API の助けを借りることで、彼はその答えを明らかにすることができました。詳しくは本ケーススタディをご覧ください。
</Note>

<div id="challenge">
  ## 課題
</div>

カナダのライアソン大学における移民・統合分野の Canada Excellence Research Chair (CERC) のリサーチフェローである Stein Monteiro にとって、仕事は統計、文化、時事問題が交差するところにあります。経済学の専門性が学問的な成功の土台を作った一方で、移民研究への情熱をかき立てたのは、彼自身の経験でした。

インドで生まれ育った Stein は、高校時代をドバイで過ごし、その後カナダに渡って学部課程を修了しました。彼にとって、異国で「新参者」である感覚は、非常に馴染み深いものでした。新しい社会制度や気候、生活様式を渡り歩いていく経験は、彼にとって決して他人事ではありませんでした。新たな文化や環境に適応していくプロセスは、移民の統合プロセスへの関心を呼び起こしました。経済学の研究を続ける中でも、こうしたより個人的な経験の魅力は衰えることなく、彼はこの二つの世界を結びつけようと考えるようになります。経済学の博士号を取得するにあたり、彼は移民研究の文献をテーマに学位論文をまとめました。

この学位論文に取り組むなかで、Stein は独学でコーディングを習得しました。その後 Humber College で統計学者として働いた際には、その知識を実務で活用することができました。コーディングによって業務の自動化が可能となり、データの整理や集計を通じて、仕事を大きく単純化できたのです。

自分の業務プロセスをコードという形に落とし込むことで、Stein は問題をより容易に特定できるようになり、自分の業務内容や思考プロセスを他者に対して段階的に説明できるようになりました。これは、問題を捉え、解決策を構築するまったく新しい視点をもたらしました。彼はこのスキルセットが持つ無限の可能性に徐々に気づき始め、この知識を Canada Excellence Research Chair (CERC) にも持ち込みました。 

CERC 在籍中、Stein はソーシャルメディア上で目撃したオンライン・アクティビズム、特に（2020 年 8 月に導入された 3 つの物議を醸す法律をきっかけに起こった）インドの農民抗議運動に強く関心を引かれました。彼には、これらの抗議活動と自身の研究との間に関連が見て取れました。これは重要な瞬間でした。リアルタイムで起きている出来事が、移民およびディアスポラ全体の経験についての示唆を与えてくれる可能性があったのです。

議論が一気に盛り上がるなかで、彼は、オンライン・アクティビズムがカナダ、英国、オーストラリア、湾岸諸国に住むインド系ディアスポラの人々の行動にどのような影響を与えているのかを考えました。ソーシャルネットワークは、移民をどのようにインド国内の議論と結びつけているのか。 

ついに本格的に取り組むときが来ました。 

<div id="solution">
  ## ソリューション
</div>

インド系ディアスポラと農民抗議運動との関係性をよりよく理解するためには、オンライン上の会話に耳を傾ける必要があると Stein は考え、Twitter 上の動きに注目することにしました。

Stein によると、「こうした現地で起きている出来事が国際的な人々に届いたのは、Twitter が国際的なプラットフォームだからというだけでなく、人々、つまりごく普通の人たちがオンラインで互いに話し、ハッシュタグさえあればつながることができたからです」。彼にとって Twitter とは、「インドで起きている農民抗議運動に、インド人でなくても、あるいはインド人でも世界の別の場所にいても参加できる場」でした。

Stein は、抗議参加者、抗議に反対する言説、政府寄りの組織、そして現場で実際に何が起きているかに関する生の体験談が集まる場所として Twitter を捉えました。そして、その渦中に自分も入り込みたいと考えました。 

数多くのやり取りを追いながら、Stein はいくつかの点に絞り込む必要がありました。インドで誰かが何かをオンラインに投稿したとき、そのポストはどのように世界の他の地域へ届くのか。情報はネットワークやハッシュタグなどを通じてどのように流れるのか。そして、インドから流れてくるこれらのポストに対して、ディアスポラはどのように反応しているのか。 

Stein は、Twitter の API の助けを借りて、こうした点を明らかにすることができました。API では Academic Research プロダクトトラックが導入され、過去のデータに加えて大量のポストや投稿者情報にアクセスし、さらに分析できるようになっていました。 

Stein にとって、この API の強みは、長期間にわたるデータへアクセスできることでした。それにより、会話の時間軸――誰が何をリツイートし、誰が誰と話しているのか――を構築することが可能になったのです。 

こうした情報をすべて手にした結果、彼は自分の発見に衝撃を受けました。インド国内の状況に反応するインド系ディアスポラによる活動が、驚くほど活発だったのです。Stein は「それは遅れて表れる活動ですらなく、ほとんど即時のものでした。1 日のうちに、インドで投稿されたものを人々がリツイートし、そのリツイートが世界各地で起きているのが見えていたのです」と述べています。

データは明白でした。ソーシャルメディアは、移民やディアスポラが母国とつながり続け、重要な議論に積極的な役割を果たすことを可能にしているのです。

<div id="impact">
  ## インパクト
</div>

Twitter のオープン API を利用することで、Stein は移民の経験という、より大きな文脈での対話に光を当てるタイムリーな出来事を研究することができました。データへのアクセスと、そのデータをフィルタリングおよびソートするための機能は、このプロジェクトにおいて極めて重要な役割を果たしました。

さらに重要な点として、Stein は学術研究全体にとってのツールとしての Twitter の意義を認識しました。彼にとって研究における最大のハードルの 1 つは、データを世界中から収集する必要があることです。幸いにも、Twitter の国際的なプレゼンスにより、世界各地で多額の費用をかけて調査やフォーカスグループを実施しなくても、グローバルな情報へアクセスすることが可能になります。

Stein は次のように述べています。「Twitter は研究者や学術コミュニティに対し、自身の研究目的のためにデータへアクセスできる新たな手段を提供しており、新しい手法を開発できるだけでなく、費用対効果の高い方法でもあります。」彼は、オンライン上で交わされている会話が有用であると考えています。その理由は、「それらはほとんどフィルタリングされておらず偏りのない会話であり、研究者がその会話に介入することなく、人々の間で自然に行われていたから」です。

Stein の研究は、Twitter が今なお世界中をつなぐツールであり続けていること、そして Twitter の API が、そのつながりから生まれる大規模なインサイトを収集するうえで役立つことを証明しました。
