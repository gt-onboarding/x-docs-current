<div id="situation">
  ## 状況
</div>

COVID-19 の流行以降、アジア系アメリカ人および太平洋諸島系住民（AAPI）に対するヘイト事件が増加しました。これらの一部はニュースで取り上げられましたが、何千人もの人々が、調査やアドボカシー、能力強化の取り組みを通じてヘイト事件に対応・対処している全米報告センター [Stop AAPI Hate](https://stopaapihate.org/) に報告を寄せていました。この時期に [The American Asian Foundation (TAAF)](https://www.taaf.org/) が設立されました。この団体は、こうした事件が標的となった人々にどのような影響を与えているのかを、X を活用して新たな形で可視化しようとしたチームを後押しする原動力となりました。

<div id="solution">
  ## 解決策
</div>

2人のエンジニアから成る少人数チームは、インシデントデータの情報源としてXに注目しました。彼らがこのプラットフォームを選んだ理由は2つあります。1つ目は、ニュース、人々の考えや思考、日常生活の出来事、コミュニティで起きていることなど、豊富で多様な情報が含まれていることです。これにより、Xはさまざまな集団の、ユニークで親密なサンプルを提供します。さらにXは、TAAFの開発者がデータにアクセスできる真にオープンなAPIを備えた唯一のプラットフォームであり、そのデータを収集・分析するための高度なツールも提供しています。 

X API v2 の search Tweets エンドポイントと Tweets lookup エンドポイントを使用して、チームはポストを検索し、人々がヘイトインシデントを報告したタイミング（さらにはどのハッシュタグを使っていたかまで）を特定することができました。これによりチームは、数十億件にのぼるポストを扱える量まで絞り込む、1023文字の検索クエリを設計することができました。 

次の課題は、そのポストが実際にヘイトインシデントに関するものかどうかを検証することでした。チームは自然言語処理を用いたツールを開発し、数千件のポストの中からプロジェクトに関連するものをふるい分けられるようにしました。学習済みの機械学習モデルを適用した後、精度をさらに高めるために、人間による検証も行いました。 

2021年には、TAAFは Stop AAPI Hate 連合と共同で [Documenting Anti-AAPI Hate Codebook](https://stopaapihate.org/wp-content/uploads/2021/12/SAH-x-TAAF-Codebook-FINAL-2021-08-19.pdf) を執筆しました。このリソースには、コミュニティ主導のデータ収集のための標準案や運用方法が含まれており、人々がX上で共有しているインシデントをチームが分類するのに役立ちました。 

そこからの次の課題は、このデータを人々が簡単に理解できる形で提示することでした。TAAFはデータ可視化ツール [Decoding Hate](https://decodinghate.taaf.org/) を作成し、膨大な量のデータをインタラクティブな形で表示できるようにしました。データ可視化スタジオの協力を得て、注釈付きのXのデータを活用し、数千件のポストを、AAPIコミュニティで何が起きていたのかという真実を明らかにする洞察に富んだストーリーへと落とし込むことができました。

<div id="impact">
  ## インパクト
</div>

Decoding Hate プロジェクトは、それまで報告されていなかった何千件ものヘイト関連事案の検出に貢献し、現在もその役割を果たし続けています。このプロジェクトは、AAPI コミュニティ内で起きている個々の経験への認知を広げるための継続的なツールとなっています。

次のフェーズでは、TAAF は Stop AAPI Hate と提携し、X 上でヘイト事案を報告した一人ひとりに対して行う、人による個別対応のパイロットを実施します。このフェーズでは、Stop AAPI Hate が、これまでに経験したヘイトについて投稿している多くの人々と積極的に関わり、[Stop AAPI Hate Reporting Center](https://stopaapihate.org/) を通じて事案を報告するよう促します。より多くの人が Stop AAPI Hate に報告することで、連合およびそのパートナーは、発生しているヘイト事案の性質をより正確に把握し、最も効果的な政策的解決策を提言できるようになります。 

AAPI コミュニティで起きているヘイト関連事案を可視化するだけでなく、このプロジェクトは X の API の価値にも光を当てています。この事例は、適切なデータを適切なツールと組み合わせて活用することで、開発者が公益のために X を使ってイノベーションを起こせることを示しています。
