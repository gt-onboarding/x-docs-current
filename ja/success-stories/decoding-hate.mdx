<div id="situation">
  ## 状況
</div>

COVID-19 の流行以降、アジア系アメリカ人および太平洋諸島系住民（AAPI）を標的としたヘイト行為が増加しました。こうした事案の一部はニュースで取り上げられましたが、何千人もの人々が、調査研究、アドボカシー、組織基盤の強化を通じてヘイト事案に対応・対処している全米報告センター [Stop AAPI Hate](https://stopaapihate.org/) に報告を行っていました。この時期に [The American Asian Foundation (TAAF)](https://www.taaf.org/) が設立されました。この団体は、こうした事案が標的とされた人々一人ひとりにどのような影響を及ぼしているかを示す新たな視点を提供するために、X を活用することを決めたチームを牽引する存在となりました。

<div id="solution">
  ## 解決策
</div>

2人のエンジニアから成る少人数のチームは、インシデントデータの情報源としてXに注目しました。彼らがこのプラットフォームを選んだ理由は2つあります。第一に、ニュース、人々のアイデアや思考、日常生活の出来事、地域社会で起きていることなど、豊かで多様な情報が含まれているからです。これにより、Xはさまざまな人々の集団からの、ユニークで身近なサンプルを提供します。さらにXは、TAAFの開発者がデータへアクセスできる真にオープンなAPIを備えた唯一のプラットフォームであり、そのデータを収集・分析するための高度なツール群も提供しています。 

X API v2 の search Tweets エンドポイントおよび Tweets lookup エンドポイントを使用し、チームはポストを検索して、人々がヘイトインシデントを報告したタイミング（さらにはどのハッシュタグを使うか）を特定することができました。これにより、チームは1023文字の検索クエリを設計し、数十億件のポストを、チームが扱える件数まで絞り込むことができました。 

続いての課題は、そのポストが本当にヘイトインシデントに関するものかを検証することでした。チームは自然言語処理を用いたツールを開発し、数千件のポストからプロジェクトに関連するものをふるい分けられるようにしました。学習済みの機械学習モデルを適用した後、ポストは人手による検証も行い、精度をさらに高めました。 

2021年には、TAAFは Stop AAPI Hate 連合と共同で [Documenting Anti-AAPI Hate Codebook](https://stopaapihate.org/wp-content/uploads/2021/12/SAH-x-TAAF-Codebook-FINAL-2021-08-19.pdf) を執筆しました。このリソースには、コミュニティ主導のデータ収集のための標準案および実務上の手順が含まれており、人々がX上で共有しているインシデントをチームが分類するうえで役立ちました。 

そこからの次の課題は、このデータを人々が容易に理解できる形で提示することでした。TAAFはデータ可視化ツール [Decoding Hate](https://decodinghate.taaf.org/) を作成し、膨大なデータ量をインタラクティブな形で表示できるようにしました。データ可視化スタジオの協力を得て、注釈付きのXデータを活用し、数千件のポストを、AAPIコミュニティ内で起きていた事象の真実を浮き彫りにする洞察に富んだストーリーへと変換することに成功しました。

<div id="impact">
  ## インパクト
</div>

Decoding Hate プロジェクトは、これまで報告されていなかった何千件ものヘイト関連事案の検出に貢献し、現在も継続的にその役割を果たしています。このプロジェクトは、AAPI コミュニティ内で起きている一人ひとりの体験への認知を広げるための継続的な手段となりました。

次のフェーズの取り組みとして、TAAF は Stop AAPI Hate と提携し、X 上でヘイト事案を報告した人々に対する、人による個別対応を試験的に実施します。このフェーズの間、Stop AAPI Hate は、自身が経験したヘイトについてのポストをしている多くの人々とかかわり、その事案を [Stop AAPI Hate Reporting Center](https://stopaapihate.org/) を通じて報告するよう積極的に促します。より多くの人が Stop AAPI Hate に報告することで、連合およびそのパートナーは発生しているヘイト事案の性質をより深く理解でき、最も効果的な政策上の解決策を提言できるようになります。 

AAPI コミュニティで起きているヘイト関連事案を可視化するだけでなく、このプロジェクトは X の API の価値にも光を当てています。この事例は、適切なツールを使って適切なデータを活用することで、開発者が社会的に有益な目的のために X を用いてイノベーションを起こせることを示しています。
