<Note>
**要約（tl;dr）**

成功への軌道: Red Cloud Financial Services は、ある鉱業会社から新しいグリーンテクノロジーのマーケティング支援を依頼された際、X のアナリティクス上に構築された Audiense のターゲティングツールを活用してデータを掘り起こし、新たなオーディエンスへとリーチすることにしました。
</Note>

<div id="situation">
  ## 状況
</div>

Red Cloud Financial Services は、上場鉱業会社であるクライアントから、新たに開発した特許出願中のテクノロジーについて、まったく新しいオーディエンスに向けた認知向上を図る支援を依頼されました。 

同社は、この新製品で成果を上げて新たな地域へと事業を拡大し、その新規顧客層にリーチできるキャンペーンを効果的に実行する方法を見つける必要がありました。 

この依頼には、2 つの主要な課題がありました。 

1. **新しいオーディエンスへのリーチ**

    既存の鉱業顧客にフォーカスするのではなく、クライアントの新しいテクノロジーは、グリーンイノベーションや社会的責任に関心を持つ、より幅広いセグメントに訴求するものでした。これは Red Cloud にとって未知の領域であり、それまでに実施してきたどのキャンペーンとも異なるものでした。

1. **具体的な ROI を提示する**

    クライアントは有料キャンペーンに慣れていませんでした。Red Cloud には、認知向上を目的としたキャンペーンの ROI を明確に示し、かつ Red Cloud 自身が直接の経験をほとんど持たないオーディエンスを相手に成果を出すことが期待されており、それは非常に難しい課題でした。 
    
    Red Cloud は、既存顧客にリーチしながら新規見込み顧客を調査するため、Audiense に助けを求めました。Audiense の X マーケティング向けプロダクトは、強力なオーディエンスターゲティングツールを活用し、インパクトのある X キャンペーンを提供することを専門としています。

<div id="solution">
  ## 解決策
</div>

Red Cloud は、まず期待値の設定から着手しました。Audiense から得られたインサイトを活用してキャンペーンのパフォーマンス予測を作成することで、Red Cloud はクライアントに対し、キャンペーンの最終目標とそれに紐づく KPI について適切な期待値を設定することができました。 

Audiense の支援を受けて、Red Cloud は X 上での調査に注力し、鉱業関連の企業ニュースや関連株式情報を収集しました。Audiense のプラットフォームにより、キャンペーンと、動画再生数の増加やウェブサイトトラフィックの増加として表れる関心の高まりとの間にある潜在的な相関関係を、容易に見つけ出すことができました。

3 段階のアプローチ：

1. まったく新しいオーディエンスを構築する。

    クライアントからの指示を踏まえ、Red Cloud Financial はキャンペーンの主な対象とする 2 つのオーディエンスを特定しました。1 つ目のセグメントは鉱業と投資、2 つ目はクリーンテック、電気自動車、ESG と責任ある投資です。X のデータは、これらのオーディエンスを今後のキャンペーンと整合させるうえで Red Cloud をさらに支援しました。

1. 新しいオーディエンスをセグメント化する。 

    新しいオーディエンスは幅広い業種をカバーしていましたが、その間には多くの共通点が存在していました。Audiense は、Red Cloud Financial がこれらの新しいセグメントを深掘りし、インフルエンサーやコンテンツ消費の傾向などを特定するのに役立ちました。これらのインサイトは、フォローすべきアカウントを見極め、相互フォローによる関係構築と、Red Cloud およびクライアントのオーディエンス拡大につなげるためにも活用されました。

1. どれが最も価値が高いかを判断する。 
    
    どれが最も価値が高いかを判断する。Audiense の [Targeting Pack 機能](https://help.audiense.com/knowledge/what-is-a-targeting-pack)を使って、Red Cloud は極めて精緻にターゲティングした有料 X 広告キャンペーンを作成し、蓄積した X セグメントのデータをオーガニックキャンペーンに活用しました。Targeting Pack 機能から得られたインサイトを活用して、Audiense と Red Cloud は 2 つのオーディエンスを対象とする 2 本の独自の動画を作成しました。これにより、どの動画が一貫して最も成果の高いキャンペーンとなっているかを特定し、ターゲティングの方向性を維持したまま進めることができました。

<div id="impact">
  ## 成果
</div>

Audiense のプラットフォームと X 上でのリサーチは、Red Cloud が適切なオーディエンスを見つけてセグメント化し、フォロワー獲得に活用するとともに、オーガニック・有料コンテンツ双方の全体的なソーシャルメディア戦略の策定に役立ちました。

Red Cloud Financial はクライアントの期待を上回りました。実際、キャンペーンの結果が非常に好調だったため、クライアントは次のキャンペーンをより大規模に展開することを決定しました。 

- 複数のキャンペーンを合算して 350 万回超のインプレッションと 280 万回超の総ビューを獲得
- クライアントの新しい特許出願中のテクノロジーに関連する会話量が増加
- 新テクノロジーに関するプロモーション記事およびプレスリリースからの 7,000 人の新規訪問者を含め、ウェブサイトへの流入トラフィックの増加が見られた

これは、X 上のオーディエンス分析とターゲティングが、新たなオーディエンスの開拓、マーケティングキャンペーンの精緻化、そして実際の成果の創出にどのように役立つかを示す代表的な事例です。クライアントのブランド構築とビジネスの成長を後押しする絶好の機会となり得ます。
