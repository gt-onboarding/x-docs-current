<Note>
**要点（tl;dr）**

成功へのトレンド: Red Cloud Financial Services は、鉱山会社から新しいグリーンテクノロジーのマーケティング支援を依頼された際、X のアナリティクスを基盤とする Audiense のターゲティングツールを活用してデータを掘り起こし、新たなオーディエンスにリーチしました。
</Note>

<div id="situation">
  ## 状況
</div>

Red Cloud Financial Services は、上場鉱業会社であるクライアントが開発した特許出願中の新技術について、新たなオーディエンスの間で認知を高める支援をする役割を担いました。 

同社は、この新製品で成果を上げる方法を見つけ、新たな市場へ進出し、これらの新しい顧客にリーチできるキャンペーンを効果的に実施する必要がありました。 

この依頼には、2つの重要な課題がありました。 

1. **新しいオーディエンスへのリーチ**

    既存の鉱業分野の顧客にフォーカスするのではなく、クライアントの新技術は、グリーン・イノベーションや社会的責任に関心を持つ、より広いセグメントに訴求するものでした。これは Red Cloud にとって未知の領域であり、それまで実施してきたキャンペーンとはまったく異なるものでした。

1. **具体的な ROI を提示すること**
    
    クライアントは有料キャンペーンの実施に慣れていませんでした。Red Cloud には、認知向上を目的としたキャンペーンでどれだけの ROI が得られるかを明確に示し、かつ、直接の知見が限られているオーディエンスを対象に結果を出すことが求められており、これは大きな課題でした。 
    
    Red Cloud は、既存顧客にリーチしつつ、新規見込み顧客をリサーチするために Audiense を活用することにしました。Audiense の X マーケティングプロダクトは、強力なオーディエンスターゲティングツールを用いて、インパクトのある X キャンペーンを実現することを専門としています。

<div id="solution">
  ## Solution
</div>

Red Cloud は、まず期待のすり合わせから着手しました。Audiense から得られたインサイトを活用してキャンペーンのパフォーマンス予測を作成することで、Red Cloud はクライアントに対し、キャンペーンの最終目標とそれにひもづく KPI について、事前に期待を明確に共有することができました。 

Audiense の支援を受け、Red Cloud は企業ニュースや関連する株式情報を収集するためのリサーチを X に集中的に行いました。Audiense のプラットフォームにより、キャンペーンと、動画再生数やウェブサイトトラフィックの増加として表れる関心の高まりとの間に存在し得る相関関係を、容易に見つけ出すことができました。

3 段階のアプローチ:

1. まったく新しいオーディエンスを構築する。

    クライアントからの要望に基づき、Red Cloud Financial はキャンペーンのターゲットとする 2 つの主要オーディエンスを特定しました。1 つ目のセグメントには鉱業と投資、2 つ目にはクリーンテック、電気自動車、ESG、および責任ある投資が含まれていました。X のデータは、Red Cloud がこれらのオーディエンスを今後のキャンペーンと整合させるうえでも役立ちました。

1. 新しいオーディエンスを細分化する。 

    新しく定義されたオーディエンスは幅広い産業分野をカバーしていたものの、その間には多くの共通項が存在していました。Audiense によって、Red Cloud Financial はこれらの新しいセグメントを掘り下げ、インフルエンサー、コンテンツ消費の傾向などを特定することができました。こうしたインサイトにより、どのアカウントをフォローすべきかを判断し、フォロー返しを期待しつつ、Red Cloud とクライアント双方のオーディエンス拡大につなげることができました。

1. どれが最も価値が高いかを判断する。 
    
    どれが最も価値が高いかを判断します。Audiense の [Targeting Pack 機能](https://help.audiense.com/knowledge/what-is-a-targeting-pack) を使用して、Red Cloud は非常に精緻にターゲティングされた有料の X 広告キャンペーンを作成する一方で、蓄積された X セグメントのデータをオーガニックキャンペーンに活用しました。Targeting Pack 機能から得られたインサイトに基づき、Audiense と Red Cloud は 2 つのオーディエンスそれぞれをターゲットにした 2 本のユニークな動画を制作しました。これにより、どの動画が一貫して最も成果の高いキャンペーンであるかを特定し、ターゲティングの方向性を維持することができました。

<div id="impact">
  ## 成果
</div>

Audiense のプラットフォームと X を活用したリサーチにより、Red Cloud は適切なオーディエンスを発見・セグメント化し、そのオーディエンスを活用してフォロワーを増やすとともに、オーガニックと有料コンテンツの両方における全体的なソーシャルメディア戦略の立案に役立てることができました。

Red Cloud Financial は、クライアントの期待を上回る成果を出しました。実際、キャンペーンの結果が非常に良好だったため、クライアントは継続を決定し、さらに別のキャンペーンの規模を拡大しました。 

- 複数のキャンペーンを合わせて、350 万回超のインプレッションと 280 万回の総ビューを獲得
- クライアントの新しい特許出願中テクノロジーに関連する会話量が増加
- 自社サイトへの参照トラフィックが増加し、新テクノロジーに関するプロモーション記事およびプレスリリースからの新規訪問者が 7,000 人にのぼった

これは、X 上のオーディエンス分析とターゲティングが、新たなオーディエンスへのリーチ、マーケティングキャンペーンの最適化、そして具体的な成果の創出にどのように貢献できるかを示す好例です。クライアントのブランド構築とビジネス成長を後押しする絶好の機会となります。
