<Note>
**要点（tl;dr）は次のとおりです**

開発者でありクリエイターでもある Aaron Ng 氏は、X の API を活用して、X 上のネットワークを検索できる検索エンジン「Flocknet」を構築しました。ユーザーは Flocknet を使ってネットワークを拡大し、優秀な人材を採用し、投資案件のリードを獲得しています。もともとは自身の個人的な利用のために構築された Flocknet ですが、今では X 上で何百万もの連絡先を管理する手助けをしています。
</Note>

<div id="challenge">
  ## 課題
</div>

Aaron Ng はニューヨークで開催されたデザインカンファレンスに参加した際、同業界の人たちとつながり、実際に会いたいと考えていました。ところがそのとき、たとえ X でつながっているデザイナー仲間であっても、思っていた以上にコンタクトを取るのが難しいことに気づきました。彼のコミュニティの多く、特に業界の友人たちは X を使っていました。では、自分の友人のうち誰が同じタイミングでその街にいるのかを知る術は本当にないのだろうか。誰が近くにいるのかわからないまま、どうやって業界の仲間たちと再会すればいいのだろうか。

Aaron はすぐに、この課題をチャンスだと捉えました。こうして彼の起業家精神から、Flocknet が生まれました。 

<div id="solution">
  ## Solution
</div>

わずか週末の間に、Aaron は、たとえ相手のポストが見えなくても、自身の X ネットワーク上の人々、友人、その他のプロフェッショナルを検索・ソートし、常に動向を追い続けられるプロダクトを構築しました。彼はこのプロジェクトのために X の API を使うことを選び、「質問があれば聞ける相手がいること、そして［X の］チームが気にかけてくれているとわかっていることが、とても心地よい体験につながります」と述べています。さらに、「作り続けようと自分を駆り立ててくれたコミュニティの多くは X 上にいました——だからこそ、X は自分にとってとても重要な場所になったし、自分の使い方を高めるためのツールを作りたいと思ったのです」と語っています。  

自身の創造性、行動力、そして経験を組み合わせることで、Flocknet が生まれました——X ネットワーク上で人を見つけるための検索エンジン兼コンタクト管理ツールです。X の follows エンドポイントと Python を活用することで、Flocknet はユーザーがフォロワーを職種、勤務先、所在地、その他多数の属性で検索できるようにしています。

当初は自分自身の個人的な利用を想定していましたが、Flocknet は非常に大きな注目を集めたため、これは共有されるべきものだと彼は確信しました。そこで彼はすぐに方針を切り替え、Flocknet を現在の姿までスケールさせることに注力しました。

<div id="impact">
  ## インパクト
</div>

わずか数か月のうちに、Flocknet はユーザーに代わって何百万件ものコンタクトを管理するようになりました。特にテック系のプロフェッショナルに重宝されており、採用から投資、その間にあるさまざまな用途まで幅広く使われています。実際、Aaron 自身も Flocknet を通じて人を採用してきました。

しかし、Aaron はまだ歩みを止めていません。寄せられる新機能のリクエストに注目しながら、プロダクトを拡張していく方法を継続的に模索しています。

最終的に Aaron は、主に X 上にコミュニティが存在する人たちの体験を向上させるために、X をメタプラットフォームとして機能させる方法を見つけました。彼はこう語ります。「とにかくいろいろなものを作っています。そして、自分にとって役に立ち、ほかの人にとっても役に立つものを作るのが大好きなんです」。そして、まさにそのとおりのことを実現しました。
