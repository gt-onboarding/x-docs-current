<Note>
**要点まとめ（tl;dr）**

ここでは、開発者兼クリエイターである Aaron Ng 氏が、X の API を活用して、X 上の自分のネットワークを検索できるエンジン「Flocknet」を構築した方法を紹介します。ユーザーは Flocknet を使ってネットワークを拡大し、優秀な人材を採用し、投資先候補や投資リードを発掘しています。もともとは自身の個人的な利用のために構築された Flocknet ですが、今では、多くの人が X 上で数百万件のつながりを管理するのに活用しています。
</Note>

<div id="challenge">
  ## 課題
</div>

Aaron Ng はニューヨークで開催されたデザインカンファレンスに参加した際、同じ業界の人たちとつながり、実際に会いたいと考えていました。ところがそのとき、たとえ X 上でつながっているデザイナー仲間であっても、会う約束をするのは想像以上に難しいことに気づきました。彼のコミュニティの多く、特に業界の友人たちは X を使っていました。それなのに、誰が同じタイミングでその街に来ているのかを知る方法はないのでしょうか？誰が近くにいるのかわからないまま、業界の知り合いたちと再会するにはどうすればいいのでしょうか？

Aaron はすぐに、この課題をビジネスチャンスだと捉えました。そして、その起業家的な精神から、Flocknet が誕生しました。 

<div id="solution">
  ## 解決策
</div>

わずか 1 週末で、Aaron は、自分の X ネットワーク上の人々、友人、他のプロフェッショナルを、ポストが見られなくても検索・ソートし、動向を追い続けられるプロダクトを構築しました。彼はこのプロジェクトのために X の API を使うことを選び、「質問ができる相手がいるとわかっていて、しかも [X の] チームが気にかけてくれているとわかるのは、体験をずっと良いものにしてくれます」と語ります。さらに「作り続けるモチベーションを与えてくれたコミュニティの多くは X 上にありました。だからこそ X は自分にとって非常に重要な場所になっていて、その使い方をもっと良くするツールを作りたいと思ったのです」と付け加えています。  

自身の創造性、行動力、経験を組み合わせることで、Flocknet は誕生しました。これは、X ネットワーク内の人を見つけるための検索エンジン兼コンタクト管理ツールです。X の follows エンドポイントと Python を活用することで、Flocknet はユーザーがフォロワーを職種、勤務先、場所、その他多数の属性で検索できるようにしています。

当初は自身の個人利用を想定していたものの、Flocknet は非常に大きな関心を集めるようになり、共有すべきものだと彼は確信しました。そのため、彼はすぐに Flocknet をスケールさせ、現在の姿へと進化させることに注力しました。

<div id="impact">
  ## インパクト
</div>

わずか数か月のうちに、Flocknet はユーザーに代わって何百万件もの連絡先を管理するまでになりました。特にテクノロジー分野のプロフェッショナルに重宝されており、採用から投資、その間にあるあらゆる用途にまで幅広く活用されています。実際、Aaron 自身も Flocknet を通じて人材を採用したことがあります。

しかし、Aaron の取り組みはまだ終わっていません。寄せられる新機能の要望に耳を傾けながら、Aaron は常にプロダクトを拡張する方法を模索し続けています。

最終的に Aaron は、コミュニティの主な活動拠点が X 上にある人々の体験を向上させるために、X をメタプラットフォームへと変える方法を見つけました。彼はこう語っています。「とにかくいろいろなものを作っています。そして、自分が便利だと思い、ほかの人にも便利だと思ってもらえるものを作るのが大好きなんです」。そして、まさにそのとおりのことを実現しました。
