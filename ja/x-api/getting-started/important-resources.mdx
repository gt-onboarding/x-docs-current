<div id="learn-about-whats-possible">
  ## できることを学ぶ
</div>

トップナビゲーションからアクセスできるユースケース、ソリューション、プロダクトページは、プラットフォーム上で何ができるのかを俯瞰的に把握するのに最適な方法です。

また、さまざまなユースケースに対応するために、1つまたは複数のプラットフォームツールをどのように活用できるかを理解するのに役立つ[チュートリアル](/ja/tutorials)も複数用意しています。

ドキュメントの最も基礎的なレベルにはエンドポイントおよびツールのドキュメントがあり、いずれも、そのツールの機能と典型的なユースケースの概要を簡潔に示す導入コンテンツを含んでいます。たとえば、[Search Posts](/ja/x-api/posts/search/introduction) に関する X API v2 のエンドポイントセクションがあります。

<div id="integrate-an-endpoint-into-your-system">
  ### エンドポイントをシステムに統合する
</div>

エンドポイントとの連携をシームレスに行えるよう、各種ツール、ライブラリ、統合ガイドを用意しています。最も簡単な統合方法は、[ツールとライブラリ](/ja/x-api/tools-and-libraries/overview) を利用することです。これらは [OAuth 1.0a User Context authentication](/ja/resources/fundamentals/authentication#oauth-1-0a-2)、ページネーション、[レート制限の処理](/ja/resources/fundamentals/rate-limits) などの複雑な機能を自動的に処理します。

一方、ゼロからソリューションを構築したい場合は、エンドポイント統合ガイドで、エンドポイントをシステムに統合する際の機能やベストプラクティスを解説しています。これらのガイドは、各エンドポイントやツールのドキュメント内で「integrate」または単に「guides」として掲載されています。例えば、[Filter stream 用のルールを構築する方法](/ja/x-api/posts/filtered-stream/integrate/build-a-rule) に関する統合ガイドがあります。

<div id="troubleshoot-an-issue">
  ### 問題のトラブルシューティング
</div>

エラーが発生した場合やご不明な点がある場合は、[サポートハブ](https://developer.x.com/en/support/twitter-api)をご覧ください。トラブルシューティングのヒント、よくある質問への回答、そのほか役立つリソースをご確認いただけます。

<div id="keep-up-to-date-with-the-latest-releases">
  ### 最新リリース情報を常に把握する
</div>

私たちは、新しい X の機能を API 経由で利用できるようにするため、プラットフォームに対するアップデートを頻繁にリリースしています。加えて、少なくとも年に 1 回はメジャーバージョンを更新することを目標としており、その際には一部に互換性が失われる変更が含まれることがあります。

最新情報を受け取るには、[フォーラムの Announcements カテゴリ](https://devcommunity.x.com/c/announcements/22) を購読し、X で [@API](https://x.com/api) と [@XDevelopers](https://x.com/XDevelopers) をフォローして通知をオンにし、[Stay informed](https://developer.x.com/en/stay-informed) ページから各種リソースを確認してください。

さらに、最新バージョンの API へ移行する際のガイドとしてご利用いただける新しい [migration hub](/ja/x-api/migrate/overview) も用意しています。
