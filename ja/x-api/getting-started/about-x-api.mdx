X API は、X のデータをプログラムで取得・分析したり、X 上の会話を支えるプロダクトや機能を構築したりするために利用できます。

これまで X API は、開発者がパブリックな会話へのアクセスをスケールさせ、活用・調査できるよう、追加のアクセスレベルを提供するかたちで発展してきました。

最近、X API v2 をリリースしました。X API v2 には、最新の基盤、新機能および高度な機能、そして [Basic access](https://developer.x.com/en/portal/products/basic) への迅速なオンボーディングが含まれています。

以下の 3 つのタブでは、X API のさまざまなバージョンとアクセスレベル、v2 の新機能、そして API を使用して取得、作成、削除、調整できる X リソースについて説明します。

  <Tab title="v2 の最新情報">
    ## v2 の新機能

    X API v2 は、2012 年以降で最大の X API のアップグレードです。これにより、多数の新機能や高度な機能に加え、高速かつ無料の API アクセスが可能になります。

    v2 で利用できる主な機能には、次のようなものがあります。

    ### 新しいエンドポイント

    X API v2 に対して、まったく新しいエンドポイントのセットをリリースしました。新しいエンドポイントを含む v2 のエンドポイント一覧は、X API エンドポイントマップガイドで確認できます。

    [ X API エンドポイントマップを見る &gt;](/ja/x-api/migrate/x-api-endpoint-map)

    ### 新しく、より詳細なデータオブジェクト

    データオブジェクトを最新化し、さまざまな改善を行いました。これにより、データの探索や解析をさらに簡単に行えるようになります。

    [データ辞書を見る &gt;](/ja/x-api/fundamentals/data-dictionary)

    ### 取得したいオブジェクトとフィールドだけを取得するための新しいパラメータ

    データエンドポイントに `fields` と `expansions` のパラメータを追加し、デフォルトで返されるフィールドを超えて、関連オブジェクトやフィールドをリクエストできるようにしました。

    [fields と expansions の使い方を学ぶ &gt;](/ja/x-api/fundamentals/data-dictionary#how-to-use-fields-and-expansions)

    ### 高度なメトリクス

    ポスト、ユーザー、メディア、投票のパフォーマンスを、ペイロード内から直接、より簡単に把握できます。インプレッション、動画再生数、ユーザープロフィール、URL クリック数など、公開・非公開のメトリクスを両方リクエストでき、一部はオーガニックとプロモーションのコンテキストに分けて提供されます。

    [メトリクスの詳細を見る &gt;](/ja/x-api/fundamentals/metrics)

    ### さまざまなトピックを含むポストをフィルタリングおよび特定

    search Posts や filtered stream を使用する際に、entity 演算子と context 演算子を利用してトピックでフィルタリングできるようになりました。また、分析に役立つよう、これらのトピックをポストのペイロード内にも含めて提供しています。

    [Post annotations の詳細を見る &gt;](/ja/x-api/fundamentals/post-annotations)

    ### 返信スレッドに属するポストをフィルタリングおよび特定

    search Posts、filtered stream、Post lookup を使用する際に、会話スレッドの一部であるポストをより簡単に特定できます。さらに、Post フィールド `reply_settings` により、ポストに会話の返信設定が適用されているかどうかを判定できるようになりました。

    [会話トラッキングの詳細を見る &gt;](/ja/x-api/fundamentals/conversation-id)

    ### ほかにも多数の改善があります...

    * 高精度なスパムフィルタリング
    * 短縮 URL を完全展開し、URL 分析を容易に
    * 非推奨フィールドの削除とラベルの最新化による、よりシンプルな JSON レスポンスオブジェクト
    * ストリーミングエンドポイント向けのリカバリおよび冗長性機能
    * 検索クエリで一致した公開かつ利用可能なポストを 100% 返却
    * 接続を切断せずに変更を行うことができるストリーミングの「rules」
    * filtered stream および検索向けの、より表現力豊かなクエリ言語
    * 新しいライブラリの構築と、変更をより透明性高く追跡できる OpenAPI 仕様
    * X を利用するデベロッパー、研究者、企業、一般利用者のニーズに合わせてプラットフォームが進化する中で、新機能やエンドポイントに対する API サポートをより迅速に提供
  </Tab>

  <Tab title="X API のリソース">
    ## X API プラットフォームのリソース

    API 設計において、リソースとは、それに関連付けられたデータや他のリソースとの関係、それに対して操作を行うメソッドの集合を持つエンティティを指します。たとえば、ポストはリソースの 1 つであり、さまざまなツールを使用して作成、削除、取得できます。これには、過去のポストを検索したり、リアルタイムで取得したりすることが含まれます。

    X API では、プラットフォーム上のさまざまな種類のリソースを作成、削除、取得、更新するためのアクセスを提供しています。これには次のものが含まれます。

    | Resource            | Description                                                                                                                         |
    | :------------------ | :---------------------------------------------------------------------------------------------------------------------------------- |
    | **Posts**           | 何百万ものポストにアクセスして公開されている会話を理解したり、自分のポストを作成して会話に参加したりできます。  Current availability:  - X API v2  - Enterprise  - Standard v1.1           |
    | **Users**           | X ユーザーを管理または検索して、ネットワークを分析したり、オーディエンスを理解したり、前向きなオンライン関係を育んだりできます。  Current availability:  - X API v2  - Enterprise  - Standard v1.1 |
    | **Spaces**          | X Spaces およびその参加者を検索して、人々が興味深く関連性の高い音声会話を見つけられるように支援します。  Current availability:  - X API v2                                         |
    | **Direct Messages** | ダイレクトメッセージを送受信して、顧客からの問い合わせを振り分けたり、歓迎メッセージを送信したり、前向きな人間同士のやり取りを生み出したりできます。  Current availability:  - Standard v1.1                  |
    | **Lists**           | アカウントのリストを作成および管理して、業界の専門家や影響力のある声を追いかけたり、フォローしているアカウントを整理したりできます。  Current availability:  - X API v2  - Standard v1.1              |
    | **Trends**          | 地理的なトレンドをいち早く特定して、業界の動きを把握したり、ホットな話題を発見したり、最新の流行を先取りしたりできます。  Current availability:  - Standard v1.1                                |
    | **Media**           | メディアオブジェクトをアップロードして、クリエイティブなエネルギーを共有したり、インタラクティブな体験を作成したり、アクセシビリティツールを構築したりできます。  Current availability:  - Standard v1.1            |
    | **Places**          | 場所を検索して、近隣や世界中で何が起きているのかを把握できます。  Current availability:  - Standard v1.1                                                            |
  </Tab>
