news lookup エンドポイントを使用すると、開発者は X 上で速報として報じられているニュースやヘッドラインを取得できます。

このエンドポイントは、OAuth 1 および OAuth 2 PKCE において、app-auth と user-auth の両方の認証方式をサポートしています。

<div id="getting-started">
  ## はじめに
</div>

このエンドポイントを使用するには、[developer portal](https://developer.x.com/en/portal/dashboard) から取得した [Bearer Token](/ja/fundamentals/authentication/oauth-2-0/application-only) が必要です。

Bearer Token を取得したら、次のように News API を呼び出します。

```bash
curl 'https://api.x.com/2/news/1989418137272422538?news.fields=contexts,cluster_posts_results' --header 'Authorization: Bearer XXXXX'
```
