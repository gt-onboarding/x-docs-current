<div id="comparing-x-apis-filtered-stream-endpoints">
  ## X API の filtered stream エンドポイントの比較
</div>

v2 filtered stream エンドポイント群は、[Standard v1.1 statuses/filter](https://developer.x.com/en/docs/x-api/v1/tweets/filter-realtime/api-reference/post-statuses-filter) と [PowerTrack API](/ja/x-api/enterprise-gnip-2.0/powertrack-api) を置き換えるものです。古いバージョンの filtered stream エンドポイントを使用しているコード、アプリ、ツールがあり、最新の X API v2 エンドポイントへの移行を検討している場合は、この比較が移行の第一歩として役立ちます。 

より詳細な移行ガイドについては、以下を参照してください。

[Migrating from Standard v1.1 compared to X API v2](x-api/posts/filtered-stream#standard-v1-1-compared-to-x-api-v2)

[Migrating from PowerTrack API migration to X API v2](/ja/x-api/enterprise-gnip-2.0/powertrack-api#powertrack-operators)
