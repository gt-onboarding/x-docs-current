<div id="comparing-x-apis-filtered-stream-endpoints">
  ## X API の filtered stream エンドポイントの比較
</div>

v2 の filtered stream エンドポイント群は、[standard v1.1 statuses/filter](https://developer.x.com/en/docs/x-api/v1/tweets/filter-realtime/api-reference/post-statuses-filter) と [PowerTrack API](/ja/x-api/enterprise-gnip-2.0/powertrack-api) を置き換えるものです。既に filtered stream エンドポイントの旧バージョンを使用しているコード、アプリ、またはツールがあり、新しい X API v2 エンドポイントへの移行を検討している場合、この比較が移行を始める際の手助けとなります。 

より詳細な移行ガイドについては、次を参照してください:

[Standard v1.1 から X API v2 への移行](x-api/posts/filtered-stream#standard-v1-1-compared-to-x-api-v2)

[PowerTrack API から X API v2 への移行](/ja/x-api/enterprise-gnip-2.0/powertrack-api#powertrack-operators)
