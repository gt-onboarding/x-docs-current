このエンドポイントを使用すると、定義した条件に基づいて返信をプログラムで非表示または再表示（非表示解除）できます。X のメインエクスペリエンスと同様に、返信はメインの会話からは非表示になりますが、別ページでは引き続き表示されます。このエンドポイントを使って、ユーザーが価値がないと感じる返信を非表示にするのを支援するアプリを作成できます。

返信非表示エンドポイントは、[OAuth 1.0a User Context](/ja/resources/fundamentals/authentication#oauth-1-0a-2) または [OAuth 2.0 Authorization Code with PKCE](resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2) 認証を使用します。リクエストが成功すると、このエンドポイントは、認証されたユーザーによって開始されたポストの会話内に投稿された 1 件の返信を非表示にします。各会話で非表示にできるポストは最大 725 件です。

<Note>
  **アカウントの設定**

  これらのエンドポイントにアクセスするには、次が必要です。

  * 承認済みの [開発者アカウント](https://developer.x.com/en/portal/petition/essential/basic-info)。
  * [プロジェクト](/ja/resources/fundamentals/projects) 内にある [開発者アプリ](/ja/resources/fundamentals/developer-apps) のキーとトークンを使用して認証すること。

  X API v2 のエンドポイントへのアクセス方法の詳細は、[スタートガイド](/ja/x-api/getting-started/getting-access) を参照してください。
</Note>
