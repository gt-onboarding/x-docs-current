このエンドポイントを使用すると、指定した条件に基づいて返信をプログラムで非表示または再表示できます。通常の X の機能と同様に、返信はメインの会話からは非表示になりますが、別のページでは引き続き表示されます。このエンドポイントを使って、利用者が価値がないと感じる返信を非表示にするアプリを作成できます。

hide replies エンドポイントは、[OAuth 1.0a User Context](/ja/resources/fundamentals/authentication#oauth-1-0a-2) または [OAuth 2.0 Authorization Code with PKCE](/ja/resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2) 認証を使用します。リクエストが成功すると、このエンドポイントは、認証済みユーザーが開始したポストの会話内で公開された 1 件の返信を非表示にします。各会話ごとに最大 725 件までのポストを非表示にできます。

<Note>
  **アカウント設定**

  これらのエンドポイントにアクセスするには、以下が必要です。

  * 承認済みの[開発者アカウント](https://developer.x.com/en/portal/petition/essential/basic-info)
  * [プロジェクト](/ja/resources/fundamentals/projects) 内にある [開発者アプリ](/ja/resources/fundamentals/developer-apps) のキーとトークンを使用して認証すること

  X API v2 エンドポイントへのアクセスの取得については、[はじめにガイド](/ja/x-api/getting-started/getting-access)をご覧ください。
</Note>
