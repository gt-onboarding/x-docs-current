  ## 返信非表示エンドポイントを利用しているアプリ
</div>

ここでは、返信非表示エンドポイントを統合した開発者によって構築された注目すべきアプリを紹介します。これらのアプリは、自身の技術と X API を組み合わせることで、すべてのユーザーが公の会話をより良いものにできる有用な体験を生み出しています。

![Clarabridge](https://cdn.cms-twdigitalassets.com/content/dam/developer-twitter/hide-replies/clarabridge.jpg.twimg.1920.jpg)

**Clarabridge**

Clarabridge は返信の非表示機能を統合することで、企業が自社の X フィードを読みやすく、魅力的な状態に保てるようにします。返信は手動で非表示にできるほか、担当者は Clarabridge のテキスト分析機能を使って、不適切な表現やブランドポリシーに反する言葉を自動的に検出して非表示にすることもできます。

[Webサイトへ ▸](https://www.clarabridge.com/ "Webサイトへ ▸")

[X 上の Clarabridge](https://x.com/Clarabridge "X 上の Clarabridge")

![Perspective API テンプレートアプリ](https://cdn.cms-twdigitalassets.com/content/dam/developer-twitter/hide-replies/perspective-logo.png.twimg.1920.png)

**Perspective API テンプレートアプリ**

Perspective は、Jigsaw（Alphabet 内の一部門）によってトレーニングされた人工知能で、有害なコメントを検出します。このアプリは、ユーザーに届く返信を検出し、過去に Perspective が見てきた有害コメントにどれだけ類似しているかを示す「スコア」に基づいて、自動的に返信を非表示にします。これは X によって開発されたオープンソースのテンプレートアプリです。ユーザー向けのツールを構築するために、自由にカスタマイズしてご利用ください。

[アプリを使う ▸](https://quintessential-yoke.glitch.me "アプリを使う ▸")

[X 上の Jigsaw](https://x.com/jigsaw "X 上の Jigsaw")

![Reshuffle](data:image/svg+xml,%3Csvg xmlns=%22http://www.w3.org/2000/svg%22 viewBox=%220 0 150 150%22%3E%3C/svg%3E)

**Reshuffle**

Reshuffle は、ビジネスアプリケーションを柔軟なスクリプトで接続するためのプラットフォームです。指定したキーワードを含む返信を検出して非表示にする統合スクリプトを構築しています。試してみるには、まず「Hide replies」用に有効なアプリがあることを確認し、その後リポジトリ内の手順に従ってください。

[アプリへ ▸](https://github.com/reshufflehq/reshuffle "アプリへ ▸")

[X 上の Reshuffle](https://x.com/reshufflehq "X 上の Reshuffle")

![不要な返信を非表示](data:image/svg+xml,%3Csvg xmlns=%22http://www.w3.org/2000/svg%22 viewBox=%220 0 396 397%22%3E%3C/svg%3E)

**不要な返信を非表示**

人気アプリ QuotedReplies の開発者である Dara Oladosu は、彼が定義したいくつかの条件を満たす返信を自動的に非表示にするアプリを構築しました。これには、過去にミュートした特定のキーワードを含む返信など、攻撃的な行為を示す可能性が高いと彼が判断したものが含まれます。

[アプリへ ▸](https://hideunwantedreplies.com/ "アプリへ ▸")

[X 上の Dara Oladosu](https://x.com/dara_tobi "X 上の Dara Oladosu")

![Pandaflow](data:image/svg+xml,%3Csvg xmlns=%22http://www.w3.org/2000/svg%22 viewBox=%220 0 94 94%22%3E%3C/svg%3E)

**Pandaflow**

Pandaflow は、ワークフローを自動化するためのノーコードプラットフォームです。ユーザーが定義したカスタムフローに基づいて、プログラム的に返信を非表示にできるように、このエンドポイントを統合しています。

[アプリへ ▸](https://pandaflow.io/ "アプリへ ▸")

[X 上の Pandaflow](https://x.com/pandaflowio "X 上の Pandaflow")
