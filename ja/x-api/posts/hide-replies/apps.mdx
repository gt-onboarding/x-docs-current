  ## 返信を非表示にするエンドポイントを利用しているアプリ
</div>

こちらは、返信を非表示にするエンドポイントを組み込んで開発された注目すべきアプリの一覧です。開発者は自分たちの技術と X API を組み合わせることで、すべてのユーザーがパブリックな会話をより良いものにできる有用な体験を生み出しています。

![Clarabridge](https://cdn.cms-twdigitalassets.com/content/dam/developer-twitter/hide-replies/clarabridge.jpg.twimg.1920.jpg)

**Clarabridge**

Clarabridge は返信を非表示にする機能を統合することで、企業が自社の X フィードをクリーンで読みやすい状態に保てるようにします。返信は手動で非表示にすることもでき、エージェントは Clarabridge のテキスト分析機能を使って、不適切な表現やブランドポリシーに反する言語を自動的に検出することもできます。

[Web サイトへ ▸](https://www.clarabridge.com/ "Web サイトへ ▸")

[X 上の Clarabridge](https://x.com/Clarabridge "X 上の Clarabridge")

![Perspective API template app](https://cdn.cms-twdigitalassets.com/content/dam/developer-twitter/hide-replies/perspective-logo.png.twimg.1920.png)

**Perspective API テンプレートアプリ**

Perspective は、Jigsaw（Alphabet 内の 1 部門）によって有害なコメントを検出するためにトレーニングされた人工知能です。このアプリは、自分宛てに届く返信を検出し、Perspective が過去に見た有害なコメントとどの程度似ているかを示す「スコア」に基づいて、自動的に返信を非表示にします。これは X によって開発されたオープンソースのテンプレートアプリです。ユーザー向けツールを構築するために、自由にカスタマイズしてお使いください。

[アプリを使う ▸](https://quintessential-yoke.glitch.me "アプリを使う ▸")

[X 上の Jigsaw](https://x.com/jigsaw "X 上の Jigsaw")

![Reshuffle](data:image/svg+xml,%3Csvg xmlns=%22http://www.w3.org/2000/svg%22 viewBox=%220 0 150 150%22%3E%3C/svg%3E)

**Reshuffle**

Reshuffle は、ビジネスアプリケーションを柔軟なスクリプトで接続するためのプラットフォームです。定義したキーワードを含む返信を検出して非表示にする統合スクリプトを作成しています。試す前に、返信を非表示にする機能用に有効なアプリを持っていることを確認し、そのうえでリポジトリ内の手順に従ってください。

[アプリへ ▸](https://github.com/reshufflehq/reshuffle "アプリへ ▸")

[X 上の Reshuffle](https://x.com/reshufflehq "X 上の Reshuffle")

![Hide unwanted replies](data:image/svg+xml,%3Csvg xmlns=%22http://www.w3.org/2000/svg%22 viewBox=%220 0 396 397%22%3E%3C/svg%3E)

**Hide unwanted replies**

人気アプリ QuotedReplies の開発者である Dara Oladosu は、攻撃的な行為が表れやすいと彼が判断した条件の一部（過去にミュートした特定のキーワードを含む返信など）を満たす返信を自動的に非表示にするアプリを開発しました。

[アプリへ ▸](https://hideunwantedreplies.com/ "アプリへ ▸")

[X 上の Dara Oladosu](https://x.com/dara_tobi "X 上の Dara Oladosu")

![Pandaflow](data:image/svg+xml,%3Csvg xmlns=%22http://www.w3.org/2000/svg%22 viewBox=%220 0 94 94%22%3E%3C/svg%3E)

**Pandaflow**

Pandaflow は、ワークフローを自動化するためのノーコードプラットフォームです。このエンドポイントを統合することで、ユーザーは自分で定義したカスタムフローに基づいて、プログラム的に返信を非表示にできるようになります。

[アプリへ ▸](https://pandaflow.io/ "アプリへ ▸")

[X 上の Pandaflow](https://x.com/pandaflowio "X 上の Pandaflow")
