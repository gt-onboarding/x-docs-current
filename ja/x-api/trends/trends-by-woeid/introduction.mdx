Trends lookup エンドポイントを使用すると、where-on-earth ID (WOEID) で指定されたロケーションのトレンドを取得できます。

<Note>
  **注記**: WOEID は Yahoo によって作成されたレガシー識別子であり、現在は非推奨となっています。X API は、この数値 ID を使用して都市や国ごとのトレンドのロケーションを識別します。詳細は、以前の [ブログ記事](https://blog.x.com/engineering/en_us/a/2010/woeids-in-twitters-trends.html) または [アーカイブデータ](https://archive.org/details/geoplanet_data_7.10.0.zip0.) を参照してください。
</Note>

`tweet_count` は、利用可能な場合、多くのトレンドについて過去 24 時間分の値も返されます。

このエンドポイントは app-auth 認証をサポートしており、15 分あたり 75 リクエストのレート制限があります。

<div id="getting-started">
  ### はじめに
</div>

このエンドポイントを利用するには、[developer portal](https://developer.x.com/en/portal/dashboard) で取得した [bearer token]([https://developer.x.com(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token)](https://developer.x.com\(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token\)) が必要です。bearer token を取得したら、以下のように usage API を呼び出せます。

```
      curl 'https://api.x.com/2/trends/by/woeid/26062' --header 'Authorization: Bearer XXXXX'
```

**アカウントのセットアップ**

これらのエンドポイントにアクセスするには、以下が必要です。

* 承認済みの [開発者アカウント](https://developer.x.com/en/portal/petition/essential/basic-info)
* [プロジェクト](/ja/resources/fundamentals/projects) 内にある [開発者アプリ](/ja/resources/fundamentals/developer-apps) のキーとトークンを使用した認証

X API v2 エンドポイントへのアクセス方法の詳細については、[はじめにガイド](/ja/x-api/getting-started/getting-access) を参照してください。
