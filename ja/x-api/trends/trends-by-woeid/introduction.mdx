Trends lookup エンドポイントを使用すると、Where On Earth ID (WOEID) を使って指定したロケーションのトレンドを取得できます。

<Note>
  **注記**: WOEID は Yahoo によって作成されたレガシー識別子で、現在は非推奨です。X API では、この数値を使用して市区町村および国レベルのトレンドの場所を識別します。詳細は、過去の[ブログ記事](https://blog.x.com/engineering/en_us/a/2010/woeids-in-twitters-trends.html)や[アーカイブデータ](https://archive.org/details/geoplanet_data_7.10.0.zip0.)を参照してください。
</Note>

多くのトレンドについては、利用可能な場合、直近 24 時間分の `tweet_count` も返されます。

このエンドポイントは app-auth 認証をサポートしており、15 分あたり 75 リクエストのレート制限があります。

<div id="getting-started">
  ### はじめに
</div>

このエンドポイントを使用するには、[developer portal](https://developer.x.com/en/portal/dashboard) から取得した [bearer token]([https://developer.x.com(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token)](https://developer.x.com\(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token\)) が必要です。Bearer Token を取得したら、以下のように Usage API を呼び出すことができます。

```
      curl 'https://api.x.com/2/trends/by/woeid/26062' --header 'Authorization: Bearer XXXXX'
```

**アカウントのセットアップ**

これらのエンドポイントにアクセスするには、次のものが必要です。

* 承認済みの [開発者アカウント](https://developer.x.com/en/portal/petition/essential/basic-info)
* [プロジェクト](/ja/resources/fundamentals/projects) 内にある [開発者アプリ](/ja/resources/fundamentals/developer-apps) のキーとトークンを使用して認証すること

X API v2 エンドポイントへのアクセス方法については、[はじめにガイド](/ja/x-api/getting-started/getting-access)をご覧ください。
