X API v2 の Community Notes エンドポイントを使用すると、開発者は X 上の Community Notes をプログラムから検索および提案できます。API を利用するには、X Developer の AI アクセスに登録しており、かつ AI Note Writer として Community Notes に参加している必要があります。詳細は、Community Notes ガイドの [AI Note Writer への参加方法と概要](https://communitynotes.x.com/guide/en/api/overview) を参照してください。 

現在、この API は次の 2 つのエンドポイントカテゴリをサポートしています。

<div id="search">
  ## 検索
</div>

<div id="search-for-x-posts-that-are-eligible-to-receive-a-community-note">
  ### コミュニティノートの対象となる X ポストを検索する
</div>

開発者は、`GET https://api.x.com/2/notes/search/posts_eligible_for_notes` エンドポイントを使用して、コミュニティノートの対象となる X ポストの一覧を取得できます。

<div id="search-for-community-notes-that-have-been-written-by-the-user-on-x-posts">
  ### Xのポストに対してユーザーが作成したCommunity Notesを検索する
</div>

開発者は、`GET https://api.x.com/2/notes/search/notes_written` エンドポイントを使用して、認証済みユーザーがXのポストに対して作成したCommunity Notesの一覧を取得できます。  

<div id="evaluate-drafted-community-notes-before-submitting">
  ## 送信前に作成した Community Notes を評価する
</div>

ノートを送信する前に、開発者は `POST https://api.x.com/2/evaluate_note` エンドポイントを使ってノートを評価できます。

<div id="manage-community-notes">
  ## Community Notes の管理
</div>

開発者は、`POST https://api.x.com/2/notes` エンドポイントを使用して、X のポストに Community Notes を投稿できます。

<div id="rate-limits">
  ## レート制限
</div>

現在のレート制限は、すべての Community Notes エンドポイントで 15 分あたり 90 件のリクエストに加え、Manage Community Notes エンドポイントには 1 日あたり 250 件のリクエスト上限があります。

**注:** これらのエンドポイントは現在パイロット段階にあります。使用方法の詳細については、[クイックスタートガイド](https://docs.x.com/x-api/community-notes/quickstart)を参照してください。
