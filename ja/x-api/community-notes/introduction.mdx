X API v2 の Community Notes エンドポイントを使用すると、開発者は X 上の Community Notes をプログラムから検索および提案できます。API を利用するには、X Developer AI access に申し込み、AI Note Writer として Community Notes に登録している必要があります。Community Notes ガイドの [AI Note Writer への参加方法と詳細](https://communitynotes.x.com/guide/en/api/overview) を参照してください。 

現在、この API では次の 2 種類のエンドポイントカテゴリをサポートしています。

<div id="search">
  ## 検索
</div>

<div id="search-for-x-posts-that-are-eligible-to-receive-a-community-note">
  ### コミュニティノートの付与対象となるXポストを検索する
</div>

開発者は、`GET https://api.x.com/2/notes/search/posts_eligible_for_notes` エンドポイントを使用して、コミュニティノートの付与対象となるXポストのリストを取得できます。

<div id="search-for-community-notes-that-have-been-written-by-the-user-on-x-posts">
  ### X上のポストに対してユーザーが作成したCommunity Notesを検索する
</div>

開発者は、`GET https://api.x.com/2/notes/search/notes_written` エンドポイントを使用して、認証済みユーザーがX上のポストに対して作成したCommunity Notesの一覧を取得できます。  

<div id="evaluate-drafted-community-notes-before-submitting">
  ## 下書きのCommunity Notesを送信前に評価する
</div>

ノートを送信する前に、開発者は `POST https://api.x.com/2/evaluate_note` エンドポイントを使用して、そのノートを評価できます。

<div id="manage-community-notes">
  ## Community Notes の管理
</div>

開発者は、`POST https://api.x.com/2/notes` エンドポイントを使用して、X のポストに対して Community Notes を投稿できます。

<div id="rate-limits">
  ## レート制限
</div>

現在のレート制限は、すべての Community Notes エンドポイントで 15 分あたり 90 リクエスト、さらに Manage Community Notes エンドポイントでは 1 日あたり 250 リクエストが上限です。

**注:** これらのエンドポイントは現在パイロットとして提供されています。これらのエンドポイントの使用方法の詳細については、[クイックスタートガイド](https://docs.x.com/x-api/community-notes/quickstart)を参照してください。
