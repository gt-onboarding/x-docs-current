<div id="community-tools-and-libraries-for-v2">
  ## v2 向けコミュニティ製ツールおよびライブラリ
</div>

ここに掲載しているライブラリは、開発者コミュニティのメンバーによって作成されたものです。API 機能のカバー範囲や対応状況はライブラリごとに異なる場合があります。

独自の X API ライブラリや便利なツールを作成された方は、ぜひ [こちらからお知らせください](https://devcommunity.x.com/c/libraries-and-sdks/63)。他の方が見つけやすくなるよう、このリストに追加させていただきます。また、ご自身の README ファイルで使用できる [バージョンバッジ](https://twbadges.glitch.me/) もいくつか用意していますので、自由にご利用ください。

インスピレーションを得たい場合は、他の開発者による有用なコード例を見つけるために、GitHub の [X](https://github.com/topics/twitter?o=desc&s=updated) および [X-api-v2](https://github.com/topics/twitter-api-v2?o=desc&s=updated) トピックを閲覧・検索してみてください。

**ジャンプ先:** [C# / .NET](#csharp)、[Dart / Flutter](#dart)、[Go](#go)、[Java](#java)、[JavaScript (Node.JS) / TypeScript](#nodejs)、[Kotlin](#kotlin)、[PHP](#php)、[PowerShell](#powershell)、[Python](#python)、[R](#r)、[Ruby](#ruby)、[Rust](#rust)、[Swift](#swift)

<div id="additional-official-resources">
  ## 追加の公式リソース
</div>

以下のツールも、X API を扱う際に役立ちます。\
さらに多くのコード例が必要な場合は、[GitHub](https://github.com/xdevplatform) や [Glitch](https://glitch.com/@twitter) でもサンプルを公開しています。

| Tool / Library        | Description                                                                                                                                                                               |
| :-------------------- | :---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| twemoji               | Twitter の無償オープンソース絵文字セットで、クロスプラットフォーム対応のための JavaScript ライブラリを含みます。                                                                           |
| twitter-text          | ポスト本文の構文解析およびトークン化を標準化するためのライブラリ群です。Java、JavaScript、Objective-C、Ruby 向けに利用できます。 [ポストの文字数カウントの詳細はこちら。](#)              |
| OpenAPI specification | この仕様を使用して、[Postman](#) や [Insomnia](#) などのツールを使って **v2** API を操作できます。                                                                                         |
| twurl                 | OAuth 認証機能を含む Twitter API 用のコマンドラインツール (CLI) です。Ruby ランタイムが必要です。                                                                                          |
| Postman collection    | X API のエンドポイントをテストや開発目的で扱う際には、Postman コレクションを活用してください。                                                                                            |

<div id="community-tools-and-libraries">
  ## コミュニティ製ツールとライブラリ
</div>

以下は、複数のプログラミング言語やプラットフォームにわたって X API を扱う、コミュニティによってサポートされている多数のライブラリの一部です。これらのリソースは、すべてが Twitter チームによってテストされているわけではない点に注意してください。

ここに掲載されているライブラリは、特に記載がない限り Standard API **v1.1** のほとんどの機能を実装しているはずです。詳細や追加サポートについては、各ライブラリの作者に確認してください。\
X API **v2** をサポートするライブラリを作成された場合は、このページへの掲載を検討するために、[コミュニティフォーラム](https://devcommunity.x.com/c/libraries-and-sdks/63) を通じてお知らせください。これらの掲載内容の変更を報告する際にもフォーラムを利用できます。

お気に入りのプログラミング言語向けのライブラリやツールが見つからない場合は、[フィードバックプラットフォーム](https://twitterdevfeedback.uservoice.com/) からお知らせください。そこでは、アイデアに投票したり、新しいものを作成して投稿するためのインスピレーションを得ることもできます。
