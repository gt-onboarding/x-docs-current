<div id="community-tools-and-libraries-for-v2">
  ## v2 向けコミュニティ製ツールとライブラリ
</div>

ここに掲載しているライブラリは、開発者コミュニティのメンバーによって作成されたものです。API のカバレッジ状況は、それぞれ異なる場合があります。

独自の X API ライブラリや便利なツールを作成した場合は、ぜひ [お知らせください](https://devcommunity.x.com/c/libraries-and-sdks/63)。他の開発者が見つけやすくなるよう、このリストに追加します。README ファイルなどで利用できる [バージョンバッジ](https://twbadges.glitch.me/) も用意していますので、自由にご利用ください。

インスピレーションを探している場合は、GitHub 上の [X](https://github.com/topics/twitter?o=desc&s=updated) および [X-api-v2](https://github.com/topics/twitter-api-v2?o=desc&s=updated) トピックを閲覧・検索して、他の開発者による有用なコード例を探してみてください。

**ジャンプ先:** [C# / .NET](#csharp)、[Dart / Flutter](#dart)、[Go](#go)、[Java](#java)、[JavaScript (Node.JS) / TypeScript](#nodejs)、[Kotlin](#kotlin)、[PHP](#php)、[PowerShell](#powershell)、[Python](#python)、[R](#r)、[Ruby](#ruby)、[Rust](#rust)、[Swift](#swift)

  ## 追加の公式リソース

以下のツールも、X API を扱う際に役立ちます。\
さらにコード例をお探しですか？ [GitHub](https://github.com/xdevplatform) や [Glitch](https://glitch.com/@twitter) で、サンプルコードをご覧いただけます。

| Tool / Library        | Description                                                                                                                                                                               |
| :-------------------- | :---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| twemoji               | Twitter の無料でオープンソースの絵文字セットで、クロスプラットフォーム対応のための JavaScript ライブラリも含まれます。                                                                    |
| twitter-text          | ポスト本文の構文解析とトークナイズを標準化するためのライブラリ群です。Java、JavaScript、Objective-C、Ruby で利用可能です。[ポストの文字数カウントの詳細はこちら。](#)                        |

<div id="community-tools-and-libraries">
  ## コミュニティ製ツールとライブラリ
</div>

ここでは、さまざまなプログラミング言語やプラットフォームで X API を扱う、多数のコミュニティ製ライブラリの一部を紹介します。これらのリソースは、必ずしもすべて Twitter チームによってテストされているとは限らない点に注意してください。

ここに掲載しているライブラリは、特記がない限り Standard API **v1.1** の機能の大部分を実装しているはずです。詳細や追加サポートについては、各ライブラリの作者に確認してください。\
X API **v2** をサポートするライブラリを作成された場合は、[コミュニティフォーラム](https://devcommunity.x.com/c/libraries-and-sdks/63)からお知らせください。このページへの追加を検討します。また、これらの掲載内容に変更があった場合の報告にもフォーラムをご利用いただけます。

お好みのプログラミング言語向けのライブラリやツールが見つからない場合は、[フィードバックプラットフォーム](https://twitterdevfeedback.uservoice.com/)からお知らせください。そこでアイデアに投票したり、新しいものを構築して投稿するためのインスピレーションを得たりすることもできます。
