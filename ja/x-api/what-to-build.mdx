<div id="introduction">
  ## はじめに
</div>

私たちの開発者コミュニティは、X の社内だけでは埋めきれないギャップを補い、問題を解決し、X 上で新たなイノベーションを生み出すことのできる、独自のスキル、経験、視点を持っています。X をより良く、より健全な場にし、公共の対話を広げていくツールやプロダクトの構築に、ぜひ取り組んでください。特に、次のカテゴリでのイノベーションを歓迎します。

* [健全性と安全性のために会話をモデレートする](#moderate)
* [創造と自己表現を可能にする](#enable)
* [起きていることを計測・分析する](#measure)
* [コミュニティでの体験を向上させる](#improve)
* [コンテンツをキュレーションし推薦する](#curate)
* [より大きな公益に貢献する](#impact)

<div id="moderate-conversations-for-health-and-safety">
  ### 健全性と安全性のために会話を適切に管理する
</div>

すべての人に、X 上の会話に安心して、快適に、そして前向きな気持ちで参加してもらいたいと考えています。私たちは、ポストへの返信設定やブロック、ミュートなど、人々が自分の体験をより主体的に管理できるようにするためのツールの提供に力を入れてきました。しかし、取り組むべきことは常に残っており、プラットフォームの健全性と安全性を高める、補完的・拡張的なソリューションを構築していただきたいと考えています。以下のリソースを参考に、開発を始めてください。

<div id="enable-creation-and-personal-expression">
  ### 創造と自己表現を可能にする
</div>

X は、人々が自分の視点やアイデア、情熱を共有するために集まる場所です。個人とブランド、創業者とアスリート、著名人と CEO が交差する場でもあります。開発者には、X 上の人々がリーチを拡大し、新たな方法で自己表現し、同じ志を持つコミュニティとつながれるようにするクリエイティブなソリューションを構築する機会があります。開発者自身もクリエイターでありイノベーターであるため、便利なボットやゲーミフィケーション、クロスポストコンテンツなど、人々が X を体験する方法を広げるものを構築することもできます。以下はいくつかのアイデアであり、皆さんのイノベーションを喚起するための種となるものです。

関連するエンドポイント:

<div id="measure-and-analyze-whats-happening">
  ### 「いま起きていること」を計測し、分析する
</div>

多くの人がXを利用するのは、新規・既存のオーディエンスにリーチし、自分の声を拡げ、影響を与える機会があるためです。開発者は、こうしたクリエイター、アーティスト、ビジネス、地域コミュニティの支援者が、自身のコンテンツの影響を計測・分析し、理解できるよう支援できます。さらに開発者は、トレンドやオーディエンスなどに関する公開の会話からインサイトを引き出せる新たなソリューションを構築できます。ここでは、そのための第一歩として役立つリソースをいくつか紹介します。

<div id="improve-community-experiences">
  ### コミュニティ体験の向上
</div>

X 上で新たなコミュニティが成長し集まっていく中で、私たちは開発者のみなさんの専門知識を活用し、これらのコミュニティ固有のニーズに応えてほしいと考えています。X は世界中の人々が会話する場であり、言語、タイムゾーン、文化的な違い、その他のローカルなニーズにかかわらず、誰もが参加できるようにしたいと考えています。さらに、金融やゲームといったトピックを中心に成長するコミュニティには、それぞれの独自の体験やニーズに合わせて構築されたツールが用意されることを望んでいます。
