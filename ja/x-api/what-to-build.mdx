<div id="introduction">
  ## はじめに
</div>

私たちの開発者コミュニティは、私たちXだけでは埋めきれないギャップを補い、問題を解決し、X上で新たなイノベーションの芽を生み出せる、独自のスキルや経験、視点を持っています。Xをより良く、より健全な場にし、公共の会話を広げていくツールやプロダクトを構築していただくことを歓迎します。特に、次のカテゴリーにおけるイノベーションを期待しています。

* [健全性と安全のための会話のモデレーション](#moderate)
* [創造と個人の表現を可能にする](#enable)
* [起きていることを測定・分析する](#measure)
* [コミュニティ体験を向上させる](#improve)
* [コンテンツをキュレーションし、推薦する](#curate)
* [より大きな公益に貢献する](#impact)

<div id="moderate-conversations-for-health-and-safety">
  ### 健全性と安全性のために会話をモデレートする
</div>

X 上の会話に、誰もが安心・安全に、そして前向きな気持ちで参加できるようにしたいと考えています。私たちは、ユーザーが自分の体験をよりコントロールできるようにするため、ポストへの返信設定、ブロック、ミュートといったツールの提供に力を入れてきました。しかし、常にさらなる取り組みの余地があり、プラットフォームの健全性と安全性の向上に役立つ補完的・付加的なソリューションを、ぜひ皆さんにも構築していただきたいと考えています。以下では、取り組みを始めるうえで役立つリソースを紹介します。

関連するエンドポイント:

* [Follows](/ja/x-api/users/follows/introduction)
* [Blocks](/ja/x-api/users/blocks/introduction)
* [Mutes](/ja/x-api/users/mutes/introduction)
* [Hide replies](/ja/x-api/posts/hide-replies/introduction)
* [Manage Posts](/ja/x-api/posts/manage-tweets/introduction) (ポストへの返信設定)

<div id="enable-creation-and-personal-expression">
  ### 創造と自己表現を可能にする
</div>

X は、人々が自分の視点やアイデア、情熱を共有するために集まる場です。個人とブランド、創業者とアスリート、著名人と CEO が交わる場所でもあります。開発者には、X 上の人々がリーチを広げ、新たな形で自己表現し、志を同じくするコミュニティとつながれるようにする、創造的なソリューションを構築できる機会があります。自らもクリエイターでありイノベーターである開発者は、役立つボットやゲーミフィケーション、クロスポストされたコンテンツなど、人々が X を体験する方法を広げるしくみを構築することもできます。以下は、イノベーションのヒントとなるいくつかのアイデアです。

Relevant endpoints:

* [Manage Posts](/ja/x-api/posts/manage-tweets/introduction)
* [Spaces](/ja/x-api/spaces/introduction)
* [Retweets](/ja/x-api/posts/retweets/introduction)

<div id="measure-and-analyze-whats-happening">
  ### 「いま起きていること」を測定・分析する
</div>

多くの人がXを利用するのは、新規または既存のオーディエンスにリーチしたり、自分の声を広げたり、影響力を発揮したりできるからです。開発者は、こうしたクリエイター、アーティスト、ビジネス、地域のアドボケイトが、自身のコンテンツのインパクトを測定・分析・理解できるよう支援できます。さらに開発者は、トレンドやオーディエンスなど、公開の会話からインサイトを得るための新しいソリューションを構築することもできます。ここから始めるための参考リソースをいくつか用意しました。

<div id="improve-community-experiences">
  ### コミュニティ体験の向上
</div>

新しいコミュニティがX上で成長し集う中で、私たちは開発者のみなさんの専門知識を活かし、これらのコミュニティ特有のニーズに応えられるようにしていきたいと考えています。Xは、世界中の人々が会話を交わす場であり、言語、タイムゾーン、文化的な違い、その他の地域特有のニーズに関わらず、誰もが参加できるようにしたいと考えています。さらに、金融やゲームといったトピックを中心に成長するコミュニティには、それぞれの固有の体験やニーズに合わせて設計されたツールを利用してもらえるようにしたいと考えています。
