<div id="overview">
  ## 概要
</div>

毎日数千人の開発者が X API にリクエストを送信しています。こうした大量のリクエストを管理するため、送信できるリクエスト数には上限が設けられています。これらの上限により、開発者コミュニティが依存している、信頼性が高くスケーラブルな API を提供し続けることができます。 

許可されるリクエストの最大数は、一定の時間間隔、つまり特定の期間（ウィンドウ）に基づいて決まります。あるエンドポイントに 900 requests/15-minutes のレート制限が設定されている場合、任意の 15 分間のウィンドウにおいて最大 900 件までのリクエストが許可されます。
