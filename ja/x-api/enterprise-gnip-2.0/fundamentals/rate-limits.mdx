<div id="overview">
  ## 概要
</div>

毎日、何千人もの開発者が X API へリクエストを送信しています。この膨大な数のリクエストを管理するため、送信できるリクエスト数には上限が設けられています。これらの上限によって、開発者コミュニティが利用する信頼性が高くスケーラブルな API を提供し続けることができます。 

許可されるリクエスト数の最大値は、一定の時間間隔、つまり特定の期間や時間ウィンドウに基づいて決まります。あるエンドポイントのレート制限が 900 requests/15-minutes の場合、任意の 15 分間のウィンドウ内で最大 900 件のリクエストが許可されます。
