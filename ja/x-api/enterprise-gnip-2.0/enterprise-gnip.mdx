Enterprise API は、X のデータに大きく依存するお客様に、最高レベルのアクセスと信頼性を提供します。Premium を超える規模までスケールし、より高い信頼性のあるアクセスやカスタマイズされたパッケージ、長期契約が必要な場合に最適です。Enterprise API へのアクセスには、専任のアカウントマネージャーと個別の技術サポートが含まれます。

Basic と Pro のソリューションだけではニーズを満たせないと判断された場合は、次のリンクから Enterprise 製品へのアクセス申請を行うことができます。

  アクセスを申請する
</Button>

<div id="available-endpoints">
  ## 利用可能なエンドポイント
</div>

| リアルタイム | [PowerTrack API](/ja/x-api/enterprise-gnip-2.0/powertrack-api) | 公開ポストをリアルタイムに完全にカバーし、幅広いオペレーターとルールを使ってフィルタリングできます。 |
| :--- | :--- | :--- |
| [Decahose API](/ja/x-api/enterprise-gnip-2.0/fundamentals/decahose-api) | すべての公開ポストの 10% をランダムにサンプリングしたストリームと、その「いいね」を取得します。 |
| :--- | :--- | :--- |
| [Account Activity API](/ja/x-api/enterprise-gnip-2.0/fundamentals/account-activity) | ユーザーをサブスクライブし、ポスト、ダイレクトメッセージ、「いいね」、フォローなど、さまざまなアクティビティの通知を受信します。 |
| :--- | :--- | :--- |
| 履歴 | [Search API](/ja/x-api/enterprise-gnip-2.0/fundamentals/search-api) | 直近 30 日分、または公開ポストのフルアーカイブを、高度なフィルタリングツールを使って検索できます。 |
| [Historical PowerTrack API](/ja/x-api/enterprise-gnip-2.0/powertrack-api) | 公開ポストのフルアーカイブを、コスト効率の高いバッチアクセス方式でフィルタリングします。 |
| インサイト | [Engagement API](/ja/x-api/enterprise-gnip-2.0/fundamentals/engagement-api) | 公開ポストのインプレッション数とエンゲージメント数を計測します。 |
| コンプライアンス | [Compliance Firehose API](/ja/x-api/enterprise-gnip-2.0/fundamentals/firehouse) | すべてのコンプライアンスイベントをストリーミングします。 |
| 使用状況 | [Usage API](/ja/x-api/enterprise-gnip-2.0/fundamentals/usage) | Enterprise 製品全体におけるアクティビティ消費量へプログラムからアクセスできます。 |

<div id="important-concepts">
  ## 重要な概念
</div>

<div id="gnip-console">
  ### Gnip Console
</div>

[Gnip Console](/ja/x-api/enterprise-gnip-2.0/fundamentals/overview) から、Enterprise 製品およびアカウントへのアクセスを管理できます。 

<div id="enrichments">
  ### エンリッチメント
</div>

有料プランを利用している開発者向けに、Premium API のレスポンスペイロードにメタデータを追加しました。展開済み URL、投票（poll）データ、プロフィールの位置情報についての詳細は、[エンリッチメントのセクション](/ja/x-api/enterprise-gnip-2.0/fundamentals/data-enrichments)をご覧ください。
 

<div id="advanced-filtering-tools">
  ### 高度なフィルタリングツール
</div>

Premium Search API では、標準の検索エンドポイントでは利用できない高度な検索演算子を利用できます。これらの演算子の多くは無料の sandbox ユーザーでも利用できますが、有料ティアをご利用いただくと、より強力で充実した演算子をより幅広く使用できます。詳しくは、[ルールとフィルターのセクション](/ja/x-api/enterprise-gnip-2.0/fundamentals/rules-filtering#getting-started-with-enterprise-rules-and-queries)をご覧ください。

<div id="explore-x-api-v2">
  ## X API v2 を確認する
</div>

v2 で現在利用できる機能に興味がある方は、ドキュメントで新機能を確認し、移行ハブで最終的に X API v2 へ移行する際に役立つ各種リソースをご覧ください。新しい v2 エンドポイントセクションに掲載されている各エンドポイントごとの移行ガイドにもアクセスできます。

[詳細はこちら](/ja/x-api/migrate/overview)

サポートが必要ですか？

サポートセクションを確認するか、アカウントマネージャーに連絡して、サポートチームへのお問い合わせ方法をご確認ください。
