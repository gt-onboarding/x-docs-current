Enterprise API は、X のデータを継続的に活用するお客様に、最高レベルのアクセスと信頼性を提供します。Premium の範囲を超えてスケールし、より安定したアクセスやカスタマイズされたパッケージ、長期契約が必要になった場合に最適です。Enterprise API にアクセスすると、専任のアカウントマネージャーと個別の技術サポートが提供されます。

Basic や Pro のソリューションではニーズを満たせない場合は、次のリンクから Enterprise 向け製品へのアクセスを申請できます。

  アクセスを申請
</Button>

<div id="available-endpoints">
  ## 利用可能なエンドポイント
</div>

| リアルタイム | [PowerTrack API](/ja/x-api/enterprise-gnip-2.0/powertrack-api) | 公開ポストをリアルタイムで網羅的に取得し、幅広いオペレーターやルールでフィルタリングできます。 |
| :--- | :--- | :--- |
| [Decahose API](/ja/x-api/enterprise-gnip-2.0/fundamentals/decahose-api) | すべての公開ポストの 10% をランダムにサンプリングしたポストと、それらへの「いいね」をストリーミング配信します。 |
| :--- | :--- | :--- |
| [Account Activity API](/ja/x-api/enterprise-gnip-2.0/fundamentals/account-activity) | ユーザーをサブスクライブし、ポスト、ダイレクトメッセージ、いいね、フォローなど、さまざまなアクティビティを受信します。 |
| :--- | :--- | :--- |
| 履歴 | [Search API](/ja/x-api/enterprise-gnip-2.0/fundamentals/search-api) | 高度なフィルタリングツールを使って、直近 30 日分、または公開ポストのフルアーカイブを検索します。 |
| [Historical PowerTrack API](/ja/x-api/enterprise-gnip-2.0/powertrack-api) | コスト効率の高いバッチアクセス方式で、公開ポストのフルアーカイブをフィルタリングします。 |
| インサイト | [Engagement API](/ja/x-api/enterprise-gnip-2.0/fundamentals/engagement-api) | 公開ポストのインプレッション数およびエンゲージメント数を測定します。 |
| コンプライアンス | [Compliance Firehose API](/ja/x-api/enterprise-gnip-2.0/fundamentals/firehouse) | すべてのコンプライアンスイベントをストリーミング配信します。 |
| 利用状況 | [Usage API](/ja/x-api/enterprise-gnip-2.0/fundamentals/usage) | Enterprise 製品全体の利用状況にプログラムからアクセスできます。 |

<div id="important-concepts">
  ## 重要な概念
</div>

<div id="gnip-console">
  ### Gnip Console
</div>

[Gnip Console](/ja/x-api/enterprise-gnip-2.0/fundamentals/overview) を使用して、Enterprise 製品やアカウントへのアクセスを管理できます。 

<div id="enrichments">
  ### Enrichments（拡張情報）
</div>

有料ティアをご利用の開発者向けに、Premium API のレスポンスペイロードにメタデータを追加しました。展開済み URL、投票データ、プロフィールの位置情報について詳しくは、[Enrichments セクション](/ja/x-api/enterprise-gnip-2.0/fundamentals/data-enrichments)をご覧ください。
 

<div id="advanced-filtering-tools">
  ### 高度なフィルタリングツール
</div>

Premium Search API では、標準の検索エンドポイントでは利用できない高度なオペレーターを使用できます。これらのオペレーターの多くはサンドボックス環境の無料ユーザーでも利用できますが、有料プランを利用すると、より充実したオペレーターセットにアクセスできます。詳しくは、[ルールとフィルターのセクション](/ja/x-api/enterprise-gnip-2.0/fundamentals/rules-filtering#getting-started-with-enterprise-rules-and-queries)をご覧ください。

<div id="explore-x-api-v2">
  ## X API v2 を探索する
</div>

v2 で現在利用可能な機能を詳しく知りたい場合は、ドキュメントをご覧ください。新機能の確認に加えて、最終的に X API v2 へ移行する際に役立つ各種リソースをまとめた移行ハブも参照できます。新しい v2 エンドポイントセクションに掲載されている各エンドポイントの移行ガイドにもアクセスできます。

[詳細を見る](/ja/x-api/migrate/overview)
