X API v2 を使用すると、X 上のグローバルな会話 ― ポスト、ユーザー、スペース、DM、リスト、トレンド、メディア など ― にプログラムによるアクセスが可能です。強力で最新のエンドポイントを使って、アプリを構築し、データを分析し、新しい体験を創出しましょう。

***

<div id="key-features">
  ## 主要な機能
</div>

<CardGroup cols="3">
  <Card title="豊富なデータオブジェクト" icon="database" href="/ja/x-api/fundamentals/data-dictionary">
    ポスト、ユーザー、メディアなどに関する詳細かつ構造化されたデータに、包括的なフィールド選択に対応してアクセスできます。
  </Card>

  <Card title="高度なメトリクス" icon="chart-pie" href="/ja/x-api/fundamentals/metrics">
    エンゲージメント分析、リアルタイムのメトリクス、およびデータに関する詳細なインサイトを取得できます。
  </Card>

  <Card title="リアルタイムストリーム" icon="bolt" href="/ja/x-api/posts/filtered-stream/introduction">
    ライブ更新とリアルタイムなデータ処理のために、フィルタリング済みまたはサンプリング済みのストリームを購読できます。
  </Card>
</CardGroup>
