X API v2 を使用すると、X のグローバルな会話――ポスト、ユーザー、スペース、DM、リスト、トレンド、メディアなど――にプログラムを通じてアクセスできます。強力で最新のエンドポイントを活用して、アプリを構築し、データを分析し、新たな体験を創出できます。

***

<div id="key-features">
  ## 主な機能
</div>

<CardGroup cols="3">
  <Card title="リッチなデータオブジェクト" icon="database" href="/ja/x-api/fundamentals/data-dictionary">
    ポスト、ユーザー、メディアなどの詳細かつ構造化されたデータに、包括的なフィールド選択とともにアクセスできます。
  </Card>

  <Card title="高度なメトリクス" icon="chart-pie" href="/ja/x-api/fundamentals/metrics">
    エンゲージメント分析、リアルタイムメトリクス、データに関する詳細なインサイトを取得できます。
  </Card>

  <Card title="リアルタイムストリーム" icon="bolt" href="/ja/x-api/posts/filtered-stream/introduction">
    絞り込みまたはサンプリングされたストリームを購読して、ライブ更新とリアルタイムなデータ処理を行うことができます。
  </Card>
</CardGroup>
