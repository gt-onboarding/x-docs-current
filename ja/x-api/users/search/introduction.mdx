Pro 以上のアクセスレベルを持つ開発者が利用できます

Users Search エンドポイントは、X 上の公開ユーザーアカウントに対して、関連度にもとづくシンプルな検索インターフェースを提供します。関心のあるトピック、氏名、企業名、所在地、その他の条件でクエリを試してみてください。

このエンドポイントは現在、[User Auth](/ja/resources/fundamentals/authentication) をサポートしています。デフォルトでは 1 回のリクエストにつき 100 件のユーザーを取得し、最大 1,000 件のユーザーまで指定できます。24 時間あたり 10,000 リクエストという上限があります（これとは別に、15 分あたり 300 リクエストという制限もあります）。

<Note>
  **アカウントのセットアップ**

  これらのエンドポイントにアクセスするには、次が必要です。

  * 承認済みの[開発者アカウント](https://developer.x.com/en/portal/petition/essential/basic-info)。
  * [プロジェクト](/ja/resources/fundamentals/projects) 内にある [開発者アプリ](/ja/resources/fundamentals/developer-apps) のキーとトークンを使用して認証すること。 

  X API v2 エンドポイントへのアクセス方法については、[入門ガイド](/ja/x-api/getting-started/getting-access) を参照してください。
</Note>
