Pro アクセス以上をお持ちの開発者が利用できます。

Users Search エンドポイントは、X 上の公開ユーザーアカウントに対して、関連度に基づくシンプルな検索インターフェースを提供します。トピックの興味関心、氏名、会社名、所在地など、さまざまな条件でクエリを試すことができます。

このエンドポイントは現在、[User Auth](/ja/resources/fundamentals/authentication) をサポートしています。デフォルトでは 1 リクエストあたり 100 ユーザーが返され、最大 1,000 ユーザーまで指定できます。24 時間あたり 10,000 リクエストという上限があります（これとは別に、15 分あたり 300 リクエストの制限もあります）。

<Note>
  **アカウントの設定**

  これらのエンドポイントにアクセスするには、次が必要です。

  * 承認済みの [開発者アカウント](https://developer.x.com/en/portal/petition/essential/basic-info)。
  * [Project](/ja/resources/fundamentals/projects) 内にある [developer App](/ja/resources/fundamentals/developer-apps) から取得したキーとトークンを使用して認証すること。 

  X API v2 エンドポイントへのアクセス方法の詳細は、[はじめにガイド](/ja/x-api/getting-started/getting-access)をご覧ください。
</Note>
