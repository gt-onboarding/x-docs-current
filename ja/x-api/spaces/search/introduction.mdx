このエンドポイントでは、Spaces のタイトルで検索し、ステータスで結果をフィルタリングできます。タイトルに含まれるキーワード、メンションされているユーザー、ハッシュタグに基づいて、ライブ中または今後開催予定の Spaces を見つけるのに役立ちます。

このエンドポイントは、1 つ以上のキーワードをクエリとして受け取ります。結果は、Space のステータス（ライブまたは予約済み）でフィルタリングできます。デフォルトでは、指定したクエリに一致するライブおよび予約済みの両方の Spaces が返されます。

<Note>
  **アカウント設定**

  これらのエンドポイントにアクセスするには、次のものが必要です。

  * 承認済みの [開発者アカウント](https://developer.x.com/en/portal/petition/essential/basic-info)。
  * [プロジェクト](/ja/resources/fundamentals/projects) 内にある [開発者アプリ](/ja/resources/fundamentals/developer-apps) のキーとトークンを使用して認証すること。 

  X API v2 のエンドポイントへのアクセス方法については、[はじめに](/ja/x-api/getting-started/getting-access)を参照してください。
</Note>
