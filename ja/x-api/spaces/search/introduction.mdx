このエンドポイントを使用すると、タイトルを指定して Space を検索し、ステータスで結果を絞り込むことができます。タイトルに含まれるキーワード、メンションされているユーザー、またはハッシュタグに基づいて、配信中または今後予定されている Space を見つけるのに役立ちます。

このエンドポイントは、1つ以上のキーワードをクエリとして受け取ります。結果は Space のステータス（ライブまたはスケジュール済み）でフィルタリングできます。既定では、リクエストは指定したクエリに一致するライブおよびスケジュール済みの Space の両方を返します。

<Note>
  **アカウントのセットアップ**

  これらのエンドポイントにアクセスするには、以下が必要です。

  * 承認済みの [開発者アカウント](https://developer.x.com/en/portal/petition/essential/basic-info)。
  * [プロジェクト](/ja/resources/fundamentals/projects) 内にある [開発者アプリ](/ja/resources/fundamentals/developer-apps) のキーとトークンを使用して認証すること。 

  X API v2 エンドポイントへのアクセス方法の詳細は、 [はじめにガイド](/ja/x-api/getting-started/getting-access) を参照してください。
</Note>
