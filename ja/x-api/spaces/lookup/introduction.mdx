id でのルックアップ用エンドポイントは、ユーザーに Spaces 情報を表示するエクスペリエンスを構築したり、1 つ以上の Space の詳細を取得したりするために利用できるデータを返します。1 回のリクエストで 1 つ以上の Space をルックアップできます。

Spaces は作成者の id を指定してルックアップすることもできます。このエンドポイントは、リスト内のユーザーがホストする Spaces をルックアップしたり、複数の常連ホストが現在ライブか、または今後の Space をスケジュールしているかどうかを判定したりするのに有用です。1 回のリクエストで最大 100 人のユーザーまでルックアップできます。

**アカウントのセットアップ**

これらのエンドポイントにアクセスするには、次のものが必要です。

* 承認済みの [開発者アカウント](https://developer.x.com/en/portal/petition/essential/basic-info)。
* [プロジェクト](/ja/resources/fundamentals/projects) 内にある [開発者アプリ](/ja/resources/fundamentals/developer-apps) のキーとトークンを使用して認証すること。

X API v2 エンドポイントへのアクセスの取得方法については、[はじめにガイド](/ja/x-api/getting-started/getting-access)をご覧ください。
