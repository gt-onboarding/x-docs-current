ID によるルックアップ用エンドポイントでは、ユーザーに Spaces の情報を表示するエクスペリエンスを構築したり、1 つ以上の Spaces の詳細を取得したりするために利用できるデータが返されます。1 回のリクエストで 1 つ以上の Spaces をルックアップできます。

また、作成者 ID で Spaces をルックアップすることもできます。このエンドポイントは、リスト内のユーザーの Spaces を取得したり、複数の定期的なホストが現在ライブか、今後の Space をスケジュールしているかを確認したりするのに便利です。1 回のリクエストで最大 100 ユーザーまでルックアップできます。

**アカウントの設定**

これらのエンドポイントにアクセスするには、以下が必要です。

* 承認済みの [開発者アカウント](https://developer.x.com/en/portal/petition/essential/basic-info)。
* [プロジェクト](/ja/resources/fundamentals/projects) 内にある [開発者アプリ](/ja/resources/fundamentals/developer-apps) のキーとトークンを使用して認証を行うこと。 

X API v2 エンドポイントへのアクセス方法の詳細については、[はじめにガイド](/ja/x-api/getting-started/getting-access) を参照してください。
