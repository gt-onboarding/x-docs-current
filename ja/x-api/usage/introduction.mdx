X API v2 の Usage API を使用すると、開発者は自分のプロジェクトの利用状況をプログラム経由で取得できます。このエンドポイントを使うことで、特定の請求サイクルにおいて取得したポスト数を把握・監視できます。

開発者は、この GET エンドポイントを使用して、直近最大 90 日間のプロジェクトの日次利用状況を取得できます。利用状況は、プロジェクトに紐付いているクライアントアプリごとに集計することもできます。

この GET エンドポイントには、15 分あたり 50 リクエストのアプリのレート制限があります。

<div id="getting-started">
  ### はじめに
</div>

このエンドポイントを利用するには、[Bearer Token](https://developer.x.com/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token) を [developer portal](https://developer.x.com/en/portal/dashboard) から取得する必要があります。Bearer Token を取得したら、以下のように Usage API を呼び出すことができます。

```
      curl 'https://api.x.com/2/usage/tweets' --header 'Authorization: Bearer XXXXX'
```

リクエストが成功すると、以下のような JSON レスポンスが表示されるはずです。

```
{
    "data": {
        "cap_reset_day": 10,
        "project_cap": "5000000000",
        "project_id": "1369785403853424",
        "project_usage": "43435"
    }
}
```

<Note>
  **アカウントのセットアップ**

  これらのエンドポイントにアクセスするには、以下が必要です。

  * 承認済みの[開発者アカウント](https://developer.x.com/en/portal/petition/essential/basic-info)
  * [プロジェクト](/ja/resources/fundamentals/projects)内にある[開発者アプリ](/ja/resources/fundamentals/developer-apps)のキーとトークンを使用した認証

  X API v2 のエンドポイントへのアクセス方法の詳細は、[はじめにガイド](/ja/x-api/getting-started/getting-access)を参照してください。
</Note>
