X API v2 の Usage API を使用すると、開発者はプロジェクトの利用状況をプログラム経由で取得できます。このエンドポイントを使うことで、特定の課金サイクルにおいて取得したポスト数を把握・監視できます。

開発者は GET エンドポイントを使用して、最大で直近 90 日分のプロジェクトの日次利用状況を取得できます。利用状況は、プロジェクトに接続されているクライアントアプリごとに集計することもできます。

この GET エンドポイントには、アプリごとに 15 分あたり 50 リクエストのレート制限があります。

<div id="getting-started">
  ### はじめに
</div>

このエンドポイントを使用するには、[developer portal](https://developer.x.com/en/portal/dashboard) から取得した [Bearer Token](https://developer.x.com/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token) が必要です。Bearer Token を取得したら、以下のように usage API を呼び出せます。

```
      curl 'https://api.x.com/2/usage/tweets' --header 'Authorization: Bearer XXXXX'
```

リクエストが成功すると、次のような JSON レスポンスが表示されます。

```
{
    "data": {
        "cap_reset_day": 10,
        "project_cap": "5000000000",
        "project_id": "1369785403853424",
        "project_usage": "43435"
    }
}
```

<Note>
  **アカウントのセットアップ**

  これらのエンドポイントにアクセスするには、次のものが必要です。

  * 承認済みの [developer account](https://developer.x.com/en/portal/petition/essential/basic-info)
  * [Project](/ja/resources/fundamentals/projects) 内にある [developer App](/ja/resources/fundamentals/developer-apps) のキーとトークンを使用した認証

  X API v2 のエンドポイントへのアクセス方法の詳細については、[はじめにガイド](/ja/x-api/getting-started/getting-access)をご覧ください。
</Note>
