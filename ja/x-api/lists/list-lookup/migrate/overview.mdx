  ## X API の List 参照エンドポイントの比較
</div>

v2 の List 参照エンドポイントグループは、標準的な v1.1 の [GET lists/show](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/get-lists-show) および [GET lists/ownership](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/get-lists-ownerships) エンドポイントに置き換わります。これらのバージョンの List 参照エンドポイントのいずれかを利用しているコード、アプリ、ツールがあり、より新しい X API v2 エンドポイントへの移行を検討している場合は、このガイド一式が役立ちます。

次の表では、標準的な v1.1 と X API v2 の List エンドポイントを比較します。

**id での List 参照**

|     |     |     |
| :--- | :--- | :--- |
| 説明 | Standard v1.1 | X API v2 |
| 対応している HTTP メソッド | `GET` | `GET` |
| ホストドメイン | `https://api.x.com` | `https://api.x.com` |
| エンドポイントパス | `/1.1/lists/show.json` | `/2/lists/:id` |
| [認証](/ja/resources/fundamentals/authentication) | OAuth 1.0a User Context<br /><br />App only | OAuth 1.0a User Context<br /><br />OAuth 2.0 Authorization Code with PKCE<br /><br />App only |
| デフォルトのリクエストの[レート制限](/ja/resources/fundamentals/rate-limits) | 15 分あたり 75 回のリクエスト（OAuth 1.0a）<br /><br />15 分あたり 75 回のリクエスト（OAuth 2.0）<br /><br />15 分あたり 75 回のリクエスト（App only） | 15 分あたり 75 回のリクエスト（OAuth 1.0a）<br /><br />15 分あたり 75 回のリクエスト（OAuth 2.0）<br /><br />15 分あたり 75 回のリクエスト（App only） |

**ユーザー所有の List 参照**

|     |     |     |
| :--- | :--- | :--- |
| 説明 | Standard v1.1 | X API v2 |
| 対応している HTTP メソッド | `GET` | `GET` |
| ホストドメイン | `https://api.x.com` | `https://api.x.com` |
| エンドポイントパス | `/1.1/lists/ownerships.json` | `/2/users/:id/owned_lists` |
| [認証](/ja/resources/fundamentals/authentication) | OAuth 1.0a User Context<br /><br />App only | OAuth 1.0a User Context<br /><br />OAuth 2.0 Authorization Code with PKCE<br /><br />App only |
| デフォルトのリクエストの[レート制限](/ja/resources/fundamentals/rate-limits) | 15 分あたり 15 回のリクエスト（OAuth 1.0a）<br /><br />15 分あたり 15 回のリクエスト（App only） | 15 分あたり 15 回のリクエスト（OAuth 1.0a）<br /><br />15 分あたり 15 回のリクエスト（OAuth 2.0）<br /><br />15 分あたり 15 回のリクエスト（App only） |

X API v2 エンドポイントにアクセスするには、[開発者アカウントに登録](https://developer.x.com/en/portal/petition/essential/basic-info)する必要があります。認証を行う際は、[プロジェクト](/ja/resources/fundamentals/projects) 内にある [開発者アプリ](/ja/resources/fundamentals/developer-apps) のキーとトークンを使用する必要があります。 

X API v2 エンドポイントへのアクセス方法の詳細については、[はじめに](/ja/x-api/getting-started/getting-access) ページをご覧ください。
