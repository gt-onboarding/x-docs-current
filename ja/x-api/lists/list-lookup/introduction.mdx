指定した List を ID でルックアップする場合には 15 分あたり 75 リクエスト、ユーザー所有の List をルックアップする場合には 15 分あたり 15 リクエストのレート制限があります。

これらのエンドポイントへのリクエストを認証するには、[OAuth 1.0a User Context](/ja/resources/fundamentals/authentication)、[App only]([https://developer.x.com(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token)](https://developer.x.com\(/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token\))、または [OAuth 2.0 Authorization Code with PKCE](resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2) を使用できます。 

<Note>
  **アカウントのセットアップ**

  これらのエンドポイントにアクセスするには、次のものが必要です。

  * 承認済みの [developer account](https://developer.x.com/en/portal/petition/essential/basic-info)。
  * [Project](/ja/resources/fundamentals/projects) 内にある [developer App](/ja/resources/fundamentals/developer-apps) から取得したキーとトークンを使用して認証すること。 

  X API v2 エンドポイントへのアクセス権の取得については、[getting started guide](/ja/x-api/getting-started/getting-access) を参照してください。
</Note>
