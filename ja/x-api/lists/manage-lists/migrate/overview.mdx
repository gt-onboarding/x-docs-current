<div id="comparing-x-apis-lists-endpoints">
  ## X API の List エンドポイントの比較
</div>

v2 の List 管理エンドポイントは、最終的に [POST lists/create](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/post-lists-create)、[POST lists/destroy](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/post-lists-destroy)、および [POST lists/update](https://developer.x.com/en/docs/twitter-api/v1/accounts-and-users/create-manage-lists/api-reference/post-lists-update) を置き換える予定です。これらのエンドポイントの旧バージョンを使用しているコードやアプリ、ツールがあり、より新しい X API v2 への移行を検討している場合は、このガイドが参考になります。 

以下の表では、標準 v1.1 と X API v2 の List エンドポイントを比較しています。

**List を作成する**

| 説明 | 標準 v1.1 | X API v2 |
| :--- | :--- | :--- |
| サポートされている HTTP メソッド | POST | POST |
| ホストドメイン | https://api.x.com | https://api.x.com |
| エンドポイントパス | /1.1/lists/create.json | /2/lists |
| [認証](/ja/resources/fundamentals/authentication) | OAuth 1.0a User Context | OAuth 1.0a User Context |
| デフォルトのリクエスト[レート制限](/ja/resources/fundamentals/rate-limits) | なし | 300 requests per 15 min (per user) |

**List を削除する**

| 説明 | 標準 v1.1 | X API v2 |
| :--- | :--- | :--- |
| サポートされている HTTP メソッド | POST | DELETE |
| ホストドメイン | https://api.x.com | https://api.x.com |
| エンドポイントパス | /1.1/lists/destroy.json | /2/lists/:id |
| [認証](/ja/resources/fundamentals/authentication) | OAuth 1.0a User Context | OAuth 1.0a User Context |
| デフォルトのリクエスト[レート制限](/ja/resources/fundamentals/rate-limits) | なし | 300 requests per 15 min (per user) |

**List を更新する**

| 説明 | 標準 v1.1 | X API v2 |
| :--- | :--- | :--- |
| サポートされている HTTP メソッド | POST | PUT |
| ホストドメイン | https://api.x.com | https://api.x.com |
| エンドポイントパス | /1.1/lists/update.json | /2/lists/:id |
| [認証](/ja/resources/fundamentals/authentication) | OAuth 1.0a User Context | OAuth 1.0a User Context |
| デフォルトのリクエスト[レート制限](/ja/resources/fundamentals/rate-limits) | なし | 300 requests per 15 min (per user) |

X API v2 エンドポイントにアクセスするには、[開発者アカウントに登録](https://developer.x.com/en/portal/petition/essential/basic-info)する必要があります。認証を行う際は、[プロジェクト](/ja/resources/fundamentals/projects) 内にある [開発者アプリ](/ja/resources/fundamentals/developer-apps) のキーとトークンを使用する必要があります。 

X API v2 エンドポイントへのアクセス方法については、[はじめに](/ja/x-api/getting-started/getting-access)ページをご覧ください。
