List Posts lookup には、指定した List からポストを取得するためのエンドポイントが 1 つあります。このエンドポイントを使用すると、ユーザーがタイムラインに表示されるポストをカスタマイズ、整理、優先順位付けできるようにするソリューションを構築できます。

このエンドポイントには 15 分あたり 900 リクエストのレート制限があり、レスポンスでは指定した List について最新の 800 件のポストを取得できます。

このエンドポイントへのリクエストを認証するには、[OAuth 1.0a User Context](/ja/resources/fundamentals/authentication)、[App only](/ja/resources/fundamentals/authentication#app-only-authentication-and-oauth-2-0-bearer-token)、または [OAuth 2.0 Authorization Code with PKCE](resources/fundamentals/authentication#oauth-2-0-authorization-code-flow-with-pkce-2) を使用できます。 

**アカウントの設定**

これらのエンドポイントにアクセスするには、以下が必要です。

* 承認済みの [開発者アカウント](https://developer.x.com/en/portal/petition/essential/basic-info)
* [プロジェクト](/ja/resources/fundamentals/projects) 内にある [開発者アプリ](/ja/resources/fundamentals/developer-apps) のキーとトークンを使用して認証すること 

X API v2 エンドポイントへのアクセス方法の詳細については、[利用開始ガイド](/ja/x-api/getting-started/getting-access)を参照してください。
